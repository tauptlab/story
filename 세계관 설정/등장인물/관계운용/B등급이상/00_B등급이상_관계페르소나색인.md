# B급 이상 관계/페르소나 카드 색인

## 운용 순서

1. 현재 회차와 장면 위치를 먼저 정한다.
2. 해당 인물의 관계/페르소나 카드에서 사용 가능 범위를 확인한다.
3. 상대가 전용 관계 문서에 걸리면 전용 관계 문서를 우선한다.
4. 전용 관계 문서가 없으면 관계 색인의 관계권 문장과 이 카드의 대사/행동 변화를 결합한다.
5. 출력 뒤에는 반드시 카드의 자기 검수 질문을 통과시킨다.
