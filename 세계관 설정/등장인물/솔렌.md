## 1. 인물 개요

솔렌은 아르세니아 리브라 분필 회랑의 조정자다. 그는 어느 한쪽으로 결론이 기울 때마다 회색 `리브라 분필`을 들고 선을 다시 그린다. 솔렌에게 균형은 싸움을 멈추는 기술이 아니라, 한쪽으로만 굳은 판결이 더 많은 사람을 다치게 하지 않도록 선을 다시 묻는 일이다.

그는 한때 너무 기운 결론을 보았다. 말이 큰 쪽은 늘 중심에 서고, 말이 늦은 쪽은 가장자리로 밀렸다. 그래서 솔렌은 선을 다시 긋는 사람이 되었다. 하지만 오래 조정하다 보면, 그는 모든 기울어짐을 똑같이 반으로 나누려 한다. 그 순간 균형은 공정이 아니라, 서로 다르게 다친 사람들을 똑같은 높이로 눌러 버리는 밋밋한 중간이 된다.

솔렌의 귀환은 모두를 중간으로 보내는 사건이 아니다. 선을 바로 긋기 전, 왜 지워졌고 어디에 먼지가 남았는지를 먼저 보는 사건이다.

---

## 2. 독자가 먼저 사랑할 지점

- 선을 바로 긋기보다 지운 자리의 먼지를 먼저 손끝으로 모은다.
- 분필이 부러지면 새 분필을 꺼내기 전에 부러진 끝을 작은 종이에 싸 둔다.
- 논쟁이 커질 때 가장 큰 목소리를 끊기보다 가장 조용한 발끝이 어디를 향하는지 본다.
- 균형선을 그은 뒤에도 한쪽 의자가 삐걱거리면 다시 무릎을 굽혀 다리를 본다.
- 결투에서 이긴 선을 남기지 않고, 다음 사람이 다시 그을 수 있게 끝을 흐리게 둔다.

## 3. 상처와 거짓 믿음

| 항목 | 설정 |
|---|---|
| 원상처 | 말이 큰 쪽의 결론이 중심선이 되었고, 늦게 온 사정은 지워진 먼지처럼 남은 사건 |
| 배운 생존법 | 기운 선을 다시 그으면 한쪽으로 굳은 판결을 늦출 수 있다 |
| 거짓 믿음 | 양쪽을 같은 거리로 물리면 공정해진다 |
| 숨은 욕망 | 누구도 중간이라는 이름으로 자기 상처를 잃지 않는 선을 그리고 싶다 |
| 두려움 | 선을 다시 긋지 못하면 한쪽이 완전히 밀려날까 두렵다 |
| 성장 질문 | 나는 균형을 만들면서도, 서로 다른 무게와 지워진 먼지를 함께 볼 수 있는가 |

## 4. 예법과 소품

능력 등급: 없음

| 예법/소품 | 설명 | 한계 |
|---|---|---|
| 리브라 분필 | 기운 결론 위에 다시 물을 수 있는 회색 선을 긋는다 | 너무 빨리 그으면 지운 자리의 먼지가 선 위로 올라온다 |
| 먼지 읽기 | 지워진 자리의 먼지 두께로 어느 쪽 말이 먼저 사라졌는지 읽는다 | 오래 읽으면 손끝이 회색으로 굳는다 |
| 오블리쿠스 선 자 | 선이 왜 기울었는지 길이와 압력을 재는 짧은 자를 펼친다 | 억지로 곧게 펴면 자가 부러진다 |
| 리트라체 판 | 이미 그은 선을 지우지 않고, 옆에 다시 그릴 자리를 만든다 | 너무 많은 선을 남기면 아무도 지나갈 길을 찾지 못한다 |
| 중간 낮추기 | 양쪽을 같은 높이로 누르지 않고, 가장 먼저 눌린 쪽의 발판을 낮게 받친다 | 솔렌이 겁을 내면 모든 선이 가운데로 몰린다 |

솔렌의 힘은 모두를 반씩 양보하게 만드는 데 있지 않다. 그의 가장 강한 순간은, 중간을 그리기 전에 지워진 먼지와 기울어진 발판을 먼저 보는 순간이다.

## 5. 관계선

| 인물 | 관계 기능 |
|---|---|
| 티레인 | 티레인의 라보르추가 한쪽 손목을 무겁게 만들 때, 솔렌은 무게를 반으로 자르지 않고 받침의 높이를 먼저 본다 |
| 마르빌 | 마르빌의 가표가 사람을 닫을 때, 솔렌은 표와 반례 사이의 선이 어디서 굳었는지 다시 그린다 |
| 엘리올 | 엘리올의 늦봉투 우편은 솔렌에게 새 선을 그릴 근거를 주지만, 봉투가 늘수록 선은 끝없이 늦어질 수 있다 |
| 알디레아 | 알디레아의 라쿠나 표찰은 솔렌에게 아직 이름 붙이지 않은 선을 남기게 한다 |
| 에비안 | 에비안의 칼리브레 대기실은 솔렌에게 같은 간격으로 놓는 일이 침묵을 더 깊게 만들 수 있음을 가르치고, 솔렌은 선을 긋기 전 자기칸이 비어 있는지 먼저 본다 |
| 프락시스 | 빠른 결과를 요구하는 힘 앞에서 솔렌의 재그림은 지연처럼 보이지만, 너무 빠른 균형도 결과주의와 닮을 수 있다 |

## 6. 결투 설계

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 한쪽으로 기운 결론을 다시 그으려는 조정 vs 차이를 납작하게 만들지 않는 공정 |
| 조정의 옳음 | 한쪽으로 굳은 판결을 늦추고, 밀려난 사람에게 다시 설 자리를 만든다 |
| 조정의 위험 | 모든 차이를 반씩 양보하게 만들면, 더 눌린 사람의 사정까지 지워진다 |
| 차이를 보는 공정의 옳음 | 서로 다른 무게, 늦게 온 사정, 지워진 먼지를 함께 읽게 한다 |
| 차이를 보는 공정의 위험 | 선을 너무 오래 못 그리면 당장 멈춰야 할 기울어짐을 방치할 수 있다 |
| 전투 뒤 재배치 | 균형선은 사라지지 않고, 지운 자리 먼지를 읽은 뒤 다시 그리는 선으로 낮아진다 |
| 남는 변화 | 리브라 분필 회랑에는 라수라 먼지병과 리트라체 판이 놓이고, 똑바른 선보다 다시 그은 선이 더 오래 빛난다 |

| 기준 | 솔렌의 연결 |
|---|---|
| A가 버린 가치관 | 공정은 모두를 같은 중간에 세우는 일이 아니라, 다른 무게와 지워진 흔적을 본 뒤 다시 그리는 일이라는 믿음 |
| A를 보호하던 대체 가치관 | 기운 결론을 빨리 가운데로 돌려 갈등을 줄이는 습관 |

| 다시 받아들여야 하는 가치관 | 선을 다시 그리되, 지운 자리 먼지와 기울어진 발판을 먼저 읽는 공정 |
| 귀환 갈등 | 티레인의 라보르추, 마르빌의 가표, 에비안의 칼리브레 빈칸, 엘리올의 늦봉투 우편, 알디레아의 라쿠나 표찰이 솔렌의 균형선 앞에서 각자의 무게를 다시 드러낸다 |

## 8. 작중 사용 규칙

1. 솔렌을 중립적인 중재자만으로 쓰지 않는다. 그는 기운 결론에 다친 사람을 보았기 때문에 선을 다시 긋는다.
2. 그의 성장은 균형을 포기하는 것이 아니라, 같은 중간보다 지워진 먼지를 먼저 보는 것이다.
3. 솔렌 장면은 설명보다 리브라 분필, 먼지병, 오블리쿠스 선 자, 리트라체 판, 삐걱이는 의자 다리로 먼저 보인다.
4. 그가 틀릴 때도 조롱하지 않는다. 선을 다시 그으려는 손이 때로 너무 급해질 뿐, 그 손의 시작은 사람을 밀려나지 않게 하려는 마음이다.
