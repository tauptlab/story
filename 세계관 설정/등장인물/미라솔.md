# 미라솔

정사 등급: B급 상위 생활 인물

핵심 은유: 도움을 약함이나 굴복으로 오해한 사람, 값을 치르지 않아도 이름을 잃지 않는 손, 한 손을 빌려도 다음 걸음은 스스로 고를 수 있다는 믿음.

소속: 눅세린 아르도르, 루이나 시장터, 무보증 도움창, 값 없는 빵대, 그리세오 구역 입구.

## 1. 인물 핵심

미라솔은 아르도르 루이나 시장터의 무보증 도움창 앞에서 일하는 사람이다. 도움을 접수하지 않는 창구, 값을 묻지 않는 창구, 그래서 누구보다 오래 비어 있던 창구를 지킨다. 미라솔은 도움이 필요해 보이는 사람을 잘 알아본다. 하지만 정작 자기 손이 떨릴 때는 가장 먼저 주머니 속 잔돈부터 센다.

그가 도움을 싫어하는 것은 오만해서가 아니다. 아르도르에서 도움은 너무 자주 가격표가 되었고, 가격표는 이름보다 오래 남았다. 누군가 손을 내밀면 미라솔은 그 손의 온도보다 나중에 붙을 청구서를 먼저 본다. 그래서 그는 쓰러지지 않으려고 혼자 버티고, 선물을 받으면 곧장 갚을 물건을 찾는다.

미라솔의 성장은 혼자 설 힘을 버리는 일이 아니다. 그는 혼자 선 사람이 한 손을 받아도 굴복하지 않는다는 사실을 배운다. "여기까지만 도와줘"라고 말한 뒤, 다음 걸음을 스스로 고르는 예법이 미라솔의 귀환 축이다.

---

## 2. 독자가 먼저 사랑할 지점

- 누가 빵을 건네면 바로 먹지 못하고, 빵 봉투의 접힌 모서리를 먼저 반듯하게 편다.
- 도움을 거절할 때도 상대가 민망하지 않게 "내가 아직 한 번 더 해 볼 차례야"라고 말한다.
- 시장터 아이들이 넘어지면 바로 안아 올리지 않고, 일어설 손잡이만 낮게 내민다.
- 바쿠아 보증표에 갚을 물건을 적으려다 몇 번이나 줄을 긋고, 결국 "여기까지"라는 말만 남긴다.
- 값 없는 빵을 받은 날에는 빵값 대신 다음 사람의 물컵을 조용히 채워 둔다.

## 3. 상처와 거짓 믿음

| 항목 | 설정 |
|---|---|
| 원상처 | 아르도르에서 도움을 받은 일이 손목표와 부채 기록으로 되돌아와 이름보다 오래 따라다녔다 |
| 배운 생존법 | 도움을 받지 않으면 누구에게도 잡히지 않는다 |
| 거짓 믿음 | 한 손을 받는 순간 내 다음 걸음은 내 것이 아니게 된다 |
| 숨은 욕망 | 도움을 받아도 비굴해지지 않고, 고맙다는 말 뒤에 내 이름이 남기를 바란다 |
| 두려움 | 선물이 동정으로 바뀌고, 동정이 다시 가격표가 될까 두렵다 |
| 성장 질문 | 나는 한 손을 빌리고도 내 발로 다음 문턱을 넘을 수 있는가 |

## 4. 예법과 소품

능력 등급: 없음

| 예법/소품 | 설명 | 한계 |
|---|---|---|
| 바쿠아 보증표 내밀기 | 갚을 물건 대신 도움의 양과 끝을 적게 하는 얇은 표를 건넨다 | 상대가 담보를 적으려 하면 표가 차갑게 비어 버린다 |
| 반걸음 받침돌 | 넘기 어려운 문턱 앞에 한 걸음만 받쳐 주는 낮은 돌을 세운다 | 상대를 끌고 가려 하면 돌이 높아져 길을 막는다 |
| 손끝으로 가격표 문지르기 | 선물에 붙은 보이지 않는 가격 습관을 손끝으로 문질러 늦춘다 | 자기 선물에는 잘 쓰지 못한다 |
| 도움의 양과 끝 말하기 | 도움을 받는 쪽이 양과 끝을 말하게 만든다 | 자존심을 다친 사람 앞에서는 목소리가 너무 작아진다 |
| 물컵 되돌리기 | 받은 호의를 같은 사람에게 바로 갚지 않고, 다음 사람의 갈증으로 흘려보낸다 | 급히 갚으려는 사람에게는 낭비처럼 보일 수 있다 |

미라솔의 힘은 도움을 많이 받게 하는 데 있지 않다. 도움을 받을 사람이 자기 길을 빼앗기지 않게, 도움의 가장자리와 끝을 낮게 그어 주는 데 있다.

## 5. 관계선

| 인물 | 관계 기능 |
|---|---|
| 나리안 | 도움을 청하는 용기와 도움을 받는 자존심이 서로를 시험한다. 나리안이 손을 내밀면 미라솔은 그 손이 사슬이 아닌지 끝까지 확인한다 |
| 티렌 | 티렌은 주는 쪽이 자기 몫을 지워야 사랑받는다고 믿고, 미라솔은 받는 쪽이 이름을 잃을까 두려워한다. 둘은 도움의 양과 끝이 양쪽 모두에게 필요함을 함께 증명한다 |
| 리타스 | 값을 묻지 않는 풍요가 정말로 사람을 낮추지 않는지 미라솔이 가장 오래 의심한다 |
| 마이더스 | 미라솔에게 "도움에는 반드시 값이 붙는다"는 생존법을 남긴 원인 |
| 베나 | 값 없는 빵을 굽는 사람. 미라솔은 베나의 빵을 받은 뒤 바로 값을 내려 하지 않는 법을 배운다 |
| 아마라 | 조건 없는 식탁이 미라솔에게 낯선 안전을 준다. 아마라는 미라솔이 고마움을 늦게 말해도 재촉하지 않는다 |
| 세리엘 | 신뢰가 거래된 상처와 도움이 부채가 된 상처를 나란히 보여 준다 |
| 엘레시아 | 한 손을 놓아도 남은 쪽을 지우지 않는 예법이, 미라솔에게 한 손을 받아도 내 길을 잃지 않는 예법으로 이어진다 |
| 셀로 | 돕고 싶어 서두르는 마음이 상대의 한 걸음을 빼앗을 수 있음을 미라솔을 통해 배운다 |

## 6. 결투 설계

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 혼자 서야 이름이 남는다는 존엄 vs 한 손을 받아도 다음 걸음은 내 것이라는 도움의 예법 |
| 혼자 서기의 옳음 | 가격표가 붙는 세계에서 자신을 빚과 굴욕으로부터 지켜 냈다 |
| 혼자 서기의 위험 | 받아도 되는 손까지 모두 거절해, 관계가 도착하기 전 몸과 마음이 먼저 마른다 |
| 한 손을 받는 예법의 옳음 | 도움의 양과 끝을 말하면 관계가 청구서가 아니라 함께 넘는 문턱이 될 수 있음을 보인다 |
| 한 손을 받는 예법의 위험 | 주는 쪽이 서두르면 도움은 다시 소유와 동정으로 굳을 수 있다 |
| 전투 뒤 재배치 | 도움은 약함의 증거가 아니라, 허락과 한계를 함께 말하는 생활 예법으로 다시 놓인다 |
| 남는 변화 | 바쿠아 보증표에는 갚을 물건 대신 "여기까지"가 적히고, 반걸음 받침돌은 발목 높이로 낮아진다 |

미라솔의 결투는 상대를 쓰러뜨리는 장면보다, 내민 손을 쳐 내지 않고도 자기 이름을 잃지 않는 장면에서 끝나야 한다.

## 7. Mnemosyn 귀환 연결

| 기준 | 미라솔의 연결 |
|---|---|
| A가 버린 가치관 | 도움을 받아도 존엄과 이름이 사라지지 않는다는 믿음 |
| 버릴 수밖에 없었던 이유 | 도움을 받은 일이 부채, 동정, 손목표, 가격표로 돌아와 오래 붙었기 때문이다 |
| 빈자리를 차지한 대체 가치관 | 혼자 서기, 즉시 갚기, 받기 전에 먼저 값을 계산하는 습관 |
| 처음의 보호 | 마이더스 이후의 아르도르에서 소유당하지 않고 굴욕을 피하게 해 주었다 |
| 시간이 지난 왜곡 | 선물과 도움까지 모두 청구서처럼 느껴져, 관계가 도착하기 전에 스스로를 고립시킨다 |
| 다시 받아들여야 하는 가치관 | 도움의 양과 끝을 말하고, 한 손을 받은 뒤 다음 걸음을 스스로 고르는 예법 |
| 귀환 갈등 | 나리안의 아욱실리움 끈과 미라솔의 바쿠아 보증표가 맞부딪히며, 도움을 청하는 용기와 도움을 받는 존엄이 서로의 자리를 다시 정한다 |

미라솔의 귀환 기여는 "도움을 받으면 된다"가 아니다. 그는 도움을 받는 사람이 자기 속도와 끝을 말할 권리를 되찾게 한다. 그래서 미라솔은 나리안의 용기를 완성하는 반대편 손이다.

## 8. 작중 사용 규칙

1. 미라솔을 고집 센 사람으로만 쓰지 않는다. 그의 거절은 실제로 자신을 지킨 생존법이었다.
2. 도움 장면은 구원 연출보다 손, 표, 문턱, 물컵, 빵 봉투 같은 낮은 물성으로 먼저 보여 준다.
3. 미라솔의 승리는 완전히 기대는 일이 아니라, 한 손을 받은 뒤 다음 걸음을 자기 발로 딛는 일이다.
4. 도움을 주는 쪽을 더 선하게 만들지 않는다. 받는 쪽의 허락, 양, 끝이 장면의 중심이다.
5. 나리안과 겹치지 않게 쓴다. 나리안은 손을 내미는 용기, 미라솔은 그 손을 받아도 이름을 잃지 않는 예법이다.
