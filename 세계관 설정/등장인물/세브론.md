## 1. 인물 개요

세브론은 레비오렌 피니투스 종 공장의 반장이다. 그는 '물건이 다 됐다'고 말하기보다, 그 물건을 마지막으로 만진 사람의 이름을 먼저 부른다. 세브론에게 완료는 숫자가 아니라 약속이다. 끝낸 일이 있어야 누군가는 밤에 문을 잠글 수 있고, 누군가는 기다리던 물건을 들고 집으로 돌아갈 수 있다.

그가 완성을 믿게 된 데에는 이유가 있다. 끝나지 않은 약속 때문에 누군가의 밤이 계속 밀리던 때가 있었고, 아무도 종을 울리지 못해 모두가 작업대 앞에서 잠을 미뤘다. 그래서 세브론은 피니투스 종을 지키는 사람이 되었다. 그러나 시간이 지나며 그는 종이 울리지 않는 밤을 견디지 못한다. 피니스 패에서 아직 채워지지 않은 빈 줄이 사람의 숨결보다 크게 보이고, 일을 끝낸 사람의 이름만 도시 한가운데 남을 때, 완성은 안심이 아니라 압박이 된다.

세브론의 귀환은 완료를 버리는 사건이 아니다. 끝낸 일을 존중하되, 종을 울리기 전에 만든 손과 남은 손을 함께 부르는 사건이다.

---

## 2. 독자가 먼저 사랑할 지점

- 종을 치기 전 망치 손잡이를 닦지 않고, 마지막 작업자의 손때가 남은 홈만 조심스럽게 만진다.
- 피니스 패에 이름이 빠지면 종 앞에서 한참 망설이다가, 빈 줄을 먼저 펴 둔다.
- 늦게 들어온 견습공의 작은 흠집도 "끝을 늦춘 손"이 아니라 "끝을 같이 만든 손"이라고 고쳐 부른다.
- 종소리벌이 천장에 부딪히면 종을 더 세게 치지 않고, 공장 불을 한 칸 낮춘다.
- 자신은 잠을 미루면서도, 다른 사람이 쉬겠다고 말하면 작업대 아래에 오티움 나사를 먼저 끼워 준다.

## 3. 상처와 거짓 믿음

| 항목 | 설정 |
|---|---|
| 원상처 | 끝나지 않은 약속 때문에 모두가 잠을 미루고, 아무도 돌아갈 수 없었던 밤 |
| 배운 생존법 | 피니투스 종을 울리면 기다림이 끝나고 다음 하루를 시작할 수 있다 |
| 거짓 믿음 | 종이 울리지 않으면 아무도 안심할 자격이 없다 |
| 숨은 욕망 | 끝낸 일이 사람을 지우지 않고, 함께 만든 손을 불러 주는 약속이 되기를 바란다 |
| 두려움 | 내가 종을 늦추면 누군가의 밤이 또 밀릴까 두렵다 |
| 성장 질문 | 나는 끝냄을 지키면서도, 아직 남은 손과 쉬어야 할 손을 함께 부를 수 있는가 |

## 4. 예법과 소품

능력 등급: 없음

| 예법/소품 | 설명 | 한계 |
|---|---|---|
| 피니투스 종 치기 | 완성된 물건 앞에서 낮은 종을 울려 약속이 끝났음을 알린다 | 이름을 빼고 치면 종소리벌이 길을 잃는다 |
| 마누스 울티마 호명 | 마지막으로 닿은 손의 이름을 작게 불러 피니스 패를 또렷하게 만든다 | 손을 순위처럼 부르면 이름이 종소리에 묻힌다 |
| 피니스 패 펴기 | 만든 손과 남은 손을 한 표에 적어 성과가 사람을 지우지 않게 한다 | 빈 줄을 수치로 읽으면 표 가장자리가 차갑게 굳는다 |
| 틴니투스 끈 | 종을 칠 때 힘이 아니라 기다린 손의 수만큼 끈을 당긴다 | 조급하게 당기면 끈이 손목을 감는다 |
| 레스타 바쿠아줄 | 아직 불리지 않은 손을 위해 피니스 패 한 줄을 비워 둔다 | 영원히 비우면 끝내야 할 약속이 흐려진다 |

세브론의 힘은 더 많은 종을 울리는 데 있지 않다. 그의 가장 강한 순간은, 종을 치기 직전 빈 줄을 보고도 부끄러워하지 않고 "아직 한 손이 남았다"고 말하는 순간이다.

## 5. 관계선

| 인물 | 관계 기능 |
|---|---|
| 마딘 | 마딘은 되돌아온 물건도 다시 맡는 책임임을 보여 주고, 세브론은 완료가 수정의 가능성을 닫지 않아야 함을 배운다 |
| 오렐라 | 오렐라의 미완성 선반은 세브론에게 아직 끝나지 않은 것이 곧 실패는 아님을 가르친다 |
| 타비온 | 타비온의 빈 줄은 세브론의 피니스 패가 숫자보다 사람을 먼저 보게 만든다 |
| 프락시스 | 프락시스의 빠른 결과는 세브론에게 매력적으로 보이지만, 종이 사람보다 먼저 울리는 위험을 드러낸다 |
| 에르바크 | 에르바크의 회송봉투는 이미 울린 종 뒤에도 늦게 온 손이 있을 수 있음을 보여 준다 |
| 에이도스 | 에이도스는 피니스 패의 빈 줄을 질문으로 읽고, 세브론은 질문이 종을 늦추는 방해만은 아님을 배운다 |

## 6. 결투 설계

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 끝낸 약속의 안심 vs 남은 손을 지우지 않는 완료 |
| 끝낸 약속의 옳음 | 완료가 있어야 기다림이 끝나고 다음 하루로 갈 수 있다 |
| 끝낸 약속의 위험 | 종소리가 사람보다 먼저 울리면, 끝낸 사람만 남고 남은 손은 수치가 된다 |
| 남은 손을 보는 완료의 옳음 | 완성품 뒤에 있는 손과 아직 남은 손을 함께 불러 준다 |
| 남은 손을 보는 완료의 위험 | 모든 손을 기다리다 끝내야 할 약속을 계속 미룰 수 있다 |
| 전투 뒤 재배치 | 피니투스 종은 사라지지 않고, 마누스 울티마 명부와 레스타 바쿠아줄을 확인한 뒤 낮게 울린다 |
| 남는 변화 | 피니투스 종 공장에는 틴니투스 끈과 마누스 울티마 명부가 놓이고, 종소리벌은 사람 이름이 불린 뒤에만 낮게 맴돈다 |

| 기준 | 세브론의 연결 |
|---|---|
| A가 버린 가치관 | 끝낸 일은 사람을 지우는 결과가 아니라 함께 만든 손을 불러 주는 약속이라는 믿음 |
| A를 보호하던 대체 가치관 | 빨리 완료를 확정해야 기다림과 불안을 끝낼 수 있다는 습관 |
| 시간이 지나 왜곡된 가치관 | 완료가 안심이 아니라 압박이 되고, 끝낸 사람만 이름을 얻으며 남은 손은 수치가 된다 |
| 다시 받아들여야 하는 가치관 | 피니투스 종을 울리되, 마누스 울티마 명부와 레스타 바쿠아줄을 먼저 확인하는 성실함 |
| 귀환 갈등 | 타비온의 빈 줄, 오렐라의 미완성 선반, 마딘의 리파브라 장갑, 에르바크의 회송봉투가 세브론의 피니투스 종 앞에서 끝냄의 의미를 다시 묻는다 |

## 8. 작중 사용 규칙

1. 세브론을 성과 강박만 가진 인물로 쓰지 않는다. 그는 끝낸 일이 실제로 누군가의 밤을 구했다는 기억 때문에 종을 지킨다.
2. 그의 성장은 완료를 포기하는 것이 아니라, 종을 치기 전 만든 손과 남은 손을 함께 부르는 것이다.
3. 세브론 장면은 설명보다 피니투스 종, 피니스 패, 마누스 울티마 명부, 틴니투스 끈, 종소리벌의 낮은 비행으로 먼저 보인다.
4. 완료되지 않은 것은 무조건 아름답지 않다. 다만 아직 끝나지 않았다는 이유만으로 사람의 이름이 실패가 되면 안 된다.
