# 엘레시아

정사 등급: B급 상위 접경 생활 인물

소속: 눅세린/센시르 두손 마당, 두물빛 잔 탁자, 한쪽발 돌마당

핵심 은유: 서로 다른 쪽을 모두 품으려는 다정함, 선택을 미루는 보호의 위험, 한쪽을 골라도 남은 쪽을 저주하지 않는 책임

## A의 생각과 태고의 뿌리 (작가용·원고 비노출)

- **힘을 주는 태고의 본능:** 사랑(타인애)·두려움
- **개화시킨 A의 생각:** "한쪽을 고르면 다른 쪽을 버리는 거니까, 아무도 놓지 말고 답을 미뤄야 한다."
- 이 생각이 강해질수록 약한 발현이 짙어지고, 흐려지면 옅어진다(세계 작동 원리 `../정식바이블/01A_정사_편찬_규칙.md` §1.5). 이 문서 「3. 상처와 거짓 믿음」의 거짓 믿음 `선택은 곧 배신이며, 분명한 한 발은 다른 한 발의 추락이다`과 같은 뿌리다.
- 원고에서는 이 생각을 직접 말하지 않고 발현·선택·소품으로만 보인다.

## 1. 인물 개요

엘레시아는 두 나라의 말투를 모두 배운 접경의 안내자다. 눅세린 사람에게는 너무 느리게 보이고, 센시르 사람에게는 너무 많은 약속을 품은 사람처럼 보인다. 그녀는 갈등하는 양쪽 사이에서 먼저 손을 놓는 사람이 되지 않으려 한다.

엘레시아의 선의는 진짜다. 그녀는 한쪽을 고르는 순간 다른 쪽이 버려진다고 믿는 사람들 곁에서 오래 서 있었다. 그래서 양손을 동시에 내밀고, 두 잔을 동시에 식히고, 어느 쪽 손도 갑자기 차갑게 놓지 않으려 한다.

그러나 엘레시아의 위험도 바로 그 다정함에서 온다. 아무도 잃지 않으려는 마음은 처음에는 관계를 살렸지만, 시간이 지나면 아무도 분명한 답을 받지 못하는 긴 기다림이 된다. 엘레시아의 귀환은 하나의 가치가 다시 세계 질서 안으로 돌아오는 사건이다. 선택은 배신이 아니라, 남은 쪽을 기억한 채 한 발을 내딛는 책임일 수 있다는 가치다.

에레시아는 실로에르의 좋은 기억을, 오늘 꺼내도 좋다는 허락을 받아 꺼내는 인물이다. 엘레시아는 눅세린/센시르 접경에서 양쪽을 다 잃지 않으려는 귀환 인물이다. 두 이름은 가깝지만 맡은 갈등은 다르다.

## 2. 독자가 먼저 사랑할 지점

- 다툼이 커지면 누구 편인지 말하기 전, 두 사람의 컵이 모두 식지 않았는지 본다.
- 한쪽 손을 잡으면 반드시 남은 손의 온도도 확인하지만, 결투 전까지는 어느 손을 먼저 놓아야 하는지 말하지 못한다.
- 약속을 적을 때 문장을 반으로 접어 두고, 접힌 안쪽에 "남은 쪽도 사라지지 않는다"라고 작게 쓴다.
- 누가 배신이라고 말하면 바로 반박하지 않고, 두물빛 잔의 물을 한 모금씩 따로 마신다.
- 길을 안내할 때 가운데만 걷지 않고, 일부러 한쪽발 돌 위에 먼저 서서 다른 쪽을 향해 고개를 숙인다.

## 3. 상처와 거짓 믿음

| 항목 | 설정 |
|---|---|
| 원상처 | 한쪽을 선택한 날, 남은 쪽에서 자신을 버린 사람이라 불렀다 |
| 배운 생존법 | 어느 쪽도 먼저 놓지 않고, 답을 조금 더 늦추면 관계가 찢어지지 않는다 |
| 거짓 믿음 | 선택은 곧 배신이며, 분명한 한 발은 다른 한 발의 추락이다 |
| 숨은 욕망 | 한쪽을 고르더라도 다른 쪽이 완전히 사라지지 않는 관계를 믿고 싶다 |
| 두려움 | 누군가의 손을 놓는 순간, 지금까지의 다정함까지 모두 거짓으로 판정될까 두렵다 |
| 성장 질문 | 나는 한 발을 딛고도 남은 손의 온도를 부정하지 않을 수 있는가 |

## 4. 약한 발현과 예법

능력 등급: 약한 발현
발현 표지: 두손끈
작동: 두 손의 연결을 잠시 유지하다가 한쪽 발을 딛는 순간 남은 손의 표식을 남긴다

| 예법/소품 | 설명 | 한계 |
|---|---|---|
| 두손 잇기 | 갈라진 두 사람의 손목에 느슨한 끈을 걸어, 서로가 아직 완전히 끊기지 않았음을 알게 한다 | 오래 걸어 두면 끈이 두 손목을 모두 아프게 조인다 |
| 두물빛 잔 나누기 | 같은 잔 안에 서로 다른 두 물빛을 잠시 나누어 담는다 | 두 물을 동시에 마시려 하면 목이 막히고 잔이 흐려진다 |
| 한쪽발 딛기 | 한쪽발 돌 위에 먼저 서서 선택이 시작되었음을 보인다 | 선택한 뒤 남은 쪽을 저주하면 돌이 차갑게 갈라진다 |
| 남은손 인사 | 선택받지 못한 쪽을 삭제하지 않고, 그쪽에도 짧은 인사를 남긴다 | 인사가 변명으로 길어지면 다시 유예가 된다 |

엘레시아는 모두를 이해하려는 인물이지만, 모두를 이해한다고 모두에게 같은 답을 줄 수 있는 것은 아니다. 그녀가 겨누는 것은 선택 자체가 아니라, 선택을 배신으로만 부르는 오래된 두려움이다.

## 5. 관계선

| 인물 | 관계 기능 |
|---|---|
| 나리안 | 도움을 청하는 용기와 선택을 말하는 용기가 서로 닮았음을 보여 준다 |
| 세리엘 | 배신당한 믿음과 선택받지 못한 쪽의 상처를 함께 드러낸다 |
| 리타스 | 대가 없는 환대가 한쪽 선택 뒤에도 남을 수 있는지 묻는 기준이 된다 |
| 카시안 | 모두 듣는 일과 대신 결론을 내리지 않는 일의 경계를 엘레시아에게 보인다 |
| 살루스 | 쉬게 하는 다정함이 선명한 선택을 흐릴 수 있음을 대비한다 |
| 세이르 | 맞춰 주는 다정함이 자기 대답을 잃을 때의 위험을 비춘다 |
| 릴센 | 빌린 얼굴을 반납하는 일과, 양쪽 기대를 모두 입지 않는 일이 이어진다 |
| 셀로 | 통합하려는 마음이 선택을 미루는 방식으로 기울 때 어떤 후폭풍이 생기는지 보게 한다 |

## 6. 결투 설계

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 모두를 잃지 않으려는 포용 vs 한쪽을 선택하고 책임지는 용기 |
| 포용의 옳음 | 서둘러 갈라진 관계를 다시 붙잡고, 버려졌다고 느끼는 쪽에 손을 남겨 준다 |
| 포용의 위험 | 답을 늦추는 동안 양쪽 모두 기다림의 벌을 받으며, 어느 누구도 분명한 길을 받지 못한다 |
| 선택의 용기의 옳음 | 관계가 앞으로 움직이려면 한 발은 실제 바닥에 닿아야 한다 |
| 선택의 용기의 위험 | 남은 쪽을 지워 버리면 선택은 다시 폭력이 되고, 이해는 승자의 변명이 된다 |
| 전투 뒤 재배치 | 선택은 배신이라는 오래된 말에서 벗어나, 남은 쪽을 기억한 채 딛는 한 발로 다시 놓인다 |
| 남는 변화 | 두손 끈은 묶는 끈에서 인사를 남기는 끈으로 바뀌고, 한쪽발 돌은 차갑게 갈라지지 않는다 |

| A가 버린 가치관 | 선택해도 관계 전체가 사라지지 않을 수 있다는 믿음 |

| 시간이 지나 왜곡된 가치관 | 배신은 줄었지만 결정도 사라져, 양쪽 모두 끝나지 않는 대기 속에 묶인다 |
| 다시 받아들여야 하는 가치관 | 한쪽을 고르되 남은 쪽을 지우지 않고, 선택 뒤의 인사와 책임을 남기는 용기 |
| 귀환 갈등 | 모두를 품으려는 포용과 한쪽을 딛는 책임이 결투 뒤 서로의 자리를 다시 정한다 |

## 8. 작중 사용 규칙

1. 엘레시아를 우유부단한 사람으로만 쓰지 않는다. 그녀는 실제로 많은 관계를 찢어지지 않게 지킨 사람이다.
2. 선택 장면은 큰 선언보다 손, 잔, 발, 남은 쪽을 향한 짧은 인사로 먼저 보인다.
3. 엘레시아의 승리는 양쪽을 모두 만족시키는 것이 아니라, 선택 뒤에도 남은 쪽을 사라진 쪽으로 만들지 않는 것이다.
4. 에레시아와 혼동되지 않도록 엘레시아의 무대는 늘 눅세린/센시르 접경의 손, 잔, 발 물성으로 고정한다.
