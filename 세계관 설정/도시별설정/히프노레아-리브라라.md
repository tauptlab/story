# 히프노레아 - 리브라라

정사 등급: B급

소속: 히프노레아 비바 기억숲 바깥 고원, 페쿤디아 문턱로 향하는 외곽 생활권

핵심 은유: 생명을 지키려는 손이, 아직 오지도 않은 그 생명의 미래까지 대신 정해 버릴 때 생기는 다정한 통제

---

## 1. 개요

리브라라는 비바 기억숲 바깥 고원에 자리한 씨앗 돌봄 도시다. 성역 입구가 누군가를 안으로 들일 수 있는지 묻는 문턱이라면, 리브라라는 아직 문턱까지 가지 못한 씨앗과 어린 뿌리를 오래 돌보는 생활권이다. 이곳 사람들은 바람을 낮추고, 그늘을 나누고, 이름을 늦추며, 작은 생명이 다시 세계를 믿을 수 있을 때까지 기다린다.

그러나 리브라라의 위험은 그 다정함에서 온다. 다치지 않게 하려는 손은 쉽게 앞질러 길을 정하고, 안전하다는 이유로 씨앗의 이름도, 맺을 열매도, 자랄 방향까지 미리 붙여 둔다. 이 도시는 생명을 살리는 보호와, 생명이 자기 방향을 찾을 권리 사이에서 흔들린다.

---

## 2. 첫인상

- 고원 전체가 온실과 바람막이 난간으로 이어져 있다.
- 어린 뿌리 옆에는 이름표가 있지만, 가운데는 비어 있다.
- 주민들은 씨앗을 들어 보이기 전에 먼저 손바닥 온도를 낮춘다.
- 예비열매 저장고에는 아직 맺히지 않은 열매 모양의 투명한 껍질이 걸려 있다.
- 바깥바람 난간에서는 아주 약한 바람만 한 줄씩 들어오고, 아이들은 그 바람에 잎이 어느 쪽으로 기우는지 기록한다.

## 3. 구조

| 구역 | 설명 | 장면 기능 |
|---|---|---|
| 노멘 백온실 | 씨앗에 이름을 붙이지 않고 빈 이름표만 꽂아 두는 온실 | 보호가 미래 확정으로 굳지 않게 함 |
| 바깥바람 난간 | 성역 바깥의 바람을 한 줄씩 들이는 고원 난간 | 위험을 완전히 차단하지 않고 견딜 만큼 만나게 함 |
| 모라 심기 마당 | 심는 날짜보다 씨앗의 준비를 먼저 보는 마당 | 빠른 희망과 기다리는 생명의 차이 |
| 노멘 예비서가 | 붙일 수 있었지만 아직 붙이지 않은 이름들을 보관하는 서가 | 좋은 이름도 때가 아니면 짐이 됨 |
| 예비열매 저장고 | 언젠가 맺힐 수 있는 열매 모양 껍질을 걸어 두는 창고 | 가능성과 기대의 경계 |
| 그늘나눔 계단 | 너무 밝은 기억과 너무 차가운 흙 사이의 그늘을 나누는 계단 | 보호의 양을 조절하게 함 |
| 스스로길 화단 | 뿌리가 자기 방향으로 틀어도 지지대를 늦게 옮기는 화단 | 자라는 방향을 대신 정하지 않음 |

## 4. 유기물과 소품

| 이름 | 설정 | 활용 |
|---|---|---|
| 벡토르덩굴 | 열린 흙 쪽으로 천천히 몸을 돌리는 덩굴 | 생명이 스스로 향하는 곳을 보임 |
| 예비열매 | 아직 맺히지 않은 열매를 닮은 투명한 껍질 | 기대가 약속처럼 굳는 위험 |
| 바람잠풀 | 겁먹은 씨앗 곁에서 바람을 낮추는 풀 | 회복 초기에 필요한 완충 |
| 느린뿌리꽃 | 뿌리가 원래 방향을 바꾸어도 시들지 않을 때 피는 꽃 | 달라진 미래를 실패로 보지 않게 함 |
| 길묻는이끼 | 갈림 흙에서 질문이 많을수록 낮게 자라는 이끼 | 정답보다 다시 묻는 시간을 만듦 |
| 노멘 백표 | 가운데를 비워 둔 얇은 이름표 | 이름 붙임을 보호가 아니라 기다림으로 둠 |
| 바람틈 삽 | 온실 벽 아래 아주 얇은 바람길을 파는 삽 | 위험을 전부 막지 않고 작은 통로를 냄 |
| 늦심기 달력 | 날짜보다 흙과 씨앗의 준비를 함께 보는 달력 | 늦음이 실패가 아님을 보임 |
| 예비그늘 천 | 어린 잎 위에 잠시 덮는 얇은 그늘 천 | 보호의 밝기를 조절 |
| 발아계약 | 옮겨 심기 전 화분에 느슨히 묶는 끈 | 옮김에도 허락과 때가 필요함 |

## 5. 주요 인물

| 인물 | 이 장소와의 관계 |
|---|---|
| 리엘라 | 노멘 백온실의 돌봄이. 겁이 나면 이름을 붙이고 싶어 하지만, 노멘 백표를 매일 닦으며 참는다 |
| 보르미 | 바깥바람 난간 지기. 바람을 막는 법보다 한 줄씩 들이는 법을 더 오래 배웠다 |
| 아델룬 | 노멘 예비서가 서기. 아름다운 이름을 잘 만들지만, 붙이지 않은 이름을 접어 두는 손이 더 조심스럽다 |
| 세라온 | 예비열매 저장고 관리인. 아이들에게 열매를 약속하기보다 껍질이 아직 비어 있음을 보여 준다 |
| 루비안 | 모라 심기 마당 달력지기. 남들이 늦었다고 재촉해도 흙의 온도와 씨앗의 기울기를 먼저 본다 |
| 오넬 | 스스로길 화단의 어린 길잡이. 뿌리가 지지대와 다른 쪽으로 틀면 작은 돌을 옮겨 길을 내준다 |
| 레기나 페쿤디아 | 리브라라를 지나며 생명을 지키는 손도 미래를 대신 정할 수 있음을 본다 |
| 아미키티아 | 죄책감 때문에 더 완벽하게 보호하려는 마음이, 씨앗의 바깥바람을 빼앗을 수 있음을 배울 수 있다 |

## 6. 스토리 활용

1. 성역 입구와 분리한다. 성역 입구는 통과 문턱이고, 리브라라는 문턱 바깥에서 오래 살아남은 씨앗 돌봄 생활권이다.
2. 레기나의 생명 가치가 단순 희망이 아니라, 보호와 미래 결정의 경계를 포함한다는 점을 보여 준다.
3. 아미키티아의 속죄가 "다시는 다치지 않게 완벽히 지키겠다"로 기울 때, 리브라라가 그 위험을 생활 장면으로 비춘다.
4. 녹티오라에서는 공포보다 답답한 안전감이 먼저 온다. 조용하고 따뜻한데, 잎이 자기 방향을 잃는 방식으로 보인다.
5. 귀환 전조에서는 씨앗의 이름을 대신 붙이지 않는 선택이, 버렸던 생명의 자율성을 다시 받아들이는 작은 사건으로 이어질 수 있다.

## 7. 규칙과 금기

| 규칙/금기 | 이유 | 어기면 생기는 일 |
|---|---|---|
| 노멘 백표에는 주인이 오기 전 이름을 쓰지 않는다 | 좋은 이름도 때가 아니면 뿌리를 묶는다 | 이름표가 뿌리 안쪽으로 파고든다 |
| 바깥바람 난간을 완전히 닫지 않는다 | 바람 없는 안전은 자라는 힘을 약하게 만든다 | 잎이 안쪽으로만 말린다 |
| 예비열매를 수확물처럼 세지 않는다 | 가능성은 아직 약속이 아니다 | 투명 껍질이 돌처럼 무거워진다 |
| 늦심기 달력을 벌로 쓰지 않는다 | 늦음은 준비와 실패를 함께 묻는 말이다 | 모든 날짜가 같은 회색 칸으로 변한다 |
| 스스로길 화단의 지지대를 먼저 묶지 않는다 | 뿌리는 지지대가 아니라 흙과 바람을 먼저 읽는다 | 뿌리가 지지대 모양으로 굳는다 |

삽입 장면 씨앗:

- 리엘라가 가장 아름다운 이름을 노멘 백표에 쓰려다 멈추고, 이름 대신 손바닥 온도만 낮게 적는다.
- 보르미가 난간을 닫아 달라는 부탁 앞에서 바람을 없애지 않고, 바람틈 삽으로 한 손가락만큼의 길을 남긴다.
- 오넬이 "이쪽이 더 안전하다"는 어른들의 말 뒤에서, 다른 쪽으로 튼 뿌리 옆에 작은 돌길을 놓는다.

## 말투와 사고방식 운용

### 지역 말결

- 기본 결은 히프노레아 계열이다. 기록과 잠, 보존의 감각이 생활 말투를 이룬다. 사람들은 감정을 밀어붙이지 않고, 덮어 둔 장부를 언제 열지 따진다.
- 리브라라에서는 기록의 습관이 말의 순서를 정한다. 인물은 감정을 바로 해설하기보다 책등, 장부, 접힌 쪽지 같은 구체물을 먼저 확인한다.
- 자주 떠오르는 어휘권은 기록, 잠, 보존이다. 단어를 표식처럼 남발하지 말고, 장면의 손동작과 물성에 묻혀서 들리게 한다.

### 장면 운용

- 낮 장면: 바로 판단하지 않고 누가 무엇을 남겼는지부터 확인한다.
- 밤 장면: 밤이 되어 눌러 둔 충동이 몸에 가까워지면, 이 장소의 생활 습관이 본능과 충돌한다. 싸움은 초현상보다 사람의 고집, 경쟁심, 보호 본능이 물건과 동선을 망가뜨리는 방식으로 먼저 발생한다.
- 외부자 반응: 외부인은 이곳 사람들의 말이 차갑거나 느리다고 느낄 수 있다. 하지만 장면은 설명 대사보다 기다리는 순서, 건네는 물건, 비워 둔 자리로 지역성을 증명한다.

### 금기

- 리브라라의 지역성을 사투리 표기만으로 처리하지 않는다. 말끝 변형보다 사고 순서, 침묵의 길이, 물건을 다루는 방식이 우선이다.
- 장소 설명을 인물 입으로 길게 해설하지 않는다. 책등, 장부, 접힌 쪽지 같은 구체물을 만지는 손, 줄 서는 순서, 숨을 고르는 박자로 먼저 보인다.
- 밤 장면에서도 바닥 균열이나 괴현상을 기본값으로 쓰지 않는다. 밤의 위험은 발현된 사람들의 본능, 충돌, 파손 여파에서 출발해야 한다.
