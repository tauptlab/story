# 눅세린 - 닐세로안

정사 등급: B급

소속: 벨라사르 동쪽 장인 공방대와 평판 심사장이 모인 공방 도시

핵심 은유: 좋은 것을 만들고 싶은 마음이 이름표와 평판에 묶일 때 생기는 기쁨과 피로, 이름 붙은 솜씨와 함께 만든 손의 긴장

---

## 1. 개요

닐세로안은 눅세린의 장인/평판 생활 도시다. 이곳의 주민들은 물건을 잘 만들고 싶어 한다. 더 오래 가는 손잡이, 덜 아픈 의자, 한 번 더 고쳐 쓸 수 있는 그릇을 만들려는 마음은 진짜다. 문제는 그 솜씨가 곧장 이름표, 도장, 심사장, 주문 순서로 옮겨 갈 때 생긴다.

닐세로안은 인정 욕망을 조롱하지 않는다. 누군가의 이름이 작품에 남는 일은 기쁨이다. 다만 그 이름이 너무 커져 함께 만든 손을 지우거나, 첫 흠을 숨겨야만 팔리는 도시가 될 때, 좋은 물건은 사람을 쉬게 하지 못하고 다시 평가의 자리로 데려간다.

---

## 2. 첫인상

- 공방마다 문 위에 작은 노멘도장이 걸려 있고, 도장 아래에는 지워진 보조자의 이름 자국이 남아 있다.
- 거리의 금속 냄새 사이로 오래 문지른 나무 손잡이 냄새가 섞인다.
- 심사장 앞에는 완성품보다 다시 고쳐 온 물건을 든 사람이 더 오래 서 있다.
- 마누스꽃은 박수받은 전시대보다 오래 쓰인 작업대 밑에서 핀다.
- 아이들은 새 망치보다 첫 흠이 남은 망치를 더 조심스럽게 건넨다.

## 3. 구조

| 구역 | 설명 | 장면 기능 |
|---|---|---|
| 손끝 심사장 | 만든 사람 이름을 가린 채 물건의 손끝 감각부터 보는 심사장 | 이름보다 실제 쓰임을 먼저 묻는 장소 |
| 노멘도장 거리 | 공방마다 자기 노멘도장을 걸어 두는 긴 거리 | 인정의 기쁨과 이름 독점의 위험 |
| 리디투스 작품창고 | 반품된 물건과 버려진 작품을 고쳐 두는 창고 | 실패한 솜씨의 회복 |
| 프리마 시카트릭스 공방 | 처음 생긴 흠을 숨기지 않고 쓰임을 다시 맞추는 공방 | 완벽보다 버티는 물건의 윤리 |
| 무명 손길 우물 | 공방 사람들이 도장 가루를 씻는 낮은 우물 | 함께 만든 손의 이름을 되찾는 자리 |
| 평판 종탑 | 도시의 주문과 소문에 따라 종소리가 달라지는 탑 | 칭찬과 압박의 빠른 전파 |
| 도르숨 도장천 걸이 | 심사 전 노멘도장을 덮는 얇은 천이 걸린 처마 | 이름보다 쓰임을 먼저 만지게 하는 예법 |
| 소키우스 손자국 작업대 | 여러 손이 잡은 자리가 희미하게 남는 긴 작업대 | 공동 작업을 한 이름 아래 지우지 않게 함 |
| 노멘 모라끈 서랍 | 도장을 찍기 전 잠시 묶어 두는 끈을 보관하는 서랍 | 이름의 기쁨과 조급함을 분리 |

## 4. 유기물과 소품

| 이름 | 설정 | 활용 |
|---|---|---|
| 시길룸 나방 | 과하게 빛나는 노멘도장 위에 앉아 금빛 가루를 갉아 먹는 작은 나방 | 이름이 작품보다 커질 때의 표식 |
| 마누스꽃 | 오래 쓰인 손잡이와 작업대 밑에서 피는 작은 꽃 | 박수보다 사용된 시간이 남긴 아름다움 |
| 프리마 시카트릭스 망치 | 첫 흠을 더 크게 깨지 않고 어디를 고쳐야 하는지 울림으로 알려 주는 망치 | 실패를 숨기지 않는 수선 |
| 노멘도장 | 공방 이름을 찍는 도장 | 이름을 남기는 기쁨과 지우는 위험 |
| 리디투스 별표 | 되돌아온 작품에 붙이는 작은 별표 | 실패가 아니라 다시 만질 자리라는 표시 |
| 무명쇳결 | 여러 손이 함께 다듬어 이름 하나로 닫히지 않는 쇳결 | 공동 작업의 흔적 |
| 평판녹 | 칭찬이 오래 쌓인 도장 주변에 생기는 얇은 녹 | 이름이 무게가 되는 징후 |
| 도르숨 도장천 | 완성품의 노멘도장을 잠시 덮어 쓰임을 먼저 만지게 하는 얇은 천 | 이름보다 쓰임을 먼저 보게 함 |
| 소키우스 손자국 | 도구 손잡이에 희미하게 떠오르는 여러 손의 자국 | 함께 만든 손을 지우지 않게 함 |
| 노멘 모라끈 | 도장을 찍기 전 함께 만든 손을 셀 때까지 묶어 두는 낮은 끈 | 이름을 잃지 않고 늦추는 예법 |
| 프리마 흠못 | 첫 흠 옆에 박아 고칠 방향을 낮게 울리는 작은 못 | 흠을 숨기지 않고 수선의 방향으로 바꿈 |
| 리디투스 온도장갑 | 되돌아온 물건에 마지막으로 남은 손 온도를 읽는 장갑 | 반품을 낙인이 아니라 사용자의 부탁으로 읽음 |
| 아노니마 결빗 | 쇳결과 나뭇결 사이의 함께 든 손길을 빗어 드러내는 빗 | 한 이름으로 모이지 않는 결을 나란히 보임 |

## 5. 주요 인물

| 인물 | 이 장소와의 관계 |
|---|---|
| 니프렌 | 손끝 심사장의 젊은 장인. `169` 기준 도르숨 도장천과 노멘 모라끈으로 이름 붙은 솜씨와 함께 만든 손의 긴장을 드러낸다 |
| 도멜 | 리디투스 작품창고 수선인. 리디투스 별표를 실패표가 아니라 다시 만질 별자리처럼 붙인다 |
| 라후 | 노멘도장 거리의 도장 파수. 노멘도장을 찍기 전 함께 만든 손의 수를 묻는다 |
| 세비르 | 아르티잔스와 닐세로안을 잇는 인물. 혼자 빛나는 작품과 함께 버티는 작품 사이에서 흔들릴 수 있다 |
| 콘코르디아 | 자기 목소리를 찾은 뒤, 자기 이름을 작품 위에 어떻게 남길지 다시 묻는 장면에 들어올 수 있다 |

## 6. 밤 변형

- 손끝 심사장에서는 이름을 먼저 말하면 심사대가 차갑게 굳고, 손잡이를 먼저 만지면 낮은 울림이 난다.
- 노멘도장 거리는 도장이 너무 크면 간판보다 그림자가 먼저 커진다.
- 리디투스 작품창고에서는 리디투스 별표가 실패한 날짜가 아니라 다시 고친 손의 온도로 빛난다.
- 프리마 시카트릭스 망치는 흠을 숨기려는 손에서는 무거워지고, 흠을 보려는 손에서는 작게 울린다.
- 무명 손길 우물은 혼자 만든 척한 사람이 손을 씻으면 물 위에 다른 손자국을 띄운다.
- 도르숨 도장천은 쓰임을 먼저 만지려는 손에서는 부드럽게 접히고, 다른 손을 숨기려는 손에서는 빳빳하게 굳는다.
- 소키우스 손자국은 손잡이 아래에서 낮게 떠오르지만, 한 사람 이름만 외치면 금가루처럼 흩어진다.
- 노멘 모라끈은 함께 만든 손을 셀 때까지 느슨하게 남고, 빨리 끊으면 시길룸 나방이 몰려온다.
- 프리마 흠못은 고칠 방향을 낮게 울리지만, 흠을 숨기면 나무 안쪽으로 가라앉는다.
- 리디투스 온도장갑은 되돌아온 물건의 마지막 온도를 읽지만, 수치로 대하면 손가락 끝이 차갑게 붙는다.
- 아노니마 결빗은 여러 결을 나란히 드러내지만, 한 이름으로 억지로 모으면 빗살이 휘어진다.

---

## 7. 스토리 활용

1. 눅세린의 인정 욕망을 가격과 박수만이 아니라, 솜씨와 평판의 생활 노동으로 확장한다.
2. `169` 기준 니프렌을 통해 인정받고 싶은 마음이 왜 사랑스럽고 위험한지, 그리고 이름이 함께 든 손을 가리지 않으려면 어떤 예법이 필요한지 동시에 보여 준다.
3. 세비르와 콘코르디아를 연결하면, "내 이름을 지키는 일"과 "함께 만든 손을 지우지 않는 일"의 긴장을 만들 수 있다.
4. 세리카 매듭골목과 연결하면 숨은 손노동, 닐세로안은 이름 붙은 손노동으로 대비된다.

## 8. 규칙과 금기

| 규칙/금기 | 이유 | 어기면 생기는 일 |
|---|---|---|
| 손끝 심사장에서는 이름보다 쓰임을 먼저 본다 | 이름은 물건의 쓰임을 대신할 수 없다 | 심사대가 차갑게 굳고 시길룸 나방이 모두 날아간다 |
| 첫 흠을 완전히 숨기지 않는다 | 흠은 실패뿐 아니라 고쳐야 할 방향도 알려 준다 | 프리마 시카트릭스 망치가 손에서 무거워진다 |
| 함께 만든 손의 이름을 지우지 않는다 | 좋은 물건은 한 사람의 빛만으로 오래 버티지 않는다 | 무명 손길 우물 위에 지워진 손자국이 떠오른다 |
| 리디투스 별표를 수치로 쓰지 않는다 | 되돌아온 물건은 다시 만질 기회를 품는다 | 리디투스 별표가 검게 식어 창고 길을 막는다 |
| 도르숨 도장천을 숨김막으로 쓰지 않는다 | 이름보다 쓰임을 먼저 보는 예법이지 책임 회피가 아니다 | 천이 빳빳하게 굳어 물건의 손잡이를 덮는다 |
| 노멘 모라끈을 조급하게 끊지 않는다 | 이름은 늦게 찍어도 사라지지 않는다 | 시길룸 나방이 몰려와 이름 가루를 갉아 먹는다 |
| 프리마 흠못을 빼내지 않는다 | 첫 흠은 수치가 아니라 고칠 방향을 알려 준다 | 못이 나무 안쪽으로 가라앉아 울림이 멎는다 |

삽입 장면 씨앗:

- 니프렌이 손끝 심사장 앞에서 자기 이름을 말하려다, 먼저 낡은 의자 팔걸이를 만져 보고 입을 다문다.
- 니프렌이 도르숨 도장천을 걷으려다 소키우스 손자국이 떠오르자, 노멘 모라끈을 다시 묶는다.
- 니프렌이 리디투스 온도장갑으로 되돌아온 손잡이를 읽고, 실패표 대신 프리마 흠못을 조심스럽게 박는다.
- 아노니마 결빗이 휘어지는 순간 세비르와 콘코르디아가 한 이름 아래 사라진 손을 서로 다르게 부른다.
- 도멜이 실패한 작품에 붙은 리디투스 별표를 떼지 않고, 새 손잡이 밑에 살짝 옮겨 붙인다.
- 라후가 콘코르디아에게 노멘도장을 건네기 전, "이 이름 아래 들어갈 손은 몇 명인가"라고 묻는다.
