# 프론티아 - 트리아넬

정사 등급: B급

소속: 에딕툼 감시탑 하단, 흰 프라이토르창 그림자에 붙은 생활 구역

핵심 은유: 다치지 않기 위해 말을 먼저 접는 조심성, 그리고 그 조심성이 자기 자신을 지우는 순간

---

## 1. 개요

트리아넬은 에딕툼 감시탑 아래에서 생겨난 낮은 문장들의 구역이다. 감시탑이 말을 규약 문장으로 번역한다면, 트리아넬 사람들은 탑이 판정하기 전에 스스로 말을 낮추고 접고 고른다.

이들은 겁이 많아서만 말을 줄인 것이 아니다. 한때 거친 말 하나가 군중을 다치게 하고, 급한 선언 하나가 실제 위험을 불렀던 시기가 있었다. 그래서 트리아넬의 조심성은 처음에는 보호였다. 그러나 오래 지나자 사람들은 다치게 할 말을 줄이는 법보다, 자신이 살아 있다는 말을 줄이는 법을 먼저 배우게 되었다.

> 조심스러운 말은 누군가를 살릴 수 있다. 그러나 모든 말이 미리 접히면, 살아 있는 사람은 자기 목소리보다 안전한 문장만 남긴다.

---

## 2. 첫인상

- 간판의 끝 글자가 살짝 접혀 있어, 어느 가게도 자기 이름을 큰소리로 끝내지 않는다.
- 거리에는 문을 닫는 종보다 다시 묻는 종이 더 많다.
- 주민들은 대화를 빨리 끝내지 않고, 중간에 낮은 쉼표를 남긴다.
- 아이들은 사과문보다 먼저 "지금 이 말을 받아도 되는가"를 묻는 법을 배운다.
- 바람이 불면 플리카 문장천들이 흰 빨래처럼 흔들리지만, 가까이 가면 지워진 말 대신 손때가 보인다.

## 3. 구조

| 구역 | 설명 | 장면 기능 |
|---|---|---|
| 복스 데미사 계단 | 말을 시작하기 전 한 칸씩 내려가 목소리를 낮추는 계단 | 배려와 자기삭제의 차이를 보임 |
| 우눔 발판 | 계단 중간에 놓인 작은 발판 | 아무 말도 못 하겠는 사람에게 한 단어의 자리를 줌 |
| 리토르노 난간 | 낮춘 말을 끝낸 뒤 다시 올라올 때 잡는 낮은 난간 | 낮춘 말이 자기 지움으로 굳지 않게 함 |
| 코마 공방 | 끝내지 않아도 되는 문장을 쉼표 모양 핀으로 묶는 공방 | 말이 곧 판결이 되지 않게 함 |
| 세콰르 보크스걸이 | 쉼표 뒤에 돌아올 말을 실로 걸어 두는 낮은 걸이 | 보류가 책임 있는 약속인지 보임 |
| 종언받침대 | 자기 숨으로 끝난 문장 아래 받침을 놓는 낮은 대 | 마침과 덮음의 차이를 보임 |
| 플리카 문장터 | 아직 말하지 못한 문장을 천에 적어 씻고 말리는 빨래터 | 숨긴 말과 기다린 말의 차이 |
| 스피라 마른줄 | 플리카 문장천을 햇빛보다 낮은 그늘에 말리는 줄 | 기다림과 방치의 차이를 보임 |
| 세미플리카 바늘대 | 문장을 한 번에 펴지 않고 한 겹씩 풀기 위한 낮은 바늘대 | 배려와 재촉의 차이를 보임 |
| 빈끝표 보관소 | 함부로 마침표를 찍지 못한 문장 끝을 보관하는 작은 서가 | 끝내지 못한 말도 자리를 갖게 함 |
| 문턱종탑 | 상대가 받을 준비가 되었는지 먼저 묻는 낮은 종탑 | 허락과 회피의 경계 |
| 반당김 종줄 난간 | 큰 종을 울리기 전 반만 당기는 줄이 낮게 걸린 난간 | 작은 시작과 끝없는 지연의 차이를 보임 |
| 빈자리 방울대 | 대답할 사람의 빈자리를 대신 채우지 않기 위해 방울을 거는 대 | 존중과 대리 응답의 차이를 보임 |
| 알바 여백골목 | 판정창에 걸리지 않도록 말 사이에 여백을 남기는 골목 | 보호가 자기 지움으로 바뀌는 순간 |
| 반쯤 열린 입문 | 들어왔다는 선언 없이도 머물 수 있는 작은 문 | 정체를 당장 확정하지 않아도 되는 자리 |

## 4. 유기물과 소품

| 이름 | 설정 | 활용 |

| 코마풀 | 끝난 문장과 다음 문장 사이의 틈에서 자라는 낮은 풀 | 말을 끝장내지 않고 이어 둘 수 있음을 보임 |
| 낮말초 | 낮게 말했지만 숨기지는 않은 문장 곁에서 피는 작은 흰 꽃 | 배려와 침묵의 차이를 보여줌 |
| 괄호나방 | 잠시 접어 둔 말 주변을 맴도는 옅은 나방 | 오래 갇힌 말과 다시 꺼낼 말을 구분 |
| 복스 데미사 패 | 지금은 낮게 말하겠다는 뜻의 작은 표 | 말하지 않을 권리와 영원히 숨는 습관을 분리 |
| 우눔 표석 | 오늘 말할 수 있는 한 단어를 올려 두는 작은 표석 | 작아져도 자기 말의 주인이 남게 함 |
| 보크스 높이끈 | 자기 말이 닿을 수 있는 높이를 묶어 표시하는 얇은 끈 | 배려의 높이를 남이 정하지 못하게 함 |
| 보크스 등잔 | 말하는 이가 아직 자기 자리에 있는지를 비추는 낮은 등잔 | 큰소리가 아니라 남은 자리를 확인 |
| 리토르노 표 | 낮춘 뒤 다시 자기 높이로 돌아올 길을 남기는 표 | 낮춤이 한쪽 방향으로만 굳지 않게 함 |
| 도르숨 쉼표 | 말하기 전 손등에 찍어 잠깐 멈추는 작은 쉼표 | 쉼과 영원한 미룸을 구분 |
| 코마 핀 | 문장을 완결하지 않고 이어 두는 은색 핀 | 판결 대신 보류를 만드는 소품 |
| 세콰르 말실 | 쉼표 뒤에 돌아와야 할 말을 느슨히 묶어 두는 실 | 보류가 떠넘김이 아니라 맡은 말이 되게 함 |
| 종언먹 | 문장이 자기 숨으로 끝났는지 잘려 끝났는지 남기는 먹 | 끝과 절단의 차이를 보임 |
| 리디투스 날짜표 | 아직 끝내지 못한 문장에 다시 만날 날을 붙이는 표 | 기다림과 방치를 구분 |
| 종언받침 | 돌아온 말이 자기 숨을 다했을 때 마침표 아래 놓는 받침 | 끝낼 수 있는 말을 다음 하루로 보냄 |
| 플리카 문장천 | 아직 말할 준비가 되지 않은 문장을 싸 두는 천 | 숨김이 아니라 기다림으로 쓰일 때 따뜻해짐 |
| 베스티기움 비누 | 접힌 자국의 먼지만 씻는 작은 비누 | 글자를 지우지 않고 접힘의 무게만 덜어 냄 |
| 도미누스 손집게 | 말한 사람의 손이 닿기 전까지 천을 잡아 두는 집게 | 기다림과 영원한 보관을 구분 |
| 스피라 마른줄 | 말할 준비가 될 때까지 천의 무게와 숨을 말리는 낮은 줄 | 준비와 방치의 시간차를 보임 |
| 세미플리카 바늘 | 문장을 감당 가능한 만큼 한 겹씩 푸는 바늘 | 폭로가 아니라 단계적 펼침을 만듦 |
| 플리카 온도석 | 접힌 문장이 오늘 펼칠 온도인지 확인하는 작은 돌 | 준비와 호기심을 구분 |
| 빈끝표 | 마침표 대신 비워 둔 문장 끝 표식 | 끝내지 못한 말도 버리지 않게 함 |
| 문턱종 | 말하기 전 상대의 준비를 묻는 낮은 종 | 조심성이 회피가 되지 않게 하는 예법 |
| 초문각인 | 종을 치기 전 자기 손바닥에 적는 오늘 말할 한 줄 | 묻는 사람도 자기 말의 무게를 손에 올리게 함 |
| 반당김 종줄 | 큰 종을 울리기 전 반만 당기는 낮은 줄 | 작은 시작과 끝없는 지연을 구분 |
| 맞손 종추 | 종 안쪽에 매달린 작은 추 | 말하는 손과 듣는 손의 무게를 함께 읽음 |
| 빈자리 방울 | 대답할 사람의 자리를 비워 두기 위해 난간에 다는 방울 | 존중과 대신 대답하기를 구분 |
| 접령분 | 질문이 진짜로 닿았는지 종 아래에 떨어지는 은색 가루 | 질문과 핑계를 손바닥에 드러냄 |

## 6. 규칙과 금기

| 규칙/금기 | 이유 | 어기면 생기는 일 |
|---|---|---|
| 복스 데미사 패를 입막음 표로 쓰지 않는다 | 말하지 않을 권리와 말하지 못하게 하는 일은 다르다 | 표 가장자리가 입술 모양으로 굳어 떨어지지 않는다 |
| 우눔 표석에 남의 단어를 대신 올리지 않는다 | 한 단어라도 말한 사람의 것이어야 한다 | 표면이 희게 비어 발자국만 비춘다 |
| 리토르노 표를 버린 채 계단을 내려가지 않는다 | 낮춘 말은 다시 올라올 길을 가져야 한다 | 계단이 한쪽으로만 이어져 목소리보다 발자국만 많아진다 |
| 코마 핀으로 남의 말을 영원히 묶지 않는다 | 보류는 계속 묻기 위한 자리이지, 끝없는 지연이 아니다 | 핀이 곧은 못으로 변해 문장천을 찢는다 |
| 리디투스 날짜표 없이 코마 핀을 새로 꽂지 않는다 | 이어 둔 말은 다시 만날 책임을 가져야 한다 | 코마 핀이 곧은 못이 되어 공방 벽에 박힌다 |
| 종언받침을 너무 일찍 놓지 않는다 | 마침표는 덮개가 아니라 자기 숨으로 끝난 말의 받침이다 | 받침이 뚜껑처럼 닫혀 남은 말이 안쪽에서 울린다 |
| 플리카 문장천을 주인 없이 펴지 않는다 | 기다린 말도 말할 사람의 손이 먼저 닿아야 한다 | 글자가 흰 먼지처럼 흩어진다 |
| 도미누스 손집게를 핑계로 영원히 물려 두지 않는다 | 말하지 않을 권리도 돌아올 길을 가져야 한다 | 집게가 천에 파고들어 접힌 자국이 닫힌다 |
| 플리카 온도석을 남의 호기심으로 쥐지 않는다 | 펼칠 준비는 구경꾼이 아니라 말한 사람의 손에서 확인해야 한다 | 돌이 식어 천의 글자가 더 낮게 가라앉는다 |
| 문턱종을 핑계로만 울리지 않는다 | 준비를 묻는 일은 대화를 피하는 일이 아니다 | 종소리가 안쪽에서만 울려 아무에게도 닿지 않는다 |
| 초문각인 없이 종을 크게 울리지 않는다 | 묻는 사람도 자기 말의 무게를 손에 올려야 한다 | 글씨 없는 손금 사이로 종소리가 안쪽으로 빨려 들어간다 |
| 반당김 종줄을 오래 잡고만 있지 않는다 | 작은 시작은 영원한 지연이 아니다 | 종줄이 손목에 감겨 손을 내리지 못한다 |
| 빈자리 방울에 대신 대답하지 않는다 | 빈자리는 상대의 자리지, 누군가 대신 채울 구멍이 아니다 | 방울 속이 비어 차갑게 돈다 |

삽입 장면 씨앗:

- 테아르가 계단 아래로 내려가려는 아이를 붙잡지 않고, 같은 높이까지 내려가 우눔 표석을 놓고 "여기서 한 단어만 말해도 된다"고 말한다.
- 모디안이 너무 빨리 닫힌 판결문 끝에 코마 핀을 꽂고, 세콰르 말실과 리디투스 날짜표를 함께 묶어 문장 아래에 작은 여백을 남긴다.
- 세루아가 오래 플리카 문장천을 바로 펴지 않고, 도미누스 손집게와 플리카 온도석을 확인한 뒤 세미플리카 바늘로 한 겹만 푼다.
- 릴메가 문턱종을 치려다 멈추고, 자기 손바닥에 남은 글자 없는 줄을 먼저 세루아에게 보여 준 뒤 초문각인을 한 줄 적는다.

## 7. 스토리 활용

1. 에딕툼 감시탑의 거대한 판정이 주민의 일상 언어에 어떻게 스며드는지 보여준다.
2. 옵세르바 A-1이 외부 시선의 압박이라면, 트리아넬은 스스로 말을 접는 습관으로 구분한다.
3. 도그마 축을 납작한 악으로 만들지 않고, 다치지 않게 하려던 문장이 어떻게 살아 있는 말을 줄였는지 보여준다.
4. 카산드라나 루멘이 이곳을 지나갈 때, 진실을 말하는 일과 받을 수 있게 묻는 일이 다르다는 장면을 만들 수 있다.
5. 테아르 장면은 큰소리와 침묵의 대립이 아니라, 낮춘 말에도 자기 주인과 되돌아올 높이가 남는지를 우눔 표석과 리토르노 표로 보인다.
6. 모디안 장면은 결론과 미룸의 대립이 아니라, 아직 끝낼 수 없는 말에 돌아올 책임을 붙이고 끝낼 수 있는 말에는 종언받침을 놓는 선택으로 보인다.
7. 세루아 장면은 비밀과 폭로의 대립이 아니라, 말하지 않을 권리를 지키되 영원히 접힌 말이 사라지지 않도록 돌아올 손과 온도를 남기는 선택으로 보인다.
8. 릴메 장면은 질문과 침묵의 대립이 아니라, 받을 준비를 묻는 선의가 자기 말을 미루는 핑계로 굳지 않도록 초문각인과 맞손 종추를 함께 확인하는 선택으로 보인다.

## 말투와 사고방식 운용

### 지역 말결

- 기본 결은 프론티아 계열이다. 절차, 계산, 보고가 말의 뼈대를 만든다. 다만 좋은 프론티아 장면은 규칙 뒤에 남은 피로와 망설임을 함께 보여준다.
- 트리아넬에서는 장소의 생활 리듬이 말의 순서를 정한다. 인물은 감정을 바로 해설하기보다 문턱, 손, 작은 표식 같은 구체물을 먼저 확인한다.
- 자주 떠오르는 어휘권은 확인, 보고, 오차이다. 단어를 표식처럼 남발하지 말고, 장면의 손동작과 물성에 묻혀서 들리게 한다.

### 장면 운용

- 낮 장면: 장소의 기능을 해설하지 말고, 사람이 움직이는 순서로 먼저 보여준다.
- 밤 장면: 같은 공간이 밤의 차원으로 넘어가면, 이 장소의 생활 습관이 본능과 충돌한다. 싸움은 초현상보다 사람의 고집, 경쟁심, 보호 본능이 물건과 동선을 망가뜨리는 방식으로 먼저 발생한다.
- 외부자 반응: 외부인은 이곳 사람들의 말이 차갑거나 느리다고 느낄 수 있다. 하지만 장면은 설명 대사보다 기다리는 순서, 건네는 물건, 비워 둔 자리로 지역성을 증명한다.

### 금기

- 트리아넬의 지역성을 사투리 표기만으로 처리하지 않는다. 말끝 변형보다 사고 순서, 침묵의 길이, 물건을 다루는 방식이 우선이다.
- 장소 설명을 인물 입으로 길게 해설하지 않는다. 문턱, 손, 작은 표식 같은 구체물을 만지는 손, 줄 서는 순서, 숨을 고르는 박자로 먼저 보인다.
- 밤 장면에서도 바닥 균열이나 괴현상을 기본값으로 쓰지 않는다. 밤의 위험은 발현된 사람들의 본능, 충돌, 파손 여파에서 출발해야 한다.
