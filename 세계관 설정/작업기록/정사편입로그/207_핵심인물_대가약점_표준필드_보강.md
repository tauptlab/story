# 핵심 인물 대가/약점 표준 필드 보강

## 1. 작업 배경

능력 구조 감사에서 오래된 핵심 인물 문서 일부가 능력의 위험과 대가를 본문에는 갖고 있지만, 표준 표 항목 `대가/약점`으로는 분리하지 않은 상태임을 확인했다.

최신 운용 기준상 고유 특수능력은 강한 가치관의 화신과 핵심 결투 인물에게만 허용되며, 해당 인물은 반드시 다음 필드를 분리해야 한다.

- 능력명
- 작가용 의미
- 전투 기능
- 대가/약점
- 소품/물성
- 녹티오라 변형
- 결투 기능

## 2. 반영 대상

다음 25개 핵심 인물 문서에 `대가/약점` 표준 행을 추가했다.

| 인물 | 보강 방향 |
|---|---|
| 나르키소스 | 반사면이 타인의 욕망에 맞춰지며 본래 얼굴을 잃는 위험 |
| 도그마 | 규약이 자신과 타인의 변화를 막는 위험 |
| 레기나 페쿤디아 | 재촉된 성장이 유리처럼 굳고 책임 없는 낙관이 되는 위험 |
| 레마니스 | 과거의 아름다움이 현재 책임을 밀어내는 위험 |
| 레툼 | 끝의 선언이 살아 있는 선택까지 누르는 위험 |
| 로고스 | 빈칸을 결론으로 덮고 현재의 고통을 지나치는 위험 |
| 루멘 | 진실의 빛이 구원보다 처벌의 밝기가 되는 위험 |
| 리버 | 끊은 뒤의 방향을 세우지 못해 해방이 추락로가 되는 위험 |
| 리타스 | 환대가 끝없는 착취 유출로 바뀌는 위험 |
| 마이더스 | 가격표가 값 없는 관계와 자기 결핍을 덮는 위험 |
| 므네몬 | 보호 봉인이 감금과 열람권 박탈로 굳는 위험 |
| 블라드 드라쿨 | 빼앗은 이름이 많아질수록 자기 이름이 흐려지는 위험 |
| 사일러스 | 진심이 계약 조항으로 늦춰지고 위약 사슬이 길어지는 위험 |
| 세리엘 | 가시가 진심까지 문턱 밖에 세우는 위험 |
| 셀레네 | 이름을 고정하는 빛이 길표가 아니라 닫힌 원이 되는 위험 |
| 소키우스 | 협력이 빈칸 없는 역할표로 굳는 위험 |
| 실바누스 | 숨긴 지식의 뿌리가 길을 막고 자신도 묶는 위험 |
| 아마라 | 환대가 자기 몫을 태우는 고리로 닫히는 위험 |
| 아미키티아 | 동의 없는 연결이 철사가 되고 관계가 동원령이 되는 위험 |
| 오블라투스 | 이름 없는 무게가 복수를 정당화하며 자기 이름도 흐리는 위험 |
| 이놉스 | 정확한 절망이 모든 첫 발을 무의미하게 만드는 위험 |
| 파불라 | 부드러운 이야기가 사실의 모서리와 증언의 도착을 늦추는 위험 |
| 팔수스 | 거짓 희망을 깨다 회복 가능성까지 조롱하는 위험 |
| 포르티스 | 불씨를 독점해 생존의 빛이 감옥 중심불이 되는 위험 |
| 포보스 | 두꺼운 경계가 진짜 표정을 숨기고 보호가 타인의 밤이 되는 위험 |

## 3. 운영 메모

- 이번 보강은 새 능력 추가가 아니라 표준 필드 정렬이다.
- 이 25명은 모두 고유 특수능력 보유자로 남긴다. 다만 능력은 강한 가치관/핵심 결투 조건 때문에 허용되는 예외이며, 평범한 생활 인물에게 확장하지 않는다.
- 이후 새 고유 특수능력을 추가할 때는 `대가/약점` 필드를 누락하면 아직 집필 참조 불가로 본다.
