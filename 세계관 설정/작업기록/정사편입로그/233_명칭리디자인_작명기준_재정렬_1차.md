## 1. 새 판단 기준

| 기준 | 내용 |
|---|---|
| 외래어 일괄화 금지 | 외래어는 이미지가 선명하고 입에 붙을 때만 유지한다 |
| 사물 물성 보존 | `주사위`, `열쇠`, `문`, `꽃`, `가위`, `끈`, `방패`, `잔` 같은 물건 이름은 가능한 한 남긴다 |
| 직접어 제거 | 작가용 의미가 그대로 보이는 이름은 교체한다 |
| 짧고 강한 한국어 허용 | `파열동`, `결류`, `야장가`처럼 장면성이 있으면 한국어/한자어 조어도 사용한다 |
| 능력 강도 구분 | 강한 가치관만 고유 특수능력을 가진다. 약한 발현과 사소한 감각에는 굳이 기술명을 붙이지 않는다 |

---

## 2. 유지한 명칭

| 명칭 | 판정 | 이유 |
|---|---|---|
| `프리즈마 알바` | 유지 | 외래어지만 빛/분광 이미지가 즉각적이고 전투 장면에서 부르기 쉽다 |
| `코로나 루브라` | 유지 | 왕관/붉은 고리 이미지가 선명하고 장르적 기술명으로 기능한다 |
| `크로노 스타시스` | 유지 | 시간 정지/정체 이미지가 바로 떠오르며 발음이 안정적이다 |
| `오컴의 면도날` | 유지 | 철학적 의미와 물성 있는 전투 기능이 함께 살아 있다 |
| `파열동` | 유지 | 짧고 강하며 장면의 충격이 먼저 떠오른다 |
| `결류` | 유지 | 직접 해설은 아니지만 흐름과 결합의 이미지가 남는다 |
| `야장가` | 유지 | 밤, 장송, 노래 이미지가 겹치되 작가용 의미를 바로 설명하지 않는다 |

## 3. 교체한 명칭

| 기존명 | 새 정식명 | 분류 | 이유 |
|---|---|---|---|
| `알레아 프로스펙트` | `선견의 주사위` | 고유 특수능력/소품 겸용 | 주사위라는 핵심 물성이 사라져 복원 |
| `프로노이아 주사위` | `선견의 주사위` | 소품/물성 | 과도한 외래어 중복을 줄이고 기존 기억 가능명을 복원 |
| `크리스탈리스 살베온` | `크리스탈리스 보호처` | 특수공간 | 보호처라는 공간 기능이 자연스럽고 독자 기억성이 높음 |
| `아이기스권` | `보호 대상` | 판정 명칭 | 현장 행정어로서의 차가움이 더 적합함 |
| `오르도권` | `교정 대상` | 판정 명칭 | 기능이 복잡한 외래어보다 명확하고 장면에 붙음 |
| `피니스권` | `제거 대상` | 판정 명칭 | 판정의 폭력성이 직접적 행정어로 더 잘 드러남 |
| `세쿤다 열쇠` | `두 번째 열쇠` | 소품/물성 | 열쇠 물성과 질문 예법이 살아남 |
| `세쿤다 문` | `두 번째 문` | 장소/물성 | 문이라는 기능이 중심이므로 과한 외래어를 제거 |

## 4. 보류한 명칭

| 명칭 | 보류 이유 | 다음 판단 |
|---|---|---|
| `허무의 그림자` | 의미가 보이지만 기능명보다 시적 이명에 가깝다 | 실제 장면에서 너무 설명적으로 느껴지면 교체 |
| `크리스탈리스 보호처` | `보호`가 보이지만 공간 기능명으로 자연스럽다 | 능력명으로 쓰지 않고 장소명으로만 운용 |
| `선견의 주사위` | `선견`이 의미를 일부 드러내지만 사물명과 장면성이 강하다 | 카우투스의 능력명/핵심 소품으로 유지하되 작가용 해설을 줄임 |
| `두 번째 열쇠` | 기능이 드러나지만 바레노스의 확인 예법과 물성이 살아 있다 | 밤 변형에서 직접 해설이 늘어나지 않도록 주의 |

---

## 5. 능력명에서 소품/물성으로 재분류한 항목

| 항목 | 기존 문제 | 재분류 |
|---|---|---|
| `선견의 주사위` | 외래어 기술명으로 치환되며 물성이 약해짐 | 카우투스의 고유 특수능력이자 핵심 소품으로 병행. 실제 장면에서는 주사위를 굴리는 행위로 전투 기능을 보임 |
| `크리스탈리스 보호처` | 공간명이 외래어 시설명처럼 변해 기능이 흐려짐 | 능력명이 아니라 특수공간/보호 윤리 시험장으로 고정 |
| `보호 대상/교정 대상/제거 대상` | 판정 명칭이 외래어 권역명으로 변하며 행정 폭력성이 흐려짐 | 셀로/프라이토르 계열의 판정어 및 녹티오라 분류선으로 운용 |
| `두 번째 열쇠` | 열쇠가 기술명처럼 외래어화됨 | 바레노스 확인 예법을 보여주는 소품/문 물성으로 운용 |

## 6. 아직 추가 판단이 필요한 항목

| 항목군 | 판단 필요 이유 |
|---|---|
| `인터발룸 회랑`, `프리마 소누스 교실`, `모라 플라우수스석` | 외래어가 연속되어 기억성이 약해질 수 있음 |
| `프락타 라에티아 탑`, `테피다 티툴루스 시장`, `안티테르마 결정` | 의미와 이미지는 있으나 한국 독자 입에 붙는지 재검토 필요 |
| `모라 클렙시드라`, `모라 스페쿨룸방` | 장면 이미지는 있지만 외래어 밀도가 높음 |
| `프리마 라수라`, `데클리나 핀`, `프로템 패` | 사물명은 남아 있으나 기술/소품명으로 너무 낯설 수 있음 |
| 최근 도시별 생활 물성명 전체 | 도시/출신 말맛과 작명 방식이 일관되는지 2차 점검 필요 |

---

## 7. 다음 리디자인 우선순위

1. 정식바이블의 소품/생태/물성 도감에서 외래어가 연속된 항목을 유지/보류/교체로 판정한다.
2. 도시별설정 문서에서 각 도시의 문화권과 말맛을 기준으로 명칭군을 다시 묶는다.
3. 등장인물 문서에서 고유 특수능력, 약한 발현, 사소한 감각, 능력 없음의 구분을 재확인한다.
4. 능력명으로 남겨야 할 것과 소품/물성/녹티오라 변형으로 내려야 할 것을 다시 분류한다.
5. 변경할 때마다 기존명/새 이름 대응표를 새 작업기록에 남긴다.
