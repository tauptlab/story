# 라에나르 정사 승격 1차

정사 등급: 작업 기록

역할: 루트 초기 자료의 `라에나르` 씨앗을 로루멘 표정 대여소의 생활 인물로 승격하고, "인정받기 위해 자신을 바꾸는 성실함"을 맞춤의 보호성과 자기 삭제의 위험 사이에서 정리한다.

---

## 1. 승격 요약

라에나르는 기존 자료에서 `라에나르(구 라에나)`로 남아 있던 로루멘 갈등 인물 후보였다. 이번 승격에서는 그를 단순히 남에게 맞추는 인물이 아니라, 표정을 빌리는 일이 실제로 사람을 지켜 준다는 사실을 아는 견습 안내인으로 정리했다.

핵심 이동은 아래와 같다.

| 구분 | 승격 전 | 승격 후 |
|---|---|---|
| 인물 기능 | 로루멘 갈등 인물 후보 | 표정 대여소의 맞춤/쉼 생활 인물 |
| 핵심 질문 | 인정받기 위해 자신을 바꾸는가 | 맞춤을 권하기 전 쉬어도 되는 자리를 줄 수 있는가 |
| 주 무대 | 로루멘 후보 | 표정 대여소, 맞춤자 리본 서랍, 쉼물컵 분장대 |
| 주요 물성 | 미정 | 맞춤자 리본, 권유서랍 손잡이, 쉼물컵, 민얼굴풀 |
| 결투 기능 | 미정 | 자기 맞춤의 보호성과 고치지 않은 얼굴의 머물 권리 재배치 |

## 2. 124 기준 반영

| 왜 버릴 수밖에 없었는가 | 있는 그대로 들어간 자리에서 얼굴이 맞지 않는다는 판정을 받았고, 먼저 고쳐 맞추는 일이 생존법으로 남았다 |
| 대체 가치관 | 상대가 좋아할 표정과 말투를 먼저 골라 거절당할 위험을 줄이는 습관 |
| 처음의 보호 기능 | 거칠고 빠른 관계 안에서 덜 다치게 하고, 첫 문턱을 넘게 했다 |
| 시간이 지나며 생긴 왜곡 | 모든 만남이 맞춤실이 되고, 사랑받기 전 자기 얼굴을 계속 잘라 내게 된다 |
| 귀환 계기 | 표정 대여소의 밤 서랍이 먼저 열리는 결투에서 라에나르가 서랍 대신 쉼물컵을 먼저 내민다 |
| 필연성 | 맞춤만으로는 사람이 자기 얼굴을 잃고, 맞춤을 모두 부정하면 생존의 옷까지 수치가 되므로 쉬는 순서가 필요해진다 |
| 귀환 후 후폭풍 | 로루멘은 표정을 빌리기 전 쉼물컵 분장대를 지나가게 되고, 표정 대여소의 권유서랍은 손님의 첫 숨을 기다리게 된다 |

## 3. 녹티오라 설계

| 낮의 요소 | 왜 남았는가 | 밤의 형태 | 선택 시험 |
|---|---|---|---|
| 권유서랍 손잡이 | 알맞은 얼굴을 빨리 찾으면 덜 다칠 수 있다는 믿음이 남았다 | 손님이 앉기도 전에 서랍이 열리고 맞지 않는 얼굴들이 떨어진다 | 보호인가, 재촉인가 |
| 맞춤자 리본 | 방에 맞는 얼굴을 재던 습관이 목의 선으로 남았다 | 오래 두르면 목을 조이고, 쉼물컵 곁에서는 손목끈처럼 느슨해진다 | 맞춤인가, 삭제인가 |
| 쉼물컵 | 표정을 고르기 전 잠시 쉬고 싶었던 갈증이 남았다 | 마시지 않고 서랍을 열면 식고, 한 모금 뒤에는 손잡이 온도를 돌려준다 | 쉼인가, 핑계인가 |
| 민얼굴풀 | 아무 표정도 고르지 못한 의자 아래의 시간이 남았다 | 서두른 권유 앞에서는 접히고, 기다림 앞에서는 낮게 자란다 | 실패인가, 시작 전 자리인가 |

## 5. 결투 설계 메모

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 인정받기 위한 자기 맞춤 vs 고치지 않은 얼굴로 잠시 머물 권리 |
| 두 가치의 옳음 | 맞춤은 사람을 덜 다치게 하고, 머물 권리는 맞추기 전의 사람을 지운 얼굴로 만들지 않는다 |
| 두 가치의 위험 | 맞춤은 자기 삭제가 되고, 머물 권리는 모든 생존의 옷을 수치로 만들 수 있다 |
| 끝난 뒤 재배치 | 맞춤은 사랑받기 위한 증명이 아니라 잠시 빌릴 수 있는 옷이 되고, 쉬는 얼굴은 실패가 아니라 시작 전 자리로 남는다 |
| 도시 변화 | 표정 대여소는 서랍을 열기 전 쉼물컵 분장대를 먼저 지나가고, 맞춤자 리본은 목을 조이지 않고 손목에 느슨히 묶인다 |
| 인물 변화 | 라에나르는 손님이 다칠까 봐 서랍을 먼저 열던 손을 멈추고, 물컵과 의자를 먼저 내민다 |
