# 1부 중위 누락 카드 상세 초안

이 문서는 `19_1부_통합요소_추출과_누락카드.md`에서 중위 우선도로 분류한 요소를 세부 카드로 승격하기 위한 초안이다. `20_1부_누락카드_상세초안.md`가 1부 상위 후보를 다룬다면, 이 문서는 장면의 생활감과 모순 방지에 필요한 중간 요소를 담당한다.

---

## 1. 샘터

정사 등급: B급 후보

| 항목 | 설정 |
|---|---|
| 분류 | 장소/의식 원형 |
| 소속 | 눅세린-암디르 접경 마을, 길목의 등불 뒤뜰 우물길 |
| 낮의 형태 | 얕은 돌샘. 물은 맑지만 바닥에 아주 작은 빛 조각들이 섞여 있다 |
| 밤의 변형 | 물보다 먼저 사람의 시선이 수면에 뜬다. 무엇을 먼저 보느냐에 따라 샘의 빛이 달라진다 |
| 은유 | 첫 욕망, 첫 비교, 첫 세르가 한 자리에서 엉키는 장소 |
| 관련 인물 | 셀로, 마르네, 익명의 광물 우선 인물, 에이도스 |
| 대가 | 샘터에서 오래 물끄러미 보면 자신이 원한 것과 남이 부러워한 것을 헷갈린다 |
| 장면 기능 | 셀로가 처음으로 "사람보다 물건을 먼저 보는 시선"을 목격하게 한다 |

활용:

- 샘터는 센시르의 반향의 샘이나 몸울림 우물보다 훨씬 작고 생활적이어야 한다.
- 이곳의 물은 진실을 말하지 않는다. 다만 시선의 우선순위를 보여준다.
- 후반 회수 시, 셀로가 같은 샘을 다시 떠올릴 때 반짝이는 광물보다 그 곁에 서 있던 사람의 손을 기억하면 좋다.

## 2. 안대

정사 등급: B급 후보

| 항목 | 설정 |
|---|---|
| 분류 | 무기물/관습 소품 |
| 소속 | 녹티오라 통행 관습, 밤의 순찰자, 길목의 등불 |
| 낮의 형태 | 검은 천 안대. 안쪽에는 아주 가는 은실이 꿰매져 있다 |
| 밤의 변형 | 보지 않기 위해 쓴 것이 아니라, 너무 많이 보지 않기 위해 쓰는 통행 필터가 된다 |
| 은유 | 회피와 보호의 경계 |
| 일상 사용자 | 밤의 순찰자, 접경 주민, 겁이 많은 여행자 |
| 관련 인물 | 리버, 테른, 셀로, 마르네 |
| 대가 | 오래 쓰면 위험을 덜 보는 대신 도움을 청하는 표정도 놓친다 |
| 파손 방식 | 누군가 스스로 보겠다고 결심하면 안쪽 은실이 느슨해진다 |
| 장면 기능 | 녹티오라가 단순한 어둠이 아니라 감당 가능한 만큼만 세계를 보게 하는 관습을 가졌음을 보여준다 |

사용 원칙:

- 안대를 쓴 사람을 비겁하게만 그리지 않는다.
- 어떤 인물에게는 안대가 생존 기술이고, 리버에게는 감금처럼 느껴진다.
- 셀로는 안대의 옳고 그름보다, 누가 왜 그것을 쓰게 되었는지를 먼저 배워야 한다.

## 3. 속삭임 파편

정사 등급: B급 후보

| 항목 | 설정 |
|---|---|
| 분류 | 무기물/전언 매개체 |
| 소속 | 눅세린 벨라사르, 소문 거래망, 멘다키우스 계열 |
| 낮의 물성 | 손톱만 한 얇은 금속성 결정 조각. 귓가에 대면 멀리서 한 번 말한 문장이 아주 작게 되울린다 |
| 밤의 변형 | 들은 말이 원문 그대로가 아니라, 듣는 사람이 가장 믿고 싶어 하는 형태로 변한다 |
| 은유 | 전달, 소문, 왜곡된 공감 |
| 일상 사용자 | 소문꾼, 항구 정보상, 마이더스 하수인, 멘다키우스 |
| 관련 인물 | 셀로, 에이도스, 콘코르디아, 멘다키우스 |
| 대가 | 오래 지니면 직접 확인하기보다 "누가 그렇게 말했다"에 기대게 된다 |
| 파손 방식 | 두 사람이 같은 말을 서로 다른 이유로 믿는 순간 금이 간다 |
| 장면 기능 | 현대적 통신 장치 조각을 작중어로 치환하고, 벨라사르의 소문 경제를 물질화한다 |

작중 사용:

- 에이도스는 파편을 분석하려 하지만, 파편은 논리보다 평판의 방향을 먼저 보여준다.
- 콘코르디아가 파편을 들으면 여러 목소리가 합의처럼 들린다. 이것이 그의 약점을 찌른다.

## 4. 질문자들

정사 등급: B급 후보

작중 별칭 후보: 문답지기, 틈문지기, 빈칸지기

| 항목 | 설정 |
|---|---|
| 분류 | 집단/의식 집행자 |
| 소속 | 암디르 그림자 골짜기, 질문 의식의 방들 |
| 선물 | 도망치던 질문을 말이 아닌 공간과 물건으로 바꿔 보여준다 |
| 상처 | 한때는 사람들의 공포를 대신 물어주던 존재였으나, 너무 많은 답 없는 비명을 맡으며 얼굴을 잃었다 |
| 거짓 믿음 | 질문받는 자가 아파할수록 답에 가까워진다 |
| 낮의 흔적 | 탑 벽면의 작은 구멍, 방 안쪽에 접힌 종이, 돌문 아래의 낡은 손자국 |
| 밤의 형태 | 키가 제각각이고 얼굴이 비어 있는 질문 집행자. 직접 공격보다 질문의 구조를 바꾼다 |
| 사랑스러운 습관 | 답을 들은 뒤, 틀린 답이라도 한 번은 접어 주머니에 넣는다 |
| 관련 인물 | 포보스, 셀로, 에이도스, 리버, 콘코르디아 |
| 반작용 | 질문받는 자가 질문을 되묻는 순간 힘이 약해진다 |
| 장면 기능 | 질문 의식이 추상 장치가 아니라 살아 있는 압박과 오래된 노동으로 보이게 한다 |

주의:

- 질문자들은 우스꽝스러운 장난꾼이 아니라, 공포 처리 제도의 오래된 하급 집행자다.
- 완전히 악하지 않다. 그들은 답을 모으고 싶어 하지만, 답을 어떻게 돌려줘야 하는지 잊었다.

## 5. 석상 파편과 망치 자국

정사 등급: B급 후보

| 항목 | 설정 |
|---|---|
| 분류 | 무기물/역사 단서 |
| 소속 | 암디르 그림자 골짜기, 포보스의 탑 주변 |
| 낮의 물성 | 깨진 얼굴, 손, 날개, 뿔이 흩어진 검은 석상 조각. 표면마다 같은 망치 자국이 있다 |
| 밤의 변형 | 조각들이 완성되려 할수록 더 큰 공포의 형상을 만들고, 망치 자국은 이를 다시 멈추는 틈이 된다 |
| 은유 | 포보스가 공포를 만들고, 동시에 완성되기 전에 부수어 온 역사 |
| 관련 인물 | 포보스, 셀레네, 에이도스, 셀로 |
| 대가 | 조각을 맞추려는 사람은 포보스의 실패까지 함께 재현하게 된다 |
| 파손 방식 | 이미 깨져 있으나, 애도 없이 조립하면 조각 사이에서 검은 소리가 난다 |
| 장면 기능 | 포보스가 살해자가 아니라 통제에 실패한 관리자였다는 단서를 초반에 심는다 |

브라다만테와의 관계:

- 브라다만테는 단순 무기가 아니라 "완성되면 위험해지는 형상"을 부수던 망치다.
- 망치 자국은 파괴의 흔적이면서 보호의 흔적이다.

## 6. 교정 집행체

정사 등급: B급 후보

| 항목 | 설정 |
|---|---|
| 분류 | 집단/장치/반자동 집행자 |
| 소속 | 프론티아 시그마-7, 관찰구역 A-1, 재교정실 |
| 선물 | 혼란 속에서 즉각적인 안전선을 만든다 |
| 상처 | 과거 한 번의 예외가 큰 사고로 이어졌다는 기록을 반복 주입받았다 |
| 거짓 믿음 | 다치지 않게 하려면 먼저 움직이지 못하게 해야 한다 |
| 낮의 형태 | 사람처럼 걷지만 얼굴보다 규약판이 먼저 보이는 집행자 |
| 밤의 변형 | 명령 문장이 몸의 관절이 되어, 스스로도 멈출 수 없는 절차가 된다 |
| 사랑스러운 균열 | 어린 주민이 넘어지면 아주 짧게 손을 내밀었다가, 뒤늦게 규약 위반 경고를 띄운다 |
| 관련 인물 | 프레펙투스, 리버, 리네아, 셀로, 에이도스 |
| 대가 | 집행체가 오래 활동할수록 내부의 작은 망설임도 오류로 기록된다 |
| 반작용 | 리버의 절차 밖 기다림, 리네아의 작은 친절, 셀로의 곡선 세르 |
| 장면 기능 | 프론티아의 폭력이 개인의 악의만이 아니라 자동화된 보호 논리에서 나온다는 점을 보여준다 |

정예 교정 집행체:

- 정예형은 규약판이 더 선명하고, 망설임 기록이 더 깊이 삭제되어 있다.
- 프레펙투스가 두려워하는 것은 집행체의 실패가 아니라, 집행체가 망설이기 시작하는 순간이다.

## 7. 중앙 기록 회랑

정사 등급: B급 후보

| 항목 | 설정 |
|---|---|
| 분류 | 장소/건축물/제도 |
| 소속 | 프론티아 시그마-7 내부 |
| 낮의 형태 | 긴 은색 회랑. 벽마다 시민의 동선, 교정 기록, 실험 구역 번호가 얇은 빛으로 저장된다 |
| 밤의 변형 | 기록이 사람을 따라오는 것이 아니라, 사람이 자기 기록 안으로 걸어 들어간다 |
| 은유 | 기억이 아니라 허가된 기록으로만 존재를 증명하려는 질서 |
| 관련 인물 | 리버, 리네아, 에이도스, 프레펙투스 |
| 대가 | 이름보다 번호를 오래 읽으면, 이름을 불렸을 때도 돌아보는 속도가 늦어진다 |
| 파손 방식 | 기록되지 않은 친절이 반복될 때 벽면의 빛이 잠깐 끊긴다 |
| 회복 방식 | 기록을 삭제하는 것이 아니라, 빠진 이름을 주석처럼 덧붙일 때 회랑이 덜 차갑게 울린다 |
| 장면 기능 | 코어 로그, 72-B 문, 리네아의 이름 회복, 에이도스의 다음 목적지 단서를 한 장소에 묶는다 |

활용:

- 리버가 이 회랑에서 자기 번호를 볼 때, 분노보다 먼저 익숙함을 느끼면 상처가 깊어진다.
- 리네아가 처음 자기 이름을 쓰는 장소로 쓰면, 시그마-7 해방의 규모가 한 줄의 글씨로 압축된다.
