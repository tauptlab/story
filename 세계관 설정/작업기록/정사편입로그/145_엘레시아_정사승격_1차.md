# 엘레시아 정사 승격 1차

정사 등급: 작업 기록

역할: 루트 초기 자료의 `엘레시아` 씨앗을 눅세린/센시르 접경 귀환 인물로 승격하고, "서로 다른 쪽을 모두 품으려는 다정함"을 선택 회피의 위험과 남은 쪽을 지우지 않는 책임 사이에서 정리한다.

---

## 1. 승격 요약

엘레시아는 기존 자료에서 "서로 다른 쪽을 모두 품으려던 인물"로 남아 있었다. 이번 승격에서는 그녀를 단순한 중재자가 아니라, 양쪽을 잃지 않으려는 선의가 오래되면 모두를 기다리게 하는 유예가 될 수 있음을 드러내는 접경 귀환 인물로 정리했다.

핵심 이동은 아래와 같다.

| 구분 | 승격 전 | 승격 후 |
|---|---|---|
| 인물 기능 | 눅세린/센시르 접경 귀환 후보 | 두손접경마당의 귀환 인물 |
| 핵심 질문 | 서로 다른 쪽을 모두 품을 수 있는가 | 한쪽을 선택하고도 남은 쪽을 지우지 않을 수 있는가 |
| 주 무대 | 눅세린/센시르 후보 | 두손다리, 반반 잔 탁자, 한쪽발 돌마당, 남은손 난간 |
| 주요 물성 | 미정 | 두손 끈, 반반 잔, 한쪽발 돌, 두빛풀, 남은손 표식 |
| 결투 기능 | 미정 | 모두를 잃지 않으려는 포용과 한쪽을 딛는 책임의 재배치 |

## 2. 124 기준 반영

| 기준 | 엘레시아 반영 |
|---|---|
| A가 버린 가치관 | 선택해도 관계 전체가 사라지지 않을 수 있다는 믿음 |
| 왜 버릴 수밖에 없었는가 | 한쪽을 선택한 일이 남은 쪽에서 배신으로 불리며, 선택 자체가 관계 파괴처럼 각인되었다 |
| 대체 가치관 | 양쪽을 모두 붙잡고, 분명한 답을 조금 더 늦추는 습관 |
| 처음의 보호 기능 | 성급한 단절을 막고, 버려졌다고 느끼는 쪽에 손을 남겨 관계를 살렸다 |
| 시간이 지나며 생긴 왜곡 | 답을 늦추는 동안 양쪽 모두 끝나지 않는 대기 속에 묶이고, 아무도 앞으로 가지 못한다 |
| 귀환 계기 | 두손다리와 한쪽발 돌마당이 겹치는 결투에서 엘레시아가 처음으로 한 발을 딛고 남은 쪽에 인사를 남긴다 |
| 필연성 | 포용만으로는 관계가 움직이지 않고, 선택만으로는 남은 쪽이 지워지기 때문에 선택 뒤의 인사와 책임이 필요해진다 |
| 귀환 후 후폭풍 | 눅세린의 약속 문화는 유예표에 이유를 남기게 되고, 센시르의 회복 예법은 모두를 듣되 대신 답하지 않는 선을 더 분명히 묻게 된다 |

## 3. 녹티오라 설계

| 낮의 요소 | 왜 남았는가 | 밤의 형태 | 선택 시험 |
|---|---|---|---|
| 두손다리 | 어느 쪽도 놓지 않으려던 손목의 통증이 오래 남았다 | 양쪽 난간이 서로 반대편으로 당겨져 가운데 선 사람의 팔을 늘인다 | 붙잡음인가, 속박인가 |
| 반반 잔 | 두 감정을 동시에 삼키려다 목이 막힌 경험이 남았다 | 오래 들수록 무거워지고, 동시에 마시려 하면 마른 흙처럼 걸린다 | 포용인가, 목막힘인가 |
| 한쪽발 돌 | 선택을 시작하려던 발끝의 두려움이 강하게 남았다 | 선택을 말하지 않으면 작아지고, 남은 쪽에 인사하면 따뜻해진다 | 배신인가, 책임인가 |
| 두빛풀 | 선택받지 못한 길도 완전히 사라지지 않기를 바란 마음이 남았다 | 저주받은 길가에서는 눕고, 인정받은 길가에서는 두 색 잎을 편다 | 삭제인가, 남김인가 |
| 유예표 | 답을 미루며 관계를 살린 날과 망친 날이 모두 남았다 | 이유 없는 표부터 딱딱하게 붙어 떨어지지 않는다 | 기다림인가, 회피인가 |

## 5. 결투 설계 메모

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 모두를 잃지 않으려는 포용 vs 한쪽을 선택하고 책임지는 용기 |
| 두 가치의 옳음 | 포용은 성급한 단절을 막고, 선택은 관계가 실제로 움직일 바닥을 만든다 |
| 두 가치의 위험 | 포용은 끝없는 대기가 되고, 선택은 남은 쪽을 지우면 폭력이 된다 |
| 끝난 뒤 재배치 | 선택은 배신의 다른 이름이 아니라, 남은 쪽을 기억한 채 딛는 한 발로 다시 놓인다 |
| 도시 변화 | 두손접경마당의 유예표에는 미루는 이유를 적게 되고, 한쪽발 돌은 남은손 표식이 놓인 뒤에만 따뜻해진다 |
| 인물 변화 | 엘레시아는 양손을 모두 붙잡는 대신, 한 손을 놓은 뒤에도 그 손이 사라지지 않도록 짧은 인사를 남긴다 |
