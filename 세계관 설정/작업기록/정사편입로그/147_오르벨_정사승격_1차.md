# 오르벨 정사 승격 1차

정사 등급: 작업 기록

역할: 루트 초기 자료의 `오르벨` 씨앗을 로루멘 박수 보류석의 생활 인물로 승격하고, "감정을 전략처럼 쓰는 보호 본능"을 빠른 박수의 보호성과 조종의 위험 사이에서 정리한다.

---

## 1. 승격 요약

오르벨은 기존 자료에서 로루멘의 `전략적 감정을 쓰는 인물` 후보로 남아 있었다. 이번 승격에서는 그를 단순히 감정을 계산하는 사람이 아니라, 침묵이 한 사람에게 실패 판결처럼 내려앉는 일을 막기 위해 먼저 박수를 마련하는 연출가로 정리했다.

핵심 이동은 아래와 같다.

| 구분 | 승격 전 | 승격 후 |
|---|---|---|
| 인물 기능 | 로루멘 생활 인물 후보 | 박수 보류석의 박수/질문 생활 인물 |
| 핵심 질문 | 감정을 전략처럼 쓰는가 | 보호하려는 박수가 늦은 질문을 덮지 않게 할 수 있는가 |
| 주 무대 | 로루멘 후보 | 박수 보류석, 보류등 조정실, 박수 매듭 난간 |
| 주요 물성 | 보류 박수표 | 박수끈, 보류등, 접은 박수표, 손뼉이끼 |
| 결투 기능 | 미정 | 감정 연출의 보호성과 진짜 응답의 도착 시간 재배치 |

## 2. 124 기준 반영

| 기준 | 오르벨 반영 |
|---|---|
| A가 버린 가치관 | 반응이 늦게 와도 관계가 바로 무너지지 않을 수 있다는 믿음 |
| 왜 버릴 수밖에 없었는가 | 침묵과 조롱이 누군가에게 실패 판결처럼 남는 장면을 보았고, 늦은 반응은 이미 상처라고 배웠다 |
| 대체 가치관 | 먼저 분위기와 박수와 웃음을 마련해 상처받을 틈을 줄이는 습관 |
| 처음의 보호 기능 | 무대 위에서 혼자 버려지는 사람을 줄이고, 어색한 방에 잠시 난간을 세웠다 |
| 시간이 지나며 생긴 왜곡 | 모든 환대가 미리 짜인 장면이 되고, 사람들은 자기 마음보다 먼저 박수치게 된다 |
| 귀환 계기 | 박수 보류석의 밤 결투에서 오르벨이 박수끈을 당기는 대신 보류등을 켜고, 늦은 질문을 객석에 남긴다 |
| 필연성 | 박수만으로는 진짜 응답이 사라지고, 박수를 모두 금지하면 침묵에 버려지는 사람이 생기므로 박수와 질문의 순서가 다시 정해져야 한다 |
| 귀환 후 후폭풍 | 로루멘은 빠른 박수를 금지하지 않지만, 박수 보류석마다 보류등과 늦은 질문의 자리를 함께 둔다 |

## 3. 녹티오라 설계

| 낮의 요소 | 왜 남았는가 | 밤의 형태 | 선택 시험 |
|---|---|---|---|
| 박수끈 | 누군가를 침묵에서 건져 내기 위해 손뼉의 속도를 맞추던 습관이 남았다 | 난간과 손목을 묶고, 사람들은 자기 마음보다 먼저 손을 든다 | 보호인가, 조종인가 |
| 보류등 | 박수를 치기 전 잠깐 기다리고 싶었던 마음이 불빛으로 남았다 | 너무 오래 켜면 그늘이 고이고, 늦은 질문이 오면 낮게 맑아진다 | 기다림인가, 방치인가 |
| 접은 박수표 | 찢지 못한 인정과 미처 묻지 못한 질문이 날짜별 접힘으로 남았다 | 찢으면 환호가 되고, 접어 두면 가장자리에 작은 질문이 적힌다 | 인정인가, 확인인가 |
| 손뼉이끼 | 강요된 손뼉과 진짜 환대의 차이가 객석 아래의 생장으로 남았다 | 재촉 앞에서는 납작해지고, 질문이 도착하면 잎끝이 선다 | 환대인가, 압박인가 |

## 5. 결투 설계 메모

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 상처를 막기 위한 감정 연출 vs 늦게 도착하는 진짜 반응을 기다릴 권리 |
| 두 가치의 옳음 | 박수는 침묵에 버려지는 사람을 지키고, 늦은 질문은 조율되지 않은 응답을 관계 안에 남긴다 |
| 두 가치의 위험 | 박수는 조종이 되고, 늦은 질문은 방치가 될 수 있다 |
| 끝난 뒤 재배치 | 박수는 명령이 아니라 잠시 기대는 난간이 되고, 질문은 박수를 부끄럽게 만들지 않는 두 번째 응답이 된다 |
| 도시 변화 | 박수 보류석은 박수끈을 당기기 전 보류등을 켜고, 접은 박수표를 찢기보다 질문 가장자리를 남긴다 |
| 인물 변화 | 오르벨은 무대 위 사람을 지키려는 마음을 버리지 않되, 손뼉을 먼저 유도하던 손을 낮추고 질문을 기다린다 |
