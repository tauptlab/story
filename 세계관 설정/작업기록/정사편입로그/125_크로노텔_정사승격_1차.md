# 크로노텔 정사승격 1차

이 문서는 `106_루트초기자료_씨앗추출_1차.md`에서 중위 후보로 분류한 크로노텔을 히프노레아와 오라쿨룸 접경의 기록 극장 도시로 승격한 기록이다.

---

## 1. 승격 이유

크로노텔은 `124`의 상위 기준을 도시 승격에 적용하는 첫 사례다. 이곳의 후회는 단순한 슬픔이나 과거 미련이 아니라, 이미 벌어진 말과 행동을 어떻게 다시 볼 것인가의 문제다. 히프노레아가 장면을 보존하고 오라쿨룸이 의미를 다시 울린다면, 크로노텔은 그 둘 사이에서 장면을 무대에 올린다.

이 도시는 후회의 선의를 보존한다. 잘못한 말과 행동을 아무 일도 아닌 것처럼 묻으면, 같은 일이 다시 반복된다. 다만 후회가 벌이 되면 사람은 같은 장면을 끝없이 재연하며 현재의 책임으로 나가지 못한다. 그래서 크로노텔의 핵심 긴장은 다시 보는 일과 다시 벌하는 일의 차이다.

## 3. 124 기준 적용

| 기준 | 크로노텔의 처리 |
|---|---|
| 버린 가치관 | 잘못한 뒤에도 말하고 수습하려는 용기 |
| 보호하던 대체 가치관 | 잊지 않는 기록과 의미 해석 |
| 왜곡된 가치관 | 후회를 반복 벌로 만들고 현재 행동을 늦추는 재상연 |
| 다시 받아들일 가치관 | 들을 권리를 먼저 묻고, 지금 가능한 책임으로 나아가는 사과 |
| 녹티오라 흔적 | 빈 객석, 늦은 대사, 오지 않은 답장, 닫힌 마지막 줄 |
| 귀환 갈등 | 과거를 바꾸려는 반복과 현재를 감당하려는 선택의 충돌 |

| 기능 | 운용 |

## 5. 주의점

- 크로노텔은 과거로 돌아가는 시간 도시로 쓰지 않는다.
- 후회는 벌이 아니라 다음 행동을 묻는 기록 극장으로 다룬다.
- 사과는 말한 사람의 속도만으로 완성되지 않는다. 듣는 쪽의 자리와 온도를 먼저 확인해야 한다.
- 브리멜론은 기록에 붙은 감정의 어긋남, 실렌카르는 지나간 가능성의 애도, 크로노텔은 이미 벌어진 말과 행동의 재상연 윤리로 분리한다.
