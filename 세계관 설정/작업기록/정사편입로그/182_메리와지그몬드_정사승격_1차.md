# 메리와 지그몬드 정사 승격 1차

작성일: 2026-06-24

역할: 루트 초기 자료의 `메리와 지그몬드` 씨앗을 현 주연 구도와 직접 병합하지 않고, 호수지역의 `별편지 전승`으로 낮춰 정리한다. 이번 승격은 사랑, 보호, 왕권, 구출 실패, 오해를 하나의 독립 관계 신화로 묶어, 구하려는 사랑과 침묵하는 책임의 충돌을 Mnemosyn 귀환 갈등에 연결한다.

---

## 1. 승격 요약

메리와 지그몬드는 과거의 왕가 비극을 그대로 되살리는 인물이 아니다. 둘은 A가 한때 생존을 위해 버렸던 관계의 예법, 곧 사랑했던 이를 실패 뒤에도 명령이나 경멸이나 침묵으로 바꾸지 않는 힘을 다시 받아들이게 하는 관계 신화다.

| 항목 | 내용 |
|---|---|
| 정사명 | 메리와 지그몬드 |
| 별칭 | 별편지 전승, 양쪽구출문 전승 |
| 소속 | 특수공간-호수지역, 별편지 선착장, 양쪽구출문, 방패비침 여울, 왕관물금 나루 |
| 정사 등급 | B급 상위 독립 관계 신화 |
| 핵심 가치 | 사랑은 실패한 구출 뒤에도 상대의 현재 선택을 물을 수 있어야 한다 |
| 주요 물성 | 별편지, 양쪽열쇠, 늦은대답 잔, 대신맞는 방패, 왕관물금, 늦은별갈대 |
| 밤 변형 | 한쪽만 구하려는 문, 변명으로 얼어붙는 잔, 묻지 않고 던지면 엉뚱한 이를 보호하는 방패가 선택을 시험한다 |

| 버린 이유 | 구해지지 못한 기억과 늦게 도착한 말은 너무 아팠고, 누군가를 잃지 않으려면 먼저 지키거나 먼저 닫는 편이 더 안전해 보였기 때문이다 |
| 대체 가치관 | 소중한 것은 반드시 지켜야 하며, 더 큰 피해를 막기 위해서는 상대가 모르는 사이에라도 먼저 결정해야 한다는 습관 |
| 처음의 보호 | 이 습관은 A를 무력한 방관자로 남기지 않았고, 위험 앞에서 방패를 들거나 문을 닫게 했다 |
| 왜곡 | 시간이 지나 보호는 상대를 구해야 할 대상으로 바꾸고, 책임은 침묵과 위에서 내린 결정으로 굳어 사랑받는 사람의 현재 선택을 지웠다 |
| 다시 받아들여야 하는 가치 | 늦었더라도 직접 대답하고, 구하려는 마음이 상대의 오늘 말을 지우지 않게 하며, 사랑했던 이를 옛 약속의 모양으로 강요하지 않는 예법 |
| 녹티오라 흔적 | 별편지 선착장, 한쪽만 돌아가지 않는 양쪽구출문, 변명 앞에서 어는 늦은대답 잔, 묻지 않으면 엉뚱한 이를 덮는 대신맞는 방패 |
| 귀환 계기 | 아흐의 빈자리, 켈의 빌린 힘, 세리엘의 상처 입은 믿음, 에레시아의 따뜻한 보존, 리버의 떠나는 힘이 메리와 지그몬드 앞에서 구출과 선택의 경계를 묻는다 |

## 3. 밤 변형 설계

| 요소 | 왜 남았는가 | 밤의 변형 | 선택 시험 |
|---|---|---|---|
| 별편지 | 멀리서도 서로에게 닿으려 했던 말과, 정작 가장 필요할 때 도착하지 못한 대답이 함께 남았다 | 오늘의 말을 쓰면 잉크가 낮게 빛나고, 옛 약속만 반복하면 배가 출발하지 않는다 | 기억인가, 현재 대답인가 |
| 양쪽열쇠 | 구출은 한쪽 손만으로 완성되지 않는다는 뒤늦은 깨달음이 남았다 | 안팎 손잡이가 함께 돌아가야 문이 열리고, 한쪽에서 밀면 문살이 더 무거워진다 | 구출인가, 선택의 삭제인가 |
| 늦은대답 잔 | 늦게 온 말이 변명인지 책임인지 가르기 어려웠던 감각이 남았다 | "내가 늦었다"는 말을 먼저 담으면 따뜻해지고, "어쩔 수 없었다"부터 담으면 얼어붙는다 | 책임인가, 핑계인가 |
| 대신맞는 방패 | 사랑하는 이를 지키려다 그의 선택을 묻지 못한 순간이 강하게 남았다 | 묻지 않고 던지면 엉뚱한 사람을 덮고, 이름과 현재 뜻을 물으면 손바닥만 한 틈을 남긴다 | 보호인가, 대신 살기인가 |
| 왕관물금 | 위에 서서 지키려는 마음이 사랑하는 이의 얼굴을 보지 못하게 만든 기억이 남았다 | 물금 위에 서서 대신 결정하면 물속 얼굴이 사라지고, 같은 높이에 앉으면 반영이 돌아온다 | 책임인가, 높은 자리의 고립인가 |
| 늦은별갈대 | 낮에는 사소한 편지의 마지막 문장이었지만, 밤에는 가장 오래 남은 미완의 대답이 되었다 | 두 사람의 현재 이름을 모두 부르면 양쪽 기슭으로 눕고, 한쪽 이름만 부르면 마른다 | 추억인가, 현재의 호명인가 |

## 5. 결투 기능

메리와 지그몬드 결투는 사랑의 크기를 재는 장면이 아니다. 메리의 보호는 선하다. 사랑하는 이를 잃지 않으려는 힘은 무력한 방관을 깨고 실제 행동을 만든다. 그러나 보호가 명령이 되면 상대는 말할 사람이 아니라 구해야 할 대상이 된다.

지그몬드의 책임도 선하다. 모르는 채 움직인 선의가 더 큰 피해를 만들 수 있다는 두려움은 가볍지 않다. 그러나 책임이 침묵과 대신 결정으로 굳으면, 사랑하는 이를 지키려는 손이 바로 그 사람의 선택을 지워 버린다.

| 충돌 | 설명 |
|---|---|
| 구하려는 사랑 | 소중한 것을 잃지 않으려면 직접 움직이고 방패를 들어야 한다 |
| 침묵하는 책임 | 더 큰 피해를 막으려면 감정만으로 움직이지 않고 상황 전체를 보아야 한다 |

결투 뒤에는 별편지 선착장이 사라지지 않는다. 다만 편지는 옛 약속을 반복하지 않고 오늘의 대답을 요구한다. 양쪽구출문은 한 사람이 미는 문이 아니라 양쪽에서 함께 돌리는 문으로 남고, 대신맞는 방패는 빈틈을 남겨 보호받는 사람이 손을 내밀 수 있게 한다. 이것이 Mnemosyn 귀환의 직접 치유가 아니라, 각 인물에게 다르게 남는 후폭풍이다.

## 6. 다음 후보

메리와 지그몬드는 `182` 기준으로 독립 관계 신화 승격을 완료했고, 루지에로는 `183` 기준으로 독립 전승 인물 승격을 완료했다. 다음 정사 승격 후보는 `미니`가 적절하다. 미니는 루지에로가 흔들릴 때 곁에 남던 옛 친구 씨앗으로, 우정이 돌봄인지 붙잡기인지 정리할 수 있다.
