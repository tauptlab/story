# 장면시퀀스 상위 누락카드 상세초안

이 문서는 `25_장면시퀀스_통합요소_추출과_누락카드.md`에서 상위 우선도로 분류한 요소를 실제 정사 카드로 승격하기 위한 상세 초안이다. 현재 단계는 실제 연재분 집필이 아니라 세계 성경 편찬 단계이므로, 각 항목은 향후 `02`, `03`, `04`, `07`, `11`, `12`, `15`, 도시별설정, 인물 개별 문서로 분산 반영한다.

장면시퀀스 상위 카드의 핵심 원칙은 "위험한 힘도 처음에는 누군가를 살리려는 마음에서 출발했다"는 것이다. 셀로의 심판, 카시안의 조율, 드라쿨의 인정 갈증, 이놉스의 무력감은 모두 파괴적일 수 있지만, 독자는 그 안에 있는 선물과 상처를 함께 보아야 한다.

---

## 0. 상위 카드 요약

| 카드 | 분류 | 핵심 가치 | 왜곡 위험 | 우선 반영 |
|---|---|---|---|---|
| 대공명실 | 장소/의식 공간 | 모두의 울림을 한자리에 듣는 책임 | 복잡한 고통을 하나의 판결로 줄임 | `02`, `07`, 리테노아 문서 |
| 고요의 이끼 | 유기물/치유 생태 | 말 없는 고통도 살아 있다는 믿음 | 치유를 자원처럼 분배하려 함 | `11`, `04`, 리테노아 문서 |
| 자기 왕관의 영역 | 능력/특수공간 | 충돌 없는 평화를 만들고 싶은 욕망 | 감정과 질문을 삭제한 질서 | `03`, `07`, 특수공간 문서 |
| 잿빛 균열 | 현상/공간 균열 | 억눌린 감정의 귀환 | 분노를 오직 파괴로만 표출 | `07`, `16`, `01` |
| 피의 계약 | 능력/맹약 | 기억되고 싶은 절박함 | 타인의 기억을 빼앗아 존재 증명 | `07`, 드라쿨 문서 |
| 내면의 계약 | 능력/성장 맹약 | 타인의 시선 없이 자신을 인정함 | 면죄부처럼 오해될 위험 | `07`, 드라쿨 문서 |
| 무력장 | 능력/전장 | 멈춤으로 더 큰 파국을 피하려는 본능 | 아무것도 하지 못하게 만드는 체념 | `07`, 이놉스 문서 |
| 방향 노드 | 능력/작중어 | 자유에 방향과 책임을 부여함 | 타인의 선택까지 돌리려는 유혹 | `15`, `07`, 리버 문서 |
| 중앙 원형단 | 장소/의식 구조 | 상실을 함께 둘러싸고 견디는 의식 | 추모가 영원한 전투장으로 굳음 | 상실의 계곡 문서, `07` |

## 1. 대공명실

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/건축/의식 공간 |
| 소속 | 센시르 리테노아, 반향의 샘 하층 또는 인접 심부 |
| 낮의 형태 | 여러 개의 물길, 현, 유리 같은 공명판이 둥근 방 안에 겹쳐진다. 누군가 말하면 말보다 먼저 숨의 떨림이 벽에 닿는다 |
| 밤의 변형 | 방 안의 모든 고통이 빛줄기처럼 떠오르지만, 서로 섞이지 않은 고통은 검은 매듭으로 굳어 바닥을 끌어당긴다 |
| 선물 | 말이 약한 사람, 늦게 우는 사람, 자기 고통을 설명하지 못하는 사람의 울림도 듣게 한다 |
| 상처 | 과거 리테노아가 너무 늦게 들었기 때문에 이미 잃은 사람들이 있었다 |
| 거짓 믿음 | 모든 울림을 한 방에 모으면 가장 옳은 답도 한 번에 들릴 것이다 |
| 관련 인물 | 카시안, 셀로, 페이단, 고요의 정원지기, 넥서스 후원자들 |
| 반작용 | 카시안의 공명 현, 셀로의 심판의 영역, 고요의 이끼, 잿빛 균열 |
| 장면 기능 | 셀로의 실패를 "악의"가 아니라 "복잡함을 견디지 못한 선의"로 보이게 한다 |

의식 구조:

| 단계 | 내용 | 장면 기능 |
|---|---|---|
| 입장 | 각 공동체 대표가 자기 소리를 현 하나에 맡긴다 | 갈등을 추상 논쟁이 아니라 몸의 떨림으로 만든다 |
| 분리 청취 | 카시안이 각 울림을 섞지 않고 따로 듣는다 | 느림과 책임의 미덕을 보여준다 |
| 공명 대조 | 서로의 울림이 닿을 때 생기는 불협과 화음을 확인한다 | 한쪽의 고통이 다른 쪽의 악함을 자동 증명하지 않음을 보여준다 |
| 조율 보류 | 답이 아직 없으면 의식을 끝내지 않고 보류한다 | 셀로가 견디지 못하는 지점 |
| 강제 판결 | 셀로가 심판의 영역으로 매듭을 자르려 한다 | 대공명실이 붕괴하고 자기 왕관의 영역으로 이어진다 |

독자가 붙잡을 디테일:

- 방 한가운데에는 의자가 하나도 없다. 누구도 위에서 판결하지 못하게 하기 위해서다.
- 가장 오래된 현은 끊어질 듯 떨리지만, 완전히 끊어지지는 않는다.
- 카시안은 의식이 끝난 뒤에도 한동안 손끝을 벽에 대고 남은 떨림을 듣는다.

사용 원칙:

- 대공명실은 재판장이 아니라 청취의 방이다.
- 셀로가 이곳을 망가뜨리는 장면은 잔혹한 폭주가 아니라, "더는 아무도 울지 않게 하겠다"는 조급한 마음에서 출발해야 한다.
- 카시안이 느려 보일수록, 독자는 그 느림이 무능이 아니라 여러 사람을 동시에 놓치지 않으려는 태도임을 느껴야 한다.

모순 주의:

- `심판의 영역`과 겹치지 않는다. 대공명실은 듣는 공간이고, 심판의 영역은 자르는 공간이다.
- `반향의 샘`과 겹치지 않는다. 반향의 샘은 기억과 후회를 비추고, 대공명실은 현재의 복수 울림을 조율한다.

## 2. 고요의 이끼

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 유기물/치유 생태/기억 흡수체 |
| 소속 | 센시르 리테노아, 대공명실 주변 정원과 습한 벽면 |
| 낮의 형태 | 물빛과 잿빛이 섞인 낮은 이끼. 손으로 만지면 차갑지 않고, 오래 참은 숨처럼 미세하게 부푼다 |
| 밤의 변형 | 주변의 말하지 못한 감정이 이끼의 결 사이에 작은 방울로 맺힌다. 억지로 뜯으면 방울이 검게 타며 침묵의 껍질을 만든다 |
| 선물 | 과하게 말하지 못하는 고통을 천천히 받아들이고, 고통의 주인이 다시 견딜 수 있는 만큼만 돌려준다 |
| 상처 | 과거 전쟁과 실험의 후유증을 너무 많이 머금어, 스스로도 쉬지 못한다 |
| 거짓 믿음 | 아픈 이를 살릴 수 있다면 이끼의 침묵은 잠시 희생해도 된다 |
| 관련 인물 | 고요의 정원지기, 넥서스 후원자들, 카시안, 셀로 |
| 반작용 | 균형 회복 의식, 침묵의 껍질, 역공진 파동 |
| 장면 기능 | 치유와 자원 분배의 갈등을 선악 구도로 만들지 않는다 |

갈등 구조:

| 집단 | 지키려는 것 | 독자가 공감할 이유 | 위험 |
|---|---|---|---|
| 고요의 정원지기 | 이끼가 가진 느린 생명과 축적된 고통 | 이끼를 물건이 아니라 돌봄의 주체로 본다 | 사람의 급한 생명을 외면하는 것처럼 보일 수 있다 |
| 넥서스 후원자들 | 당장 죽어가는 아이들과 가족 | 눈앞의 생명을 살리고 싶은 요구는 정당하다 | 이끼를 뜯어 쓰는 순간 다른 생명을 침묵시킨다 |
| 카시안 | 두 절박함을 동시에 듣는 절차 | 늦더라도 누구도 도구화하지 않으려 한다 | 너무 늦어 보인다 |
| 셀로 | 더는 고통을 방치하지 않는 판결 | 독자가 처음에는 속 시원함을 느낄 수 있다 | 복잡한 생명을 하나의 답으로 자른다 |

독자가 붙잡을 디테일:

- 정원지기들은 이끼를 "그것"이라 부르지 않고, 어린아이에게 하듯 낮은 소리로 인사한다.
- 넥서스 후원자 중 한 사람은 이미 아이를 잃었고, 그래서 더 거칠게 요구한다.
- 이끼가 건강할 때는 빛나지 않는다. 조용히 어두운 편이 오히려 살아 있다는 뜻이다.

사용 원칙:

- 고요의 이끼는 만능 치유약이 아니다. 고통을 없애는 것이 아니라, 견딜 수 있는 형태로 되돌려주는 생명체다.
- 이끼를 뜯어내는 장면에서는 악당의 약탈보다 "시간이 없는 사람들의 절박함"이 먼저 느껴져야 한다.
- 셀로의 실패는 이끼를 지키는 쪽과 쓰려는 쪽 중 하나를 고르는 데서 오지 않는다. 둘 다 온전히 듣지 못하고 판결하려 한 데서 온다.

## 3. 자기 왕관의 영역

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/특수공간/왜곡 왕국 |
| 소속 | 셀로, 심판의 영역 붕괴 이후 형성 |
| 낮의 형태 | 현실에서는 얇은 은빛 관처럼 보이는 경계선. 그 안에 들어간 사람은 다툼이 사라진 듯한 평온을 느낀다 |
| 밤의 변형 | 은빛 수정 성채를 중심으로 모든 길이 왕관의 살처럼 배열된다. 주민은 질문 없이 이해하고, 상처 없이 공감하는 순응 존재군으로 변한다 |
| 선물 | 더는 서로 해치지 않는 세계를 만들고 싶어 한다 |
| 상처 | 셀로가 복잡한 갈등 속에서 사랑받지 못하거나 실패할까 두려워한 기억 |
| 거짓 믿음 | 모두가 나를 중심으로 조용해지면, 그것이 모두를 위한 평화다 |
| 관련 인물 | 셀로, 에이도스, 카시안, 페이단, 순응 존재군 |
| 반작용 | 잿빛 균열, 페이단 씨앗, 진홍의 진단서 불꽃, 에이도스의 질문 |
| 장면 기능 | 자기애를 단순 허영이 아니라 "상처 없는 세계를 만들고 싶은 통제 욕망"으로 깊게 만든다 |

공간 규칙:

| 규칙 | 작동 방식 | 위험 |
|---|---|---|
| 질문 약화 | 누군가 질문하려 하면 문장이 부드럽게 칭찬으로 바뀐다 | 비판이 사라지고 성장도 멈춘다 |
| 색 흡수 | 강한 감정의 색은 은빛 벽에 흡수된다 | 분노, 슬픔, 질투, 수치가 모두 잿빛 균열로 밀려난다 |
| 중심 회귀 | 길을 잃으면 결국 성채 앞으로 돌아온다 | 선택이 있는 듯하지만 모든 방향이 셀로에게 귀속된다 |
| 상처 미화 | 아픈 기억은 아름다운 문양으로 정리된다 | 치유가 아니라 전시가 된다 |

독자가 붙잡을 디테일:

- 순응 존재군은 셀로에게 친절하지만, 누구도 자기 이야기를 먼저 시작하지 않는다.
- 성채의 창문은 밖을 비추는 듯하지만 사실 안쪽 얼굴만 반사한다.
- 영역 안에서는 싸움이 없어서 처음엔 안심되지만, 웃음소리도 점점 같은 높이가 된다.

사용 원칙:

- 자기 왕관의 영역은 명백히 끔찍한 감옥처럼 시작하면 약하다. 처음에는 정말 편안하고 안전해 보여야 한다.
- 셀로를 우스운 독재자로 만들지 않는다. 그는 진심으로 상처를 줄이고 싶어 한다.
- 이 공간의 공포는 폭력이 아니라 감정의 납작함이다.

모순 주의:

- `심판의 영역`은 판결하는 힘이고, `자기 왕관의 영역`은 판결 뒤 남은 세계를 자기 중심으로 재배열하는 힘이다.
- 순응 존재군은 암디르의 질문자들과 다르다. 질문자는 아픈 진실을 묻고, 순응 존재군은 질문 자체를 부드럽게 삼킨다.

## 4. 잿빛 균열

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 현상/공간 균열/억압 감정의 통로 |
| 소속 | 자기 왕관의 영역 바깥층, 근원층에 가까운 틈 |
| 낮의 형태 | 은빛 벽 아래 생기는 가는 금. 가까이 보면 먼지처럼 보이지만, 귀를 대면 말하지 못한 소리가 난다 |
| 밤의 변형 | 금이 벌어지며 잿빛과 진홍빛이 섞인 균열지대가 된다. 균열 안쪽에는 삭제된 감정의 잔해가 떠다닌다 |
| 선물 | 억눌린 감정이 사라진 것이 아니라 아직 돌아올 수 있음을 알려준다 |
| 상처 | 오랫동안 안전과 평온이라는 이름으로 밀려난 감정들이 통로를 잃었다 |
| 거짓 믿음 | 감정이 돌아오려면 반드시 부수고 태워야 한다 |
| 관련 인물 | 페이단, 셀로, 에이도스, 순응 존재군 |
| 반작용 | 페이단 씨앗, 진홍의 진단서 불꽃, 자기 왕관의 영역 |
| 장면 기능 | 분노를 악으로만 보지 않고, 생명 부재를 알리는 경보로 만든다 |

균열 안의 잔해:

| 잔해 | 형태 | 의미 |
|---|---|---|
| 말끝 조각 | 끝나지 않은 문장의 작은 파편 | 질문이 칭찬으로 바뀌기 전 사라진 말 |
| 색 없는 눈물 | 흐르지 못하고 공중에 굳은 방울 | 슬픔을 아름답게 정리당한 흔적 |
| 뜨거운 재 | 만지면 늦은 분노가 손바닥에 남는다 | 페이단 강림의 전조 |
| 작은 발자국 | 안쪽으로만 이어지는 흔적 | 순응 존재군이 한때 개별 존재였다는 단서 |

사용 원칙:

- 잿빛 균열은 페이단을 위한 악마 소환문이 아니다. 억압된 감정들이 "여기 살아 있다"고 두드리는 문이다.
- 균열이 커질수록 무섭지만, 독자는 동시에 안도해야 한다. 사라진 줄 알았던 감정이 아직 남아 있기 때문이다.
- 셀로가 균열을 막으려 할 때, 그는 거짓을 지키는 것이 아니라 자신이 만든 평화가 가짜라는 사실을 견디지 못하는 것이다.

명칭 정리:

- 눅세린의 `잿빛 구역`과 혼동하지 않기 위해, EP13 특수 현상은 `잿빛 균열` 또는 `잿빛 균열지대`로 통일한다.

## 5. 피의 계약

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/맹약/전투 진화 |
| 소속 | 블라드 드라쿨 |
| 낮의 형태 | 손등과 목덜미에 붉은 서명처럼 떠오르는 맹약 흔적. 가까이 보면 피가 아니라 이름의 획처럼 흔들린다 |
| 밤의 변형 | 상대의 기억, 시선, 인정 욕구가 피빛 실로 변해 드라쿨의 검과 몸을 강화한다 |
| 선물 | 누구도 자신을 보지 않을 때도 살아남게 하는 절박한 생존력 |
| 상처 | 사랑받거나 기억되려면 누군가에게 강하게 각인되어야 한다고 믿게 된 시간 |
| 거짓 믿음 | 나를 기억하게 만들 수 있다면, 그것은 사랑받은 것과 같다 |
| 관련 인물 | 드라쿨, 아미키티아, 셀레네, 마이더스, 욕망의 사자 |
| 반작용 | 이름 새긴 방패, 셀레네의 피팩 기억, 내면의 계약 |
| 장면 기능 | 드라쿨을 단순 흡혈귀형 악역이 아니라 "잊히는 공포와 싸우다 타인의 기억을 빼앗게 된 인물"로 만든다 |

전투 규칙:

| 작동 | 효과 | 대가 |
|---|---|---|
| 이름 물기 | 상대가 지키는 이름을 읽고 전투 패턴을 모사한다 | 타인을 하나의 기억 자원으로 보게 된다 |
| 피빛 반향 | 자신을 향한 공포와 증오까지 힘으로 바꾼다 | 미움도 인정처럼 착각한다 |
| 과거 검술 모사 | 상대가 배운 사람들의 움직임을 빼앗아 쓴다 | 자기만의 검술이 더 흐려진다 |
| 계약 과식 | 너무 많은 기억을 삼키면 몸의 윤곽이 여러 사람의 그림자로 갈라진다 | 존재감은 커지지만 자아는 흐려진다 |

독자가 붙잡을 디테일:

- 드라쿨은 상대의 이름을 빼앗기 전 반드시 한 번 정확히 불러준다. 그것이 그에게는 폭력이자 애원이다.
- 셀레네와 관련된 기억만은 피의 계약으로 삼키지 못한다. 처음으로 조건 없이 보인 시선이기 때문이다.
- 아미키티아가 방패의 이름들을 하나씩 부를 때, 드라쿨은 처음으로 자신이 훔친 것이 얼굴들이었다는 사실을 본다.

사용 원칙:

- 피의 계약은 멋진 흡수 능력으로만 보이면 안 된다. 쓸수록 드라쿨이 더 외로워져야 한다.
- 아미키티아 사망 장면에서 피의 계약은 승리처럼 보이지만, 곧 드라쿨에게 가장 큰 공허를 남겨야 한다.

## 6. 내면의 계약

| 밤의 변형 | 타인의 기억을 끌어오지 않고, 자신이 외면한 기억들이 작은 촛불처럼 켜진다 |

각성 조건:

| 조건 | 내용 | 장면 기능 |
|---|---|---|
| 무관중 | 아무도 보지 않는 장소에서 시작된다 | 인정 욕구와 결별 |
| 이름 되돌림 | 빼앗은 이름을 하나씩 자기 밖으로 돌려보낸다 | 피해를 기억으로 복구 |
| 첫 문장 | "나를 봐라"가 아니라 "내가 했다"로 시작한다 | 책임의 방향 전환 |
| 검의 변화 | 검이 더 강해지기보다, 먼저 겨누지 않는 무게를 얻는다 | 힘보다 사용 조건이 달라짐 |

독자가 붙잡을 디테일:

- 내면의 계약이 시작되면 박수나 빛나는 승리 연출이 없다. 오히려 방이 조용해진다.
- 드라쿨은 처음으로 자기 이름을 낮은 목소리로 부른다.
- 의지의 검은 피를 마시지 않는다. 대신 칼등에 되돌려준 이름들이 작은 흠처럼 남는다.

사용 원칙:

- 내면의 계약은 면죄가 아니다. 독자가 드라쿨을 이해하게 할 수는 있지만, 아미키티아의 죽음을 지우면 안 된다.
- 성장 후 드라쿨은 즉시 동료가 되기보다, 갚을 수 없는 것을 들고 함께 걸을 수 있는지 시험받아야 한다.
- 이 능력의 아름다움은 화려한 변화가 아니라 "남에게 보이지 않아도 책임지는 태도"에 있다.

## 7. 무력장

| 밤의 변형 | 전장 전체가 회색 수면처럼 변하고, 움직이려는 의지가 표면 아래로 가라앉는다 |
| 선물 | 충동적인 파괴를 멈추고, 아무것도 하지 않는 시간의 필요성을 알려준다 |

| 장면 기능 | 무력감을 추상 감정이 아니라 실제 전투 규칙으로 만든다 |

전장 규칙:

| 규칙 | 효과 | 대응 |
|---|---|---|
| 의지 지연 | 마음먹은 행동이 몸에 도착하기까지 늦어진다 | 리버의 방향 노드가 지연된 의지에 길을 만든다 |
| 이유 마모 | "왜 해야 하는가"가 흐려진다 | 포르티스의 불꽃이 이유를 다시 점화한다 |
| 소리 둔화 | 외침은 들리지만 절박함이 전달되지 않는다 | 포보스의 경계선이 침묵의 균열을 만든다 |
| 승리 무효감 | 작은 성공도 곧 의미 없는 것처럼 느껴진다 | 중앙 원형단의 기억들이 성공을 붙들어준다 |

독자가 붙잡을 디테일:

- 무력장 안에서는 무기가 무거워지는 것이 아니라, 들어야 할 이유가 가벼워진다.
- 이놉스는 비웃지 않는다. 그는 정말로 모두가 곧 포기할 것이라고 믿는다.
- 무력장이 깨질 때 가장 먼저 돌아오는 것은 함성이 아니라 발소리다.

사용 원칙:

- 이놉스는 게으름의 악역이 아니다. 그는 "해도 안 된다"는 오래된 상처가 얼굴을 얻은 인물이다.
- 무력장의 대항은 의지 한 방이 아니라, 의지, 방향, 공포의 인정, 기억의 공동 지지가 함께 작동해야 한다.

## 8. 방향 노드

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/작중어/전술 결절 |
| 소속 | 리버 |
| 원문 초안어 | Redirection Node |
| 정사명 | 방향 노드 |
| 장면 호칭 후보 | 결돌림, 방향 돌림, 길매듭 |
| 낮의 형태 | 리버가 손끝이나 발끝으로 허공에 짧은 매듭을 만든다. 매듭은 곧 사라지지만, 주변의 흐름이 한 박자 방향을 얻는다 |
| 밤의 변형 | 무너진 길, 빗나간 파동, 주저앉은 의지가 하나의 통과점으로 모인다 |
| 선물 | 자유가 도망이 아니라 선택이 되게 한다 |
| 상처 | 어디에도 묶이지 않으려다, 아무곳에도 닿지 못했던 시간 |
| 거짓 믿음 | 내가 방향을 만들 수 있다면, 타인의 길도 내가 돌려도 된다 |
| 관련 인물 | 리버, 포르티스, 포보스, 이놉스, 셀로 |
| 반작용 | 멈춤의 훈련, 중심돌, 무력장, 공간 간섭 경계선 |
| 장면 기능 | 리버의 자유가 즉흥과 회피를 넘어 책임 있는 이동으로 성장했음을 보여준다 |

능력 규칙:

| 규칙 | 내용 | 제한 |
|---|---|---|
| 흐름만 돌림 | 이미 있는 움직임의 방향을 바꾼다 | 없는 의지를 새로 만들 수 없다 |
| 짧은 지속 | 노드는 오래 남지 않고 한 번의 선택만 돕는다 | 통제 장치가 되지 않는다 |
| 자기 통과 | 리버 자신도 노드를 통과해야 안정된다 | 안전한 외부 조종이 불가능하다 |
| 대가의 잔향 | 돌린 방향의 결과 일부가 리버에게 돌아온다 | 무책임한 회피를 막는다 |

독자가 붙잡을 디테일:

- 리버가 미숙할 때는 노드가 화살표처럼 보이지만, 성장 후에는 매듭처럼 보인다.
- 포르티스와의 훈련 뒤 리버는 먼저 움직이지 않고, 한 번 멈춘 뒤 방향을 만든다.
- 이놉스 전투에서 방향 노드는 공격을 맞히는 기술보다 "다시 걷게 하는 통과점"으로 빛난다.

작중어 원칙:

- 설정 문서에서는 `방향 노드`를 정사명으로 사용한다.
- 원고 대사에서는 인물이 성격에 따라 `길매듭`, `결돌림`, `방향 돌림`처럼 부를 수 있다.
- 영어식 `Redirection Node`는 원고 본문에 남기지 않는다.

## 9. 중앙 원형단

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/의식 구조/전투 무대 |
| 소속 | 암디르 상실의 계곡 심부 |
| 낮의 형태 | 오래된 기억의 돌들이 원형으로 박혀 있고, 가운데에는 꺼지지 않는 낮은 불씨가 있다. 바닥에는 검으로 생긴 홈과 발자국이 겹겹이 남아 있다 |
| 밤의 변형 | 각 돌이 잃어버린 사람의 목소리를 아주 짧게 되돌려준다. 불씨는 싸움이 격해질수록 커지는 것이 아니라, 누군가 포기하지 않을 때 더 또렷해진다 |
| 선물 | 상실을 혼자 견디지 않고, 둘러앉아 기억하게 한다 |
| 상처 | 너무 많은 추모가 이곳에 쌓여 계곡 자체가 싸움을 끝내지 못한다 |
| 거짓 믿음 | 잃은 이를 기억하려면 계속 같은 전투를 반복해야 한다 |
| 관련 인물 | 포르티스, 포보스, 리버, 이놉스, 셀레네 |
| 반작용 | 기억의 돌, 불꽃, 무력장, 방향 노드, 불꽃의 코어 검격 |
| 장면 기능 | 포르티스/리버/포보스가 각자의 힘을 한곳에 모아 이놉스의 무력감을 넘는 무대 |

공간 요소:

| 요소 | 형태 | 장면 기능 |
|---|---|---|
| 기억의 돌 | 이름 없는 돌, 일부는 손때가 묻어 반질하다 | 상실이 개인 사연을 넘어 공동 기억임을 보여준다 |
| 중앙 불씨 | 작지만 꺼지지 않는 불 | 포르티스의 의지가 폭발보다 지속에 가깝다는 점을 보여준다 |
| 검 아래의 홈 | 오래전부터 반복된 칼자국 | 싸움이 계속되었음을 시각화한다 |
| 외곽 그림자선 | 원형단 밖을 감싸는 어두운 경계 | 포보스의 공간 간섭 경계선과 연결된다 |

전투 사용:

| 단계 | 내용 | 장면 기능 |
|---|---|---|
| 침묵 | 무력장이 원형단의 소리를 눌러 모두의 이유를 흐린다 | 이놉스의 힘을 체감시킨다 |
| 발소리 | 리버가 방향 노드로 첫 통과점을 만든다 | 자유가 다시 행동으로 이어진다 |
| 경계 | 포보스가 공포를 외부 투사가 아니라 경계선으로 쓴다 | 수호자 전환을 보여준다 |
| 점화 | 포르티스가 불꽃의 코어 검격으로 무력장의 핵을 태운다 | 의지가 추모와 연결된다 |
| 잔향 | 원형단의 돌들이 짧은 목소리를 되돌려준다 | 승리가 아니라 계속 걸을 이유를 남긴다 |

사용 원칙:

- 중앙 원형단은 보스전 경기장이 아니라, 상실을 둘러싼 공동 의식 공간이다.
- 이놉스가 이곳을 장악하는 이유는 힘이 세서만이 아니라, 이곳에 쌓인 "해도 되돌릴 수 없다"는 감정이 무력장과 잘 맞기 때문이다.
- 포르티스의 승리는 과거를 되살리는 것이 아니라, 과거가 현재의 발을 붙잡지 않게 하는 것이다.

## 11. 장면 설계 원칙

1. 대공명실과 고요의 이끼는 선악 재판이 아니라, 서로 다른 보호 방식의 충돌로 쓴다.
2. 자기 왕관의 영역은 처음엔 독자도 잠깐 안심할 만큼 매혹적이어야 한다.
3. 잿빛 균열과 페이단은 파괴의 쾌감보다 억압 감정의 귀환으로 다룬다.
4. 피의 계약은 드라쿨의 카리스마를 만들되, 쓸수록 그가 더 외로워진다는 대가를 보여준다.
5. 내면의 계약은 면죄가 아니라 책임의 시작이다.
6. 무력장은 게으름이나 비겁함이 아니라 오래된 체념의 물리 법칙이다.
7. 방향 노드는 리버가 타인을 조종하는 힘이 아니라, 자기까지 통과해야 하는 책임 있는 자유다.
8. 중앙 원형단은 이기는 장소가 아니라 상실을 함께 견디는 장소다.
