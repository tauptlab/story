# 195 고유 특수능력 운용 필드 보강 1차

작성일: 2026-06-24

역할: 고유 특수능력 보유자 중 초중반 세계 인상을 강하게 담당하는 인물 5명의 개별 문서에 `작가용 의미`, `소품/물성`, `녹티오라 변형`, `결투 기능`을 명시한 보강 기록이다.

## 보강 대상

| 인물 | 능력명 | 보강 초점 |
|---|---|---|
| 나르키소스 | 카토프론 | 아름다움/시선/자기상 왜곡의 전투화 |
| 마이더스 | 아우룸 임페라 | 가격화 질서와 가치 부채의 물성화 |
| 도그마 | 클라우수라 | 닫힌 문장과 예외 봉인의 전장 기능 |
| 므네몬 | 팔림프세스트 | 기록 보존/봉인/열람권의 결투 기능 |
| 루멘 | 알레테이아 | 진실을 밝히는 힘과 기다리는 윤리의 분리 |

## 적용 원칙

- 기존 능력명은 유지한다.
- 이미 문서에 있던 장면 기능은 보존하되, 전투 기능과 작가용 의미를 분리해 실제 결투 설계에서 바로 볼 수 있게 한다.
- 녹티오라 변형은 공포 연출이 아니라 A에게 강하게 남은 흔적이 밤에 어떻게 시험이 되는지 중심으로 적는다.
