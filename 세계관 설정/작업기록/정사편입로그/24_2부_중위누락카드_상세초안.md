# 2부 중위 누락 카드 상세 초안

이 문서는 `22_2부_통합요소_추출과_누락카드.md`의 중위 후보를 실제 장면에 투입 가능한 설정 카드로 확장한다. `23_2부_누락카드_상세초안.md`가 2부의 핵심 무대와 능력을 다룬다면, 이 문서는 그 무대가 "살아 있는 세계"처럼 보이게 하는 기반 시설, 정치 집단, 보조 장치, 밤 생태를 담당한다.

정사 등급: B급에서 A급 승격 후보

| 항목 | 설정 |
|---|---|
| 분류 | 무기물/단서/작중어 후보 |
| 소속 | 잊혀진 지식의 평원, 고대 도서관 터, 히프노레아 심층부 연결 |
| 낮의 물성 | 빛이 아니라 울림이 갇힌 작은 결정. 귀에 대면 소리가 들리는 것이 아니라, 발밑의 길이 아주 미세하게 떨린다 |
| 밤의 변형 | 결정이 가진 울림과 같은 방향의 기록문, 문, 뿌리, 별자리 틀이 한순간 함께 밝아진다 |
| 은유 | 잊힌 지식은 사라진 것이 아니라, 아직 맞는 귀와 발걸음을 기다리는 오래된 울림이라는 감각 |
| 관련 인물 | 로고스, 에이도스, 셀로, 므네몬 |

| 파손 방식 | 억지로 분석하면 소리가 여러 갈래로 찢어져 거짓 길을 만든다 |
| 회복 방식 | 모르는 것을 모른다고 인정하고 조용히 들을 때 울림이 한 줄로 모인다 |
| 장면 기능 | 로고스가 히프노레아 심층으로 가는 길잡이를 남기는 물질 단서 |

- 정사명은 `고대 공명결`로 둔다.
- 장면 중 짧게 부를 때는 `오래된 울림결` 가능.
- 현대식 수치 울림 표현은 원고에서 피한다.

| 낮의 형태 | 땅 위와 아래를 번갈아 지나는 반투명 관. 오래 멈춰 있어 안쪽에는 먼지와 빛 찌꺼기가 층처럼 쌓여 있다 |
| 밤의 변형 | 과거에 흐르던 세르의 길이 잠시 살아나, 잘못 연결된 관에서는 어긋난 결론이 누수된다 |
| 은유 | 지식과 힘이 흐르던 길이 끊기면, 남는 것은 폐허가 아니라 잘못된 연결의 위험이다 |

| 파손 방식 | 충격을 주면 빛이 터지는 것이 아니라, 주변의 논리 길을 엉키게 한다 |
| 회복 방식 | 출발점과 도착점을 모두 확인하고, 중간의 끊긴 구간을 인정할 때만 낮게 흐른다 |
| 장면 기능 | 소포스의 닫힌 역장과 로고스의 열린 법칙을 공간 구조로 대비 |

활용:

- 에이도스는 처음에 관을 효율적인 경로로 본다.
- 로고스는 "흐르지 않는 관도 기억한다"고 말하며, 멈춘 구조에도 해석이 남는다는 점을 보여준다.

## 3. 프락시스파와 카우투스파

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 집단/생활 정치 |
| 소속 | 프론티아 아스트리온 |
| 낮의 형태 | 거리 연설, 회랑 낙서, 색이 다른 기록 끈, 관료들의 작은 표정 차이로 드러나는 파벌 |
| 밤의 변형 | 프락시스파의 길은 직선으로 짧아지고, 카우투스파의 길은 위험 표식 때문에 끝없이 돌아간다 |
| 은유 | 실행과 신중함이 국가 안에서 생활 습관과 계급 감정으로 굳는 과정 |
| 관련 인물 | 프락시스, 카우투스, 조일루스, 아미키티아, 에이도스 |
| 대가 | 파벌이 강해질수록 서로의 장점보다 상대의 실패 사례만 수집한다 |
| 장면 기능 | 프론티아가 단색 논리 국가가 아니라 내부 정치와 생활 감정을 가진 사회임을 보여준다 |

파벌 구분:

| 집단 | 선물 | 왜곡 | 사랑스러운 디테일 |
|---|---|---|---|
| 프락시스파 | 오래 지연된 문제를 실제로 움직인다 | 느린 회복과 보이지 않는 고통을 비용 처리한다 | 회의 전에 의자 배치를 전부 앞으로 돌려놓는다 |
| 카우투스파 | 위험과 반례를 끝까지 기억한다 | 실패 가능성만 보며 행동을 늦춘다 | 지도에 사라진 길도 지우지 않고 옅은 선으로 남긴다 |

사용 원칙:

- 어느 쪽도 완전히 어리석게 쓰지 않는다.
- 독자는 프락시스파의 답답하지 않은 실행력도, 카우투스파의 조심스러운 품격도 이해해야 한다.

## 4. 세르 공급 시설

정사 등급: B급에서 A급 승격 후보

| 항목 | 설정 |
|---|---|
| 분류 | 건축/장치/착취 기반 시설 |
| 소속 | 눅세린 아르도르, 잿빛 구역과 황금 궁전 사이 |
| 낮의 형태 | 가치 결정체를 정제해 도시 조명과 방어 장치에 공급하는 거대한 지하 시설 |
| 밤의 변형 | 사람들의 결핍과 가치 부채가 금빛 연료관을 따라 흐르며, 웃음소리처럼 들리는 증기가 새어 나온다 |
| 은유 | 화려한 도시가 감정과 노동을 실제 연료로 태우는 구조 |
| 관련 인물 | 마이더스, 아마라, 에이도스, 한신, 셀로 |
| 대가 | 시설을 파괴하면 착취는 멈추지만, 그 연료에 의존해 하루를 버티던 사람들도 즉시 흔들린다 |
| 약점 | 가치 결정체가 아니라 조건 없는 도움의 잔향이 섞이면 관이 역류한다 |
| 장면 기능 | 마이더스 체계가 감정적 선동만이 아니라 물질적 기반 시설임을 보여준다 |

활용:

- 저항군이 이곳을 공격할 때, 단순 폭파보다 "무엇을 멈추고 무엇을 남길 것인가"가 갈등이 된다.
- 아마라는 시설을 증오하지만, 시설 노동자들이 잿빛 구역 주민이라는 사실 때문에 망설인다.

| 밤의 변형 | 보는 사람이 아름답다고 느끼는 방향으로 움직이며, 감탄하는 순간 탈출 경로를 막는다 |

| 파손 방식 | 강제로 부수면 더 작은 장식 파편이 되어 주변 거울에 붙는다 |

| 장면 기능 | 나르키소스의 피상성이 단순 허영이 아니라 타인의 시선을 포획하는 폭력임을 보여준다 |

활용:

- 에이도스는 조각상의 패턴을 분석해 약점을 찾지만, 콘코르디아는 자신도 모르게 조각상의 자세를 따라 하려 한다.
- 조각상을 깨는 장면은 "아름다움 파괴"가 아니라 "시선 감금 해제"처럼 보여야 한다.

## 6. 검은 가시덤불

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 유기물/밤 변형 생태 |
| 소속 | 수정 호수 정원, 세리엘의 녹티오라 |
| 낮의 형태 | 평소에는 보이지 않거나, 맹세석 주변에 아주 얇은 검은 뿌리로만 남아 있다 |
| 밤의 변형 | 배신감이 커질수록 땅에서 솟아오르며, 다가오는 사람의 말보다 망설임에 먼저 반응한다 |
| 은유 | 믿음의 상처가 개인 몸을 넘어 환경 전체를 찌르는 생태로 번지는 과정 |
| 관련 인물 | 세리엘, 사일러스, 셀로, 에이도스 |
| 대가 | 가시덤불이 넓어질수록 세리엘은 안전해지지만, 누구도 그의 목소리를 가까이서 들을 수 없다 |
| 파손 방식 | 불로 태우면 검은 재가 되어 다음 거짓 사과에 더 빨리 자란다 |
| 회복 방식 | 억지 화해가 아니라 기다림이 반복될 때 가시 끝에 투명한 물방울이 맺힌다 |
| 장면 기능 | EP8의 풍경 변형을 세리엘의 내면과 직접 연결한다 |

주의:

- 검은 가시덤불은 세리엘의 악의를 표현하는 장식이 아니다.
- 상처가 너무 커져 주변 세계까지 방어 자세를 취한 결과로 써야 한다.

## 8. 골드바 전투원

|---|---|

| 낮의 물성 | 묵직한 금괴. 표면에는 가치 등급과 소유 이력이 작게 새겨져 있다 |
| 밤의 변형 | 금괴의 가치와 소유자의 갈망에 비례해 금빛 전투원으로 일어선다 |
| 은유 | 가치가 사람을 지키는 것이 아니라 사람 대신 폭력을 수행하는 순간 |

| 대가 | 전투원이 쓰러질 때마다 그 금괴에 기대던 누군가의 작은 욕망도 함께 부서진다 |

| 장면 기능 | 마이더스의 욕망이 추상 권력이 아니라 실제 폭력으로 변환되는 클라이맥스 장치 |

하위 변종:

- 장신구 무기: 반지, 목걸이, 브로치가 검, 사슬, 방패로 변한다. 개인의 과시가 즉시 공격성이 되는 장면에 쓴다.
- 금가루 병사: 골드바 전투원을 부순 뒤 생기는 작은 잔재. 완전히 없애려 할수록 더 넓게 묻어난다.

활용:

- 콘코르디아가 멘다키우스의 능력을 복제하려 할 때, 기만의 장막은 그의 약점도 함께 건드린다. 모두의 목소리를 흉내 내면 자기 목소리가 다시 흐려질 수 있다.
- 에이도스는 장막을 찢기보다, 장막이 바꾼 순서의 규칙을 찾아 원문에 닿는다.
