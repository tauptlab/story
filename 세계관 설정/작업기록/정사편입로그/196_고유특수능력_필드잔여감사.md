# 196 고유 특수능력 필드 잔여 감사

작성일: 2026-06-24

역할: 전체 인물 문서 110개를 기준으로 `능력 등급:` 표준 필드와 능력 운용 필드 보강 현황을 점검한 기록이다.

## 1. 현재 판정

| 구분 | 문서 수 | 상태 |
|---|---:|---|
| 고유 특수능력 | 48 | 12명 보강 완료, 36명 잔여 |

| 사소한 감각 | 4 | 생활 감각으로 유지 |
| 능력 없음 | 42 | 예법/소품/습관 중심으로 유지 |

## 2. 이번 라운드 완료

- 핵심 누락 7명 표준 필드 추가: 셀로, 에이도스, 프락시스, 루지에로, 켈, 아흐, 카라카사.
- 약한 발현 9명 녹티오라/예법/대가 보강: 살리엔, 아나트, 아우토노에, 에라타, 에렉투스, 에토스, 엘리올, 인터라, 프리오르.
- 고유 특수능력 1차 운용 필드 보강: 나르키소스, 마이더스, 도그마, 므네몬, 루멘.

## 3. 고유 특수능력 잔여 대상

아래 인물은 개별 문서 안에 능력명과 전투 기능은 있으나, `작가용 의미`, `소품/물성`, `녹티오라 변형`, `결투 기능`이 표준 필드로 아직 충분히 분리되지 않았다.

| 우선 묶음 | 인물 |
|---|---|
| 기억/생명/이야기 축 | 레기나 페쿤디아, 로고스, 실바누스, 파불라, 팔수스 |
| 관계/우정/공동체 축 | 레마니스, 소키우스, 아미키티아, 아마라, 오블라투스 |
| 공포/상실/죽음 축 | 포보스, 포르티스, 레툼, 이놉스, 셀레네 |
| 자유/풍요/결핍 반작용 축 | 리버, 리타스, 사일러스, 세리엘, 블라드 드라쿨 |
| 논리/기만/비판 축 | 멘다키우스, 조일루스, 소포스, 포에두스, 프레펙투스, 한신 |
| 보존/신중/진실 축 | 카우투스, 쿠스토스 세넥스, 카산드라, 카시안, 알디레아, 릴센 |
| 감각/흐름 보강 축 | 리르, 살루스, 콘코르디아, 페이단 |

## 4. 다음 작업 방식

1. 각 묶음마다 4~6명씩 처리한다.
2. 기존 능력명은 유지하고, 직접적인 설명어는 작가용 의미로만 둔다.
3. `능력 운용 보강` 표를 추가해 작가용 의미, 소품/물성, 녹티오라 변형, 결투 기능을 분리한다.
4. 보강 뒤 `고유 특수능력` 감사에서 잔여 수를 다시 줄인다.
