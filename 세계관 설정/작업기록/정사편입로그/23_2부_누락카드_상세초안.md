# 2부 누락 카드 상세 초안

이 문서는 `22_2부_통합요소_추출과_누락카드.md`에서 상위 우선도로 분류한 요소를 실제 정사 카드로 승격하기 위한 상세 초안이다. 2부의 핵심은 욕망을 단순 악으로 처리하지 않고, 인정받고 싶음, 결과를 만들고 싶음, 믿고 싶음, 기억되고 싶음이 어떻게 뒤틀리는지를 보여주는 데 있다.

아직 최종 원고가 아니라 설정 성경 단계의 작업물이므로, 각 항목은 향후 `02`, `03`, `07`, `12`, 도시별설정, 인물 개별 문서로 분산 반영한다.

## 1. 판결의 투기장

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/제도/의식 건축 |
| 소속 | 프론티아 아스트리온, 왕좌 교체 의식 |
| 낮의 형태 | 원형 경기장처럼 보이지만 관중석보다 계산석이 많다. 바닥은 거대한 은색 눈금판이고, 중앙에는 왕좌 대신 빈 기준점이 있다 |
| 밤의 변형 | 싸우는 자의 주장이 바닥의 선으로 변하고, 선이 닿은 곳마다 실제 지형과 규칙이 바뀐다 |
| 은유 | 결정권을 얻기 위해 먼저 자신이 무엇을 희생할지 증명해야 하는 제도 |
| 제도가 믿는 거짓말 | 결과가 명확하면 책임도 명확해진다 |
| 관련 인물 | 카우투스, 프락시스, 아미키티아, 에이도스, 셀로 |
| 대가 | 승자가 얻는 것은 왕좌가 아니라, 이후 모든 실패를 자기 계산으로 떠안아야 하는 책임이다 |
| 반작용 | 아미키티아의 우정, 셀로의 존재의 방벽, 카우투스의 선견의 주사위 |
| 장면 기능 | 카우투스를 무능한 보수주의자가 아니라 너무 많은 위험을 본 통치자로, 프락시스를 단순 야심가가 아니라 실제 결과를 갈망하는 인물로 보이게 한다 |

의식 구조:

| 단계 | 내용 | 장면 기능 |
|---|---|---|
| 기준점 선언 | 도전자가 자신이 세계를 어디로 끌고 갈지 한 문장으로 선언한다 | 프락시스의 결론 집착을 드러낸다 |
| 반례 소환 | 현 왕좌가 그 결론에서 빠진 사람과 실패 가능성을 불러낸다 | 카우투스의 신중함이 힘으로 작동한다 |
| 결과 투척 | 도전자가 자기 결과를 무기처럼 던진다 | `결과의 창` 첫 전면 사용 |
| 책임 회수 | 바닥에 남은 선들이 승자의 몸으로 되돌아온다 | 왕좌는 권력보다 부채에 가깝다는 감각 |

사용 원칙:

- 투기장은 피를 흘리는 경기장이기보다 결론이 사람을 베는 장소다.
- 카우투스가 이곳을 지킨 이유는 권력을 놓기 싫어서만이 아니라, 젊은 확신이 감당할 실패의 무게를 너무 많이 보았기 때문이다.
- 프락시스가 이긴다 해도 완전한 승리가 아니라, 자신이 계산에서 지운 얼굴들이 바닥의 선으로 남는 승리여야 한다.

## 2. 결과의 창

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/무기/프론티아 무기물 변형 |
| 소속 | 프락시스 |
| 낮의 물성 | 은청색 장창. 창날보다 손잡이의 눈금이 더 복잡하며, 던지기 전 이미 도착점이 희미하게 빛난다 |
| 밤의 변형 | 사용자가 정한 결과를 향해 창, 방패, 지지대, 분신, 다리로 변형된다 |
| 은유 | 과정을 사람으로 보지 않고 결과에 이르는 경로로만 보는 추진력 |
| 선물 | 망설임 속에서 실제 행동을 만들어낸다 |
| 상처 | 아무도 결정을 내리지 못해 누군가가 오래 다쳤던 기억 |
| 거짓 믿음 | 좋은 결과를 만들 수 있다면 중간의 얼굴들은 잠시 접어도 된다 |
| 관련 인물 | 프락시스, 카우투스, 에이도스, 셀로, 아미키티아 |
| 대가 | 창을 많이 쓸수록 아직 끝나지 않은 사람도 이미 결론 난 변수처럼 보인다 |
| 반작용 | 선견의 주사위, 에이도스의 질문, 아미키티아의 이름 부르기, 리버의 즉흥 |
| 장면 기능 | 결과주의의 매력과 폭력을 동시에 보여준다 |

전투 응용:

| 형태 | 기능 | 위험 |
|---|---|---|
| 투창 | 가장 짧은 경로로 목표에 도달 | 경로 위의 사람을 장애물로 분류 |
| 방패 | 목적을 방해하는 변수를 차단 | 보호와 배제가 구분되지 않음 |
| 지지대 | 무너질 구조를 임시로 떠받침 | 오래 유지하면 임시방편이 제도가 됨 |
| 분신 | 같은 결론을 여러 방향에서 밀어붙임 | 반례를 들을 귀가 줄어듦 |

후반 회수:

- 프락시스가 진짜로 성장할 때 결과의 창은 더 날카로워지는 것이 아니라, 손잡이에 빈 눈금을 남기기 시작한다.
- 그 빈 눈금은 아직 계산하지 않은 사람, 아직 결론 내리지 않은 가능성의 자리다.

## 3. 리타스의 정원 지하공간

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/은신처/생활 공동체 |
| 소속 | 눅세린 아르도르, 잿빛 구역 아래 |
| 낮의 형태 | 무너진 정원 배수로와 뿌리 창고를 이어 만든 지하 거점. 벽에는 오래전 리타스가 심었다는 뿌리 문양이 남아 있다 |
| 밤의 변형 | 사람들의 대가 없는 호의가 작은 씨앗빛으로 떠오르지만, 숨긴 두려움이 많을수록 빛은 흙 속으로 가라앉는다 |
| 은유 | 값이 매겨진 도시 아래에서도 아직 가격표 없이 살아남은 보살핌 |
| 관리 인물 | 아마라와 저항 공동체 |
| 관련 인물 | 아마라, 아미키티아, 리타스, 루멘, 마이더스, 셀로 |
| 대가 | 너무 오래 숨어 있으면 선의도 자기가 부끄러운 것처럼 낮은 목소리로만 말하게 된다 |
| 반작용 | 마지막 화롯가, 화롯가 민들레, 바랜 가치 결정체 |
| 장면 기능 | 저항군을 단순 반란 조직이 아니라 오래된 풍요의 마지막 생활권으로 만든다 |

공간 요소:

| 요소 | 형태 | 사용법 |
|---|---|---|
| 뿌리 기둥 | 천장을 붙드는 마른 뿌리와 금이 간 돌 | 공동체가 버티는 힘을 시각화 |
| 흙잔 선반 | 각자 자기 이름 없이 쓰는 작은 잔 | 소유보다 나눔을 보여줌 |
| 씨앗 저장칸 | 황금 베일을 피해 숨긴 씨앗 | 리타스 귀환과 후반 회수 |
| 낮은 노래방 | 말보다 허밍으로 서로의 생존을 확인하는 공간 | 아미키티아의 온기와 연결 |

사용 원칙:

- 아마라 공동체는 숭고하기만 한 조직이면 안 된다. 굶주림, 피로, 질투, 작은 계산도 있어야 살아 있는 공동체가 된다.
- 그럼에도 누군가 마지막 조각을 숨기지 않고 건네는 순간, 이 공간은 2부에서 가장 따뜻한 장소가 된다.

## 4. 선전 방송탑

정사 등급: B급에서 A급 승격 후보

| 항목 | 설정 |
|---|---|
| 분류 | 건축물/정보 장치/도시 제도 |
| 소속 | 눅세린 아르도르, 마이더스 체제 |
| 낮의 형태 | 금빛 스피커와 거울판이 겹친 높은 탑. 도시 어디서나 보이지만 가까이 가면 표면이 낡아 있다 |
| 밤의 변형 | 들은 말을 각자의 결핍에 맞춰 바꿔 들려준다. 굶주린 이는 식량 공지를, 외로운 이는 인정의 약속을 듣는다 |
| 은유 | 욕망을 만들어내기보다 이미 있는 결핍에 이름표를 붙여 조종하는 선전 |
| 운영 인물 | 마이더스의 정보 통제실, 멘다키우스 계열 전언 조작자 |
| 관련 인물 | 마이더스, 멘다키우스, 아마라, 한신, 셀로 |
| 대가 | 오래 들으면 자기 욕망과 주입된 욕망의 목소리 높낮이를 구분하지 못한다 |
| 반작용 | 리타스의 정원 지하공간, 마지막 화롯가, 속삭임 파편의 원문 확인 |
| 장면 기능 | 마이더스를 금만 탐하는 폭군이 아니라, 사람들의 결핍을 알고 조율하는 위험한 통치자로 만든다 |

장면 씨앗:

- 방송탑은 거짓말만 하지 않는다. 가장 위험한 방송은 사실 한 조각을 포함한다.
- 아마라는 탑을 부수려 하지만, 탑 아래에서 방송을 들으며 하루를 버틴 사람들을 보고 망설인다.
- 셀로가 탑의 파동을 들을 때, 그것은 거대한 명령이라기보다 수천 개의 작은 "나도 봐줘"가 겹친 소리여야 한다.

## 5. 수정 호수 정원

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/맹세의 정원 |
| 소속 | 눅세린 신뢰 거래소 외곽, 세리엘과 사일러스의 과거 장소 |
| 낮의 형태 | 얕은 수정 호수와 흰 돌길, 작은 맹세석이 놓인 정원. 물은 맑지만 바닥의 금은 잘 보이지 않는다 |
| 밤의 변형 | 호수에 비친 두 사람의 모습이 서로 다른 시기의 약속을 보여준다 |
| 은유 | 믿음은 깨끗한 감정이 아니라, 서로 다른 기억 위에 놓인 위험한 다리다 |
| 관련 인물 | 세리엘, 사일러스, 셀로, 에이도스, 마이더스 |
| 대가 | 이곳에서 한 맹세는 마음이 변해도 형태가 남는다. 형태가 남은 약속은 쉽게 무기가 된다 |
| 반작용 | 신뢰의 가시, 호의 장부, 신뢰 거래소 어린 서기 |
| 장면 기능 | 세리엘의 비극을 단순 배신 사건이 아니라, 아름다웠던 약속이 변질되는 과정으로 만든다 |

사용 원칙:

- 정원은 처음부터 음산하면 안 된다. 독자가 "여기서라면 누군가를 믿고 싶다"고 느껴야 배신이 아프다.
- 사일러스는 처음부터 거짓말만 한 사람이 아니라, 약속을 지키고 싶었던 시기가 있었던 인물로 남겨둔다.
- 셀로는 정원에서 배신의 증거보다 먼저 둘의 오래된 웃음을 보아야 한다. 그래야 이후 판단이 단순해지지 않는다.

## 6. 신뢰의 가시

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/무기/감정 변형 |
| 소속 | 세리엘 |
| 낮의 물성 | 투명한 수정 가시. 가까이서 보면 표면에 약속 문장이 가느다랗게 새겨져 있다 |
| 밤의 변형 | 믿었던 사람이 건넨 말일수록 더 깊이 박히고, 뽑으려 하면 그 말을 믿었던 자신까지 함께 찢긴다 |
| 은유 | 배신당한 신뢰가 타인을 찌르기 전에 먼저 자신을 찌르는 구조 |
| 선물 | 누가 진심으로 다가왔는지 알아보는 예민함 |
| 상처 | 가장 믿은 사람이 자신의 믿음을 거래 대상으로 만든 기억 |
| 거짓 믿음 | 다시 믿지 않으면 다시 배신당하지 않는다 |
| 관련 인물 | 세리엘, 사일러스, 셀로, 에이도스, 아미키티아 |
| 대가 | 가시가 강해질수록 보호와 공격의 방향이 구분되지 않는다 |
| 반작용 | 아미키티아의 이름 부르기, 콘코르디아의 합의가 아닌 경청, 셀로의 판단 유예 |
| 장면 기능 | 세리엘을 사랑에 속은 피해자도, 배신에 미친 적도 아닌, 믿고 싶었던 마음이 상처로 무장한 인물로 만든다 |

전투 규칙:

| 작동 | 효과 | 회복 단서 |
|---|---|---|
| 약속 반사 | 상대가 한 말을 가시로 되돌림 | 같은 말을 다른 의미로 다시 말해야 약해짐 |
| 거리 봉쇄 | 가까워지려는 인물을 밀어냄 | 억지 접근보다 기다림이 필요 |
| 자기 갑옷 | 세리엘 주변에 수정 가시가 자람 | 세리엘이 스스로 한 문장을 철회할 때 틈이 생김 |

후반 회수:

- 신뢰의 가시는 완전히 사라지는 것이 아니라, 세리엘이 다시 믿을 대상을 고르는 감각으로 바뀌는 편이 좋다.
- 상처를 없애는 결말보다, 상처가 더 이상 모든 손을 적으로 읽지 않는 결말이 설득력 있다.

## 7. 심층 기록 보관소

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/건축/봉인 구조 |
| 소속 | 히프노레아 이렐사드 지하 |
| 낮의 형태 | 공식 기록 서가 아래 숨겨진 봉인 보관소. 결정 책장 사이로 살아있는 뿌리기둥이 내려와 있다 |
| 밤의 변형 | 기록을 읽는 사람이 아니라 기록이 읽는 사람을 고른다. 숨긴 기억이 많을수록 길이 길어진다 |
| 은유 | 보존이라는 이름으로 묻어둔 죄와, 잊히지 않으려는 생명의 저항 |
| 수호 인물 | 므네몬, 실바누스의 잔향, 어린 이렐리안 사서들 |
| 관련 인물 | 셀로, 에이도스, 므네몬, 실바누스, 팔수스, 레기나 페쿤디아 |
| 대가 | 깊이 들어갈수록 무엇을 기억해야 하는지보다 무엇을 기억할 자격이 있는지를 묻게 된다 |
| 반작용 | 히프노레아 전통 의복, 얼굴 없는 초상화, 세계수 뿌리 |
| 장면 기능 | 다산의 일족 숙청, 팔수스 침공, 실바누스 수호축을 하나의 지하 구조로 묶는다 |

하위 구조:

| 구역 | 기능 | 장면 사용 |
|---|---|---|
| 무음 열람실 | 봉인 기록을 소리 없이 읽는 곳 | 셀로의 이상적 형태 투사 전조 |
| 뿌리 계단 | 세계수 뿌리가 만든 비공식 통로 | 실바누스의 흔적과 연결 |
| 삭제 수로 하류 | 폐기 기록 결정체가 흘러드는 곳 | 공식 역사 밖 단서 발견 |
| 봉인된 심층문 | 다산의 일족 기록으로 가는 마지막 문 | 므네몬의 보호와 은폐가 충돌 |

사용 원칙:

- 보관소는 단순 던전이 아니라 죄책감이 건축이 된 장소다.
- 므네몬은 악한 은폐자라기보다, 기억을 공개했을 때 다시 학살이 반복될 가능성을 두려워하는 관리자여야 한다.

## 8. 이상적 형태 투사

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/상태/셀로 성장 단계 |
| 소속 | 셀로 |
| 낮의 형태 | 셀로의 몸 바깥에 한 겹 더 선명하고 아름다운 빛의 윤곽이 생긴다 |
| 밤의 변형 | 실제 셀로보다 빠르고 바르고 강한 "되어야 할 셀로"가 앞서 움직인다 |
| 은유 | 아직 자기 자신을 사랑하지 못한 마음이, 사랑받을 만한 이상형을 먼저 만들어내는 현상 |
| 선물 | 압도적인 위기에서 잠시 더 나은 선택을 상상하게 한다 |
| 상처 | 지금의 자신으로는 충분하지 않다는 감각 |
| 거짓 믿음 | 더 완벽한 내가 나타나면 지금의 나는 덜 미워해도 된다 |
| 관련 인물 | 셀로, 에이도스, 리버, 콘코르디아, 세리엘, 므네몬 |
| 대가 | 사용할수록 실제 셀로의 떨림, 망설임, 도움 요청이 뒤처진 것으로 취급된다 |
| 반작용 | 리버의 불완전한 즉흥, 에이도스의 모르는 것을 인정하는 질문, 콘코르디아의 불협화음 수용 |
| 장면 기능 | 셀로의 성장이 곧바로 건강한 통합이 아니라, 자기 이상화라는 위험한 단계도 거친다는 점을 보여준다 |

능력 구분:

| 능력 | 핵심 | 위험 |
|---|---|---|
| 존재의 방벽 | 지금의 자신을 지키는 보호막 | 방어에만 머물면 고립 |
| 이상적 형태 투사 | 되어야 할 자신을 앞세우는 투사 | 현재의 자신을 부끄러워함 |
| 심판의 영역 | 자기 기준으로 세계를 판정하는 장 | 보호가 독선적 처벌로 바뀜 |

장면 사용:

- EP9에서 처음 뚜렷하게 나타날 때는 독자가 멋있다고 느껴도 된다. 다만 에이도스나 리버가 아주 작은 위화감을 보아야 한다.
- 이상적 형태는 셀로를 도와주지만, 동시에 셀로의 약함을 뒤로 밀어낸다. 이 모순을 남겨야 EP10의 심판의 영역이 설득된다.

## 9. 욕망의 옥좌실

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/건축/클라이맥스 전장 |
| 소속 | 눅세린 아르도르 황금 궁전 |
| 낮의 형태 | 황금 왕좌, 가치 결정체 기둥, 정보 통제실과 이어진 긴 계단을 가진 방. 눈부시지만 의자가 하나뿐이다 |
| 밤의 변형 | 들어온 사람마다 가장 부끄러워하는 욕망을 왕좌 주변에 형상으로 띄운다 |
| 은유 | 인정받고 싶은 마음이 타인을 자리 없는 관객으로 만들 때 생기는 외로움 |
| 지배 인물 | 마이더스 |
| 관련 인물 | 마이더스, 아마라, 셀로, 나르키소스, 멘다키우스, 드라쿨, 한신 |
| 대가 | 왕좌에 가까워질수록 원하는 것을 얻는 장면은 선명해지지만, 그것을 함께 기뻐할 사람의 얼굴은 흐려진다 |
| 반작용 | 리타스의 정원 지하공간, 아마라의 마지막 선택, 셀로의 판단 유예 |
| 장면 기능 | 마이더스와 셀로가 모두 "내 기준으로 세계를 다시 세우고 싶다"는 유혹에 닿는 장소가 된다 |

공간 요소:

| 요소 | 형태 | 사용법 |
|---|---|---|
| 황금 왕좌 | 앉은 자의 욕망을 도시 명령으로 증폭 | 마이더스의 권력과 고독을 함께 보여줌 |
| 가치 결정체 기둥 | 시민들의 인정 욕구가 쌓인 기둥 | 부수면 해방과 공허가 동시에 옴 |
| 정보 통제실 문 | 왕좌 뒤편의 낮은 문 | 선전 방송탑, 속삭임 파편 조작과 연결 |
| 빈 관중석 | 누군가를 세워두기 위한 계단식 자리 | 욕망의 최종 관객이 사라졌음을 시각화 |

사용 원칙:

- 옥좌실은 화려하지만 승리감보다 피로감이 먼저 남아야 한다.
- 마이더스가 후퇴해도 그의 체제는 즉시 끝나지 않는다. 욕망은 왕 한 명보다 오래 산다.

## 10. 심판의 영역

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/전장/셀로 왜곡 단계 |
| 소속 | 셀로 |
| 낮의 형태 | 셀로 주변에 희미한 원이 생기고, 원 안의 대상들이 위협/비위협처럼 단순한 빛으로 분류된다 |
| 밤의 변형 | 원 안의 모든 것이 셀로의 기준에 따라 보호 대상, 교정 대상, 제거 대상으로 나뉜다 |
| 은유 | 상처받은 자기보호가 세계 전체를 자기 기준으로 판결하려는 순간 |
| 선물 | 혼란 속에서 누군가를 즉시 지킬 수 있는 결단 |
| 상처 | 자신이 놓친 사람, 구하지 못한 사람, 배신당한 믿음을 한꺼번에 만회하고 싶은 압박 |
| 거짓 믿음 | 내가 충분히 정확히 판단하면 아무도 더 다치지 않는다 |
| 관련 인물 | 셀로, 마이더스, 아마라, 세리엘, 에이도스, 리버 |
| 대가 | 판단 속도가 빨라질수록 질문, 사과, 기다림이 불필요한 것으로 밀려난다 |
| 반작용 | 에이도스의 질문 지연, 리버의 절차 밖 움직임, 아미키티아의 우정, 콘코르디아의 불완전한 합 |
| 장면 기능 | 셀로의 힘이 독자를 설레게 하면서도 두렵게 해야 하는 2부 후반 핵심 장치 |

작동 규칙:

| 판정 | 겉보기 효과 | 실제 위험 |
|---|---|---|
| 보호 대상 | 방벽이 감싸고 공격을 밀어냄 | 보호받는 사람의 선택권까지 줄어듦 |
| 교정 대상 | 움직임이 느려지고 말이 짧아짐 | 프론티아식 폭력과 닮아감 |
| 제거 대상 | 세르가 날카로운 궤적으로 응축 | 적을 없애며 자기 두려움도 함께 덮음 |

사용 원칙:

- 심판의 영역은 악한 능력이 아니라, 너무 절박한 보호 욕구가 판단권을 독점할 때 생기는 위험이다.
- 독자는 셀로가 왜 이 능력을 쓰는지 이해해야 하지만, 동시에 "이 방식은 무섭다"고 느껴야 한다.
- EP10 이후 셀로가 이 힘을 부끄러워하기만 하면 안 된다. 부끄러움에서 멈추면 또 다른 회피가 되므로, 이후에는 판단하기 전에 묻는 법을 배워야 한다.
