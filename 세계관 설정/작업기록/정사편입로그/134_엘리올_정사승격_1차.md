# 엘리올 정사 승격 1차

정사 등급: 작업 기록

역할: 루트 초기 자료의 `엘리올` 씨앗을 프론티아 아르세니아와 라세드린의 귀환 인물로 승격하고, "정답을 요구하는 성실함"을 빠진 사정을 다시 받는 공정과 Mnemosyn 귀환 축에 연결한다.

---

## 1. 승격 요약

엘리올은 기존 자료에서 아르세니아 반례 우편함 광장의 논쟁자로 배치되어 있었다. 이번 승격에서는 그를 단순한 정답 강요자가 아니라, 흐린 책임 때문에 누군가가 다시 묻지 못했던 사건을 본 뒤 선명한 답을 보호로 믿게 된 인물로 정리했다.

핵심 이동은 아래와 같다.

| 구분 | 승격 전 | 승격 후 |
|---|---|---|
| 인물 기능 | 프론티아 논쟁형 후보 | 아르세니아의 판결 예법과 라세드린의 말 도착 예법을 잇는 귀환 인물 |
| 핵심 질문 | 정답을 요구하는 태도는 어떻게 바뀌는가 | 답을 지키면서도 늦게 온 사정을 다시 받을 수 있는가 |
| 주 무대 | 아르세니아 후보 | 반례 우편함 광장, 첫줄 개정대, 라세드린 이유 원탁 |
| 주요 물성 | 반례 우편함, 노력추, 역제출 봉투 | 첫줄 지우개, 유예 은핀, 기준꽃, 반례이끼 |
| 결투 기능 | 미정 | 명확한 정답의 보호와 빠진 사정을 다시 받는 공정의 재배치 |

## 2. 124 기준 반영

| 기준 | 엘리올 반영 |
|---|---|
| A가 버린 가치관 | 모르는 상태를 견디고, 늦게 온 사정을 다시 받을 수 있다는 믿음 |
| 왜 버릴 수밖에 없었는가 | 흐린 책임과 애매한 결론 뒤에서 억울한 사람이 다시 묻지 못한 기억이 남았다 |
| 대체 가치관 | 선명한 답과 기준을 세워 누구도 책임을 피하지 못하게 하는 습관 |
| 처음의 보호 기능 | 애매함 속에 버려진 사람을 줄이고, 판결의 책임선을 보이게 했다 |
| 시간이 지나며 생긴 왜곡 | 답이 사람보다 먼저 움직이고, 늦은 사정은 예외나 방해로 밀려난다 |
| 귀환 계기 | 반례 우편함의 늦은 봉투와 라세드린 이유잔이 엘리올의 첫 줄을 다시 쓰게 만드는 결투 |
| 필연성 | 정답만으로는 빠진 사람을 보호할 수 없고, 끝없는 보류만으로는 당장 다친 사람을 지킬 수 없기 때문에 고칠 수 있는 답이 필요해진다 |
| 귀환 후 후폭풍 | 프론티아는 판결 앞에 개정 절차를 남기고, 라세드린은 이기는 말보다 도착하는 말의 책임을 더 무겁게 묻는다 |

## 3. 녹티오라 설계

| 낮의 요소 | 왜 남았는가 | 밤의 형태 | 선택 시험 |
|---|---|---|---|
| 반례 우편함 | 빠진 사정을 뒤늦게라도 받아야 한다는 공정의 흔적 | 결론이 빨라질수록 입구가 높아지고, 기다리면 아이 손 높이로 낮아진다 | 지금 닫을 것인가, 늦은 봉투를 받을 것인가 |
| 첫줄 지우개 | 완벽한 문장도 첫 전제가 틀리면 고쳐야 한다는 습관 | 모든 문장을 지우지 않고 첫 줄만 물빛으로 흐린다 | 자존심을 지킬 것인가, 첫 줄을 다시 쓸 것인가 |
| 유예 은핀 | 끝내지 못한 문장을 잠시 붙들기 위한 작은 은핀 | 오래 꽂으면 문장이 차갑게 굳고, 책임을 말하면 부드럽게 빠진다 | 보류를 회피로 쓸 것인가, 남은 책임으로 쓸 것인가 |
| 기준꽃 | 같은 기준과 다른 사정이 함께 있어야 한다는 생활 생태 | 꽃잎 수는 그대로지만 향을 잃고, 빠진 이름이 불리면 향이 돌아온다 | 동일함만 볼 것인가, 달라진 사정을 함께 맡을 것인가 |

## 5. 결투 설계 메모

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 명확한 정답의 보호 vs 빠진 사정을 다시 받는 공정 |
| 두 가치의 옳음 | 정답은 흐린 책임을 막고, 다시 받는 공정은 늦게 온 사람을 예외로 밀지 않는다 |
| 두 가치의 위험 | 정답은 사람보다 먼저 움직일 수 있고, 다시 받는 공정은 결론을 끝없이 늦출 수 있다 |
| 끝난 뒤 재배치 | 답은 버려지지 않고, 고칠 수 있는 문장으로 낮아진다 |
| 도시 변화 | 아르세니아의 반례 우편함은 낮아지고, 라세드린의 이유 원탁은 이긴 말보다 첫 이유를 더 오래 비춘다 |
| 인물 변화 | 엘리올은 완벽한 문장을 이기기 위해 읽는 대신, 늦은 봉투 앞에서 첫 줄을 다시 쓴다 |
