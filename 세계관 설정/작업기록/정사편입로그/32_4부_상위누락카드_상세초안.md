# 4부 상위 누락카드 상세초안

이 문서는 `31_4부_통합요소_추출과_누락카드.md`에서 상위 후보로 고정한 4부 핵심 카드 6개를 실제 장면에 투입 가능한 설정 카드로 확장한다. 4부는 셀로의 단순 폭주편이 아니라, 네우로시아의 처리 체계 전체가 "중심에 모든 것을 모으려는 욕망"을 견디지 못하고 무너지는 시기다.

4부 상위 카드의 핵심 원칙은 "무너짐도 하나의 세계사다"이다. 셀로, 카산드라, 마이더스, 도그마, 불씨 주민들은 모두 무너지는 장치 안에서 각자의 방식으로 살아남으려 한다. 따라서 아래 카드들은 전투 장치가 아니라, 세계가 왜 이런 방식으로 고장났고 어떤 방식으로 다시 숨을 찾을 수 있는지를 보여주는 역사 장치다.

| 중앙 가치 변환소 붕괴 | 역사 사건/시설 | 가치 체계가 자기 모순으로 무너짐 | 외부 공격 한 방과 중복됨 | 마이더스 문서, `01`, `02`, `07` |

## 1. 왕좌의 파동

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/세계 사건/자기 왕관 붕괴 파동 |
| 소속 | 셀로, 자기 왕관의 영역, 4부 EP16 |
| 작중 별칭 | 왕좌의 파동, 왕관 분노, 은빛 초신성, 돌아오는 길의 폭발 |
| 낮의 형태 | 셀로 주변의 모든 선택지가 그에게 돌아오는 듯한 압력으로 먼저 나타난다 |
| 밤의 변형 | 자기 왕관의 영역 전체가 거대한 왕관살처럼 타오르고, 그 중심에서 동심원 파동이 세계로 번진다 |
| 선물 | 더는 아무도 상처받지 않게 하려는 압도적인 보호 욕망 |
| 상처 | 세상이 자신이 만든 완벽한 질서를 담아주지 않는다는 왕좌 상처 |
| 왜곡 | 세계가 자신을 받아들이지 않으면 세계를 자신과 같은 모양으로 밀어내려 한다 |
| 관련 인물 | 셀로, 페이단, 에이도스, 아미키티아, 마이더스, 카시안 |
| 반작용 | 페이단 씨앗, 영혼의 맹약 방어선, 무지갯빛 수정체, 루멘의 먼 빛 |
| 장면 기능 | 셀로의 보호가 폭력이 되는 가장 큰 전환점 |

### 1.1 작동 규칙

| 규칙 | 장면 효과 | 위험 |
|---|---|---|
| 중심 회수 | 파동이 닿은 선택, 길, 세르 흐름이 셀로의 중심을 향해 휘어진다 | 타인의 고유한 방향이 지워진다 |
| 이질성 압축 | 셀로와 다른 리듬을 가진 파동이 눌리거나 터진다 | 갈등이 해결되지 않고 폭발한다 |
| 황금 외피 박리 | 눅세린의 황금 베일과 가치 변환 장치가 먼저 벗겨진다 | 마이더스 체계의 약점이 세계 전체에 노출된다 |
| 감정 과열 | 억압 감정이 잿빛 균열과 페이단 씨앗을 통해 다시 달아오른다 | 파동을 막을수록 분노가 더 많은 통로를 찾는다 |
| 자기 소모 | 파동이 커질수록 셀로의 내부 색이 사라진다 | EP17 자기 분열과 EP18 결정화로 이어진다 |

### 1.2 장면 사용 원칙

- 파동은 악의의 공격처럼 시작하면 약하다. 처음에는 "내가 다 고치겠다"는 결의처럼 보여야 한다.
- 파동의 피해자는 적만이 아니다. 눅세린 주민, 센시르 조율자, 히프노레아 기록물, 암디르의 불씨가 모두 서로 다른 방식으로 휘말린다.
- 아미키티아가 존재해도 모든 손목을 묶을 수 없다는 한계를 보여준다. 그녀의 실패가 아니라 재앙의 규모가 문제다.
- 마이더스의 방어막이 찢길 때는 마이더스만 조롱하지 않는다. 그 황금 외피가 실제로 한때 누군가를 안심시켰음을 짧게 남긴다.
- 셀로가 파동 뒤에 느끼는 공포는 "내가 졌다"가 아니라 "내가 원한 평화가 이 모습이 아니었다"여야 한다.

### 1.3 삽입 장면 씨앗

- 파동이 지나간 리테노아의 물길은 말소리를 키우지 않고, 모든 말을 셀로의 목소리와 같은 음으로 되돌려 준다. 카시안은 그 순간 셀로가 얼마나 외로운지를 듣지만, 동시에 얼마나 위험한지도 듣는다.
- 눅세린의 황금 건물에서 먼저 떨어지는 것은 장식이 아니라 이름표다. 가격표가 지워지자, 그 아래 숨겨진 사람들의 손글씨가 드러난다.
- 에이도스는 파동을 계산하려다 멈춘다. 수치가 너무 커서가 아니라, 모든 선이 "셀로에게 돌아감"이라는 하나의 결론으로만 접히기 때문이다.

## 2. 분열된 셀로들

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 존재군/내면 파편/자기 왕관 붕괴 생태 |
| 소속 | 셀로, 자기 왕관의 영역 잔해, 4부 EP17 |
| 낮의 형태 | 셀로의 말투, 시선, 손짓이 장면마다 서로 다른 사람처럼 어긋나는 전조 |
| 밤의 변형 | 깨진 왕관의 얼굴들이 독립된 형체로 현실에 나타난다 |
| 선물 | 셀로 안에 있던 실제 욕망과 상처를 숨기지 않고 보이게 한다 |
| 상처 | 한 얼굴로만 사랑받아야 한다고 믿은 시간 |
| 왜곡 | 각 파편이 자신만이 진짜 셀로라고 주장한다 |
| 관련 인물 | 셀로, 에이도스, 아미키티아, 도그마, 콘코르디아, 리타스 |
| 반작용 | 영혼의 맹약 방어선, 카시안의 공명 현, 루멘의 진실의 빛 |
| 장면 기능 | 셀로를 하나의 폭군에서 여러 사랑 가능한 얼굴의 집합으로 확장한다 |

### 2.1 파편별 상세

| 파편 | 핵심 상처 | 사랑스러운 습관 | 왜곡 능력 | 구원 조건 |
|---|---|---|---|---|
| 공허한 이상 | 완벽해야 사랑받는다는 믿음 | 깨진 표면도 손끝으로 매만져 대칭을 맞추려 한다 | 불완전한 것을 백색 빛으로 깎는다 | 누군가 깨진 채로도 곁에 머무는 장면 |
| 침묵의 심판자 | 틀린 선택이 누군가를 다치게 했다는 공포 | 판결 전 아주 짧게 눈을 감는다 | 대답을 듣기 전 은빛 원으로 분류한다 | "보류"가 도망이 아니라 책임일 수 있음을 보는 장면 |
| 절망하는 아이 | 기억을 잃고 혼자 남은 최초의 고립 | 겁먹으면 자기 주변에 작은 유리막을 만든다 | 주변 감정을 무작위로 빨아들여 폭발한다 | 누군가 해결책 없이 옆에 앉아 주는 장면 |
| 오만한 폭군 | 통제를 놓으면 모두 무너진다는 두려움 | 무너진 왕관 조각을 계속 제자리에 올려놓는다 | 붕괴한 공간 파편을 억지로 배열한다 | 통제하지 않아도 누군가가 자기 선택으로 남는 장면 |
| 허무의 그림자 | 모든 시도가 결국 실패했다는 피로 | 희망의 말을 들으면 먼저 그 말의 끝을 본다 | 주변 온기와 의욕을 회색으로 낮춘다 | 작은 행동이 세계 전체를 고치지 않아도 의미 있음을 보는 장면 |

### 2.2 전투 규칙

| 규칙 | 설명 |
|---|---|
| 파편은 죽이면 끝나지 않는다 | 물리적 격파는 그 감정이 다른 파편에 옮겨붙는 결과를 낳는다 |
| 각 파편은 일부 진실을 가진다 | 공허한 이상의 완벽함, 심판자의 책임감, 아이의 두려움, 폭군의 보호 욕구, 그림자의 피로는 모두 진짜다 |
| 서로를 적으로 본다 | 각 파편은 다른 파편을 "나를 망친 얼굴"로 보고 공격한다 |
| 동료가 할 수 있는 일은 통합이 아니라 보호다 | 진짜 통합은 셀로 내부에서 시작되어야 하며, 동료들은 소멸을 막고 시간을 번다 |
| 도그마의 삭제는 역효과를 낳는다 | 삭제된 파편의 진실이 왜곡되어 더 거친 형태로 돌아온다 |

### 2.3 장면 사용 원칙

- 파편마다 독자가 이해할 만한 슬픔을 먼저 보여준다. 공격은 그 다음이다.
- 파편들은 "나쁜 셀로"가 아니라, 셀로가 살아남기 위해 한때 필요했던 얼굴들이다.
- 절망하는 아이는 가장 약한 파편이지만 가장 중요한 파편이다. 이 아이를 지키려는 선택이 동료들이 셀로를 포기하지 않는 이유가 된다.
- 도그마가 파편을 오류로 부를 때, 독자는 도그마의 판단이 아주 틀리지만은 않다는 긴장을 느껴야 한다. 파편들은 실제로 위험하다.
- 파편 이름은 장면 안에서 누군가가 붙여도 되고, 처음부터 능력명처럼 선언하지 않는다.

### 2.4 삽입 장면 씨앗

- 공허한 이상은 부서진 벽을 고치다가, 벽 안쪽의 아이 낙서를 지워버린다. 그는 아름답게 만들었지만, 누군가 살았던 흔적을 없앤 것이다.
- 침묵의 심판자는 아미키티아의 영혼의 맹약 실을 "분류 불가"로 보고 자르려 하지만, 실이 잘리지 않고 느슨해지는 것을 보고 처음으로 판결을 보류한다.
- 허무의 그림자가 리타스의 풍요를 비웃자, 리타스는 더 큰 샘을 열지 않는다. 대신 작은 컵 하나를 놓고 "한 명부터"라고 말한다.

## 3. 무지갯빛 수정체와 파동 수신

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 상태/무기물/능력 반전/침묵기 |
| 소속 | 셀로 결정화 상태, 4부 EP18~20 |
| 작중 별칭 | 무지갯빛 수정체, 침묵한 프리즘, 모든 비명을 듣는 돌 |
| 낮의 형태 | 빛은 있으나 움직임과 목소리가 없는 반투명 결정 |
| 밤의 변형 | 네우로시아의 감정 파동이 색실처럼 빨려 들어가 내부에서 충돌한다 |
| 선물 | 셀로가 처음으로 세계의 고통을 자기 판단 없이 듣는다 |
| 상처 | 모두를 품으려다 자기 목소리를 잃는다 |
| 왜곡 | 듣는 것과 받아들이는 것, 받아들이는 것과 살아내는 것을 구분하지 못한다 |
| 관련 인물 | 셀로, 에이도스, 아미키티아, 콘코르디아, 리타스, 카시안, 루멘 |
| 반작용 | 핵심의 파동, 루멘의 빛, 프리즘 결정 |
| 장면 기능 | 셀로의 능력을 발산에서 수신으로 뒤집어 성숙한 통합의 전조건을 만든다 |

### 3.1 파동 수신 규칙

| 수신 파동 | 수정체 내부 반응 | 장면 기능 |
|---|---|---|
| 공포 | 검은 곡선이 결정 안쪽을 빠르게 돈다 | 포보스/암디르 계열의 보호 본능을 되돌림 |
| 분노 | 진홍색 열이 균열 주변에 맺힌다 | 페이단 씨앗과 왕좌 분노의 잔류를 연결 |
| 슬픔 | 푸른 층이 천천히 가라앉는다 | 상실과 기억의 무게를 보여줌 |
| 희망 | 빛나기보다 아주 작은 핵으로 응축된다 | 5부 핵심의 파동과 루멘 전조 |
| 거짓 안정 | 표면은 매끈해지지만 내부 색이 탁해진다 | 도그마/마이더스식 안정의 위험 |
| 동의한 유대 | 분홍빛과 금빛 실이 표면을 깨지 않고 감싼다 | 아미키티아의 한계와 역할을 동시에 보여줌 |

### 3.2 회복 조건

| 단계 | 조건 | 장면 표시 |
|---|---|---|
| 수신 | 세계의 소리를 판단 없이 듣는다 | 결정 안에 너무 많은 색이 뒤섞인다 |
| 분리 | 어떤 파동이 누구의 것인지 구분한다 | 색들이 흐름을 만들기 시작한다 |
| 선택 | 모두를 대신 느끼려 하지 않고, 자신이 살아낼 한 줄기를 고른다 | 내부에 작은 핵이 생긴다 |
| 응답 | 외부로 첫 파동을 돌려보낸다 | 핵심의 파동 전조가 나타난다 |
| 통합 | 받은 파동을 지우지 않고 통과시킨다 | 후반 프리즘 결정으로 이어진다 |

### 3.3 장면 사용 원칙

- 수정체는 "감정의 쓰레기통"으로 표현하지 않는다. 그것은 비하적이고 수동적이다. 정사상 수정체는 과청취된 상태다.
- 동료들이 셀로를 흔들어 깨우려 할수록 수정체가 더 탁해질 수 있다. 회복은 자극이 아니라 보호와 분리에서 시작된다.
- 에이도스는 분석하지만, 분석만으로 셀로를 깨우지 못한다. 분석은 어떤 파동이 들어오는지 구분하는 데만 유효하다.
- 콘코르디아는 수정체의 평온이 진짜 평온이 아님을 가장 먼저 느낀다.
- 루멘의 먼 빛은 수정체를 깨뜨리는 망치가 아니라, 내부의 핵이 어느 방향으로 맥동하는지 보이게 하는 등불이어야 한다.

### 3.4 삽입 장면 씨앗

- 수정체 곁에 있던 카시안이 갑자기 자기 것이 아닌 울음을 낸다. 그는 그 울음이 네우로시아 어딘가의 이름 없는 아이에게서 온 것임을 듣고, 셀로가 얼마나 많은 소리를 견디는지 깨닫는다.
- 에이도스가 수신 파동을 분류하려고 선을 긋자, 선은 금세 녹아 서로 섞인다. 아미키티아가 실을 얹었을 때만 잠시 각 색이 자기 자리를 찾는다.
- 루멘의 빛이 멀리서 닿는 순간, 수정체는 깨지지 않는다. 대신 내부의 한 점이 처음으로 "밖"을 향해 맥동한다.

- 카산드라 연락망은 전부 선한 저항 조직이 아니다. 그래서 더 살아 있어야 한다.

- 카산드라는 연락망의 절대 지도자가 아니라, 가장 오래 듣고 가장 많이 의심받은 중심 청취자에 가깝다.
- 아미키티아는 연락망을 "믿어도 되는 관계"로 정리하려 하지만, 모든 정보가 우정 안으로 들어오지는 않는다.
- 에이도스는 연락망을 자료로 만들려 하지만, 사람의 위험을 자료로만 다루는 순간 카산드라와 충돌한다.

### 4.4 삽입 장면 씨앗

- 한 중개인이 카산드라의 말을 일부러 더 무섭게 바꿔 팔고 있다. 카산드라는 그를 처벌하려다가, 그가 실제로 잃은 가족 이야기를 듣고 진실과 공포 장사의 경계를 고민한다.
- 아미키티아가 연락망의 한 아이에게 "이걸 누가 시켰니?"라고 묻자, 아이는 "아무도요. 그래서 더 무서웠어요"라고 답한다.
- 에이도스가 완벽한 순환 오류 지도를 만들었을 때, 카산드라는 지도 밖에 남은 작은 마을 하나를 가리킨다. "저기는 아직 아무도 묻지 않았어요."

### 5.4 삽입 장면 씨앗

- 에이도스가 순환 오류를 관측하는 순간, 별 하나가 하늘에서 떨어지지 않고 위치만 바꾼다. 카산드라는 그것을 "별이 아니라 매듭"이라고 부른다.
- 순환 수호자는 일행을 공격하기 전 정중하게 길을 막는다. 그들은 악의를 품은 병사가 아니라, 오래된 장치가 자기 몸으로 만든 경비다.

| 심장로 과열 | 중앙에서 모든 흐름을 흡수하려다 스스로 버티지 못한다 | 마이더스 체계와 셀로 왕관 욕망의 닮은꼴 |
| 황금 외피 박락 | 시설 외벽의 금이 벗겨지고 낡은 지지대가 드러난다 | 금빛 도시의 그늘이 역사 사건으로 굳어짐 |
| 침묵 이후의 나눔 | 불씨 주민들이 잔해 속에서 가격표 없는 물과 빵을 나눈다 | 5부 루멘 전조와 작은 회복의 시작 |

### 6.3 장면 사용 원칙

- 중앙 가치 변환소는 나쁜 시설로만 보이면 약하다. 초반에는 실제로 혼란을 줄이고 사람들을 안심시킨 적이 있어야 한다.
- 붕괴는 페이단이나 셀로가 직접 부숴서 생긴 결과가 아니라, 누적된 모순이 버티지 못해 안에서부터 터지는 사건이어야 한다.
- 왕좌의 파동은 외부 충격으로 작용할 수 있지만, 결정타는 내부 부채와 회색 숨이다.
- 마이더스의 표정에는 당황만이 아니라 슬픔이 있어야 한다. 그는 자기 방식으로 만든 안정이 정말로 사람을 지켜준다고 믿었다.
- 루멘이 이후 이 잔해를 비출 때, 빛은 "거짓이니 사라져라"가 아니라 "무엇이 진짜로 남았는가"를 드러내야 한다.

### 6.4 삽입 장면 씨앗

- 붕괴 직전, 한 늙은 노동자가 시설 벽에 손을 대자 자기 이름이 처음으로 황금빛이 아니라 낡은 잉크색으로 떠오른다. 그는 울지 않고 손을 뗀다.
- 마이더스가 마지막으로 붙잡는 것은 가장 큰 가치 결정체가 아니라, 황금이 되지 않은 낡은 동전이다.
- 불씨 주민들은 변환소 잔해에서 나온 금을 줍지 않고, 먼저 갇힌 사람들의 손목에서 가치 부채 사슬을 풀기 시작한다.
