# 알디레아 정사 승격 1차

정사 등급: 작업 기록

역할: 루트 초기 자료의 `알디레아` 씨앗을 히프노레아 브리멜론과 프론티아 아르세니아의 귀환 인물로 승격하고, "사실만 붙드는 태도"를 읽을 속도를 지키는 정직함과 Mnemosyn 귀환 축에 연결한다.

---

## 1. 승격 요약

알디레아는 기존 자료에서 브리멜론의 깨진 기쁨 탑과 아르세니아의 후반 대립 씨앗으로 남아 있었다. 이번 승격에서는 그를 단순한 폭로자가 아니라, 좋은 제목 아래 남은 차가운 사실이 다시 묻히지 않게 붙드는 인물로 정리했다.

핵심 이동은 아래와 같다.

| 구분 | 승격 전 | 승격 후 |
|---|---|---|
| 인물 기능 | 사실만 붙드는 귀환 씨앗 | 브리멜론의 제목 아래 온도와 아르세니아의 고칠 수 있는 답을 잇는 귀환 인물 |
| 핵심 질문 | 사실을 붙드는 태도는 어떻게 바뀌는가 | 사실을 흐리지 않으면서도, 그 사실을 읽을 손의 온도와 속도를 지킬 수 있는가 |
| 주 무대 | 깨진 기쁨 탑 후보 | 깨진 기쁨 탑, 반대온도 열람실, 아르세니아 첫줄 개정대 |
| 주요 물성 | 반대온도 결정, 제목갈이 칼 | 속도등잔, 밑온도 장갑, 빈제목 표찰, 첫줄 지우개 |
| 결투 기능 | 미정 | 불편한 사실과 고칠 수 있는 답의 속도 재배치 |

## 2. 124 기준 반영

| 기준 | 알디레아 반영 |
|---|---|
| A가 버린 가치관 | 좋은 제목 아래 남은 차가운 사실을 마주할 수 있다는 믿음 |
| 왜 버릴 수밖에 없었는가 | 차가운 사실을 바로 마주하면 관계와 하루가 무너질 것 같았고, 견딜 만한 제목이 당장의 통과로 필요했다 |
| 대체 가치관 | 따뜻한 제목과 정리된 답을 붙여 장면을 지나가게 하는 습관 |
| 처음의 보호 기능 | 너무 차가운 기록을 바로 만지지 않아도 되게 하고, 하루를 계속 살 수 있게 했다 |
| 시간이 지나며 생긴 왜곡 | 따뜻한 제목이 봉인이 되고, 고친 답이 아직 만지지 못한 사실을 덮는다 |
| 귀환 계기 | 깨진 기쁨 탑의 반대온도 결정과 아르세니아 첫줄 개정대가 충돌하며, 사실도 답도 상대의 속도를 배워야 하는 결투 |
| 필연성 | 제목만으로는 차가운 사실이 계속 묻히고, 사실만으로는 읽는 손이 얼어붙기 때문에 속도를 갖춘 정직함이 필요해진다 |
| 귀환 후 후폭풍 | 브리멜론은 좋은 제목을 바로 믿지 않고 빈제목 표찰을 남기며, 아르세니아는 답을 고치기 전 제목 아래 온도를 먼저 확인한다 |

## 3. 녹티오라 설계

| 낮의 요소 | 왜 남았는가 | 밤의 형태 | 선택 시험 |
|---|---|---|---|
| 깨진 기쁨 탑 | 좋은 날이라는 제목 아래 차가운 기록이 오래 남았기 때문이다 | "좋았다"는 말이 반복될수록 금이 밝아지고, 빈제목 표찰을 놓으면 빛이 낮아진다 | 좋은 제목을 지킬 것인가, 아래 온도를 볼 것인가 |
| 반대온도 결정 | 말한 감정과 손끝 온도의 차이가 알디레아에게 가장 먼저 남았기 때문이다 | 억지로 쥐이면 검게 탁해지고, 손난로 곁에서는 서로 다른 빛을 천천히 낸다 | 바로 확인시킬 것인가, 손이 돌아올 때까지 기다릴 것인가 |
| 제목갈이 칼 | 오래 붙은 제목을 떼어 내야 다시 물을 수 있었기 때문이다 | 서두르면 아래 기록까지 긁고, 기다리면 제목표만 얇게 벗긴다 | 폭로할 것인가, 기록을 다치지 않게 열 것인가 |
| 속도등잔 | 사실을 숨기지 않으면서도 읽는 사람의 숨을 기다릴 방법이 필요했기 때문이다 | 조급해지면 유리가 금 가고, 호흡을 맞추면 불꽃이 한 칸씩 낮아진다 | 전부 밝힐 것인가, 읽을 속도를 남길 것인가 |
| 빈제목 표찰 | 새 이름을 바로 붙이는 일도 또 다른 봉인이 될 수 있기 때문이다 | 방치하면 남의 글씨가 스며들고, 주인이 돌아오면 가장자리부터 따뜻해진다 | 이름을 대신 붙일 것인가, 빈칸을 지킬 것인가 |

## 5. 결투 설계 메모

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 불편한 사실을 흐리지 않는 정직함 vs 사람이 감당할 수 있는 속도와 고칠 수 있는 답 |
| 두 가치의 옳음 | 사실은 좋은 제목 아래 묻힌 차가운 기록을 보게 하고, 고칠 수 있는 답은 본 뒤 무엇을 다시 쓸지 책임을 남긴다 |
| 두 가치의 위험 | 사실은 속도를 잃으면 강요가 되고, 답은 조급하면 아직 만지지 못한 사실을 다시 덮는다 |
| 끝난 뒤 재배치 | 제목은 바로 고쳐 붙이지 않고 빈제목 표찰로 남으며, 답은 제목 아래 온도를 확인한 뒤 첫 줄부터 다시 쓰인다 |
| 도시 변화 | 브리멜론의 깨진 기쁨 탑에는 속도등잔이 걸리고, 아르세니아 첫줄 개정대에는 빈제목 표찰을 놓는 낮은 홈이 생긴다 |
| 인물 변화 | 알디레아는 사실을 말하기 전 밑온도 장갑을 건네고, 엘리올은 답을 고치기 전 반대온도 결정을 함께 본다 |
