# 3부 상위 누락카드 상세초안

이 문서는 `27_3부_통합요소_추출과_누락카드.md`에서 상위 우선도로 분류한 3부 요소를 실제 정사 카드로 승격하기 위한 상세 초안이다. 현재 단계는 실제 연재분 집필이 아니라 세계 성경 편찬 단계이므로, 각 항목은 향후 `02`, `03`, `04`, `07`, `10`, `11`, `12`, `15`, 도시별설정, 인물 개별 문서로 분산 반영한다.

3부 상위 카드의 핵심 원칙은 "누군가를 빨리 살리고 싶은 마음도, 누군가를 천천히 돌보고 싶은 마음도 모두 사랑에서 출발할 수 있다"는 것이다. 따라서 넥서스 후원자들, 고요의 정원지기들, 오블라투스, 팔수스와 맞서는 다산의 일족은 모두 각자의 선물과 상처를 동시에 가져야 한다.

---

## 0. 상위 카드 요약

| 카드 | 분류 | 핵심 가치 | 왜곡 위험 | 우선 반영 |
|---|---|---|---|---|
| 넥서스 프로젝트 | 역사 사건/기반 시설 | 먼 고통을 빠르게 연결해 살리려는 실행력 | 흐름과 성과가 생명의 깊이를 대체함 | `01`, `07`, `16`, `24` |
| 영혼 침식증 | 질병/상태 | 고통을 느끼지 않게 해주는 임시 보호 | 의지와 감정 깊이까지 닳아 없어짐 | `04`, `11`, `12`, `16` |
| 넥서스 후원자들 | 집단/생활 인물군 | 당장 죽어가는 아이를 살리려는 절박함 | 생태와 타인의 고통을 자원으로 취급함 | `10`, `02`, 리테노아 문서 |
| 고요의 정원지기들 | 집단/생활 인물군 | 말 없는 생명도 돌봄의 주체라는 믿음 | 느린 보존이 사람의 긴급함을 외면함 | `10`, `11`, 리테노아 문서 |
| 마지막 잔 선술집 | 장소/공동체 기억 | 이름 없는 노동도 하루의 끝을 가져야 함 | 위로가 체념의 의식으로 굳음 | 도시별설정, `02` |
| 그라투스 | 인물/원형 이름 | 모두의 손이 세계를 만든다는 장인의 믿음 | 인정받지 못한 수고가 원한이 됨 | 오블라투스/카우투스 문서, `09`, `10` |
| 희생의 역류 | 능력/현상 | 묻힌 수고를 몸으로 느끼게 함 | 감사와 책임을 복수의 부채로 바꿈 | `07`, 오블라투스 문서 |
| 영혼의 맹약 | 능력/관계 의식 | 서로의 존재를 동의 아래 묶는 신뢰 | 우정을 힘의 네트워크로만 소비함 | `07`, 아미키티아 문서 |
| 다산의 일족 성역 입구 | 장소/전장 | 사라진 생명력의 문턱을 지킴 | 보호가 세계와의 단절로 변함 | 도시별설정, `02` |
| 생명의 씨앗 | 유기물/능력 매개 | 냉소 속에서도 다시 자라는 가능성 | 희망을 강요하는 도구가 됨 | `11`, `12`, 레기나 문서 |

## 1. 넥서스 프로젝트

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 역사 사건/기반 시설/국가 간 고속 연결망 |
| 소속 | 프론티아 실행 체계, 눅세린 후원 자본, 센시르 치료 수요의 결합 |
| 낮의 형태 | 세르 흐름을 빠르게 정제해 병자, 물자, 정보, 치료 요청을 각국으로 보내는 빛의 고속로 |
| 밤의 변형 | 황금빛 선로가 혈관처럼 부풀고, 지나간 생명마다 얇은 통행세 자국을 남긴다 |
| 선물 | 멀리 떨어진 고통을 빠르게 발견하고, 도움을 실제로 도착하게 만든다 |
| 상처 | 느린 절차 때문에 이미 잃은 사람들이 있었고, 그 상실이 "이번에는 늦지 않겠다"는 강박이 되었다 |
| 거짓 믿음 | 더 빠르게 연결하면 더 깊게 살릴 수 있다 |
| 관련 인물 | 프락시스, 마이더스, 카시안, 셀로, 넥서스 후원자들, 고요의 정원지기들 |
| 반작용 | 영혼 침식증, 고요의 이끼 갈등, 자기 왕관의 영역 |
| 장면 기능 | 2부의 성공 논리가 3부에서 대가로 돌아오게 하는 역사적 다리 |

작동 구조:

| 단계 | 내용 | 장면 기능 |
|---|---|---|
| 설계 | 프론티아가 병목을 계산하고, 눅세린이 비용과 홍보를 붙인다 | 선의와 이익이 처음부터 섞여 있음을 보여준다 |
| 개통 | 멀리 있던 치료와 물자가 빠르게 도착한다 | 독자가 처음에는 프로젝트의 선물을 인정하게 한다 |
| 확장 | 후원자들은 성공 사례를 근거로 더 많은 흐름을 요구한다 | "살릴 수 있다면 왜 멈추는가"라는 강한 질문 |
| 감당 너머 정제 | 세르가 너무 빨리 정제되어 어린 존재의 안쪽 울림이 얇아진다 | 영혼 침식증의 원인 |
| 사후 정산 | 센시르는 느린 조율과 책임 절차를 요구한다 | 카시안과 셀로의 충돌 |

독자가 붙잡을 디테일:

- 넥서스 선로에는 구조된 아이들의 이름패가 걸려 있다. 프로젝트가 실제로 누군가를 살렸기 때문이다.
- 선로를 오래 만지면 따뜻하지만, 손을 떼면 손바닥에 얇은 금가루 같은 피로가 남는다.
- 개통식 때 사용한 첫 리본은 아직 리테노아 보관실에 있다. 카시안은 그것을 버리지 못한다.

사용 원칙:

- 넥서스 프로젝트는 악한 실험이 아니다. 성공이 있었고, 그 성공 때문에 멈추기 어려워진다.
- 프락시스의 실행력과 마이더스의 후원이 모두 필요했지만, 둘 중 하나에게만 죄를 몰지 않는다.
- 프로젝트의 대가는 "속도" 자체가 아니라, 속도를 깊이와 같다고 믿은 데서 온다.

모순 주의:

- 2부의 프락시스/마이더스 사건과 별개 사건으로 띄우지 않는다. 2부 결과주의가 3부의 역사적 후폭풍으로 이어져야 한다.
- `세르 전송관`, `세르 공급 시설`과 겹치지 않는다. 넥서스 프로젝트는 단일 장치가 아니라 여러 기반 시설과 후원 체계가 엮인 사건군이다.

## 2. 영혼 침식증

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 질병/상태/세르 흐름 후유증 |
| 소속 | 넥서스 프로젝트 후폭풍, 센시르 리테노아 치료 갈등 |
| 낮의 형태 | 아이들이 더 이상 아파하지 않는 것처럼 보인다. 울음이 줄고, 요구가 사라지며, 주변은 잠시 평온해진다 |
| 밤의 변형 | 숨이 회색으로 옅어지고, 눈동자 안쪽의 작은 불씨가 뒤로 물러난다 |
| 선물 | 너무 큰 고통으로부터 아이를 잠시 보호한다 |
| 상처 | 보호가 길어질수록 고통뿐 아니라 바람, 반항, 상상까지 함께 닳는다 |
| 거짓 믿음 | 아파하지 않는 아이는 나아진 아이이다 |
| 관련 인물 | 넥서스 후원자들, 고요의 정원지기들, 카시안, 셀로, 레기나 |
| 반작용 | 고요의 이끼, 마지막 희망의 씨앗, 생명의 씨앗 |
| 장면 기능 | "고통 제거"와 "생명 회복"이 다르다는 3부 질문을 세운다 |

증상 단계:

| 단계 | 외형 | 내면 기능 | 장면 표현 |
|---|---|---|---|
| 회색 숨 | 말끝이 옅고 숨결이 차갑다 | 고통 반응이 줄어든다 | 부모는 처음에 안도한다 |
| 빈 선택 | 무엇을 먹고 싶은지, 어디로 가고 싶은지 대답하지 못한다 | 작은 욕구가 사라진다 | 아이가 "괜찮아요"만 반복한다 |
| 돌눈 | 눈동자의 반짝임이 뒤로 밀린다 | 호기심과 반항이 약해진다 | 장난감보다 규칙표를 오래 본다 |
| 잔향 소실 | 이름을 불러도 늦게 반응한다 | 자기 이야기의 주인이 흐려진다 | 반향의 샘에 비친 모습이 늦게 따라온다 |
| 껍질화 전조 | 몸은 살아 있으나 울림이 거의 없다 | 침묵의 껍질로 이어질 위험 | 셀로가 성급한 판결을 내리는 계기 |

회복 조건:

| 조건 | 의미 | 사용 장면 |
|---|---|---|
| 작은 선택 | 아이가 자기 욕구를 다시 말한다 | 음식, 색, 길, 놀이 같은 사소한 선택부터 시작 |
| 느린 이름 부르기 | 이름을 효율적으로 호출하지 않고 기다린다 | 카시안/정원지기 장면 |
| 고통의 일부 반환 | 고통을 완전히 지우지 않고 견딜 만큼만 돌려준다 | 고요의 이끼 장면 |
| 가능성 발아 | 아이 안에 아직 남은 바람을 씨앗처럼 키운다 | 레기나/생명의 씨앗 장면 |

사용 원칙:

- 영혼 침식증은 "감정이 없는 악한 아이"를 만드는 질병이 아니다. 너무 큰 고통에서 살아남기 위해 감정과 바람이 뒤로 물러난 상태다.
- 회복은 강한 감동이나 연설 한 번으로 끝나지 않는다. 작은 선택을 되찾는 장면을 반복해야 한다.
- 부모가 아이의 평온을 회복으로 오해하는 장면을 넣으면 넥서스 후원자들의 공포가 더 설득력 있어진다.

모순 주의:

- `침묵의 껍질`과 합치지 않는다. 영혼 침식증은 넥서스 후폭풍 질병이고, 침묵의 껍질은 셀로의 심판 실패 뒤 굳은 피해 상태다.
- `무력장`과도 다르다. 무력장은 행동 연결을 늦추는 전장 능력이고, 영혼 침식증은 존재 안쪽의 선택과 울림이 닳는 상태다.

## 3. 넥서스 후원자들

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 집단/생활 인물군/부모와 후원자 연합 |
| 소속 | 넥서스 프로젝트 수혜 도시, 리테노아 치료 요구 시위대 |
| 낮의 형태 | 깔끔한 후원 배지, 아이 이름이 적힌 팔찌, 회계 장부와 치료 요청서를 들고 움직인다 |
| 밤의 변형 | 배지의 금빛이 아이의 심장 박동처럼 깜박이고, 장부의 숫자가 작은 울음소리로 변한다 |
| 선물 | 포기하지 않고 자원을 모아 실제 생명을 살린다 |
| 상처 | 잃은 아이, 늦게 도착한 치료, "조금만 빨랐더라면"이라는 기억 |
| 거짓 믿음 | 살릴 수 있는 방법이 있다면 어떤 생태도, 어떤 절차도 뒤로 밀릴 수 있다 |
| 관련 인물 | 카시안, 셀로, 고요의 정원지기들, 넥서스 프로젝트 관련자 |
| 반작용 | 고요의 이끼, 대공명실, 균형 회복 의식 |
| 장면 기능 | 독자가 반대편의 절박함을 사랑하게 만든다 |

대표 인물 후보:

| 이름 | 역할 | 독자가 붙잡을 이유 | 위험 |
|---|---|---|---|
| 에리안 베일 | 후원회 대변인 | 회색 숨을 앓는 딸의 이름을 말할 때 목소리가 무너진다 | 공적 연설에서 타인의 생태를 숫자로 줄인다 |
| 노라 셈 | 넥서스 전직 선로 기사 | 프로젝트가 실제로 사람을 살린 순간들을 기억한다 | 선로의 대가를 기술 문제로만 보려 한다 |
| 하룬 다 | 아이를 이미 잃은 후원자 | 더는 아무도 늦어서 잃지 않기를 바란다 | 분노가 고요의 이끼를 약탈해도 된다는 주장으로 변한다 |

갈등 장면 규칙:

| 장면 | 후원자들의 정당성 | 후원자들의 위험 |
|---|---|---|
| 대공명실 공청 | 아이가 지금 죽어가고 있다는 사실을 들고 온다 | "지금"이라는 말로 모든 느린 생명을 지운다 |
| 고요의 정원 대치 | 이끼가 치료 가능성을 갖고 있다는 증거를 제시한다 | 이끼가 쌓아 온 고통과 생명을 자원으로 본다 |
| 셀로의 판결 앞 | 셀로의 빠른 답을 처음에는 환영한다 | 빠른 답이 아이의 울림을 더 얇게 만들 수 있음을 보지 못한다 |

사용 원칙:

- 넥서스 후원자들은 탐욕스러운 귀족 집단으로 단순화하지 않는다. 그들은 실제로 누군가의 부모, 보호자, 생존자이다.
- 그들이 틀리는 순간에도 독자는 "그럴 수밖에 없었다"고 느껴야 한다.
- 후원자들의 회복은 패배가 아니라, 아이를 살리기 위해 생태와 절차도 함께 살려야 한다는 인식으로 와야 한다.

## 4. 고요의 정원지기들

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 집단/생활 인물군/치유 생태 돌봄 공동체 |
| 소속 | 센시르 리테노아, 대공명실 주변 정원 |
| 낮의 형태 | 물기 어린 장갑, 낮은 종, 이끼의 숨을 듣는 얇은 청진판을 지닌 조용한 돌봄꾼들 |
| 밤의 변형 | 그들의 발자국 주변에 말하지 못한 기억 방울이 피고, 거짓말을 하면 방울이 잿빛으로 꺼진다 |
| 선물 | 말 없는 생명도 돌봄의 주체로 인정한다 |
| 상처 | 과거 치유 생태를 자원처럼 뜯어 쓴 결과 너무 많은 침묵이 남았다 |
| 거짓 믿음 | 충분히 기다리면 모든 생명은 스스로 견딜 길을 찾는다 |
| 관련 인물 | 카시안, 고요의 이끼, 넥서스 후원자들, 셀로 |

대표 인물 후보:

|---|---|---|---|

| 세트 린 | 젊은 정원지기 | 일부 이끼를 나누어 써야 한다고 생각한다 | 후원자와 정원 사이에서 양쪽 모두에게 배신자로 보인다 |
| 라움 키르 | 기록 담당 | 뜯긴 이끼 한 조각마다 어디서 왔는지 기록한다 | 기록이 돌봄을 대신할 수 있다고 착각한다 |

| 규칙 | 의미 | 장면 기능 |
|---|---|---|

- 세트 린 같은 내부 균열 인물을 두면 정원지기들이 한 목소리로만 보이지 않는다.

## 7. 희생의 역류

| 밤의 변형 | "당연한 것"으로 처리된 수고가 검은 물처럼 거꾸로 흘러나와 주변 유대와 의욕을 부식시킨다 |

| 장면 기능 | EP14 전투를 단순 화력전이 아니라 노동과 인정의 윤리전으로 만든다 |

작동 규칙:

| 규칙 | 작동 방식 | 약점 |
|---|---|---|
| 부채 호출 | 상대가 당연히 누린 편의 뒤의 수고를 끌어낸다 | 진짜 책임을 받아들이면 무게가 줄어든다 |
| 활력 역류 | 주변의 의욕과 유대가 피로로 바뀐다 | 상호 동의한 협력은 쉽게 부식되지 않는다 |
| 이름 소거 | 능력을 크게 쓸수록 오블라투스 자신도 그라투스라는 이름을 잊는다 | 이름을 정확히 불러주는 기억이 반작용이 된다 |
| 감사 왜곡 | 빈말 감사는 더 큰 분노로 변한다 | 구체적 책임과 분담만이 역류를 멈춘다 |

사용 원칙:

- 희생의 역류를 "감사하라고 혼내는 능력"으로 쓰지 않는다. 핵심은 감사보다 책임이다.
- 레마니스는 과거의 이름과 이야기를 되살리고, 소키우스는 현재의 책임 분담을 만든다. 둘 중 하나만으로는 역류를 멈출 수 없다.
- 아미키티아의 영혼의 맹약은 역류를 부정하지 않고, 그 무게를 동의한 이들이 함께 나누는 방식으로 작동해야 한다.

| 기억 | 과거의 이름과 상처를 지우지 않는다 | 힘은 커져도 관계가 빈 껍질이 된다 |
| 현재 행동 | 지금 무엇을 나눌지 실제로 선택해야 한다 | 아름다운 빛만 있고 전투를 바꾸지 못한다 |

| 축 | 인물 | 기능 |

- 셀로의 자기 왕관과도 대비한다. 자기 왕관은 모두를 중심으로 배열하지만, 영혼의 맹약은 중심을 비워 서로의 선택을 남긴다.

| 상처 | 다산의 일족 숙청 이후 바깥 세계를 다시 믿지 못한다 |
| 거짓 믿음 | 문을 닫아두면 남은 생명은 완전히 안전하다 |
| 관련 인물 | 레기나 페쿤디아, 실바누스, 파불라, 로고스, 팔수스, 아미키티아 |
| 반작용 | 생명의 씨앗, 이야기 역장, 이상의 파편, 위선의 신기루 |
| 장면 기능 | 팔수스 전투를 "희망 대 냉소"가 실제 문턱에서 부딪히는 사건으로 만든다 |

공간 구성:

| 구역 | 형태 | 장면 기능 |
|---|---|---|
| 숨결 수정길 | 발밑 수정이 걸음의 속도보다 호흡을 읽는다 | 급한 침입과 살아 있는 요청을 구분 |
| 뿌리 문턱 | 기록림 뿌리가 둥근 문살처럼 얽혀 있다 | 생명과 기록의 결합 |
| 숙청 흉터벽 | 오래전 공격의 흔적이 지워지지 않은 벽 | 아미키티아/레기나의 죄책감과 경계 |
| 심장나무 그림자 | 안쪽에서 희미하게 보이는 거대한 나무의 실루엣 | 성역 전체의 생명축 암시 |

통과 조건:

| 조건 | 의미 | 장면 기능 |
|---|---|---|
| 살아 있는 질문 | 답을 훔치려는 자가 아니라 묻고 변할 수 있는 자여야 한다 | 로고스/파불라와 연결 |
| 상처 인정 | 숙청의 역사를 부정하면 문턱이 닫힌다 | 아미키티아의 속죄축 |
| 생명 박동 | 힘의 크기보다 아직 자라려는 박동을 본다 | 레기나의 가치 기준 |
| 문턱의 허락 | 안쪽 인물이 일방적으로 여는 문이 아니다 | 성역 자체를 생명체처럼 다룬다 |

사용 원칙:

- 성역 입구는 숨겨진 던전 문이 아니라, 세계가 잊은 생명력의 문턱이다.
- 레기나 일행이 문을 닫아둔 것은 비겁함이 아니라 생존의 역사다. 다만 문을 영원히 닫으면 생명은 세계를 다시 믿지 못한다.
- 팔수스는 문턱 앞에서 생명을 조롱해야 하고, 레기나는 그 조롱을 반박하기보다 작은 씨앗 하나를 실제로 자라게 해야 한다.

## 10. 생명의 씨앗

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 유기물/능력 매개/태고 생명 결정 |
| 소속 | 레기나 페쿤디아, 다산의 일족, 살아있는 기록림 |
| 낮의 형태 | 작은 빛의 씨앗. 손바닥에 올리면 받는 이의 체온에 맞춰 색이 바뀐다 |
| 밤의 변형 | 절망이 짙을수록 더 밝게 빛나지 않고, 오히려 땅속으로 숨어 뿌리를 먼저 내린다 |
| 선물 | 냉소, 위선, 허무 속에 남은 아주 작은 가능성을 발아시킨다 |
| 상처 | 씨앗을 심을 때마다 레기나의 생명력도 일부 나뉜다 |
| 거짓 믿음 | 씨앗만 심으면 누구나 다시 희망을 가져야 한다 |
| 관련 인물 | 레기나 페쿤디아, 실바누스, 파불라, 로고스, 팔수스, 셀로 |
| 반작용 | 이상의 파편, 위선의 신기루, 마지막 희망의 씨앗 |
| 장면 기능 | 3부 후반에서 냉소를 논리로 이기는 것이 아니라, 다시 자라는 장면으로 이긴다 |

발아 규칙:

| 규칙 | 작동 방식 | 위험 |
|---|---|---|
| 체온 확인 | 씨앗은 받는 이의 손 온도에 맞춰 깨어난다 | 강제로 심으면 껍질만 남는다 |
| 진실 토양 | 거짓 위로나 과장된 희망 위에서는 자라지 않는다 | 희망 강요를 금지한다 |
| 작은 뿌리 | 처음에는 꽃이 아니라 뿌리부터 내린다 | 독자가 즉각적 기적을 기대하지 않게 한다 |
| 생명 분담 | 레기나의 생명력이 일부 나뉜다 | 무제한 치유 장치가 되지 않는다 |

팔수스 전투 적용:

| 팔수스의 공격 | 생명의 씨앗의 반응 | 의미 |
|---|---|---|
| 이상의 파편 | 파편을 부수지 않고 틈에 뿌리를 낸다 | 깨진 이상도 새 형태로 자랄 수 있다 |
| 위선의 신기루 | 신기루를 폭로하기보다 그 안의 두려움을 싹으로 바꾼다 | 냉소 뒤의 상처를 본다 |
| 모든 희망 조롱 | 씨앗은 밝게 맞서지 않고 조용히 땅속으로 숨는다 | 희망은 떠드는 감정이 아니라 지속되는 생명 |

구분 규칙:

| 항목 | 차이 |
|---|---|
| 생명의 씨앗 | 레기나가 능력으로 만들거나 품은 태고 생명 매개. 냉소와 허무를 새 가능성으로 변환 |
| 마지막 희망의 씨앗 | EP12 고요의 이끼 갈등 이후 남는 회복 단서 후보. 향후 생명의 씨앗의 작은 파편 또는 별도 치유 생태로 확정 필요 |
| 심장나무의 씨앗 | 다산의 일족 숙청 때 킹 페쿤두스가 남긴 근원 유산. 생명의 씨앗 계열의 가장 오래된 원형 |

사용 원칙:

- 생명의 씨앗은 만능 희망 장치가 아니다. 씨앗은 자라는 시간을 요구한다.
- 팔수스의 냉소를 말로 반박하기보다, 독자가 작은 뿌리가 살아남는 장면을 보게 한다.
- 레기나의 사랑스러운 습관인 손 온도 확인을 반드시 살린다. 씨앗은 대상의 현재 상태를 존중할 때만 깨어난다.
