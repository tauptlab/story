# 192 주요 인물 전투 기능 구체화

작성일: 2026-06-24

역할: 고유 특수능력을 가진 주요 인물 문서에 남아 있던 범용 템플릿 문장 `공격, 방어, 제어, 정보 획득 중 하나 이상으로 실제 결투에서 작동한다`를 실제 전투 기능으로 교체한 작업 기록이다.

## 1. 적용 원칙

- 강한 가치관을 대표하는 인물만 고유 특수능력을 가진다.
- 능력명은 작중 기술명으로 유지하고, 의미는 작가용 의미/장면 결과로 분리한다.
- 전투 기능은 공격, 방어, 제어, 정보 획득, 봉인, 지연, 증식, 변환, 반사, 추적 등 실제 장면에서 부딪히는 작동을 먼저 적는다.
- 생활 인물과 약한 가치관 인물은 이 규칙의 대상이 아니며, 예법/손기술/소품/사소한 감응으로만 운용한다.

## 2. 이번에 구체화한 인물

| 인물 | 능력명 | 구체화 방향 |
|---|---|---|
| 도그마 | 클라우수라 | 규약 문장으로 이동/발언/발동 조건을 닫고 예외를 봉인 |
| 나르키소스 | 카토프론 | 반사면과 자기상으로 공격과 시선을 굴절 |
| 레기나 페쿤디아 | 페쿤디아 | 씨앗 매개가 상처 틈에 뿌리내려 냉소/허무 공격을 지연 |
| 레툼 | 노크스 레퀴엠 | 장례 박자로 추격, 재생, 발자국을 낮춤 |
| 로고스 | 악시오마 | 전제와 균열을 드러내 방어식을 붕괴 |
| 리타스 | 카리스 플루엔스 | 등급화된 힘을 무상 흐름으로 전환 |
| 멘다키우스 | 벨라멘 | 감각/기록/궤적에 허위 베일을 겹침 |
| 마이더스 | 아우룸 임페라 | 욕망과 관계를 아우룸 결정체 무장/조건으로 전환 |
| 므네몬 | 팔림프세스트 | 지워진 장면의 층을 겹쳐 숨은 경로를 드러냄 |
| 사일러스 | 피데스 시그닛 | 약속선과 보증 조건을 잠그고 반동을 되돌림 |
| 살루스 | 소포르 | 과열 감각과 공격 충동을 수면막으로 낮춤 |
| 세리엘 | 스피나 피데이 | 피데스 가시 방어장으로 접근과 맹세를 시험 |
| 소포스 | 텔로스 거울날 | 목적과 다른 움직임을 절단/반사 |
| 실바누스 | 실바 라디쿠스 | 뿌리와 문장 가지로 방벽/속박/재생 지형 형성 |
| 아마라 | 헤스티아 결속 | 흩어진 온기와 방어를 공동체 방벽으로 묶음 |
| 오블라투스 | 폰두스 무명 | 잊힌 수고의 무게를 타격과 족쇄로 되돌림 |
| 이놉스 | 니힐룸 | 무력장으로 이유와 의지를 가볍게 만들고 방향을 지움 |
| 조일루스 | 아쿠스 논박 | 불일치점과 틀린 전제를 찔러 궤적을 꺾음 |
| 카산드라 | 시빌라 노드 | 숨은 연결선과 침묵의 균열을 정보 경로로 묶음 |
| 카우투스 | 알레아 프로스펙트 | 위험 분기를 펼쳐 덜 파국적인 방어선을 선택 |
| 쿠스토스 세넥스 | 크로노 스타시스 | 붕괴 시간을 늦춰 보존장과 방어막 형성 |
| 파불라 | 텔라 파불라 | 사건의 도착 순서와 증언 노출 순서를 엮음 |
| 팔수스 | 아포리아 | 확신 안의 모순을 신기루로 되비춰 방어를 갈라냄 |
| 페이단 | 푸리아 루브라 | 묻힌 분노를 붉은 열과 균열로 폭발 |
| 포르티스 | 코르 이그니스 | 중심불로 무력장/공포장을 버티고 핵만 태움 |
| 포에두스 | 솔레키스무스 | 발동 조건 사이에 어긋난 매듭을 삽입 |
| 프레펙투스 | 에딕툼 격자 | 전장을 규칙 칸으로 나누어 비허가 행동을 지연/봉인 |
| 한신 | 십면매복 | 한 공격을 여러 갈래 병력/궤적으로 증식 |
