# 켈 정사 승격 1차

작성일: 2026-06-24

역할: 루트 초기 자료의 `켈` 씨앗을 길을 열고 힘을 빌려 주는 반김대여 나루의 인물로 정리하고, 빌린 힘이 사람을 살리는 선의와 빌린 판단이 자기 손의 무게를 대신하는 위험을 함께 품은 접경 생활 인물로 승격한다.

---

## 1. 승격 요약

켈은 돈이나 물건의 욕망을 대표하는 인물이 아니다. 그는 준비되지 못한 사람을 버리지 않는 후견자이며, 동시에 빌린 힘이 오래되면 자기 손의 무게를 대신한다는 위험을 품은 사람이다. 이번 승격에서는 `반김보석`, `빌린손 반지`, `첫무게 저울`, `대여칼집`, `돌려잡는 손잡이`, `후견끈`, `첫무게풀`을 통해 도움과 성장, 도구와 자기 손, 후견과 조종의 경계를 분리했다.

| 항목 | 내용 |
|---|---|
| 정사명 | 켈 |
| 별칭 | 켈 아이 노스, 반기는자 |
| 소속 | 특수공간-호수지역, 반김대여 나루, 돌려잡는 선반, 첫무게 저울대 |
| 정사 등급 | B급 상위 접경 생활 인물 |
| 핵심 가치 | 빌린 힘은 사람을 살릴 수 있지만, 마지막 손잡이는 자기 손으로 돌려잡아야 한다 |
| 주요 물성 | 반김보석, 빌린손 반지, 첫무게 저울, 대여칼집, 돌려잡는 손잡이, 후견끈, 첫무게풀 |
| 밤 변형 | 반짝이는 선반과 빼기 두려운 반지가, 빌린 힘을 자기 이유로 돌려잡을 수 있는지 묻는다 |

## 2. 상위 기준 반영

| 기준 | 켈 반영 |
|---|---|
| A가 버린 가치관 | 도움과 도구를 빌리는 일은 부끄러운 일이 아니며, 빌린 힘은 언젠가 자기 손의 무게로 돌려잡을 수 있다는 믿음 |
| 버린 이유 | 아무것도 갖지 못한 상태로 버티기에는 위험이 너무 가까웠고, 먼저 주어진 힘과 답이 당장을 살렸기 때문이다 |
| 대체 가치관 | 더 강한 도구, 더 빠른 후견, 먼저 주어진 답이 안전을 만든다는 습관 |
| 처음의 보호 | 빌린 힘은 실제로 A를 넘어지지 않게 했고, 준비되지 않은 날을 건너게 했다 |
| 왜곡 | 시간이 지나 도구는 강해졌지만 손은 자기 떨림을 잊었고, 빌린 판단은 자기 이유처럼 굳었다 |
| 다시 받아들여야 하는 가치 | 도움을 받아 살아남되, 마지막 손잡이는 자기 쪽으로 돌려잡고, 빌린 판단을 자기 이유와 구분하는 예법 |
| 녹티오라 흔적 | 켈의 얼굴을 비추는 반김보석, 빼기 두려운 빌린손 반지, 한쪽으로 굳는 첫무게 저울, 열리지 않는 대여칼집 |
| 귀환 계기 | 아흐의 빈자리, 나리안의 도움끈, 미라솔의 빈보증 표, 네르완의 무게돌림 끈, 리버의 열린 길, 포르티스의 오래 남는 힘이 켈 앞에서 빌림과 성장의 경계를 묻는다 |

## 3. 밤 변형 설계

| 요소 | 왜 남았는가 | 밤의 변형 | 선택 시험 |
|---|---|---|---|
| 반김보석 | 처음 누군가의 힘을 빌려 갑자기 길이 보였던 안도와 불안이 함께 남았다 | 착용자가 자기 이유를 말하지 않으면 보석 안에 켈의 대답만 비친다 | 안내인가, 대리 답인가 |
| 빌린손 반지 | 떨리는 손이 잠시 안정되어 살아남은 기억이 강하게 남았다 | 오래 끼면 손가락이 자기 떨림을 잊고, 빼는 순간 모든 물건이 무거워진다 | 받침인가, 손의 망각인가 |
| 첫무게 저울 | 어디까지 스스로 들고 어디서부터 빌릴지 몰랐던 날이 남았다 | 전부 빌리려 하면 바늘이 켈 쪽으로 굳고, 전부 혼자 들려 하면 저울판이 닫힌다 | 분담인가, 포기인가 |
| 대여칼집 | 빌린 형과 빌린 힘을 자기 이름으로 삼고 싶었던 유혹이 남았다 | 빌린 것을 자기 이름이라 부르면 열리지 않고, 제자리로 돌려놓으면 손목이 가벼워진다 | 기록인가, 소유인가 |
| 돌려잡는 손잡이 | 마지막 선택을 누가 쥐었는지 뒤늦게 알게 된 감각이 남았다 | 자기 이유를 말하면 사용자 쪽으로 돌아오고, 남의 이유를 외우면 차갑게 고정된다 | 선택인가, 위임인가 |
| 후견끈 | 길을 잃지 않으려 붙잡았던 끈이 때로 손목을 당긴 기억이 남았다 | 느슨하면 길표가 되고, 대답까지 정하려 하면 줄이 손목을 끌어당긴다 | 안내인가, 조종인가 |
| 첫무게풀 | 빌린 것을 돌려준 뒤에야 손의 무게가 돌아온 작은 안도가 남았다 | 자기 손으로 돌려잡은 뒤 반납한 자리에서만 잎끝이 선다 | 성장인가, 반납 흉내인가 |

## 5. 결투 기능

켈 결투는 도움을 받을 것인가 거절할 것인가의 문제가 아니다. 준비되지 않은 사람에게 도구와 후견은 실제로 생존의 다리가 된다. 문제는 그 다리가 영원한 길이 되어, 걷는 사람이 자기 발의 무게를 잊는 순간이다.

| 충돌 | 설명 |
|---|---|
| 빌린 힘의 구원 | 준비되지 않은 사람도 당장 살아야 하며, 도구와 후견은 무력한 시간을 건너게 해 준다 |
| 자기 손의 무게 | 마지막 선택은 자기 손으로 들어야 하며, 빌린 판단은 자기 이유를 대신할 수 없다 |

결투 뒤에는 켈의 선반이 사라지지 않는다. 다만 모든 물건에 반납선이 생기고, 돌려잡는 손잡이는 사용자 스스로 이유를 말하기 전까지 켈 쪽으로 움직이지 않는다. 반김보석은 더 약하게 빛나지만 더 오래 따뜻하고, 빌린손 반지는 빼는 순간 손이 무너지지 않도록 손바닥에 낮은 온기를 남긴다.

## 6. 다음 후보

켈은 `181` 기준으로 1차 승격했다. 메리와 지그몬드는 `182` 기준으로 독립 관계 신화 승격을 완료했고, 루지에로는 `183` 기준으로 독립 전승 인물 승격을 완료했다. 다음 정사 승격 후보는 `미니`가 적절하다. 미니는 루지에로가 흔들릴 때 곁에 남던 옛 친구 씨앗으로, 우정이 돌봄인지 붙잡기인지 정리할 수 있다.
