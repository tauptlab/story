# 222 세렌델 생활 물성 작중명 리디자인 5차

작업일: 2026-06-25

## 목적

세렌델 생활 소품 중 기능이 선명하지만 이름이 너무 직접적으로 해설되는 항목을 작중 지역 물성처럼 들리게 조정한다. 기능, 장면 역할, 녹티오라 시험 방식은 보존한다.

## 대응표

| 기존명 | 새 이름 | 분류 | 비고 |
|---|---|---|---|
| `답례표` | `그라티아 패` | 생활 표식/소품 | 호의와 청구의 차이를 남기는 얇은 표식 기능 유지 |
| `답례 차양거리` | `그라티아 차양거리` | 도시 구역 | 작은 호의와 기대가 같은 그늘에 놓이는 시장길 기능 유지 |
| `답례나방` | `그라티아나방` | 유기물/녹티오라 흔적 | 선물 안쪽의 작은 원망에 내려앉는 기능 유지 |

| `미리단추 장터` | `프라에비아 단추장터` | 도시 구역 | 단추가 기억인지 갈고리인지 분류하는 장터 기능 유지 |
| `빈손 접시` | `바쿠아 접시` | 생활 소품 | 돌려주지 못한 사람의 자리를 남기는 기능 유지 |
