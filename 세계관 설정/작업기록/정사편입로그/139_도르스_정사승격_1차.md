# 도르스 정사 승격 1차

역할: 루트 초기 자료의 `도르스` 씨앗을 센시르 드레오렌의 귀환 인물로 승격하고, "충돌을 피하다 회피만 남은 인물"을 돌아올 길표를 남기는 우회와 Mnemosyn 귀환 축에 연결한다.

---

## 1. 승격 요약

도르스는 기존 자료에서 드레오렌 옆문거리의 수선공으로 배치되어 있었다. 이번 승격에서는 그를 단순히 충돌을 피하는 인물이 아니라, 한때 사람을 지키던 옆문이 돌아올 길표 없이 늘어날 때 어떻게 고립이 되는지 보여 주는 귀환 인물로 정리했다.

| 항목 | 기존 씨앗 | 139 기준 정사화 |
|---|---|---|
| 인물 기능 | 충돌을 피하다 회피만 남은 인물 | 피할 권리와 돌아올 한 문장을 함께 배우는 귀환 인물 |
| 주 무대 | 드레오렌 옆문거리 | 옆문거리, 정면문 마당, 되돌이 길표 보관소, 웃음보자기 빨래터 |
| 핵심 상처 | 출구 없는 대화에서 누군가 굳어 버린 사건 | 문이 많을수록 덜 다친다는 믿음 |
| 핵심 물성 | 옆문열쇠, 정면문 손잡이 | 되돌이 경첩, 한문장 못, 옆문 수선망치, 문턱숨풀 |

---

## 2. 124 기준 적용

| 기준 | 도르스의 처리 |
|---|---|
| 버린 가치관 | 불편한 말 뒤에도 관계가 완전히 끝나지 않을 수 있다는 믿음 |
| 보호하던 대체 가치관 | 농담, 돌림말, 옆문, 비스듬한 자리처럼 충돌을 늦추는 우회 |
| 왜곡된 가치관 | 문은 많아지지만 만나는 방은 줄어들고, 말은 부드러워지지만 도착하지 않는다 |
| 다시 받아들일 가치관 | 피할 권리를 인정하되, 돌아올 길표와 감당 가능한 한 문장을 함께 남기는 예법 |
| 녹티오라 흔적 | 끝없이 늘어나는 옆문, 차가운 정면문 손잡이, 문턱을 막는 문턱숨풀 |
| 귀환 계기 | 도르스가 또 하나의 옆문을 만들려는 순간, 돌아올 길표 없는 보호가 고립이 되었음을 드러내는 결투 |
| 귀환 후 후폭풍 | 드레오렌은 옆문을 금지하지 않지만, 모든 옆문에 되돌이 경첩과 길표를 요구한다 |

## 3. 녹티오라 설계

| 요소 | 왜 강하게 남았는가 | 밤 변형 | 선택 시험 |
|---|---|---|---|
| 되돌이 경첩 | 피한 뒤에도 돌아올 수 있는 문이 필요했기 때문이다 | 제대로 단 경첩은 정면문 마당의 돌빛을 비추고, 책임 없이 단 경첩은 제자리에서만 돈다 | 출구인가, 미로인가 |
| 한문장 못 | 큰 고백은 감당하기 어렵지만 작은 직접 말은 필요했기 때문이다 | 작은 문장에는 따뜻해지고, 몰아붙이는 긴 문장에는 휘어진다 | 직접 말인가, 강요인가 |
| 옆문 수선망치 | 새 문을 만드는 손과 낡은 손잡이를 고치는 손이 같은 손이기 때문이다 | 두려움으로 들면 손잡이가 길어지고, 돌아올 사람을 생각하면 낮게 울린다 | 새 문인가, 돌아올 손잡이인가 |
| 문턱숨풀 | 돌아올 길표 없는 옆문이 문턱을 막는 감각으로 남았기 때문이다 | 새 옆문만 만들면 발목 높이로 자라고, 한 문장이 놓이면 양쪽으로 갈라진다 | 보호인가, 고립인가 |

## 5. 결투 설계 메모

| 항목 | 내용 |
|---|---|
| 충돌 가치 1 | 다치지 않기 위해 피할 권리 |
| 충돌 가치 2 | 관계 안에 남기 위해 돌아오는 한 문장 |
| 우회의 선물 | 대화가 폭발하기 전 사람에게 숨 쉴 출구를 준다 |
| 우회의 위험 | 돌아오는 길표가 없으면 보호가 고립이 되고, 옆문이 미로가 된다 |
| 직접 말의 선물 | 관계 안에 다시 도착할 작은 시작을 만든다 |
| 직접 말의 위험 | 큰 고백을 강요하면 정면문은 다시 감옥이 된다 |
| 결투 뒤 재배치 | 도르스는 옆문을 없애지 않고, 되돌이 경첩과 한문장 못을 남긴다 |
| 도시 변화 | 드레오렌의 모든 옆문은 되돌이 길표를 요구하고, 정면문 마당은 긴 고백이 아니라 한 문장을 받는 장소가 된다 |
