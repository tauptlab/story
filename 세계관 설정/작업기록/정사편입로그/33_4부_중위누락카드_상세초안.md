# 4부 중위 누락카드 상세초안

이 문서는 `31_4부_통합요소_추출과_누락카드.md`에 남은 4부 중위 후보를 실제 장면에 투입 가능한 설정 카드로 확장한다. `32_4부_상위누락카드_상세초안.md`가 왕좌의 파동, 분열, 결정화, 밤낮 순환, 중앙 가치 변환소 붕괴라는 큰 기둥을 세웠다면, 이 문서는 그 기둥 아래에서 사람들이 어떻게 먹고 숨 쉬고 도망치고 질문하고 버티는지를 담당한다.

4부 중위 카드의 핵심 원칙은 "세계의 붕괴는 왕과 시설만 무너뜨리지 않고, 골목의 손잡이와 밤의 별자리와 누군가의 작은 습관까지 바꾼다"는 것이다. 따라서 아래 카드들은 전투 보조 장치가 아니라, 무너지는 세계 안에서도 사랑받을 수 있는 얼굴과 생활의 결을 만드는 장치다.

## 1. 금빛 도시의 그늘

정사 등급: B급에서 A급 승격 후보

| 항목 | 설정 |
|---|---|
| 분류 | 장소/사회 노출/역사 사건의 후폭풍 |
| 소속 | 눅세린 아르도르, 잿빛 구역, 중앙 가치 변환소 주변 |
| 낮의 형태 | 왕좌의 파동 뒤 황금 베일이 벗겨져 녹슨 골조, 세르 배출구, 미납 부채 표식, 낡은 계단이 드러난 도시 |
| 밤의 변형 | 건물의 금빛 외피가 사라진 자리에 주민들이 팔아넘긴 웃음과 이름이 얼룩처럼 떠오른다 |
| 선물 | 눅세린의 고통을 추상적 착취가 아니라 생활의 흠집으로 보이게 한다 |
| 상처 | 드러난 사람들은 해방감보다 먼저 수치심을 느낀다. 그들은 숨겨진 피해자이면서 체제 유지의 일부이기도 했다 |
| 거짓 믿음 | 진실이 드러나면 사람들은 곧바로 자유로워진다 |
| 관련 인물 | 마이더스, 아마라, 아미키티아, 루멘, 리타스, 불씨 주민들 |
| 반작용 | 중앙 가치 변환소, 가치 결정체, 마지막 화롯가, 바랜 가치 결정체 |
| 장면 기능 | EP16에서 왕좌의 파동이 마이더스 체계의 외피를 찢고, EP20 이후 루멘의 전조와 불씨 주민들의 나눔으로 이어진다 |

세부 지대:

| 지대 | 형태 | 장면 기능 |
|---|---|---|
| 박락 대로 | 금빛 포석이 벗겨져 아래의 검은 배수로가 보이는 큰길 | 왕좌의 파동 이후 첫 충격 장면 |
| 부채 세탁소 | 가치 부채자의 손목표를 임시로 닦아주는 불법 가게 | 빚을 없애지 못하고 보이지 않게만 만드는 사회 |
| 낡은 지지대 시장 | 상층 건물에서 떨어진 금 조각과 녹슨 지지대를 함께 파는 시장 | 체제의 잔해가 곧 생계가 됨 |
| 손목표 환급소 | 무너진 가치 등급표를 다시 평가받으려는 줄이 생기는 창구 | 해방 직후에도 사람들이 낡은 기준으로 돌아가려는 두려움 |
| 마지막 물통 계단 | 불씨 주민들이 가격표 없는 물을 나누기 시작하는 계단 | 루멘 이전의 작은 회복 전조 |

생활 인물 씨앗:

| 이름/별칭 | 위치 | 사랑스러운 디테일 | 이야기 기능 |
|---|---|---|---|
| 헤리온 | 손목표 환급소 서기 | 폐기된 도장을 버리지 못하고 매일 닦는다 | 체제에 기대 살던 평범한 사람의 두려움 |
| 말라 | 낡은 지지대 시장 수선공 | 금 조각보다 휘어진 못을 더 아낀다 | 쓸모와 가치의 차이를 보여줌 |
| 세비 | 마지막 물통 계단 아이 | 물을 줄 때마다 잔에 작은 금빛 흠집을 세어 본다 | 가격표 없는 나눔을 처음 배우는 세대 |

사용 원칙:

- 금빛 도시의 그늘은 "봐라, 사실은 가난했다"가 아니다. 사람들은 황금 체계 덕분에 살아남은 적도 있고, 그 체계 때문에 서로를 잃은 적도 있다.
- 상층 주민도 전부 악인으로 만들지 않는다. 어떤 이는 진짜로 자기 성공이 모두를 돕는다고 믿었다.
- 루멘의 빛은 이곳을 심판하기보다 이름을 되돌린다. 금 아래 남은 것은 죄만이 아니라 수고와 두려움이다.

## 2. 엔트로피의 한숨

정사 등급: A급 보조 현상

| 항목 | 설정 |
|---|---|
| 분류 | 세계 현상/활력 저하/후반부 기후 |
| 소속 | 네우로시아 전역, 특히 눅세린, 프론티아, 자기 왕관의 영역 잔해, 녹티오라 경계 |
| 낮의 형태 | 세르 흐름이 느려지고, 색이 바래고, 건축물의 작은 장치가 한 박자 늦게 반응한다 |
| 밤의 변형 | 숨을 내쉴 때 회색 빛가루가 떠오르고, 별매듭 일부가 맥을 놓친 것처럼 깜빡인다 |
| 선물 | 처리되지 못한 대가가 배경이 아니라 사건을 움직이는 힘으로 보인다 |
| 상처 | 무기력은 악의가 아니다. 버티던 존재들이 더 느려지고 더 작게 말할 뿐이다 |
| 거짓 믿음 | 세계가 지친 것은 누군가 더 강하게 밀어붙이지 않았기 때문이다 |
| 관련 인물 | 셀로, 에이도스, 카산드라, 레마니스, 루멘, 살루스 세넥스 |
| 반작용 | 불씨 주민들, 진실의 빛, 대가 없는 풍요, 조용한 휴식 |
| 장면 기능 | 4부와 5부 사이를 이어 후반 세계의 날씨를 만든다 |

단계:

| 단계 | 현상 | 장면 기능 |
|---|---|---|
| 미세한 식음 | 불과 조명이 평소보다 늦게 켜진다 | 독자가 아직 설명 없이 이상함을 느낌 |
| 회색 숨 | 사람의 숨과 기계 증기에 회색 가루가 섞인다 | 세계 전체가 피로해졌음을 시각화 |
| 바랜 결정 | 가치 결정체, 기록 결정체, 프리즘 결정의 색이 탁해진다 | 여러 국가의 물질 체계를 한 현상으로 묶음 |
| 정지한 박자 | 음악, 공명, 심장로, 별매듭이 한 박자씩 밀린다 | 전투와 일상 모두에서 리듬 손실 |
| 작은 불씨 | 완전한 회복이 아니라 꺼지지 않는 미세한 열이 남는다 | 루멘과 불씨 주민들의 전조 |

국가별 발현:

| 국가/장소 | 발현 | 이야기 기능 |
|---|---|---|
| 눅세린 | 황금 베일이 탁해지고 가치 결정체가 쉽게 금 간다 | 마이더스 체계의 대가 |
| 프론티아 | 문과 격자가 정확히 닫히지 않고 경고음이 늦어진다 | 완벽한 절차의 노후화 |
| 센시르 | 조율 현이 낮게 처지고 반향의 샘이 흐려진다 | 공감도 피로해질 수 있음 |
| 히프노레아 | 기록 결정체가 잠깐씩 다른 기억을 섞어 비춘다 | 밤낮 순환 오류와 연결 |
| 자기 왕관의 영역 | 은빛 구조물이 안쪽에서 서리처럼 식는다 | 셀로의 완벽주의 붕괴 |

사용 원칙:

- 엔트로피의 한숨은 악당의 주문이 아니다. 오래 미뤄 둔 대가가 세계의 호흡으로 돌아오는 현상이다.
- 무기력한 인물을 게으르거나 약하게만 쓰지 않는다. 어떤 이들은 이미 너무 오래 버텼다.
- 이 현상의 반작용은 큰 승리가 아니라 아주 작고 반복되는 돌봄, 휴식, 나눔이어야 한다.

## 3. 도그마 집행관

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 집단/전투 인물군/규율 현장직 |
| 소속 | 도그마의 정론 요새, 프론티아 규율 파견대 |
| 낮의 형태 | 흰 규약복, 얼굴을 가리지 않는 투명 면갑, 손등의 판정 문장 |
| 밤의 변형 | 각자의 등 뒤에 지워 온 예외의 그림자가 조용히 따라온다 |
| 선물 | 혼란한 현장에서 누군가를 살리기 위해 기준이 필요하다는 진실 |
| 상처 | 기준이 사람을 살리던 기억 때문에, 이제 기준이 사람을 다치게 하는 순간을 보지 못한다 |
| 거짓 믿음 | 예외를 빨리 처리할수록 더 적은 사람이 다친다 |
| 관련 인물 | 도그마, 노르마, 타키투스, 라티오, 셀로, 아미키티아, 에이도스 |
| 반작용 | 영혼의 맹약, 카산드라 네트워크, 루멘의 빛, 분열된 셀로들 |
| 장면 기능 | EP17~18에서 셀로의 파편과 수정체를 "살아 있는 상처"가 아니라 "위험한 오류"로 규정한다 |

핵심 집행관:

| 인물 | 담당 능력 | 선물 | 왜곡 | 사랑스러운 디테일 |
|---|---|---|---|---|
| 노르마 | 규격화 광선 | 혼란 속에서도 모두에게 같은 기준을 적용한다 | 다름을 위험한 돌출로 본다 | 비뚤어진 책등을 보면 저도 모르게 맞춘다 |
| 타키투스 | 절대 침묵 영역 | 공포가 번지는 현장을 조용히 만든다 | 말을 멈추면 고통도 멈춘다고 믿는다 | 아이가 울면 먼저 자기 호흡을 낮춘다 |
| 라티오 | 오류 삭제 | 구조적 결함을 빠르게 찾아낸다 | 살아 있는 예외까지 삭제 대상으로 분류한다 | 지운 이름을 작은 종이에 다시 적어 주머니에 숨긴다 |

전투 흐름:

| 단계 | 집행관 행동 | 일행의 대응 |
|---|---|---|
| 판정 | 라티오가 셀로 수정체의 불안정한 결을 오류로 명명한다 | 에이도스가 "오류"와 "상처"를 구분하려 한다 |
| 제압 | 타키투스가 절대 침묵 영역으로 파동과 대화를 차단한다 | 아미키티아가 동의한 손목끼리 아주 짧은 맹약선을 연결한다 |
| 평준화 | 노르마가 주변 파동을 같은 폭으로 깎는다 | 콘코르디아나 리타스 계열 인물이 작은 차이를 지키려 한다 |
| 삭제 | 라티오가 가장 불규칙한 결을 겨냥한다 | 절망하는 아이 파편 또는 수정체 내부 핵이 미세하게 응답한다 |

사용 원칙:

- 세 집행관은 무표정 병사가 아니다. 독자가 그들의 두려움을 이해해야 한다.
- 이들의 패배는 죽임이나 처단이 아니라, "예외를 남겨도 모두가 무너지지 않는다"는 장면을 목격하는 방식이 좋다.
- 도그마 본체와 겹치지 않게, 집행관들은 현장 노동자와 실무자의 피로를 가진 존재로 쓴다.

## 4. 규격화 광선

정사 등급: B급 전투 장치

| 항목 | 설정 |
|---|---|
| 분류 | 능력/집행 장치/표준화 광역 제압 |
| 사용자 | 노르마, 도그마 집행관, 일부 프론티아 규율 장치 |
| 낮의 형태 | 흰 직선 격자가 목표 위에 얹히고, 돌출된 색과 결이 같은 폭으로 깎인다 |
| 밤의 변형 | 목표의 그림자가 여러 개였다가 하나의 얇은 직사각형으로 접힌다 |
| 선물 | 혼란한 세르를 일정하게 만들어 구조 붕괴를 막을 수 있다 |
| 상처 | 개별 차이, 미세한 떨림, 회복 중인 불규칙성까지 잘라낸다 |
| 거짓 믿음 | 같은 모양이 되면 같은 고통을 가진 것이다 |
| 반작용 | 공명장, 영혼의 맹약, 카산드라의 속삭임, 리타스의 풍요 |
| 장면 기능 | 분열된 셀로들과 수정체 주변의 불규칙한 파동을 표준 위험으로 처리한다 |

작동 규칙:

| 규칙 | 내용 |
|---|---|
| 폭 기준 | 가장 안정된 평균값을 기준으로 삼기 때문에 약한 존재와 강한 존재 모두 잘릴 수 있다 |
| 표면 우선 | 겉으로 튀는 결부터 깎으므로, 깊은 상처는 남아도 겉보기는 안정된다 |
| 반복 사용 | 여러 번 맞으면 개성이 사라지기보다 먼저 반응 속도가 늦어진다 |
| 오작동 | 엔트로피의 한숨이 심한 곳에서는 평균값 자체가 무너져 광선이 갈라진다 |

사용 원칙:

- 규격화 광선은 화려한 공격보다 무서운 "정돈"이어야 한다.
- 맞은 대상은 폭발하지 않는다. 대신 목소리의 억양, 걸음의 리듬, 색의 번짐이 줄어든다.
- 반격은 더 큰 힘이 아니라, 평균으로 접을 수 없는 동의와 관계의 작은 차이에서 나온다.

## 5. 절대 침묵 영역

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 능력/공간/파동 차단장 |

| 낮의 형태 | 소리가 먹히는 투명한 돔. 입술은 움직이지만 말과 파동이 몇 걸음 앞에서 떨어진다 |

| 상처 | 멈춘 것은 해소된 것이 아니라 보류된 것이다. 오래 닫으면 안쪽에서 압력이 오른다 |
| 거짓 믿음 | 조용하면 안전하다 |
| 반작용 | 속삭임의 다리, 영혼의 맹약 실, 불씨 주민들의 손짓, 루멘의 낮은 빛 |

내부 규칙:

| 요소 | 작동 | 장면 기능 |
|---|---|---|
| 외음 차단 | 바깥 소리가 들리지 않는다 | 고립감 |
| 내음 증폭 | 자기 숨과 심장 소리만 커진다 | 두려움의 확대 |
| 파동 지연 | 능력이 발동해도 늦게 도착한다 | 전투 템포 붕괴 |
| 침묵 부채 | 오래 유지할수록 해제 순간 밀린 말이 터진다 | 후반 폭발 위험 |

사용 원칙:

- 절대 침묵 영역은 평화로운 무음 공간처럼 쓰지 않는다. 조용해서 더 시끄러운 곳이어야 한다.
- 타키투스의 선의가 중요하다. 그는 누군가의 비명을 멈춰 살린 기억 때문에 침묵을 믿는다.
- 아미키티아의 대응은 더 크게 외치는 것이 아니라, 허락받은 손짓과 짧은 접촉으로 침묵을 건너는 방식이 좋다.

## 6. 오류 삭제

정사 등급: B급에서 A급 위험 후보

| 항목 | 설정 |
|---|---|
| 분류 | 능력/판정/부분 삭제 |
| 사용자 | 라티오, 도그마 계열 판정관 |
| 낮의 형태 | 목표의 결 위에 붉은 작은 괄호가 생기고, 괄호 안의 파동만 종이 찢기듯 사라진다 |
| 밤의 변형 | 삭제된 자리에 흰 여백이 남고, 주변 기억이 그 여백을 당연한 것으로 맞추려 한다 |
| 선물 | 진짜 치명적 결함이나 붕괴 지점을 빠르게 제거할 수 있다 |
| 상처 | 살아 있는 존재의 예외와 위험한 결함을 구분하지 못하면 존재 일부를 지운다 |
| 거짓 믿음 | 더 이상 보이지 않으면 해결된 것이다 |
| 반작용 | 루멘의 빛, 기억의 돌, 카산드라의 관측 기록, 절망하는 아이 |
| 장면 기능 | 셀로 파편과 수정체를 삭제 대상으로 취급하며, "치료와 제거의 차이"를 묻는다 |

삭제 단계:

| 단계 | 내용 | 위험 |
|---|---|---|
| 표식 | 불규칙한 파동에 붉은 괄호가 생긴다 | 이름 붙이기가 곧 판결이 됨 |
| 분리 | 괄호 안 파동만 주변과 끊긴다 | 관계와 맥락이 함께 잘릴 수 있음 |
| 삭제 | 파동이 흰 여백으로 바뀐다 | 존재의 일부가 부재로 굳음 |
| 정합화 | 주변 기억이 여백을 자연스럽게 받아들인다 | 삭제 사실 자체가 잊힘 |

사용 원칙:

- 오류 삭제는 치료가 아니다. 정사상 치료는 응답을 기다리고, 삭제는 응답할 가능성을 닫는다.
- 라티오가 완전히 악해서 위험한 것이 아니다. 그는 실제로 무너지는 구조를 많이 막아 왔다.
- 루멘이나 카산드라가 이 능력에 맞설 때는 "지워진 것이 무엇인지"를 다시 보이게 하는 방향이 좋다.

## 7. 수정체 보호처

정사 등급: B급에서 A급 승격 후보

| 항목 | 설정 |
|---|---|
| 분류 | 장소/은신처/임시 공동체 |

| 낮의 형태 | 무너진 성소와 수로를 이어 만든 낮은 방. 중앙에는 무지갯빛 수정체가 직접 빛을 받지 않도록 천과 물그릇이 둘러져 있다 |
| 밤의 변형 | 수정체가 주변의 비명을 모두 빨아들이지 않도록 바닥의 물그릇들이 파동을 나누어 담는다 |
| 선물 | 셀로가 처음으로 세계를 지키는 자가 아니라 세계에게 지켜지는 자가 된다 |
| 상처 | 보호라는 이름으로 셀로를 다시 침묵시킬 위험이 있다 |
| 거짓 믿음 | 안전하게 숨기면 회복도 저절로 온다 |

| 장면 기능 | EP18~20에서 동료들이 셀로 없는 선택을 하고, 셀로를 대상화하지 않는 보호법을 배운다 |

공간 요소:

| 요소 | 형태 | 장면 기능 |
|---|---|---|
| 물그릇 둘레 | 작은 물그릇들이 원형으로 놓여 파동을 나누어 담는다 | 수정체가 모든 비명을 혼자 듣지 않게 함 |
| 낮은 천장 | 일부러 고개를 숙여야 들어갈 수 있다 | 지배와 과시의 반대 |
| 맹약 매듭 | 동의한 동료들의 실이 벽에 느슨하게 묶여 있다 | 강제 보호가 아닌 합의 |
| 빈 자리 | 셀로가 깨어났을 때 앉을 자리 | 수정체를 물건이 아니라 돌아올 사람으로 대함 |
| 불씨 선반 | 주민들이 가져온 작은 음식과 편지가 놓인다 | 생활 인물과 루멘 전조 |

보호 규칙:

| 규칙 | 내용 |
|---|---|
| 자극 금지 | 수정체를 깨우려 흔들거나 큰 빛을 쏘지 않는다 |
| 파동 분리 | 주변의 두려움과 수정체 내부 응답을 분리해 듣는다 |
| 동의 우선 | 방어선에 참여하는 동료는 반드시 자기 한계를 말해야 한다 |
| 자리 남김 | 셀로를 대신해 결정을 끝내지 않는다. 돌아올 자리를 남긴다 |

사용 원칙:

- 수정체 보호처는 대기실이 아니다. 동료들이 셀로 없이도 윤리적 선택을 배우는 작은 정치 공간이다.
- 아미키티아는 모든 사람을 연결하려는 유혹을 경계해야 한다. 보호처에서는 연결보다 분리와 휴식도 중요하다.
- 불씨 주민들이 가져오는 작은 물건이 루멘의 등장을 준비한다.

## 8. 순환 수호자

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 집단/경계자/반자동 수호체 |

| 밤의 변형 | 검푸른 망토와 별빛 관절을 가진 경비자. 얼굴은 개인이 아니라 그날의 하늘 모양을 닮는다 |
| 선물 | 세계가 감당하지 못할 기억과 상처를 무분별한 관측에서 보호한다 |
| 상처 | 보호할 대상을 잊고, 순환 장치 자체의 보존을 목적으로 삼는다 |
| 거짓 믿음 | 보지 못하게 막으면 안전하다 |
| 관련 인물 | 카산드라, 에이도스, 므네몬, 도그마, 포보스, 아미키티아 |
| 반작용 | 순환 오류 관측 기록, 속삭임의 다리, 밤길 안대 |
| 장면 기능 | EP19 금지된 관측의 추격전에서 단순 적이 아니라 오래된 장치의 윤리적 딜레마를 구현한다 |

수호자 유형:

| 유형 | 역할 | 사랑스러운 디테일 | 위험 |
|---|---|---|---|
| 밤문지기 | 관측점 입구를 막는다 | 길을 막기 전 먼저 안대를 건넨다 | 선택권을 준 척하면서 길을 닫음 |
| 별매듭 수선자 | 고장 난 노드를 임시로 묶는다 | 끊어진 별빛을 바느질하듯 꿰맨다 | 사람보다 장치를 먼저 고침 |
| 순환 집행자 | 금지된 관측자를 추격한다 | 공격 전 오래된 예법으로 고개를 숙인다 | 정중한 폭력 |
| 기록 회수자 | 관측 기록을 되가져간다 | 빼앗은 기록에 먼지를 털어 준다 | 증거를 보호 명분으로 은폐 |

전투/추격 규칙:

| 규칙 | 내용 |
|---|---|
| 첫 경고 | 즉시 공격하지 않고 길, 안대, 침묵 표식 중 하나를 제시한다 |
| 관측량 반응 | 더 많이 보려 할수록 더 강하게 추격한다 |
| 장치 우선 | 사람을 죽이는 것보다 기록 회수와 노드 안정화를 우선한다 |
| 오류 취약 | 순환 오류 관측 기록이 정확하면 잠깐 멈춰 자기 명령을 재검토한다 |

사용 원칙:

- 순환 수호자는 악의적 비밀경찰이 아니다. 오래된 보호 장치가 사람의 고통보다 절차를 먼저 보게 된 결과다.
- 카산드라와 에이도스는 수호자를 이기는 것이 아니라, 수호자가 보호하던 원래 이유를 떠올리게 해야 한다.
- 수호자의 예의와 폭력이 동시에 보여야 한다. 독자는 "이들이 틀렸지만 이해된다"고 느껴야 한다.

사용 원칙:

- 이 기록은 모든 진실을 한 번에 폭로하지 않는다. 오히려 "여기까지만 확실하다"는 경계를 만들어야 한다.
- 에이도스는 빈칸을 견디기 어려워하고, 카산드라는 빈칸을 너무 신비화할 위험이 있다. 둘의 긴장이 좋다.
- 루멘이 후반에 이 기록을 비출 때, 기록이 더 강해지는 것이 아니라 기록 안의 거짓 연결이 떨어져 나가야 한다.

| 층 | 기능 | 붕괴 양상 |
|---|---|---|
| 관측층 | 도시의 말, 표정, 파동을 규약 문장으로 번역한다 | 말과 표정이 서로 다른 판정을 받는다 |
| 보정층 | 기준에서 벗어난 행동에 경고를 보낸다 | 경고가 서로 상충해 길이 막힌다 |
| 침묵층 | 위험한 논쟁을 임시 봉인한다 | 봉인된 말들이 탑 내부에서 반향한다 |
| 판결층 | 최종 오류 여부를 선언한다 | 탑 자신이 예외가 되어 삭제 명령을 받는다 |

중앙 가치 변환소와 구분:

| 항목 | 중앙 가치 변환소 | 규율 감시탑 |
|---|---|---|
| 소속 | 눅세린/마이더스 | 도그마/프론티아 |
| 핵심 왜곡 | 모든 관계를 가격으로 환산 | 모든 삶을 완결 문장으로 번역 |
| 붕괴 원인 | 가치 부채와 엔트로피의 역류 | 예외를 지우려는 규칙이 자기 자신을 예외로 판정 |

사용 원칙:

- 정사 기본안은 4부 클라이맥스 시설을 중앙 가치 변환소로 둔다. 규율 감시탑은 도그마/프론티아 축을 강화하거나 병렬 붕괴 장면이 필요할 때 쓰는 대체/보조 카드다.
- 탑의 붕괴는 폭파가 아니라 모순 판정의 누적이어야 한다.
- 루멘의 빛이 탑을 부술 때도 "너희는 틀렸다"가 아니라 "너희가 지우려 한 예외가 너희를 살릴 문장일 수 있다"는 방향이 좋다.
