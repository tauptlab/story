# 라모 정사 승격 1차

역할: 드레오렌 생활 인물 `라모`를 돌림말 우물의 관리인으로 승격하고, "말하지 못한 문장을 기다리는 예법"을 방치가 아니라 귀환물결 표식을 남기는 책임으로 정리한다.

---

## 1. 승격 요약

라모는 기존 자료에서 드레오렌 돌림말 우물 관리인으로 배치되어 있었다. 이번 승격에서는 그를 단순히 조용히 기다리는 단역이 아니라, 말할 시간을 빼앗지 않는 선의와 아무 표식 없는 기다림이 방치로 굳는 위험을 함께 가진 생활 인물로 정리했다.

| 항목 | 기존 씨앗 | 142 기준 정사화 |
|---|---|---|
| 인물 기능 | 직접 말하지 못한 문장을 기다리는 우물 관리인 | 기다림의 보호성과 방치의 위험을 함께 보이는 생활 인물 |
| 주 무대 | 드레오렌 돌림말 우물 | 돌림말 우물, 젖은단어 돌계단, 되돌이 길표 보관소, 정면문 마당 |
| 핵심 상처 | 재촉받은 사람이 자기 말을 잃는 장면 | 기다리면 말은 저절로 돌아온다는 믿음 |
| 핵심 물성 | 돌림표, 돌림말 우물 | 젖은단어 돌, 낮은두레박, 귀환물결 표식, 물결숨이끼 |

## 2. 124 기준 적용

| 기준 | 라모의 처리 |
|---|---|
| 버린 가치관 | 말하지 못한 문장도 주인의 때에 다시 돌아올 수 있다는 믿음 |
| 보호하던 대체 가치관 | 침묵, 돌림말, 기다림, 물밑에 두기 |
| 왜곡된 가치관 | 기다림에 표식이 없어 말이 보호가 아니라 방치 속으로 가라앉는다 |
| 다시 받아들일 가치 | 말할 시간을 지키되, 돌아올 물결과 감당 가능한 단어 하나를 남기는 책임 |
| 녹티오라 흔적 | 깊어지는 돌림말 우물, 마르지 않는 젖은단어 돌, 작아지는 낮은두레박 |
| 귀환 계기 | 라모가 또 기다리려는 순간, 젖은단어 돌의 단어들이 마르지 않고 우물 물결이 검게 흐리는 장면 |
| 귀환 후 후폭풍 | 드레오렌은 침묵을 벌하지 않지만, 돌림말마다 귀환물결 표식과 다시 올 물가를 요구한다 |

## 3. 녹티오라 설계

| 요소 | 왜 강하게 남았는가 | 밤 변형 | 선택 시험 |
|---|---|---|---|
| 젖은단어 돌 | 긴 문장을 말할 수 없어도 단어 하나는 남기고 싶었던 감각 때문이다 | 주인이 돌아오면 마르고, 남이 대신 읽으면 갈라진다 | 보존인가, 증거화인가 |
| 낮은두레박 | 말 대신 물 한 컵만 건네도 사람이 우물가에 남을 수 있었기 때문이다 | 문장은 건지지 못하고 물만 길어 올린다 | 돌봄인가, 회피인가 |
| 귀환물결 표식 | 돌아오라는 재촉이 아니라 돌아와도 된다는 허락이 필요했기 때문이다 | 명령처럼 붙이면 둥글게 말리고, 허락으로 남기면 길처럼 낮아진다 | 허락인가, 강요인가 |
| 물결숨이끼 | 기다림이 돌봄인지 방치인지 우물가에서 먼저 말라붙는 흔적 때문이다 | 방치가 길면 돌에 붙고, 돌아올 표식이 있으면 잎끝에 물빛 숨을 맺는다 | 기다림인가, 방치인가 |

## 5. 결투 의미

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 말할 시간을 지키는 기다림 vs 영영 가라앉지 않게 돌아올 표식을 남기는 책임 |
| 기다림의 옳음 | 준비되지 않은 말을 끌어내면 사람은 자기 말을 잃을 수 있다 |
| 기다림의 위험 | 아무 표식 없이 기다리기만 하면 말은 보호가 아니라 방치 속으로 가라앉는다 |
| 돌아올 표식의 옳음 | 말하지 않아도 된다는 허락과 돌아와도 된다는 허락을 함께 남긴다 |
| 돌아올 표식의 위험 | 표식이 재촉이 되면 다시 말하기 강요가 된다 |
| 결투 뒤 재배치 | 라모는 문장을 퍼 올리지 않고, 낮은두레박과 젖은단어 돌, 귀환물결 표식을 함께 남긴다 |
| 도시 변화 | 드레오렌의 돌림말 우물은 기다림만의 장소가 아니라, 돌아올 물가를 잃지 않는 예법의 장소가 된다 |
