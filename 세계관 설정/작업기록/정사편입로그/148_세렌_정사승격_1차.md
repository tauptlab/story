# 세렌 정사 승격 1차

정사 등급: 작업 기록

역할: 루트 초기 자료의 `세렌` 씨앗을 실렌카르 뒤늦은 별표 서고의 생활 인물로 승격하고, "잊은 감정을 기록하는 일"을 확정 판결이 아니라 다시 물을 여백으로 정리한다.

---

## 1. 승격 요약

세렌은 기존 자료에서 `잊은 감정을 기록하는 인물` 후보로 남아 있었다. 이번 승격에서는 그를 잊은 감정을 크게 되살리는 사람이 아니라, 틀린 예언과 놓친 가능성 옆에 작은 별표와 오늘의 질문을 남기는 기록자로 정리했다.

핵심 이동은 아래와 같다.

| 구분 | 승격 전 | 승격 후 |
|---|---|---|
| 인물 기능 | 실렌카르 보조 인물 후보 | 뒤늦은 별표 서고의 별표/질문 생활 인물 |
| 핵심 질문 | 잊은 감정을 기록하는가 | 아직 이름 붙이지 못한 감정에 다시 물을 자리를 남길 수 있는가 |
| 주 무대 | 실렌카르 후보 | 뒤늦은 별표 서고, 오늘질문 난간, 현재 마당 |
| 주요 물성 | 뒤늦은 별표 | 오늘질문 잉크, 여백별핀, 되묻기 칸, 별표이끼 |
| 결투 기능 | 미정 | 빠른 이름 붙이기의 보호성과 여백의 책임 재배치 |

## 2. 124 기준 반영

| 기준 | 세렌 반영 |
|---|---|
| A가 버린 가치관 | 아직 이름 붙이지 못한 감정도 관계 안에 잠시 남을 수 있다는 믿음 |
| 왜 버릴 수밖에 없었는가 | 모르는 마음을 설명하라는 요구 앞에서, 감정을 빨리 이름 붙이지 못하면 죄처럼 느껴졌다 |
| 대체 가치관 | 감정과 가능성에 빠르게 이름을 붙여 추궁과 혼란을 줄이는 습관 |
| 처음의 보호 기능 | 흩어진 후회에 임시 모양을 주고, 비난을 막을 최소한의 기록을 만들었다 |
| 시간이 지나며 생긴 왜곡 | 모든 잊은 감정이 판결표가 되고, 지나간 가능성이 현재 사람을 벌한다 |
| 귀환 계기 | 뒤늦은 별표 서고의 밤 결투에서 세렌이 확정 제목 대신 여백별핀과 오늘질문 잉크를 남긴다 |
| 필연성 | 빠른 이름만으로는 감정이 판결이 되고, 여백만으로는 책임이 사라지므로 별표와 오늘질문이 함께 필요해진다 |
| 귀환 후 후폭풍 | 실렌카르는 틀린 예언 옆에 별표만 붙이지 않고, 오늘 다시 물을 칸을 함께 남긴다 |

## 3. 녹티오라 설계

| 낮의 요소 | 왜 남았는가 | 밤의 형태 | 선택 시험 |
|---|---|---|---|
| 뒤늦은 별표 | 틀림을 벌로 만들지 않으려는 작은 표식이 남았다 | 비난의 문장 옆에서는 검게 식고, 다시 묻는 문장 옆에서는 은빛을 낸다 | 면죄인가, 재검토인가 |
| 오늘질문 잉크 | 과거의 후회를 오늘 물을 수 있는 문장으로 낮추려는 손끝이 남았다 | 과거형 문장 위에서는 말라 버리고, 오늘형 질문 옆에서만 낮게 번진다 | 질문인가, 지연인가 |
| 여백별핀 | 확정하지 못한 감정을 잠시 여백에 붙들던 습관이 남았다 | 사람 이름 위에 꽂히면 굳고, 문장 가장자리에서는 부드럽게 빠진다 | 배려인가, 판결인가 |
| 되묻기 칸 | 답 대신 다시 물을 자리를 남기려던 예법이 남았다 | 대신 답을 적으면 칸 안쪽이 겹치고, 비워 두면 작은 빛을 품는다 | 여백인가, 회피인가 |

## 5. 결투 설계 메모

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 감정을 빨리 이름 붙여 정리하려는 책임 vs 아직 모르는 감정을 여백에 남길 권리 |
| 두 가치의 옳음 | 빠른 이름은 혼란을 줄이고, 여백은 모르는 마음을 관계 밖으로 밀어내지 않는다 |
| 두 가치의 위험 | 빠른 이름은 판결이 되고, 여백은 책임 회피가 될 수 있다 |
| 끝난 뒤 재배치 | 이름은 임시 표식이 되고, 여백은 오늘 다시 물을 칸을 가진 책임으로 남는다 |
| 도시 변화 | 뒤늦은 별표 서고는 별표 옆에 오늘질문 칸을 함께 두고, 별표이끼는 비난의 문장보다 다시 묻는 문장 곁에서 자란다 |
| 인물 변화 | 세렌은 확정 제목을 붙이던 손을 낮추고, 여백별핀과 오늘질문 잉크를 함께 남긴다 |
