# 티렌 정사 승격 1차

작성일: 2026-06-23

역할: 루트 초기 자료의 `티렌` 씨앗을 아르도르 남겨둔 몫 회랑의 "주는 손의 한계와 자기 몫" 생활 인물로 승격한다.

---

## 1. 승격 대상

티렌은 구판 자료에서 아르도르의 헌신 인물로 남아 있던 씨앗이다. 구판 티렌은 자기 마음을 반복해서 내주다가, 도움을 받는 사람과 주는 사람 모두의 경계를 잃는 인물이었다. 이번 승격에서는 그 기능을 살리되, 마음을 밀어 넣는 인물이 아니라 `남겨둔 잔`, `온기국자`, `되묻는 담요`, `몫잎풀`로 장면화되는 생활 인물로 낮췄다.

| 항목 | 내용 |
|---|---|
| 정사명 | 티렌 |
| 소속 | 눅세린 아르도르, 무보증 도움창 뒤편, 남겨둔 몫 회랑 |
| 정사 등급 | B급 상위 생활 인물 |
| 핵심 가치 | 주는 손의 한계, 자기 몫을 남기는 돌봄, 헌신의 예법화 |
| 주요 물성 | 남겨둔 잔, 온기국자, 되묻는 담요, 몫잎풀, 자기 의자 |

## 2. 상위 기준 반영

| 기준 | 티렌 반영 |
|---|---|
| A가 버린 가치관 | 사랑하는 사람을 돌보면서도 자기 몫과 이름을 남길 수 있다는 믿음 |
| 버릴 수밖에 없었던 이유 | 아픈 사람 곁에서 자기 필요를 말하는 일이 이기심처럼 느껴졌고, 필요로 여겨질 때만 관계 안에 남을 수 있었다 |
| 대체 가치관 | 먼저 알아차리기, 먼저 주기, 자기 몫을 줄여 관계를 붙드는 습관 |
| 보호 기능 | 관계의 급한 균열을 막고, 말하지 못한 필요를 빠르게 살폈다 |
| 왜곡 | 도움받는 사람의 차례를 빼앗고, 주는 사람은 필요가 끝나면 보이지 않는 사람으로 남는다 |
| 다시 받아들일 가치 | 도움은 양과 끝을 물을 때 오래 지속되고, 자기 몫을 남겨도 사랑은 끊기지 않는다는 믿음 |
| 녹티오라 흔적 | 투명해지는 남겨둔 잔, 짧아지는 온기국자, 입을 가리는 되묻는 담요, 반으로 펼쳐지는 몫잎풀 |
| 귀환 계기 | 나리안의 도움끈, 미라솔의 빈보증 표, 벨키르의 부담 저울, 티렌의 남겨둔 잔이 맞물려 도움의 양과 관계의 무게를 다시 배치한다 |

## 3. 밤 변형 설계

| 요소 | A의 기억에 남은 이유 | 밤 변형 | 시험 |
|---|---|---|---|
| 남겨둔 잔 | 남을 먼저 채우느라 자기 컵이 식은 채 남던 순간이 남았다 | 다른 사람의 컵을 채울수록 투명해지고, 자기 이름을 부르면 바닥 글자가 돌아온다 | 헌신인가, 자기 삭제인가 |
| 온기국자 | 누군가의 떨림을 빨리 멈추게 하고 싶던 손의 습관이 남았다 | 계속 쓰면 손잡이가 짧아져 손바닥 안으로 파고든다 | 도움인가, 대신 살기인가 |
| 되묻는 담요 | 덮어 주는 일이 늘 다정함이라고 믿었던 기억이 남았다 | 묻지 않고 덮으면 티렌의 입을 가리고, 허락을 들으면 어깨까지만 내려앉는다 | 보호인가, 덮어 버림인가 |
| 몫잎풀 | 자기 몫을 남기면 누군가 서운해할까 두려웠던 순간이 남았다 | 자기 몫을 남긴 자리에서 잎을 반으로 펼치고, 억지 희생 앞에서는 금빛으로 마른다 | 이기심인가, 지속 가능한 돌봄인가 |
| 자기 의자 | 늘 남에게 밀어 주던 자리가 몸에 남았다 | 티렌이 앉지 않으면 회랑 끝으로 밀려나고, 앉으면 문고리가 낮아진다 | 곁인가, 사라짐인가 |

## 5. 결투 기능

티렌 결투는 헌신을 그만두느냐 계속하느냐의 단순 대립이 아니다. 빠르게 알아차리고 먼저 내미는 힘은 실제로 필요하다. 문제는 그 힘이 자기 몫과 상대의 차례를 모두 지울 때 생긴다.

| 충돌 | 설명 |
|---|---|
| 전부 주는 돌봄 | 말하지 못한 필요를 먼저 알아차리지만, 도움받는 사람의 차례와 주는 사람의 이름을 지울 수 있다 |
| 자기 몫을 남기는 돌봄 | 도움을 오래 지속시키지만, 급한 고통 앞에서는 차갑거나 부족해 보일 수 있다 |

결투 뒤에는 남겨둔 잔이 사라지지 않는다. 대신 잔 바닥에 티렌의 이름이 돌아오고, 온기국자는 자기 잔을 완전히 비우면 더는 떠지지 않는다.
