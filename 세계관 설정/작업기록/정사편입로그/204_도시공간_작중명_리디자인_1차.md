# 204 도시/공간 작중명 리디자인 1차

작성일: 2026-06-24

역할: 도시, 특수공간, 핵심 장소 이름 중 작가용 의미가 직접 드러나는 항목을 작중에서 자연스럽게 부를 수 있는 명칭으로 1차 리디자인한 기록이다.

## 작업 원칙

- 레거시 파일 경로는 즉시 바꾸지 않는다.
- 백틱 안에 들어간 파일명과 경로 표기는 유지한다.
- 실제 연재분에서 독자가 보게 될 본문 명칭, 문서 제목, 아틀라스 노출명부터 새 작중명으로 바꾼다.
- 기존명은 삭제하지 않고 작업기록의 대응표로 보존한다.
- 새 이름은 라틴어/그리스어/작중어 계열을 섞되, 발음과 이미지가 너무 어렵지 않은 쪽을 우선한다.

## 1차 대응표

| 기존명 | 새 작중명 | 리디자인 사유 |
|---|---|---|
| 선전 방송탑 | 프라에코 탑 | 선전이라는 기능어를 피하고, 전령/공표의 이미지를 살림 |
| 정보 통제실 | 베르붐 관제실 | 통제라는 작가용 의미를 낮추고, 말의 경로를 다루는 시설로 보이게 함 |
| 검은 물 저장고 | 아테르 수조 | 공포/어둠의 직접어를 라틴어 계열 물성으로 우회 |
| 그림자 골짜기 | 움브라 골짜기 | 그림자 의미를 작중어로 낮춰 장소명처럼 만듦 |
| 레툼의 장례 들판 | 레퀴엠 들판 | 장례 기능은 남기되 의식적 명칭으로 우회 |
| 근원의 샘 | 오리고 샘 | 근원이라는 해설어를 작중 고유명으로 전환 |
| 수정체 보호처 | 크리스탈리스 보호처 | 보호처 기능은 유지하되 더 지역/시설명처럼 조정 |
| 자기 왕관의 영역 | 코로나 내정원 | 자기애/왕관 은유를 직접 말하지 않고 내부 정원 이미지로 전환 |
| 수리 인계 광장 | 리토르노 광장 | 수리/인계라는 운영어를 돌려주기/귀환 이미지로 우회 |
| 잊혀진 정원 | 라쿠나 정원 | 잊힘을 직접 말하지 않고 결여/빈자리의 작중어로 전환 |
| 다산의 일족 성역 입구 | 페쿤디아 문턱 | 다산의 의미를 인물/생명축 작중어로 통일 |
| 망각의 강 지류 | 레테 지류 | 망각을 신화어로 우회 |
| 보존 구역 | 쿠스토디아 구역 | 보존이라는 해설어를 관리/수호 계열 작중어로 전환 |
| 빈칸 서가 | 라쿠나 서가 | 빈칸이라는 직접어를 라쿠나 계열로 통일 |
| 살아있는 기록림 | 비바 기록림 | 살아있다는 해설을 생명감 있는 작중어로 낮춤 |
| 심층 기록 보관소 | 아르카눔 보관소 | 심층/보관 기능을 비밀 기록 시설의 명칭처럼 우회 |
| 느린 손 공방 | 모라 손공방 | 느림의 가치 설명을 작중어로 낮춤 |
| 정밀의 대장간 | 아쿠스 대장간 | 정밀함을 직접 설명하지 않고 날/첨단 이미지로 우회 |
| 형태의 전당 | 모르파 전당 | 형태라는 주제어를 그리스어 계열로 전환 |
| 대공명실 | 디아파손 홀 | 공명 기능은 남기되 음악 용어로 작중명화 |
| 조정의 광장 | 템페라 광장 | 조율/조정의 의미를 작중 도시명처럼 우회 |
| 살루스 안식 마을 | 소포르 마을 | 안식의 직접어를 살루스 능력명 계열로 통일 |
| 경계 너머 관측점 | 울트라 리멘 관측점 | 경계 너머라는 해설어를 작중 관측 시설명으로 조정 |
| 호수지역 | 라쿠스 권역 | 임시 설명형 지명에서 권역명으로 승격 |
| 풀이름 마당 | 노멘초 마당 | 이름 붙이기 기능은 남기되 노멘 계열 작중어로 통일 |

## 적용 범위

| 범위 | 결과 |
|---|---:|
| 갱신된 활성 문서 | 100개 |
| 백틱 내부 파일명/경로 | 유지 |
| 작업기록 내부 구명 | 유지 |

## 후속 작업

1. 소품/유기물/무기물 명칭도 같은 방식으로 2차 대응표를 만든다.
2. 구명 잔여는 파일 경로, 작업기록, 구판 보존 문서인지 먼저 구분한다.
3. 실제 원고 진입 전에는 새 작중명 우선, 구명은 경로/작업 로그에서만 남긴다.
