# 5부 중위 누락카드 상세초안

이 문서는 `34_5부_통합요소_추출과_누락카드.md`에 남은 5부 중위 후보를 실제 장면에 투입 가능한 설정 카드로 확장한다. `35_5부_상위누락카드_상세초안.md`가 루멘의 한계, 셀로의 위험한 도움, 마이더스의 마지막 결핍, 리타스의 귀환, 태초의 메아리, 최초의 순환, 위대한 왜곡을 큰 기둥으로 세웠다면, 이 문서는 그 기둥 아래에서 빛이 어떻게 낮아지고, 장소가 어떻게 숨 쉬고, 기록이 어떻게 침묵을 견디며, 동료들이 어떤 방식으로 남는지를 담당한다.

5부 중위 카드의 핵심 원칙은 "밝음은 강도보다 배치와 시간이 중요하다"이다. 너무 강한 빛은 상처가 되고, 너무 늦은 빛은 방치가 된다. 따라서 아래 카드들은 필살기 목록이 아니라, 인물이 자기 선물의 속도와 대가를 배우는 장면 장치다.

## 1. 심장의 거울

정사 등급: A급 보조 능력

| 항목 | 설정 |
|---|---|
| 분류 | 능력/거울/진실의 시간성 |
| 소속 | 루멘, 진실의 빛, 5부 EP21~24 |
| 작중 별칭 | 심장의 거울, 돌아온 빛, 손등의 거울, 낮춘 흰금빛 |
| 낮의 형태 | 루멘의 손등에서 낮아진 흰금빛이 상대에게 직접 쏘이지 않고 한 번 접혀 돌아간다 |
| 밤의 변형 | 거울 속에는 상대가 숨긴 진실뿐 아니라 루멘이 보고 싶어 한 결론도 함께 비친다 |
| 선물 | 상대를 굴복시키지 않고, 스스로 보게 하는 시간을 준다 |
| 상처 | 루멘은 진실을 빨리 보여주면 사람이 더 빨리 구원된다고 믿어 왔다 |
| 거짓 믿음 | 자신을 보면 누구나 곧바로 바뀔 수 있다 |
| 관련 인물 | 루멘, 셀로, 마이더스, 멘다키우스, 도그마, 므네몬 |
| 반작용 | 핵심의 파동, 준비되지 않은 기억, 침묵의 도서관, 밤의 비밀 기록 |
| 장면 기능 | 셀로의 핵심의 파동을 악한 힘으로 단정하지 않고, 셀로 자신에게 되돌려 보게 한다 |

### 1.1 작동 단계

| 단계 | 내용 | 장면 기능 |
|---|---|---|
| 빛 낮춤 | 루멘이 먼저 손등의 빛을 낮춘다 | 빛의 윤리가 힘 조절에서 시작됨 |
| 반사면 만들기 | 상대 앞에 직접 심판선이 아니라 얇은 거울빛을 놓는다 | 강제 폭로와 구분 |
| 자기 그림자 동반 | 상대의 진실과 루멘의 절박함이 함께 비친다 | 루멘도 심판자 자리에 서고 싶어 했음을 드러냄 |
| 선택의 틈 | 본 것을 말할지 침묵할지 상대에게 남긴다 | 진실은 통제보다 응답을 요구함 |
| 잔광 회수 | 루멘이 결과를 밀어붙이지 않고 빛을 거둔다 | 성장한 루멘의 기다림 |

### 1.2 실패 장면

| 상황 | 실패 | 배움 |
|---|---|---|
| 약탈자 심문 | 두려움까지 공개되어 주변의 분노가 더 커진다 | 보이는 진실과 말해야 할 진실은 다르다 |
| 셀로 조우 | 셀로의 상처를 보려다 루멘 자신의 구원 욕구도 비친다 | 남을 비추는 빛은 자신도 숨기지 못한다 |
| 므네몬 기록 | 봉인된 장면을 열려 하자 작은 신발의 잔상이 나타난다 | 보호가 언제 은폐가 되는지 더 천천히 물어야 한다 |

사용 원칙:

- 심장의 거울은 정답을 보여주는 판이 아니다. 상대가 감당할 수 있는 만큼만 돌아오는 빛이어야 한다.
- 이 능력은 아름답지만 불편해야 한다. 본다는 것은 늘 곧바로 좋아지는 일이 아니다.
- 루멘은 이 거울로 이기기보다, 자신이 서 있던 높은 자리를 내려오는 쪽으로 성장한다.

장면 씨앗:

- 루멘이 셀로에게 거울빛을 놓는 순간, 셀로의 무지갯빛 창이 멈추지 않는다. 다만 창끝이 처음으로 셀로 쪽을 향해 조금 떨린다.
- 멘다키우스가 거울을 보고 웃는다. "나도 사실을 말했습니다." 거울은 그의 말이 아니라, 사실을 배열하던 손가락의 떨림을 비춘다.

## 2. 수호의 방벽

정사 등급: B급에서 A급 승격 후보

| 항목 | 설정 |
|---|---|
| 분류 | 능력/방어/시간 벌기 |
| 소속 | 루멘, 진실의 빛, 불씨 주민들 |
| 작중 별칭 | 수호의 방벽, 흰금빛 방벽, 숨 쉬는 벽, 기다림의 빛 |
| 낮의 형태 | 루멘 앞에 단단한 벽이 생기는 것이 아니라, 빛이 여러 겹의 얇은 천처럼 겹쳐 충격을 늦춘다 |
| 밤의 변형 | 방벽에 닿은 공격의 이유가 희미한 그림자로 떠오른다 |
| 선물 | 공격을 없애기보다 다치지 않고 다음 선택을 할 시간을 만든다 |
| 상처 | 루멘은 누군가를 지키려면 자기 빛이 항상 앞에 있어야 한다고 믿는다 |
| 거짓 믿음 | 막을 수 있다면 내가 계속 막아야 한다 |
| 관련 인물 | 루멘, 셀로, 아미키티아, 리버, 불씨 주민들, 마이더스 잔당 |
| 반작용 | 핵심의 파동, 폭로의 광선, 영혼의 맹약, 작은 등불 |
| 장면 기능 | EP21의 첫 충돌에서 루멘의 방어가 셀로를 정화하지 않고, 잠시 멈추게만 하는 장치 |

### 2.1 방벽의 형태

| 형태 | 쓰임 | 한계 |
|---|---|---|
| 얇은 빛천 | 주민과 약탈자 사이의 첫 충돌을 늦춘다 | 오래 버티면 루멘의 손등이 식는다 |
| 반원 방벽 | 동료가 후퇴할 길을 만든다 | 도망과 회피를 혼동할 수 있다 |
| 낮은 처마 | 아이나 부상자 위에 빛을 낮게 드리운다 | 큰 공격은 막지 못한다 |
| 깨지는 방벽 | 강한 공격을 일부러 부서지게 받아낸다 | 루멘이 패배처럼 느낄 수 있다 |

### 2.2 방벽과 관계

| 대상 | 루멘의 초기 태도 | 성장한 태도 |
|---|---|---|
| 주민 | 내가 막아 주어야 한다 | 스스로 설 시간을 벌어 준다 |
| 동료 | 내 뒤에 있어야 안전하다 | 각자의 능력이 펼쳐질 틈을 만든다 |
| 셀로 | 그의 빛을 막아야 한다 | 셀로가 자기 창끝을 보도록 잠시 거리를 만든다 |
| 자신 | 무너지면 안 된다 | 깨진 방벽 뒤에서도 도움을 청할 수 있다 |

사용 원칙:

- 수호의 방벽은 무적 장치가 아니다. 좋은 방벽은 적을 이기기보다, 누군가가 숨을 고르게 한다.
- 방벽이 깨지는 장면을 두려워하지 않는다. 깨진 방벽은 루멘의 실패가 아니라, 그녀가 혼자 모든 것을 막을 수 없다는 진실이다.
- 방벽 뒤 주민들이 아무것도 하지 않으면 안 된다. 손을 잡고, 문을 열고, 물을 나르는 생활 행동이 함께 있어야 한다.

장면 씨앗:

- 셀로의 핵심의 파동이 방벽을 뚫고 들어오자, 방벽은 부서지는 대신 여러 작은 거울 조각으로 흩어진다. 그 조각마다 다른 사람의 고통이 비쳐 루멘은 한 발 물러난다.
- 불씨 주민 오린은 방벽 뒤에서 처음으로 물값을 받지 않는다. 루멘은 그 작은 선택이 자기 빛보다 오래간다는 사실을 본다.

## 3. 폭로의 광선

정사 등급: A급 전투 보조 능력

| 항목 | 설정 |
|---|---|
| 분류 | 능력/해체/거짓 권위 노출 |
| 소속 | 루멘, 진실의 빛, 눅세린 최종전 |
| 작중 별칭 | 폭로의 광선, 거짓을 벗기는 빛, 그림자 없는 빛 |
| 낮의 형태 | 흰금빛이 곧게 뻗지만, 살을 태우기보다 깃발, 인장, 장식, 허위 문장을 먼저 벗긴다 |
| 밤의 변형 | 빛이 닿은 거짓은 사라지지 않고, 그것이 기대고 있던 작은 진실 한 조각을 먼저 드러낸다 |
| 선물 | 거짓 권위가 왜 설득력을 얻었는지까지 보여 준다 |
| 상처 | 루멘은 거짓을 벗기면 사람도 곧바로 풀려날 거라 믿었다 |
| 거짓 믿음 | 거짓이 드러나면 권력은 즉시 무너진다 |
| 관련 인물 | 루멘, 마이더스, 멘다키우스, 나르키소스, 도그마, 한신 |
| 반작용 | 기만의 장막, 선전 방송탑, 살아있는 조각상, 황금 허기 |
| 장면 기능 | 5부 전투의 시각적 쾌감을 주되, 처벌보다 해체와 책임의 방향을 남긴다 |

### 3.1 대상별 반응

| 대상 | 광선이 벗기는 것 | 남는 것 |
|---|---|---|
| 마이더스의 황금 구조물 | 값비싼 외피와 허위 보증 | 갚지 않은 수고와 빈 그릇 |
| 멘다키우스의 장막 | 사실의 뒤바뀐 순서 | 누군가 속고 싶어 한 이유 |
| 나르키소스의 조각상 | 완벽한 표정과 자세 | 내면을 보였을 때 조롱당한 기억 |
| 도그마의 규약 | 절대 문장의 권위 | 혼란을 막고 싶었던 두려움 |
| 한신의 군중 | 숫자의 위압 | 작게 남으면 무시당할 것이라는 불안 |

### 3.2 금지 방향

| 위험 | 교정 기준 |
|---|---|
| 광선이 적을 소멸시키는 처벌로만 보임 | 벗겨진 뒤 남은 상처를 반드시 보여 준다 |
| 루멘이 모든 거짓을 즉시 판별함 | 빛도 작은 진실이 섞인 거짓 앞에서는 흔들려야 한다 |
| 독자가 적을 조롱하게 됨 | 거짓을 지탱한 결핍과 책임을 동시에 둔다 |

사용 원칙:

- 폭로의 광선은 화려해야 하지만 잔인한 승리감으로 끝나면 안 된다.
- 거짓을 벗긴 뒤 더 어려운 일이 시작된다. 사과, 보상, 거절, 재건이 남아야 한다.
- 루멘의 광선은 멘다키우스의 거짓말보다, 그 말을 믿어야 살 수 있었던 사람들의 표정 앞에서 약해진다.

장면 씨앗:

- 멘다키우스의 금빛 문장이 광선에 벗겨지자, 문장 사이에서 "우리가 이미 버림받은 건 아니라고 말해 줘"라는 작은 요청이 떨어진다.
- 나르키소스의 조각상이 부서질 때, 가장 마지막까지 남은 것은 아름다운 얼굴이 아니라 먼지를 닦던 손등이다.

### 4.1 등불이 다시 켜는 것

| 대상 | 회복 장면 | 의미 |
|---|---|---|
| 닫힌 우물 손잡이 | 한 바퀴 늦게 돌아가며 물 한 컵이 나온다 | 완전한 풍요 전의 작은 생존 |
| 부서진 가게 간판 | 가격 대신 이름이 흐릿하게 보인다 | 거래 이전의 관계 회복 |
| 아이의 색연필 | 회색만 나오던 끝에서 노란 점이 찍힌다 | 선택 감각의 귀환 |
| 빈 화롯가 | 불꽃이 아니라 따뜻한 재가 남는다 | 이미 꺼진 것 안에도 잔열이 있음 |
| 낡은 길표 | 목적지가 아니라 "돌아올 길"만 밝힌다 | 회복은 앞으로만 가는 일이 아님 |

### 4.2 등불과 리타스

| 구분 | 희망의 등불 | 풍요의 홍수 |
|---|---|---|
| 규모 | 한 사람, 한 사물, 한 습관 | 도시와 결핍 언어 전체 |
| 방식 | 남아 있는 불씨를 확인한다 | 넘침으로 가격 언어를 멈춘다 |
| 대가 | 너무 작게 느껴져 루멘이 조급해진다 | 경계가 없으면 착취된다 |
| 연결 | 등불이 먼저 켜진 곳에 홍수가 닿으면 주민이 도망치지 않는다 | 홍수 뒤에도 등불은 생활을 이어 간다 |

사용 원칙:

- 희망의 등불은 "모두가 용기를 얻었다"로 쓰지 않는다. 누가 무엇을 다시 했는지 구체적으로 쓴다.
- 등불은 회색 숨을 끝내지 못한다. 다만 회색 숨 속에서도 아직 꺼지지 않은 지점을 찾는다.
- 루멘은 등불이 작은 것을 부끄러워하다가, 후반에는 그 작음이 오래가는 방식임을 배운다.

장면 씨앗:

- 등불이 켜진 뒤 한 노인이 처음으로 가게 셔터를 다 올리지 않는다. 그는 손바닥 하나만큼 열고, 그 틈으로 물잔을 내민다.
- 루멘이 더 큰 빛을 내려고 하자 아이가 눈을 감는다. 아이가 다시 눈을 뜬 것은 손가락 끝만한 등불이 남았을 때다.

| 인물 | 잘못된 대응 | 성장한 대응 |

- 쇠락 기생체는 끔찍하지만 불쌍해야 한다. 그것은 남을 해치려기보다 붙어 있지 않으면 사라질까 봐 두려운 존재다.
- 전투 뒤에는 "어떻게 이겼나"보다 "무엇이 아직 식어 있나"를 남긴다.

장면 씨앗:

- 루멘의 광선에 망령이 흩어지자, 그 안에서 오래전 쓰다 만 장보기 목록이 떨어진다. 적의 심장 대신 생활이 남는다.
- 아미키티아가 영혼의 맹약을 걸려 하자 기생체는 손목이 없어 맴돈다. 그녀는 처음으로 실 대신 이름을 부른다.

| 소속 | 눅세린 황금 궁전, 선전 방송탑 하층, 멘다키우스 |
| 낮의 형태 | 금빛 문장판, 원문 보관실, 방송관, 말의 도착지를 바꾸는 천막 장치가 겹친 방 |
| 밤의 변형 | 같은 문장이 듣는 사람마다 다른 색으로 도착하고, 원문은 점점 낮은 곳으로 가라앉는다 |
| 선물 | 거짓말보다 위험한 것은 사실의 순서와 도착지를 바꾸는 일임을 보여 준다 |
| 상처 | 사람들은 때로 진실보다 견딜 수 있는 배열을 원했다 |
| 거짓 믿음 | 올바른 원문만 찾으면 모두가 같은 결론에 도달한다 |
| 관련 인물 | 에이도스, 아미키티아, 멘다키우스, 마이더스, 콘코르디아 |
| 반작용 | 홑날, 논리 간섭, 진정한 우정의 빛, 기만의 장막, 원문 보관실 |
| 장면 기능 | EP23 C팀 전투. 에이도스의 분석과 아미키티아의 관계 빛이 함께 있어야 기만을 벗길 수 있음을 보인다 |

### 8.1 공간 구조

| 구역 | 형태 | 장면 기능 |
|---|---|---|
| 금문장 관로 | 문장들이 금빛 관을 타고 도시로 흘러간다 | 말도 기반 시설임을 보여 줌 |
| 원문 보관실 | 가장 낮은 방에 원래 문장이 젖은 종이처럼 쌓여 있다 | 진실은 높지 않고 낮은 곳에 있음 |
| 반응 거울벽 | 사람들이 믿고 싶어 한 표정이 먼저 비친다 | 멘다키우스의 강점 |
| 침묵 탁자 | 말하지 않은 증언이 놓이는 긴 탁자 | 아미키티아의 맹약이 기다려야 하는 자리 |
| 도착지 사다리 | 같은 말이 다른 대상에게 가도록 길을 바꾸는 사다리 | 에이도스의 논리 간섭 전장 |

### 8.2 전투 역할

| 인물 | 초기 방식 | 성장한 방식 |
|---|---|---|
| 에이도스 | 홑날로 거짓 경로를 잘라낸다 | 자르면 안 되는 증언을 남겨 둔다 |
| 아미키티아 | 진정한 우정의 빛으로 모두를 한 영역에 묶으려 한다 | 동의한 증언만 빛에 올린다 |
| 멘다키우스 | 사실 하나를 앞세워 거짓 배열을 숨긴다 | 자기 말의 원문도 잃어 간다 |
| 콘코르디아 | 장막의 파동을 흉내 내려 한다 | 자기 목소리로 첫 반박을 말한다 |

사용 원칙:

- 정보 통제실은 낡은 기계실처럼 보이면 안 된다. 말이 어디로 가고, 누구에게 어떤 모양으로 닿는지를 다루는 궁전의 입이어야 한다.
- 에이도스가 모든 것을 베어 이기면 안 된다. 베지 않는 판단이 그의 성장이다.
- 아미키티아의 빛은 기만을 걷어내지만, 상대를 한 편으로 강제하지 않는다. 허락 없는 연결은 다시 과거의 잘못이 된다.

장면 씨앗:

- 에이도스가 홑날을 들자 금문장 관로가 한순간 조용해진다. 그는 승리를 확신하지만, 잘린 관 끝에서 누군가의 도움 요청도 같이 새어 나오는 것을 본다.
- 아미키티아가 침묵 탁자에 손을 올리자 탁자 위 증언 중 몇 개는 빛나지 않는다. 그녀는 처음으로 빛나지 않는 증언도 존중한다.

### 9.1 능력별 상세

| 능력 | 사용자 | 선물 | 왜곡 위험 | 5부 사용법 |
|---|---|---|---|---|
| 절대 자유 선언 | 리버 | 묶인 길을 끊고 새 경로를 만든다 | 책임 없는 탈주 | 폐허 시장터에서 복제 물량의 길을 흐트러뜨리되 주민의 탈출로를 먼저 연다 |
| 심연 통찰 | 포보스 | 두려움의 가장 깊은 방향을 본다 | 공포로 다시 통제 | 한신이 진짜로 두려워하는 "작아짐"을 찾아 과잉 증식을 멈춘다 |
| 공포 투영 | 포보스 | 경계가 필요한 곳에 어둠을 세운다 | 위협의 재생산 | 주민이 아니라 복제 군중의 길목에만 그림자를 둔다 |
| 공명 파열 | 콘코르디아 | 불협화음의 빈 곳을 흔든다 | 타인의 약점만 흉내 | 조각상의 공허함을 깨되 나르키소스의 상처를 조롱하지 않는다 |
| 세계수 각성 | 실바누스 | 살아 있는 지식이 뿌리로 돌아온다 | 복수의 나무가 됨 | 미술관 바닥을 깨고 전시된 아름다움 아래의 생명 흔적을 되돌린다 |
| 논리 간섭 | 에이도스 | 거짓 배열의 빈틈을 드러낸다 | 사람까지 구조로만 봄 | 정보 통제실에서 말의 길을 바로잡되 증언을 베지 않는다 |
| 진정한 우정의 빛 | 아미키티아 | 동의한 관계를 지킨다 | 우리 편 강제 | 허락한 증언과 손목에만 빛을 잇는다 |

### 9.2 팀별 감정 결산

| 팀 | 표면 승리 | 내면 결산 |
|---|---|---|
| 리버/포보스 대 한신 | 증식 병력 분산 | 자유와 공포가 둘 다 수호가 될 수 있음을 증명 |
| 콘코르디아/실바누스 대 나르키소스 | 조각상 붕괴 | 보이는 아름다움과 살아 있는 기억을 화해 직전까지 밀어붙임 |
| 에이도스/아미키티아 대 멘다키우스 | 기만의 장막 해체 | 진실은 분석과 신뢰가 함께 있어야 도착함 |

사용 원칙:

- 조연 반격은 "각자 필살기를 썼다"로 나열하지 않는다. 각 능력이 그 인물의 상처를 어떻게 바꾸어 쓰는지 보여 준다.
- 간부들도 전부 우스운 적으로 만들지 않는다. 한신은 작아질까 봐, 나르키소스는 보일까 봐, 멘다키우스는 아무도 움직이지 않을까 봐 두려워한다.
- 전투 뒤 조연들이 소진되어 마이더스 최종전에 바로 합류하지 못하는 이유가 감정적으로도 납득되어야 한다.

장면 씨앗:

- 리버가 길매듭을 열자 포보스가 처음으로 그 길의 끝을 막지 않는다. 그는 막는 공포가 아니라 지나간 뒤 닫아 주는 공포가 된다.
- 에이도스가 멘다키우스의 말길을 끊으려 할 때, 아미키티아가 그의 손목을 잡고 묻는다. "그 말 끝에 아직 사람이 있나요?"

### 10.1 기록 조각

| 조각 | 발견자 | 의미 |
|---|---|---|
| 세 번 놓친 별매듭 | 카산드라 | 현재 밤낮 법도가 자연 리듬과 어긋났다는 단서 |
| 물결만 남은 기록판 | 에이도스 | 말로 설명할 수 없는 것을 지우지 않는 용기 |
| 별 없는 별자리 | 로고스 | 법칙이 비어 있을 때도 형태가 남는다는 통찰 |
| 씨앗이 잠든 유리 | 레기나 페쿤디아 | 최초의 순환이 낙원이 아니라 살아 있는 미숙함도 품었음 |
| 울지 않는 신발 | 므네몬 | 봉인이 누군가를 살리려 했던 대가도 가졌음 |
| 검은 달의 흠 | 포보스 | 공포가 한때 닫힌 문을 지켰다는 흔적 |

### 10.2 공개 원칙

| 원칙 | 이유 |
|---|---|
| 한 장면에서 전부 설명하지 않는다 | 밤의 비밀은 해답보다 오래된 상처다 |
| 인물마다 다른 결론을 낸다 | 진실은 같은 빛이어도 서로 다른 책임을 요구한다 |
| 증거와 감각을 함께 둔다 | 기록만 있으면 딱딱하고, 감각만 있으면 흐려진다 |
| 틀린 기록도 남긴다 | 오류가 아니라 당시 누군가의 생존 방식일 수 있다 |

사용 원칙:

- 밤의 비밀 기록은 설정 해설용 문서가 아니다. 독자가 "그래서 이 세계가 아팠구나"를 느끼게 하는 증거 조각이다.
- 에이도스와 로고스가 독점하지 않는다. 카산드라의 불안, 카시안의 청취, 레기나의 씨앗, 포보스의 침묵도 같은 무게로 둔다.
- 기록은 포보스나 므네몬을 면죄하지 않는다. 다만 그들의 선택이 완전히 단순하지 않았음을 보인다.

장면 씨앗:

- 카산드라가 별매듭 지도를 펼치자 세 지점이 동시에 다른 박자를 친다. 로고스가 답을 말하려 할 때, 카시안은 먼저 그 박자를 듣자고 한다.
- 에이도스가 기록판에 "설명 불가"라고 적는다. 그는 부끄러워하지 않고 그 아래에 "그러나 같은 떨림을 세 사람이 보았다"라고 덧붙인다.

사용 원칙:

- 포보스는 "사실 좋은 사람이었다"가 아니다. 그는 나쁜 방식을 써서 정말로 무언가를 막으려 했던 인물이다.
- 독자가 그를 사랑하게 되는 지점은 면죄가 아니라, 자신이 지킨 문 앞에서 가장 오래 떨고 있었다는 사실이다.
- 피해자는 포보스를 용서하지 않아도 된다. 세계관의 윤리는 용서 강요가 아니라 책임의 여지를 남기는 쪽이다.

장면 씨앗:

- 루멘의 빛이 포보스가 닫은 문 앞에서 멈춘다. 문 안에는 괴물이 아니라, 너무 일찍 깨어나면 부서질 기억의 잔향이 있다.
- 포보스는 변명하지 않는다. 다만 검은 달의 흠집을 손으로 가리고, "그날 나는 다른 길을 몰랐다"고 말한다.

| 인물 | 내려놓는 것 | 남는 책임 |
|---|---|---|

### 14.2 의식 절차

| 단계 | 행위 | 의미 |
|---|---|---|
| 이름 부르기 | 내려놓을 대상을 명령이 아니라 이름으로 부른다 | 없애는 것이 아니라 관계를 바꿈 |
| 손 펴기 | 물건이나 빛을 바닥에 두지 않고 손바닥 위에서 힘을 푼다 | 버림과 구분 |
| 빈자리 인정 | 각자 앞의 빈자리를 다른 것으로 바로 채우지 않는다 | 결핍을 견딤 |
| 증언 남기기 | 에이도스나 카산드라가 완성 문장이 아닌 목격을 남긴다 | 기록의 윤리 |
| 다음 역할 선택 | 남겨진 이들이 각자의 작은 책임을 고른다 | 6부 연결 |

사용 원칙:

- 내려놓음 의식은 자기희생을 아름답게 꾸미는 장면이 아니다. 그것은 붙잡으려는 사랑도 진짜였음을 인정한 뒤, 그 방식의 대가를 보는 의식이다.
- 의식은 한 번으로 끝나지 않는다. 6부에서 반복될 작별, 보존, 안식의 원형으로 둔다.
- 내려놓은 뒤에도 그리움은 남아야 한다. 비워진 자리가 아무렇지 않다면 의식은 실패다.

장면 씨앗:

- 프락시스가 창을 내려놓지 못하고 손잡이의 빈 눈금만 하나 더 판다. 그것도 그의 첫 내려놓음이다.
- 므네몬은 봉인 열쇠를 모두 넘기지 않는다. 대신 처음으로 열쇠마다 "누구를 위해 닫았는가"를 적는다.
