역할: 칼르렌 `레야` 씨앗을 돌려치는 훈련장의 "다친 자리를 증언하는 분노와 체면으로 굳은 빠른 손이 충돌하는" 생활 인물로 승격한다.

## 1. 승격 요약

레야는 칼르렌 문서에서 주먹을 들기 전 바닥의 "지킬 것" 선을 보고도 자주 늦게 멈추는 견습 전투인으로 남아 있던 인물이다. 이번 승격에서는 `지킬 것 선`, `태우지 말 것 표`, `세숨 매듭`, `늦은주먹 모래`, `화상읽기 잿가루`, `되돌림 손등띠`를 통해 분노가 실제로 지켜야 할 선을 알리는 힘이면서도, 방향을 잃으면 체면과 복수의 손이 된다는 기준을 정리했다.

| 항목 | 내용 |
|---|---|
| 정사명 | 레야 |
| 소속 | 암디르 칼르렌, 돌려치는 훈련장, 식힘 물통 거리, 재갈림 광장 |
| 핵심 가치 | 분노는 다친 자리를 알리지만, 지킬 것과 태우지 말 것을 나누지 않으면 보호의 얼굴로 상처를 남긴다 |
| 주요 물성 | 지킬 것 선, 태우지 말 것 표, 세숨 매듭, 늦은주먹 모래, 화상읽기 잿가루, 되돌림 손등띠 |
| 밤 변형 | 붉은 모래가 찬 훈련장에서 방향망치와 손목 매듭, 비어 있는 표가 분노의 방향을 묻는다 |

| 왜곡 | 지킬 사람의 이름보다 체면과 복수가 먼저 오고, 빠른 손이 보호의 얼굴로 상처를 남긴다 |

| 녹티오라 흔적 | 발끝을 붙잡는 지킬 것 선, 검은 연기를 내는 태우지 말 것 표, 손목을 감는 세숨 매듭, 눈가로 튀는 늦은주먹 모래, 풀리지 않는 되돌림 손등띠 |

## 3. 밤 변형 설계

| 요소 | 왜 남았는가 | 밤의 변형 | 선택 시험 |
|---|---|---|---|
| 지킬 것 선 | 화가 난 손이 먼저 누구를 지키려는지 말해야 한다는 예법이 남았다 | 지킬 이름을 말하면 낮게 따뜻해지고, 체면만 말하면 발끝을 붙잡는다 | 보호인가, 체면인가 |
| 태우지 말 것 표 | 분노에도 해치지 말아야 할 이름이 있다는 흔적이 남았다 | 적힌 이름은 낮게 보호하지만, 비워 두면 재갈림 접시의 두 칸을 섞는다 | 경계인가, 방치인가 |
| 세숨 매듭 | 즉각 반응과 선택한 반응 사이의 숨이 남았다 | 기다리는 손에서는 느슨해지고, 세 숨 전에 휘두르면 손목을 더 무겁게 감는다 | 기다림인가, 폭발인가 |
| 늦은주먹 모래 | 멈춘 손과 늦은 손의 차이를 손 아래 남기려는 흔적이 남았다 | 멈춘 손 아래 한 알만 남고, 휘두른 손 앞에서는 눈가로 튄다 | 멈춤인가, 흐림인가 |
| 화상읽기 잿가루 | 잘잘못보다 어디가 탔는지 먼저 보려는 예법이 남았다 | 탄 위치를 드러내지만, 죄목을 먼저 쓰면 재덩어리로 굳는다 | 상처인가, 판정인가 |
| 되돌림 손등띠 | 해치기 전 자기 손등에도 무게를 돌려보려는 손길이 남았다 | 자기 손등에 따가움을 돌려보내고, 복수만 말하면 풀리지 않는다 | 책임인가, 되갚음인가 |

## 5. 결투 기능

레야 결투는 화낼 것인가 참을 것인가의 문제가 아니다. 분노는 실제로 다친 자리와 넘어진 선을 빠르게 알려 주고, 누군가를 늦지 않게 지킬 수 있다. 문제는 그 열이 누구를 지키는지 말하기 전에 체면과 복수의 이름으로 굳을 때 생긴다.

| 충돌 | 설명 |
|---|---|
| 다친 자리를 증언하는 분노 | 넘어진 선을 가장 빨리 알리고 약한 사람을 지키지만, 이유를 말하기 전 손이 먼저 도착하면 같은 불 안에 사람을 넣는다 |
| 체면과 복수로 굳은 빠른 손 | 위험 앞에서 필요한 즉각 반응을 가능하게 하지만, 지킬 사람의 이름을 잃으면 공격이 보호의 얼굴을 한다 |

결투 뒤에는 레야의 분노가 사라지지 않는다. 대신 그는 지킬 것 선 앞에서 발을 멈추고, 태우지 말 것 표에 남길 이름을 적은 뒤, 세숨 매듭이 풀릴 때까지 방향망치를 내리지 않는다.
