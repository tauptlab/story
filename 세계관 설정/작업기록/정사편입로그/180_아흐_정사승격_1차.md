# 아흐 정사 승격 1차

작성일: 2026-06-24

역할: 루트 초기 자료의 `아흐` 씨앗을 호수지역의 뱃사람이자 그리움의 안내자로 정리하고, 사랑했던 것을 경멸하거나 잊는 대신 현재의 길표로 접는 접경 생활 인물로 승격한다.

---

## 1. 승격 요약

아흐는 거대한 달 전설이나 사랑의 판정자가 아니라, 호수지역에서 빈자리와 물길을 다루는 뱃사람이다. 이번 승격에서는 `빈자리 노`, `정박못`, `달비늘 찌`, `그리움그물`, `되돌이 등잔`, `물결이끼`, `빈자리갈대`를 통해 그리움이 사람을 붙드는 순간과, 그리움이 현재의 길표가 되는 순간을 분리했다.

| 항목 | 내용 |
|---|---|
| 정사명 | 아흐 |
| 소속 | 특수공간-호수지역, 빈자리 선착장, 되돌이 등탑, 현재물결 나루 |
| 정사 등급 | B급 상위 접경 생활 인물 |
| 핵심 가치 | 사랑했던 것을 경멸하지 않고도 떠날 수 있으며, 그리움은 현재 길의 표식이 될 수 있다 |
| 주요 물성 | 빈자리 노, 정박못, 달비늘 찌, 그리움그물, 되돌이 등잔, 물결이끼, 빈자리갈대 |
| 밤 변형 | 배가 같은 물길만 돌고 등탑이 뒤쪽을 비추는 밤에서, 빈자리를 남긴 채 다음 나루로 떠날 수 있는지 묻는다 |

## 2. 상위 기준 반영

| 기준 | 아흐 반영 |
|---|---|
| A가 버린 가치관 | 그리움은 약함이나 미련이 아니라, 사랑했던 것이 남긴 의미를 현재의 길에 데려오는 힘일 수 있다는 믿음 |
| 버린 이유 | 그리움은 너무 아팠고, 떠올리면 다시 멈출 것 같았기 때문이다 |
| 대체 가치관 | 잊거나 경멸해야 움직일 수 있고, 떠난 것을 현재에서 끊어내야 산다는 습관 |
| 처음의 보호 | 미움과 잊음은 당장의 고통을 줄였고, 멈춘 발을 다시 움직이게 했다 |
| 왜곡 | 시간이 지나 사랑했던 것의 온도와 약속까지 미움 속에 묻히고, 빈자리는 지워야 할 흠처럼 취급되었다 |
| 다시 받아들여야 하는 가치 | 그리움을 현재의 길표로 접고, 잃은 것을 같은 모습으로 되찾으려 하지 않은 채 앞으로 가는 힘 |
| 녹티오라 흔적 | 같은 물길만 도는 배, 깊게 박히는 정박못, 건너편을 지우는 달비늘 찌, 발밑을 비추는 되돌이 등잔 |
| 귀환 계기 | 메리의 경멸, 세리엘의 상처 입은 믿음, 에레시아의 따뜻한 보존, 에밀드의 빈 답장, 리버의 떠나는 힘이 아흐 앞에서 그리움과 현재 길의 경계를 묻는다 |

## 3. 밤 변형 설계

| 요소 | 왜 남았는가 | 밤의 변형 | 선택 시험 |
|---|---|---|---|
| 빈자리 노 | 부재를 억지로 채우지 못했던 기억이 강하게 남았다 | 대신 들 사람이 정해지면 물에 가라앉고, 빈자리로 인정되면 배 안쪽에서 낮게 흔들린다 | 부재인가, 대체인가 |
| 정박못 | 당장 떠나면 무너질 것 같아 잠시 머물러야 했던 시간이 남았다 | 얕게 박히면 손으로 뽑히지만, 돌려받기 명령 앞에서는 물속 깊이 박힌다 | 쉼인가, 묶임인가 |
| 달비늘 찌 | 사랑했던 얼굴과 말이 물 위의 작은 반짝임처럼 오래 남았다 | 오래 바라보면 건너편 기슭을 지우고, 짧게 건지면 다음 물길의 표식이 된다 | 기억인가, 정지인가 |
| 그리움그물 | 놓치기 싫은 말을 버리지 못한 마음이 남았다 | 세게 당기면 현재의 목소리까지 묶고, 물에 헹구면 길표처럼 가벼워진다 | 보존인가, 붙잡기인가 |
| 되돌이 등잔 | 뒤돌아보아야 할 때와 떠나야 할 때를 구분하지 못한 밤이 남았다 | 울어야 할 때 뒤쪽을 비추고, 떠날 때가 오면 발밑 나루를 밝힌다 | 애도인가, 명령인가 |
| 물결이끼 | 빈자리 곁에서도 길이 다시 자랄 수 있다는 작은 믿음이 남았다 | 빈자리를 인정하면 물빛을 내고, 미움으로 덮으면 회색으로 마른다 | 인정인가, 덮기인가 |
| 빈자리갈대 | 누군가 대신 앉으려 할 때 말없이 흔들린 자리의 감각이 남았다 | 빈 의자를 강제로 채우면 소리를 죽이고, 비워 두면 물길 방향으로 눕는다 | 존중인가, 대리인가 |

## 5. 결투 기능

아흐 결투는 그리워할 것인가 말 것인가의 문제가 아니다. 사람은 때로 그리움 때문에 무너지고, 때로 그리움 덕분에 미워하지 않을 수 있다. 문제는 그리움이 현재를 멈추는 명령이 되는 순간이다.

| 충돌 | 설명 |
|---|---|
| 경멸하거나 잊어 앞으로 가는 힘 | 너무 아픈 기억에서 발을 빼게 하지만, 사랑했던 것의 선물과 약속까지 미움 속에 묻을 수 있다 |
| 그리움을 길표로 보존하는 힘 | 잃은 것을 지우지 않게 하지만, 같은 모습으로 돌려받으려 하면 현재의 사람과 길을 과거의 모양에 맞추려 든다 |

결투 뒤에는 아흐의 호수가 마르지 않는다. 다만 정박못은 얕게 다시 박혀 머무름과 묶임을 구분하고, 되돌이 등잔은 뒤쪽 얼굴이 아니라 다음 나루의 발밑을 비추며, 빈자리 노는 배 안쪽에 남은 채로도 배가 움직인다.

## 6. 다음 후보

아흐는 `180`, 켈은 `181` 기준으로 1차 승격했다. 메리와 지그몬드는 `182` 기준으로 독립 관계 신화 승격을 완료했고, 루지에로는 `183` 기준으로 독립 전승 인물 승격을 완료했다. 다음 정사 승격 후보는 `미니`가 적절하다. 미니는 루지에로가 흔들릴 때 곁에 남던 옛 친구 씨앗으로, 우정이 돌봄인지 붙잡기인지 정리할 수 있다.
