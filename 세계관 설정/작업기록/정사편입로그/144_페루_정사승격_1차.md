# 페루 정사 승격 1차

역할: 드레오렌 생활 인물 `페루`를 웃음보자기 빨래터의 세탁인으로 승격하고, "웃음 뒤 젖은 말을 짜내지 않는 예법"을 폭로하지 않는 돌봄과 영원한 보류의 위험 사이에서 정리한다.

---

## 1. 승격 요약

페루는 기존 자료에서 드레오렌 웃음보자기 빨래터 세탁인으로 배치되어 있었다. 이번 승격에서는 그를 단순히 젖은 보자기를 말리는 단역이 아니라, 웃음 뒤 남은 말을 찢지 않는 선의와 그 말을 주인에게 돌려주지 못하는 위험을 함께 가진 생활 인물로 정리했다.

| 항목 | 기존 씨앗 | 144 기준 정사화 |
|---|---|---|
| 인물 기능 | 젖은 보자기를 짜지 않고 그늘에서 오래 말리는 세탁인 | 폭로하지 않는 돌봄과 영원한 보류의 위험을 함께 보이는 생활 인물 |
| 주 무대 | 드레오렌 웃음보자기 빨래터 | 웃음보자기 빨래터, 그늘빨랫줄, 접힌선이끼 둔덕, 정면문 마당 |
| 핵심 상처 | 젖은 말을 너무 빨리 펼쳐 찢어진 장면 | 완전히 마를 때까지는 돌려주면 안 된다는 믿음 |
| 핵심 물성 | 웃음보자기 | 그늘빨랫줄, 보류집게, 접힌선이끼, 젖은끝 매듭 |

---

## 2. 124 기준 적용

| 기준 | 페루의 처리 |
|---|---|
| 버린 가치관 | 젖은 말도 돌봄을 거쳐 다시 주인의 손으로 돌아갈 수 있다는 믿음 |
| 보호하던 대체 가치관 | 웃음보자기, 보류, 그늘, 오래 말리기 |
| 왜곡된 가치관 | 말은 찢어지지 않지만 주인에게 돌아가지 못하고 빨래터의 무게가 된다 |
| 다시 받아들일 가치 | 폭로하지 않되, 완전히 마르기 전에도 주인의 손에 돌려주는 책임 |
| 녹티오라 흔적 | 끝없이 늘어나는 그늘빨랫줄, 무거워지는 웃음보자기, 짙어지는 접힌선이끼 |
| 귀환 계기 | 페루가 보자기를 계속 말리려는 순간, 주인이 젖은끝 매듭을 직접 잡아도 된다는 사실을 드러내는 장면 |
| 귀환 후 후폭풍 | 드레오렌은 웃음보자기를 짜내지 않지만, 일정한 때가 오면 주인에게 젖은끝 매듭을 돌려준다 |

## 3. 녹티오라 설계

| 요소 | 왜 강하게 남았는가 | 밤 변형 | 선택 시험 |
|---|---|---|---|
| 그늘빨랫줄 | 밝은 곳에 말하면 문장이 찢어질 수 있었기 때문이다 | 너무 밝히면 천이 딱딱해지고, 그늘에 두면 첫 문장이 떠오른다 | 보호인가, 은폐인가 |
| 보류집게 | 보자기를 누르지 않고 붙잡아 둘 작은 손이 필요했기 때문이다 | 세게 물리면 멍이 들고, 느슨하면 물방울 길을 만든다 | 보류인가, 붙잡음인가 |
| 접힌선이끼 | 오래 묶인 말의 접힌 자국이 먼저 눅눅해졌기 때문이다 | 영원히 묶이면 짙어지고, 주인이 한쪽 끝을 잡으면 물빛 선을 남긴다 | 기다림인가, 유기인가 |
| 젖은끝 매듭 | 완전히 마르지 않아도 주인의 손으로 돌아갈 수 있어야 했기 때문이다 | 주인이 쥐면 가벼워지고, 남이 대신 풀면 다시 무거워진다 | 반환인가, 방치인가 |

## 5. 결투 의미

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 젖은 말을 폭로하지 않고 말리는 돌봄 vs 말린 뒤 주인에게 돌려주는 책임 |
| 말리는 돌봄의 옳음 | 준비되지 않은 말을 밝은 곳에 펼치면 말은 해명보다 상처로 찢어질 수 있다 |
| 말리는 돌봄의 위험 | 끝없이 말리기만 하면 보호는 영원한 보류가 된다 |
| 돌려주는 책임의 옳음 | 어떤 말은 완전히 마르지 않아도 주인의 손으로 돌아가야 한다 |
| 돌려주는 책임의 위험 | 너무 빨리 돌려주면 주인은 아직 감당할 수 없는 물기를 혼자 떠안게 된다 |
| 결투 뒤 재배치 | 페루는 보자기를 짜내지 않고, 젖은끝 매듭을 묶어 주인의 손에 건넨다 |
| 도시 변화 | 드레오렌의 웃음보자기 빨래터는 폭로하지 않는 보류와 주인에게 돌려주는 책임을 함께 가진다 |
