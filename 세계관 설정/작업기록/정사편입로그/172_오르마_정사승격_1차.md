작성일: 2026-06-24

역할: 칼르렌 `오르마` 씨앗을 식힘 물통 거리의 "화난 사람을 부끄럽게 만들지 않는 낮춤과 다친 선을 분명히 말하는 책임이 충돌하는" 생활 인물로 승격한다.

## 1. 승격 요약

오르마는 칼르렌 문서에서 말다툼이 나면 먼저 물을 밀어 주되, "화내지 말라"는 말을 하지 않는 물지기로 남아 있던 인물이다. 이번 승격에서는 `먼저물 바가지`, `두손 물자리`, `손끝열선 수반`, `방향갈대`, `부끄럼가림 수건`, `검은물 비움잔`을 통해 분노를 부끄러운 결함으로 만들지 않고, 물 위에 손의 방향을 드러내는 기준을 정리했다.

| 항목 | 내용 |
|---|---|
| 정사명 | 오르마 |
| 소속 | 암디르 칼르렌, 식힘 물통 거리, 두손 물자리, 손끝열선 수반 난간 |
| 핵심 가치 | 화난 사람을 모욕하지 않고 잠시 손을 낮추되, 낮아진 열 위에 다친 선과 지킬 이름을 분명히 드러낸다 |
| 주요 물성 | 먼저물 바가지, 두손 물자리, 손끝열선 수반, 방향갈대, 부끄럼가림 수건, 검은물 비움잔 |
| 밤 변형 | 검게 식는 물통과 끊기는 손끝 열선, 기우는 물자리, 빙글 도는 방향갈대가 낮춤과 모욕의 차이를 묻는다 |

| A가 버린 가치관 | 화난 자신도 모욕당하지 않은 채 잠시 낮아질 자리를 가질 수 있다는 믿음 |
| 버린 이유 | 화를 낮추라는 말이 자주 다친 말을 지우고 부끄럽게 만드는 명령으로 쓰였기 때문이다 |
| 대체 가치관 | 부끄러워지기 전에 더 크게 반응해야 하고, 낮추는 말은 곧 굴복이라는 습관 |
| 처음의 보호 | 모욕당하기 전 먼저 선을 세우게 했고, 화난 이유를 눌러 삼키지 않게 했다 |
| 왜곡 | 큰 반응은 처음에는 모욕을 막지만, 모든 물자리를 굴복으로 읽으면 손의 방향을 볼 기회가 사라진다 |
| 다시 받아들여야 하는 가치 | 먼저물 바가지와 두손 물자리로 모욕을 낮추고, 손끝열선 수반 위에 남은 방향을 보고 말할 선을 고르는 책임 |
| 녹티오라 흔적 | 검게 식는 물통, 기우는 두손 물자리, 끊기는 손끝 열선, 빙글 도는 방향갈대, 무거워지는 수건, 다시 뜨거워지는 빈 잔 |
| 귀환 계기 | 레야의 빠른 손, 카르도의 식는 시간, 하스렌의 화상표, 페이단의 큰 불씨, 라카의 겁난 주먹이 오르마의 식힘 물통 앞에서 낮춤과 모욕의 경계를 묻는다 |

## 3. 밤 변형 설계

| 요소 | 왜 남았는가 | 밤의 변형 | 선택 시험 |
|---|---|---|---|
| 먼저물 바가지 | 판정 전에 모욕을 낮출 물을 먼저 내미는 손길이 남았다 | 입을 막는 물로 쓰이면 가장자리가 갈라지고, 물을 받은 사람의 목이 더 뜨거워진다 | 배려인가, 입막음인가 |
| 두손 물자리 | 양쪽 손이 같은 자리에서 잠시 낮아져야 한다는 예법이 남았다 | 한쪽 손만 올리면 기울고, 비어 있는 쪽 물이 더 뜨거워진다 | 함께 낮춤인가, 한쪽 굴복인가 |
| 손끝열선 수반 | 손이 어디로 가려 했는지 보려는 흔적이 남았다 | 지킬 이름을 말하면 얇은 선을 띄우고, 비웃음이 섞이면 검게 번진다 | 방향인가, 조롱인가 |
| 방향갈대 | 화난 손의 방향을 부끄러워하지 않고 보려는 예법이 남았다 | 손의 방향을 숨기면 빙글 돌고, 지킬 이름을 말하면 낮게 눕는다 | 확인인가, 몰아세움인가 |
| 부끄럼가림 수건 | 얼굴을 숨기지 않아도 잠시 눈을 낮출 수 있는 자리가 남았다 | 잠시 눈을 낮출 때는 가볍지만, 상처를 덮는 핑계가 되면 어깨를 누른다 | 존중인가, 덮음인가 |
| 검은물 비움잔 | 비웃음과 모욕만 따로 비워 내야 한다는 흔적이 남았다 | 비웃음만 비우면 식지만, 화 자체를 버리려 하면 다시 뜨거워진다 | 모욕 비움인가, 분노 삭제인가 |

## 5. 결투 기능

오르마 결투는 화를 낼 것인가 참을 것인가의 문제가 아니다. 화는 이미 다친 자리를 가리키고 있다. 문제는 그 화가 부끄러움으로 눌려 사라지거나, 반대로 비웃음과 모욕을 태우는 말로 변할 때 생긴다.

| 충돌 | 설명 |
|---|---|
| 화를 부끄럽게 만들지 않는 낮춤 | 화난 사람을 굴복시키지 않고 손의 가장자리를 낮추지만, 부드러운 회피가 되면 넘어간 선을 흐린다 |
| 다친 선을 분명히 말하는 책임 | 보호와 복수, 요청과 공격을 구분하게 하지만, 때와 자리를 묻지 않으면 말이 다시 모욕이 된다 |

결투 뒤에는 오르마의 물통 거리에서 물을 마시는 일이 침묵 명령이 되지 않는다. 먼저물 바가지가 모욕을 낮추고, 손끝열선 수반이 방향을 드러내며, 검은물 비움잔은 화가 아니라 비웃음만 따로 비운다.
