# 3부 중위 누락카드 상세초안

이 문서는 `27_3부_통합요소_추출과_누락카드.md`에 남은 3부 중위 후보를 실제 장면에 투입 가능한 설정 카드로 확장한다. `28_3부_상위누락카드_상세초안.md`가 3부의 핵심 사건과 상위 감정축을 다룬다면, 이 문서는 그 사건 사이를 살아 있는 세계처럼 보이게 만드는 의식, 광장, 잔류 상태, 폐허, 전투체, 보조 능력, 작중어를 담당한다.

3부 중위 카드의 핵심 원칙은 "큰 상처는 한 번의 승리로 끝나지 않고, 장소와 물건과 제도와 작은 몸짓에 잔류한다"는 것이다. 따라서 아래 항목들은 장면의 배경 장식이 아니라, 셀로의 조급한 구원, 카시안의 느린 조율, 페이단의 분노 귀환, 오블라투스의 무명 노동, 팔수스의 냉소, 이놉스의 무력감이 세계 곳곳에 남긴 생활 흔적이다.

## 1. 균형 회복 의식

정사 등급: B급에서 A급 승격 후보

| 항목 | 설정 |
|---|---|
| 분류 | 의식/제도/공동 조율 절차 |
| 소속 | 센시르 리테노아, 대공명실과 조정의 광장 |
| 낮의 형태 | 각 당사자가 자기 고통을 얇은 현, 물그릇, 숨결 돌에 맡기고 카시안 계열 조율자가 순서대로 듣는 공개 의식 |
| 밤의 변형 | 아직 듣지 못한 고통이 검은 매듭으로 남고, 성급히 자르면 매듭이 침묵의 껍질로 튄다 |
| 선물 | 고통을 크기순으로 줄 세우지 않고, 서로 다른 절박함을 동시에 놓아둔다 |
| 상처 | 과거 리테노아가 너무 오래 듣다가 늦게 움직인 기억 |
| 거짓 믿음 | 충분히 듣기만 하면 모두가 받아들일 답이 자연히 나타난다 |
| 관련 인물 | 카시안, 셀로, 넥서스 후원자들, 고요의 정원지기들 |
| 반작용 | 심판의 영역, 대공명실, 세르 불균형 정산 |
| 장면 기능 | 셀로가 느린 조율을 견디지 못하는 첫 균열을 만든다 |

의식 단계:

| 단계 | 내용 | 장면 기능 |
|---|---|---|
| 숨 등록 | 당사자들이 자기 이름보다 먼저 숨의 박자를 남긴다 | 발언권이 신분이 아니라 살아 있음에서 시작됨 |
| 고통 분리 | 각 고통을 하나의 현이나 물그릇에 따로 둔다 | 섞으면 안 되는 상처가 있음을 보여줌 |
| 반향 대조 | 서로의 고통이 닿을 때 생기는 떨림을 듣는다 | 어느 한쪽의 정당성이 다른 쪽의 삭제를 뜻하지 않음 |
| 보류 선언 | 답이 없으면 의식을 끝내지 않는다 | 카시안의 선물과 약점이 동시에 드러남 |
| 책임 배분 | 회복이 아니라 우선 부담 순서를 정한다 | 세르 불균형 정산으로 이어짐 |

사용 원칙:

- 의식은 아름다운 평화 장면이 아니라, 모두가 답답할 만큼 느리고 구체적인 절차여야 한다.
- 셀로가 이 의식을 무너뜨릴 때 독자는 답답함과 두려움을 동시에 느껴야 한다. 빠른 판결이 매혹적이어야 이후 실패가 깊어진다.
- 카시안은 절차를 사랑해서가 아니라, 절차 없이 누군가가 지워지는 것을 두려워하기 때문에 버틴다.

공간 요소:

| 요소 | 형태 | 장면 기능 |
|---|---|---|
| 숨결 수로 | 광장 바닥을 도는 얕은 물길 | 말보다 먼저 몸의 떨림을 보여줌 |
| 보류석 | 아무도 앉지 않는 작은 빈자리 | 아직 답하지 못한 사람의 몫 |
| 조율 종 | 발언을 끝낼 때 울리는 낮은 종 | 말을 끊는 것이 아니라 다음 숨으로 넘김 |
| 얼룩 벽 | 실패한 조정의 흔적이 남은 벽 | 센시르도 매번 성공하지 못했음을 보여줌 |

사용 원칙:

- 조정의 광장에는 구경꾼이 있어야 한다. 세계는 선한 당사자들만으로 구성되지 않는다.
- 광장의 위험은 폭력이 아니라, 고통이 공론장 언어로 포장되며 생활의 날것을 잃는 데 있다.
- EP11에서 광장은 셀로가 "왜 이렇게 오래 말만 하는가"라고 느끼게 만드는 공간이어야 한다.

| 항목 | 의미 | 장면 사용 |
|---|---|---|
| 구해진 생명 | 사업이 실제로 만든 선물 | 프락시스와 후원자들의 정당성 |
| 닳은 울림 | 영혼 침식증과 회색 숨 | 대가의 핵심 |
| 생태 부채 | 고요의 이끼와 정원 손상 | 정원지기들의 분노 |

사용 원칙:

- 정산은 처벌 명령이 아니라 책임의 지도다.
- 장부가 완벽해질수록 오히려 "그래도 갚을 수 없는 것"이 선명해져야 한다.
- 프락시스가 이 장부를 두려워하는 이유는 실패가 드러나서가 아니라, 성공까지도 대가와 함께 적히기 때문이다.

## 4. 침묵의 껍질

정사 등급: A급 보조 상태

| 항목 | 설정 |
|---|---|
| 분류 | 상태/무기물화/강제 치유 피해 |
| 소속 | 센시르 대공명실 붕괴, 고요의 이끼 강제 사용 후유증 |
| 낮의 형태 | 피해자의 피부와 눈가에 얇은 반투명 막이 생기고, 말은 들리지만 응답이 늦어진다 |
| 밤의 변형 | 막 안쪽에서 작은 빛이 밖으로 나오지 못하고, 주변 소리도 껍질 표면에서 미끄러진다 |
| 선물 | 너무 큰 고통이 몸 전체를 부수지 않도록 임시로 막는다 |
| 상처 | 막이 오래가면 고통뿐 아니라 욕구와 반항도 밖으로 나오지 못한다 |
| 거짓 믿음 | 조용해졌다면 안정된 것이다 |
| 관련 인물 | 셀로, 카시안, 고요의 정원지기들, 영혼 침식증 피해자 |
| 반작용 | 고요의 이끼, 영혼 침식증, 균형 회복 의식 |
| 장면 기능 | 셀로의 성급한 판결이 남긴 피해를 시각화한다 |

구분 규칙:

| 항목 | 침묵의 껍질 | 영혼 침식증 | 무지갯빛 수정체 |
|---|---|---|---|
| 원인 | 심판 실패와 강제 치유 | 넥서스 후폭풍 질병 | 셀로의 결정화 상태 |
| 피해 위치 | 응답과 의지의 표면 | 선택과 감정의 깊이 | 모든 파동의 흡수 |
| 회복 방향 | 껍질을 깨기보다 안쪽 응답을 기다림 | 작은 선택 회복 | 셀로의 통합 |

사용 원칙:

- 침묵의 껍질을 "돌처럼 굳은 피해자"로만 쓰지 않는다. 안쪽에 아직 반응이 있음을 작은 빛, 손끝 떨림, 늦은 눈물로 보여준다.
- 껍질을 강제로 깨면 피해자가 회복되는 것이 아니라, 고통이 날것으로 터진다.
- 이 상태는 셀로가 이후 자기 왕관의 영역으로 도망치는 죄책감의 물증이다.

| 씨앗 | 정사 기능 |
|---|---|

사용 원칙:

- "희망"을 큰 빛으로 보여주기보다, 누군가 버리지 않고 주머니에 넣어두는 행동으로 보여준다.

| 상처 | 실패한 이상을 근거로 모든 가능성을 조롱한다 |

- 로고스는 파편 안의 거꾸로 된 문장을 읽어, 팔수스가 왜 냉소에 도달했는지 드러낼 수 있다.

## 20. 장면 설계 원칙

1. 3부 중위 카드는 사건을 설명하는 보조 주석이 아니라, 독자가 세계를 만지는 손잡이다.
2. 의식과 제도는 선한 장치로만 쓰지 않는다. 느린 조율도 누군가에게는 폭력처럼 느껴진다.
3. 폐허와 전투체는 악의 장식이 아니라, 이전 선택이 물질화된 결과로 쓴다.
4. 팔수스 계열 능력은 거짓말만 하지 않는다. 일부 맞는 말로 독자까지 불편하게 만들어야 한다.
5. 리버와 포르티스의 능력은 큰 승리보다 "다시 한 걸음"의 감각을 우선한다.
6. 포보스의 공포는 사라지는 것이 아니라 위치가 바뀐다. 위협에서 경계로, 가학에서 수호로 전환된다.
7. 모든 회복 단서는 즉시 발아하지 않아도 된다. 늦게 자라는 설정일수록 세계의 시간이 깊어진다.
