# 벨키르 정사 승격 1차

작성일: 2026-06-23

역할: 루트 초기 자료의 `벨크` 씨앗을 `벨키르`로 이름 분리하고, 신뢰 거래소의 "관계를 거래로 오해한 생활 인물"로 승격한다.

---

## 1. 승격 대상

벨키르는 구판 자료에서 `벨크`로 남아 있던 인물 씨앗이다. 구판 벨크는 애정에 굶주려 모든 관계에서 우위를 차지하려 했고, 약함을 나누는 일을 부담 전가로 오해했다. 이번 승격에서는 그 기능을 살리되, 벨크세스와의 이름 충돌을 피하기 위해 정사명 `벨키르`로 분리했다.

| 항목 | 내용 |
|---|---|
| 정사명 | 벨키르 |
| 구판명 | 벨크 |
| 소속 | 눅세린 신뢰 거래소, 보증 홀, 호의 접수대, 무보증 대기실 |
| 정사 등급 | B급 상위 생활 인물 |
| 핵심 가치 | 관계의 무게, 갚음 강박, 비대칭 관계의 회복 |
| 주요 물성 | 관계 영수증, 부담 저울, 호의 장부, 보증 홀 의자, 무보증 번호표 |

## 2. 이름 충돌 처리

| 충돌 | 처리 |
|---|---|
| 벨크 | 구판 후보명으로만 보존한다 |
| 벨키르 | 신뢰 거래소 생활 인물의 정사명으로 사용한다 |
| 벨크세스 | 큰 물질주의 축의 이름으로 남긴다 |
| 독자 혼동 방지 | 본문과 정사 카드에서는 `벨키르`만 사용하고, `벨크`는 추출표와 승격 기록의 괄호 표기에서만 쓴다 |

## 3. 상위 기준 반영

| 기준 | 벨키르 반영 |
|---|---|
| A가 버린 가치관 | 관계는 정확히 같은 양으로 갚지 않아도 유지될 수 있다는 믿음 |
| 버릴 수밖에 없었던 이유 | 고마움이 요구로, 부탁이 부담으로, 친절이 나중의 청구로 돌아온 기억이 있었기 때문이다 |
| 대체 가치관 | 먼저 갚기, 먼저 손해 보기, 모든 관계의 무게를 장부로 맞추는 습관 |
| 보호 기능 | 한쪽만 착취당하는 관계를 피하고, 말뿐인 다정함의 위험을 빨리 보게 해 주었다 |
| 왜곡 | 모든 약함과 부탁이 짐으로 보이고, 진심이 도착하기 전 관계 영수증이 먼저 붙는다 |
| 다시 받아들일 가치 | 서로 다른 날의 무게를 묻고, 오늘 더 기댄 쪽이 내일 다른 방식으로 곁에 남을 수 있다는 믿음 |
| 녹티오라 흔적 | 길어지는 관계 영수증, 기울어진 부담 저울, 숫자를 지워도 다시 차오르는 호의 장부 |
| 귀환 계기 | 나리안의 도움끈, 미라솔의 빈보증 표, 벨키르의 관계 영수증이 맞물려 도움과 관계의 무게를 다시 배치한다 |

## 4. 밤 변형 설계

| 요소 | A의 기억에 남은 이유 | 밤 변형 | 시험 |
|---|---|---|---|
| 관계 영수증 | 친절 뒤에 무엇을 갚아야 할지 몰라 손이 굳은 순간이 남았다 | 부탁 하나마다 길어지고, 갚을 방법을 모르면 손바닥에 달라붙는다 | 기록인가, 청구인가 |
| 부담 저울 | 누가 더 많이 참고 있는지 빨리 알아차려야 안전했던 기억이 남았다 | 약함을 말할수록 기울지만, "어디까지 들까"라는 말 앞에서 기울어진 채 멈춘다 | 공정인가, 죄목인가 |
| 호의 장부 | 고마움과 빚의 경계가 흐려진 기억이 이름 옆에 남았다 | 숫자를 지우면 더 큰 숫자를 만들지만, 빈칸을 존중하면 페이지가 얇아진다 | 갚음인가, 여백인가 |
| 보증 홀 의자 | 같은 무게로 앉아야 관계가 안전하다고 믿던 몸의 기억이 남았다 | 서로 같은 무게가 아니면 등을 돌리고, 한 사람이 먼저 기다리면 조금씩 같은 방향을 본다 | 균형인가, 동행인가 |
| 무보증 번호표 | 기록이 없으면 아무도 믿어 주지 않을 것 같던 불안이 남았다 | 불러 주지 않으면 무거워지고, 이름을 묻는 순간 손에서 떨어진다 | 번호인가, 이름인가 |

## 6. 결투 기능

벨키르 결투는 관계를 계산하느냐 마느냐의 단순 대립이 아니다. 관계의 무게를 보는 힘은 필요하지만, 그 힘이 모든 약함을 죄목으로 만들 때 세계가 굳는다.

| 충돌 | 설명 |
|---|---|
| 관계의 공정한 무게 계산 | 한쪽만 계속 희생하는 관계를 드러내지만, 모든 부탁과 사과를 청구서로 만들 수 있다 |
| 약함을 나누는 비대칭 관계 | 서로 다른 날의 무게를 함께 들 수 있음을 보이지만, 허락과 한계를 묻지 않으면 착취로 굳을 수 있다 |

결투 뒤에는 신뢰 거래소의 관계 영수증이 사라지지 않는다. 대신 숫자 옆에 빈칸이 생기고, 벨키르는 그 빈칸을 억지로 채우지 않는 첫 계산사가 된다.
