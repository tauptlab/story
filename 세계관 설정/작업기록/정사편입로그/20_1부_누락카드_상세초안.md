# 1부 누락 카드 상세 초안

이 문서는 `19_1부_통합요소_추출과_누락카드.md`에서 상위 우선도로 분류한 누락 요소를 실제 정사 카드로 승격하기 위한 상세 초안이다. 아직 최종 원고가 아니라 설정 성경 단계의 작업물이므로, 각 항목은 향후 `07`, `09`, `10`, `11`, `12`, 도시별설정 문서로 분산 반영한다.

## 1. 길목의 등불

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 장소/건축물 |
| 소속 | 눅세린-암디르 접경, 이름 없는 마을 |
| 낮의 형태 | 낮은 천장, 오래된 등잔, 서로 다른 나라의 동전이 박힌 계산대가 있는 작은 선술집 |
| 밤의 변형 | 등잔불이 손님마다 다른 색으로 흔들리고, 거짓말이 많은 사람의 그림자는 문밖에 먼저 나간다 |
| 은유 | 낯선 마음이 처음으로 타인의 소문과 공포를 듣는 중간지대 |
| 관리 인물 | 등불지기 마르네. 값을 묻기 전에 손님이 앉을 의자가 있는지부터 본다 |
| 관련 인물 | 셀로, 에이도스, 리버, 밤의 순찰자, 레마니스, 소키우스 |
| 장면 기능 | 세계 설명을 대화로 밀어넣지 않고, 손님들의 말버릇과 거래 습관으로 보여준다 |
| 대가/위험 | 오래 머물면 자신이 들은 소문을 자기 기억처럼 믿게 된다 |

활용 규칙:

| 상황 | 사용법 |
|---|---|
| EP1 초반 | 셀로가 "누구의 편도 아닌 장소"에서 처음으로 편을 요구받는다. |
| 밤 전환 직전 | 안대를 쓴 손님들은 계산대 가까이, 쓰지 않은 손님들은 문가에 앉는다. |
| 리버 등장 전조 | 누군가가 리버를 범죄자처럼 말하지만, 마르네는 그가 늘 빈 잔을 되돌려놓는다고 덧붙인다. |
| 후반 회수 | 셀로가 다시 이곳을 떠올릴 때, 첫 공포보다 첫 의자가 먼저 생각난다. |

독자가 붙잡을 디테일:

- 마르네는 깨진 잔을 버리지 않고 등잔 받침으로 쓴다.
- 손님이 이름을 말하지 않으면 계산대의 동전 하나가 흐려진다.
- 가장 오래된 등불은 눅세린 금빛도 암디르 붉은빛도 아닌, 아주 약한 흰빛을 낸다.

## 2. 밤의 순찰자

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 집단/민간 질서 |
| 소속 | 눅세린-암디르 접경, 녹티오라 통행로 |
| 선물 | 밤에 길을 잃은 사람을 찾아내고, 공포에 잠식된 문을 닫아준다 |
| 상처 | 과거에 "그냥 지나가게 둔" 한 무리가 밤에 사라진 기억 |
| 거짓 믿음 | 밤에는 자유보다 통제가 먼저다 |
| 상징 소품 | 검은 천 안대, 손잡이가 짧은 등창, 이름을 적지 않는 통행패 |
| 사랑스러운 습관 | 순찰 전 서로의 신발끈을 확인한다. 끈이 풀린 사람은 그날 앞줄에 서지 않는다 |
| 관련 인물 | 리버, 셀로, 에이도스, 길목의 등불 마르네 |
| 반작용 | 리버의 방향 있는 자유, 셀로의 존재의 방벽, 에이도스의 열린 질문 |
| 장면 기능 | 안전을 바라는 마음이 어떻게 통제로 굳는지 보여준다 |

조직 구조:

| 역할 | 설명 |
|---|---|
| 등잡이 | 앞줄에서 등창을 들고 길의 균열을 확인한다. |
| 끈지기 | 일행의 발목끈과 통행패를 확인한다. 보호자와 감시자의 경계에 선다. |
| 문닫이 | 밤에 변형된 문, 창, 골목을 닫는다. 닫힌 안쪽에 누가 남았는지는 잘 묻지 않는다. |

갈등 설계:

- 이들은 나쁜 사람이 아니라 "다시는 잃지 않으려는 사람들"이어야 한다.
- 리버를 싫어하는 이유는 그가 위험해서만이 아니라, 그가 자신들이 포기한 자유를 닮았기 때문이다.
- 셀로는 처음에 이들의 통제를 이해하지 못하지만, 나중에는 이들이 지키려 한 것이 무엇인지도 보게 된다.

| 항목 | 설정 |
|---|---|

| 선물 | 누구의 손길이 들어간 작품인지 만져보면 안다 |

## 4. 시그마-7 소녀

정사 등급: B급

작업명: 리네아. 확정 전까지는 `시그마-7 소녀`와 병기한다.

| 항목 | 설정 |
|---|---|
| 분류 | 인물/생활 단역 |
| 소속 | 프론티아 시그마-7 |
| 공식 표기 | 7-LN. 시그마-7의 기록판에는 이름보다 번호가 먼저 적힌다 |
| 선물 | 정해진 줄 안에서도 작은 빈칸을 찾아 타인에게 건넨다 |
| 상처 | 규격에서 벗어난 친절 때문에 여러 번 교정 명령을 받은 기억 |
| 거짓 믿음 | 들키지 않을 만큼만 다정해야 살아남는다 |
| 사랑스러운 습관 | 음식을 반으로 나눌 때 정확히 반이 아니라, 배고파 보이는 쪽에 조금 더 준다 |
| 능력/특징 | 규율선의 미세한 끊김을 눈치챈다. 하지만 그 틈을 큰 탈출로 쓰기에는 아직 겁이 많다 |
| 관련 인물 | 셀로, 에이도스, 리버, 프레펙투스, 교정 집행체 |
| 반작용 | 셀로의 무정형 세르, 리버의 속박 파쇄, 로고스의 느린 설명 |
| 장면 기능 | 프론티아의 폭력을 한 명의 잃어버린 이름으로 체감하게 한다 |

장면 씨앗:

- 리네아는 셀로에게 음식을 나눠주며 "규정상 남은 것"이라고 말하지만, 사실은 자기 몫을 덜어낸 것이다.
- 교정 집행체가 다가오자 그는 먼저 접시의 각도를 바로잡는다. 공포가 몸에 습관으로 배어 있음을 보여준다.
- 리버가 규율선을 끊는 장면에서, 리네아는 도망치기보다 바닥에 떨어진 작은 빵 조각을 먼저 줍는다. 이 사소함이 독자에게 남는다.

후반 회수:

- 시그마-7 해방 이후 리네아가 처음으로 자기 이름을 쓰는 장면은 긴 연설보다 작은 기록판 하나면 충분하다.
- 그는 완전히 용감해지지 않아도 된다. 겁이 남아 있는 채로 누군가에게 한 숟가락 더 주는 인물로 남으면 좋다.

## 5. 질문 의식의 방들

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 분류 | 건축/의식 구조 |
| 소속 | 암디르 그림자 골짜기, 포보스의 탑 |
| 낮의 형태 | 무너진 탑 내부의 서로 맞지 않는 방들, 끊긴 계단, 둥글게 놓였지만 닫히지 않는 석판 |
| 밤의 형태 | 질문마다 구조가 바뀌는 세 개의 방. 방은 대답을 듣는 것이 아니라 대답 방식에 반응한다 |
| 은유 | 공포는 하나의 문장이 아니라 각자의 몸에 맞게 변형된 질문이다 |
| 관련 인물 | 포보스, 셀로, 에이도스, 리버, 콘코르디아 |
| 대가 | 한 방을 통과해도 다음 방에서 이전 답이 다시 시험된다 |
| 장면 기능 | 주연들의 차이를 말이 아니라 공간 통과 방식으로 보여준다 |

세부 방:

| 방 | 낮의 흔적 | 밤의 질문 | 주연별 반응 |
|---|---|---|---|
| 미완성 원의 방 | 원형으로 놓이다 만 석판 | 부족을 없애지 않고 통과할 수 있는가 | 셀로는 빈칸을 메우려 하지 않고 곡선으로 지나간다. |
| 사각 분리실 | 네 면이 서로 다른 두께의 돌벽 | 혼자 답할 것인가, 서로의 답을 기다릴 것인가 | 에이도스는 방의 모서리를 읽고 타인의 방을 추론한다. |
| 무게 없는 추의 방 | 천장에 매달린 빈 고리 | 보이지 않는 무게를 어떻게 나눌 것인가 | 리버는 추를 부수는 대신 방향을 바꾼다. |
| 한 음의 방 | 조각난 가면, 붉은 실, 검은 반달 | 같아지지 않고 함께 울릴 수 있는가 | 콘코르디아의 실패와 회복 씨앗이 동시에 놓인다. |

설계 원칙:

- 포보스의 방은 잔혹하기보다 정확해야 한다.
- 답이 맞아도 상처는 남는다. 대신 상처가 닫힌 문이 아니라 다음 문을 여는 손잡이가 된다.

## 6. 흑요석 뿔 조각

정사 등급: A급 하위 카드

| 항목 | 설정 |
|---|---|
| 분류 | 무기물/흑요석 파편 하위 변종 |
| 소속 | 암디르 그림자 골짜기, 포보스의 가면 계열 |
| 낮의 물성 | 작은 뿔처럼 굽은 흑요석 조각. 빛을 받으면 끝부분만 아주 얇게 은색으로 보인다 |
| 밤의 변형 | 놓인 방향이 조금씩 바뀌며, 피해야 할 곳이 아니라 마주해야 할 곳을 가리킨다 |
| 은유 | 공포가 남긴 날카로운 방향감 |
| 관련 인물 | 포보스, 셀로, 에이도스, 로고스, 프레펙투스 |
| 대가 | 오래 쥐고 있으면 자신이 두려워하는 방향만 선명해진다 |
| 파손 방식 | 깨지면 더 작은 조각들이 서로 다른 방향을 가리켜 판단을 흐린다 |
| 회복 방식 | 누군가 두려운 방향으로 한 걸음 걸으면 조각의 끝이 덜 날카로워진다 |
| 장면 기능 | 암디르에서 프론티아로 향하는 사건 동기를 물질로 고정한다 |

가면과의 관계:

- 흑요석 일식 가면은 얼굴을 숨기는 장치다.
- 흑요석 뿔 조각은 숨은 얼굴이 바라보던 방향을 남긴 파편이다.
- 둘을 같은 것으로 쓰면 안 된다. 가면은 정체의 문제, 뿔 조각은 방향의 문제를 담당한다.

## 7. 개념 회로도 석판

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 건축/무기물 |
| 소속 | 잊혀진 지식의 평원, 프론티아 외곽 폐허 |
| 낮의 물성 | 얇은 선과 작은 구멍이 새겨진 회백색 석판. 발소리가 닿으면 선이 잠깐 밝아진다 |
| 밤의 변형 | 선들이 공중으로 떠올라 걷는 사람의 생각 길을 미리 그리려 한다 |
| 은유 | 이해하려는 마음이 구조가 될 때 생기는 아름다움과 위험 |
| 일상 사용자 | 로고스 계열 기록자, 프론티아 옛 설계자, 시그마-7 수리공 |
| 관련 인물 | 로고스, 소포스, 에이도스, 프레펙투스 |
| 대가 | 석판을 오래 읽으면 아직 생각하지 않은 결론을 이미 낸 것처럼 느낀다 |
| 파손 방식 | 강제로 부수면 선들이 흩어져 주변 바닥에 거짓 지름길을 만든다 |
| 회복 방식 | 모르는 것을 모른다고 인정한 질문이 새겨질 때 빈 선이 안정된다 |
| 장면 기능 | 지식이 책 속 문장만이 아니라 도시의 바닥, 벽, 폐허에 남는다는 세계감을 만든다 |

소포스와의 차이:

- 로고스는 석판을 읽어도 빈칸을 남긴다.
- 소포스는 빈칸을 참지 못하고 선을 끝까지 이어 완결된 세계의 장으로 만든다.

| 분류 | 유기물/현상 생물 |
| 서식지 | 시그마-7의 규율선 틈, 잊혀진 지식의 평원 하부, 오래된 기록판 뒤 |
| 낮의 모습 | 아주 작은 반투명 벌레. 직선 위에서는 잘 보이지 않고, 금 간 곳에서만 빛난다 |
| 밤의 변형 | 한 마리가 발견되면 주변 규칙의 작은 예외들이 다리처럼 자라난다 |
| 세르 반응 | 억지로 지운 예외와 억눌린 질문에 반응해 번식한다 |
| 돌보는 인물 | 정식으로 돌보는 이는 없다. 다만 리네아는 몰래 쓸어내지 않고 작은 틈으로 옮겨준다 |
| 관련 인물 | 소포스, 프레펙투스, 리네아, 리버, 셀로 |
| 대가 | 방치하면 구조를 무너뜨리지만, 모두 없애면 도시가 숨 쉴 틈도 잃는다 |
| 장면 기능 | 오류가 단순한 결함이 아니라 생명의 틈일 수 있음을 보여준다 |

사용 규칙:

- 오류 증식 벌레는 귀엽게만 쓰지 않는다. 실제로 위험하다.
- 그러나 위험하다는 이유로 모두 박멸하면 프론티아가 더 잔혹해진다.
- 셀로가 이 벌레를 보고 "없애야 할 것"과 "살려둘 틈"을 구분하기 시작하면 좋다.

## 9. 히프노레아 전통 의복

정사 등급: B급

| 항목 | 설정 |
|---|---|
| 분류 | 소품/문화 물건 |
| 소속 | 히프노레아, 로고스의 여행 보따리 |
| 낮의 형태 | 겉감은 무채색이지만 안감에 작은 이름매듭이 촘촘히 박힌 긴 겉옷 |
| 밤의 변형 | 입은 사람이 잊고 있던 말들이 안감의 매듭 사이에서 따뜻한 숨처럼 올라온다 |
| 은유 | 기억은 갑옷이 아니라 몸을 식지 않게 하는 옷이다 |
| 관련 인물 | 로고스, 셀로, 에이도스, 므네몬, 회상나비 |
| 대가 | 옷이 오래 품은 기억이 너무 많으면 입은 사람의 현재 감각이 흐려진다 |
| 회복 방식 | 새 기억을 억지로 새기지 않고, 지나간 이름을 한 번 불러주면 매듭이 느슨해진다 |
| 장면 기능 | 프론티아의 차가운 질서에서 히프노레아의 기억 세계로 넘어가는 부드러운 연결 장치 |

디테일:

- 로고스는 옷을 접을 때 안감이 바깥으로 보이게 접는다. 기억은 숨기는 것이 아니라 식지 않게 덮는 것이라는 습관이다.
- 셀로가 처음 입으면 옷이 조금 크다. 아직 자기 기억보다 타인의 기억을 더 많이 걸치고 있기 때문이다.
- 후반에 옷이 맞아가는 순간은 셀로가 자기 이름의 무게를 받아들이는 장면과 연결할 수 있다.
