# 카르도 대장간 장인 물성화 1차

작성일: 2026-06-25

역할: 카르도와 칼르렌 진홍 대장간 계열의 외래어식 소품명을 대야, 집게, 받침돌, 실, 종의 장인 물성으로 정리한 작업 로그다.

---

## 1. 작업 기준

- 카르도는 강한 고유 특수능력자가 아니라 B급 상위 생활 인물이다. 따라서 소품을 전투 기술명으로 승격하지 않는다.
- 사물 고유명은 유지한다. 대야, 집게, 받침돌, 실, 종을 외래어로 대체하지 않는다.
- 기다림, 책임, 보호 같은 작가용 의미는 이름에 직접 넣지 않고, 물김이 낮아지고 종이 울리며 손잡이실이 끊어지는 반응으로 보인다.
- 레야/오르마/하스렌 계열과 같은 칼르렌 물성이지만, 카르도는 장인의 손과 건넬 때를 중심으로 별도 배치한다.

## 4. 후속 메모

- `물김대야`는 열을 없애는 세척 대야가 아니라, 막 만든 도구의 이름을 사람이 잡을 수 있을 만큼 낮추는 대야다.
- `손올림종`은 출정 신호가 아니라, 더 늦추면 도구 이름이 물밑으로 가라앉는 순간을 알리는 종이다.
- 하스렌의 `팔마 화상표/카르보 붓/카우사 온도먹/스티그마 천/레스타 따가움실`은 후속 로그 `263`에서 처리 완료했다.
