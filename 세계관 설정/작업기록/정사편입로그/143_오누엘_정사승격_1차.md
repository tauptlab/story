# 오누엘 정사 승격 1차

역할: 드레오렌 생활 인물 `오누엘`을 정면문 마당의 문지기로 승격하고, "한 문장만 요구하는 예법"을 작은 직접 말의 용기와 작은 강요의 위험 사이에서 정리한다.

---

## 1. 승격 요약

오누엘은 기존 자료에서 드레오렌 정면문 마당 문지기로 배치되어 있었다. 이번 승격에서는 그를 단순히 돌아온 사람을 맞는 단역이 아니라, 큰 고백을 감당 가능한 한 문장으로 낮추는 선의와 그 한 문장마저 통행료처럼 굳는 위험을 함께 가진 생활 인물로 정리했다.

| 항목 | 기존 씨앗 | 143 기준 정사화 |
|---|---|---|
| 인물 기능 | 정면으로 말하지 못한 사람에게 되돌이 길실을 건네는 문지기 | 작은 직접 말과 허락의 경계를 지키는 생활 인물 |
| 주 무대 | 드레오렌 정면문 마당 | 정면문 마당, 한문장 못대, 가능문장 패 보관대, 옆문거리 |
| 핵심 상처 | 큰 설명을 요구받은 사람이 다시 옆길로 사라진 장면 | 한 문장으로 줄이면 누구나 문을 넘을 수 있다는 믿음 |
| 핵심 물성 | 정면문 손잡이, 한문장 못 | 가능문장 패, 문턱등잔, 손잡이온이끼 |

## 2. 124 기준 적용

| 기준 | 오누엘의 처리 |
|---|---|
| 버린 가치관 | 직접 말해도 관계의 문턱에서 완전히 밀려나지 않을 수 있다는 믿음 |
| 보호하던 대체 가치관 | 옆문, 돌림말, 농담, 침묵처럼 직접 말을 늦추는 우회 |
| 왜곡된 가치관 | 직접 말의 문턱이 너무 멀어져, 돌아오려는 사람도 어떤 말부터 놓아야 할지 모른다 |
| 다시 받아들일 가치 | 큰 고백이 아니라 오늘 가능한 한 문장으로 돌아오는 용기 |
| 녹티오라 흔적 | 끝없이 넓어지는 정면문 마당, 차가운 손잡이, 흐려지는 문턱등잔 |
| 귀환 계기 | 가능문장 패를 든 사람이 아무 말도 적지 못하고, 오누엘이 한 문장 요구 자체가 문턱이 되었음을 알아차리는 장면 |
| 귀환 후 후폭풍 | 드레오렌은 한 문장을 요구하지 않고, 빈 패와 "아직 못 말해"도 마당에 놓을 수 있게 한다 |

## 3. 녹티오라 설계

| 요소 | 왜 강하게 남았는가 | 밤 변형 | 선택 시험 |
|---|---|---|---|
| 가능문장 패 | 큰 설명 대신 오늘 가능한 말 하나만 들고 서고 싶었던 감각 때문이다 | 주인이 적으면 따뜻해지고, 남이 대신 적으면 글자가 뭉개진다 | 직접 말인가, 통행료인가 |
| 문턱등잔 | 문을 열기 전 문장이 요구인지 허락인지 확인해야 했기 때문이다 | 자발적 문장 앞에서는 낮게 따뜻해지고, 강요 앞에서는 유리가 흐린다 | 허락인가, 압박인가 |
| 손잡이온이끼 | 문을 열지 못해도 곁에 머문 시간이 손잡이 아래에 남았기 때문이다 | 기다림이 놓이면 빛나고, 통행료가 되면 바싹 마른다 | 머묾인가, 배제인가 |
| 한문장 못 | 큰 고백보다 감당 가능한 작은 말이 필요했기 때문이다 | 긴 고백에는 휘고, 작은 말이나 빈 패에는 못머리만 낮게 빛난다 | 시작인가, 강요인가 |

## 5. 결투 의미

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 관계로 돌아가기 위해 필요한 작은 직접 말 vs 작은 말조차 강요가 되지 않아야 하는 허락 |
| 작은 직접 말의 옳음 | 큰 고백을 한 문장으로 낮추면 돌아오는 일이 가능해진다 |
| 작은 직접 말의 위험 | 한 문장이 통행료가 되면 사람은 자기 말 대신 문이 받을 말을 고른다 |
| 허락의 옳음 | 어떤 날의 가능한 문장은 "아직 못 말해"일 수 있다 |
| 허락의 위험 | 허락만 남고 돌아올 문턱이 없으면 관계는 다시 끝없이 미뤄진다 |
| 결투 뒤 재배치 | 오누엘은 가능문장 패를 비워 둘 수 있게 하고, 문턱등잔과 손잡이온이끼가 마르지 않을 때까지 곁에 앉는다 |
| 도시 변화 | 드레오렌의 정면문 마당은 한 문장을 받아 내는 곳이 아니라, 오늘 가능한 문장을 함께 낮추는 곳이 된다 |
