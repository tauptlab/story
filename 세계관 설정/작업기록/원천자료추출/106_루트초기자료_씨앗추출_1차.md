# 루트초기자료 씨앗추출 1차

이 문서는 `105_루트초기자료_운용주석확장_1차.md` 다음 단계다. 루트 초기 자료에서 아직 정사 카드로 완전히 흡수되지 않은 이름, 장소, 귀환 사건을 1차로 추려 둔다.

이번 문서는 확정 카드가 아니다. 목적은 버릴 씨앗과 살릴 씨앗을 구분하고, 살릴 경우 어느 국가, 도시, 인물 문서로 옮길지 정하는 것이다.

---

## 1. 추출 원칙

| 원칙 | 적용 |
|---|---|
| 이름만 먼저 건진다 | 오래된 설명은 그대로 쓰지 않는다 |
| 현재 정사 이름과 부딪히면 새 이름을 붙인다 | 이미 다른 역할을 가진 이름은 재사용하지 않는다 |
| 한 후보는 한 기능만 맡긴다 | 도시, 인물, 사건, 물건 역할을 섞지 않는다 |
| 국가 감각을 먼저 맞춘다 | 현재 국가별 설정과 도시별설정의 문장 결을 우선한다 |
| 실제 원고 투입은 나중에 판단한다 | 지금은 카드화 후보와 연결 위치만 정한다 |

## 4. 대형 구판 서사 씨앗

| 씨앗 | 살릴 수 있는 핵 | 주의 |
|---|---|---|
| 아침 왕가와 밤 왕가 | 낮과 밤 사이를 오가던 오래된 왕가 비극 | 현재 1부~6부 본줄기와 직접 병합하지 말고 전설 또는 외전 후보로 둔다 |
| 메리와 지그몬드 | 사랑, 왕권, 구출 실패, 오해가 겹친 관계 | 호수지역 별편지 전승. 현 주연 구도와 직접 병합하지 않는 독립 관계 신화로 `182` 기준 승격 완료 |
| 루지에로 | 자기 사랑이 질투와 소유로 기울어진 옛 왕 후보 | 호수지역 공로비침 여울의 독립 전승 인물. 셀로와 기능을 분리해 `183` 기준 승격 완료 |
| 미니 | 루지에로가 흔들릴 때 곁에 남던 옛 친구 | 호수지역 곁자리 마루의 접경 생활 인물. 우정이 돌봄인지 예전 모양으로 붙잡는 일인지 정리해 `184` 기준 승격 완료 |
| 켈 | 강해지게 해 주지만 의존을 키우는 인물 또는 도구 | 특수공간-호수지역 반김대여 나루의 후견/도구 인물. `181` 기준 개별 승격 완료 |
| 아흐 | 호수 지역의 안내자 또는 그리움의 인물 | 특수공간-호수지역의 빈자리 뱃사람. `180` 기준 개별 승격 완료 |
| 카라카사 | 힘과 정의를 동일시하는 옛 패권자 | 암디르/아르스 모투스 접경 옛 승리터 전설 인물. `179` 기준 전설 승격 완료 |
| 브라다만테 | 현재 정사에서는 이미 포보스의 망치 역할 | 인물명으로 되살리지 않는다. 필요하면 새 이름을 붙인다 |
