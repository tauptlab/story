# 상위 구판 문서 운용 주석 1차

## 1. 목적

이 문서는 `세계관 설정` 최상위에 남아 있는 초기 설계 문서들이 정사 성경보다 위에 놓이지 않도록 운용 기준을 고정한 기록이다.

구판 문서는 삭제하지 않는다. 오래된 발상, 관계 방향, 이름의 흔적은 이야기의 뿌리로 남긴다. 다만 실제 원고와 정사 요약에는 해당 문서의 설명 방식을 그대로 옮기지 않는다.

## 2. 이번에 주석을 단 문서

| 문서 | 보존할 것 | 원고 전환 기준 |
|---|---|---|
| `세계관 설정/세계관.md` | 전체 구조의 초기 씨앗, 국가/층계/순환의 큰 방향 | `00`, `01`, `02`, `15`, `16`, `76`을 우선하고, 바깥 대응 설명은 장소·규약·물성·선택의 대가로 바꾼다 |
| `세계관 설정/밤 (Noctiora).md` | 녹티오라의 밤 변형, 관리된 악몽, 포보스의 수문장 역할 | `밤의 결`, `뒤틀린 순환`, `회색 숨`, `상처 입은 꿈의 관리 장치` 같은 작중어로 변환한다 |
| `세계관 설정/국가이름.md` | 국가명, 도시명, 배경 배치의 초기 표 | `02_국가별_도시_건축_생활_아틀라스.md`, `국가별 설정/*.md`, `도시별설정/*.md`를 우선한다 |

## 3. 국가별 설정 폴더 판정

`세계관 설정/국가별 설정` 폴더는 현재 정사 문체로 사용할 수 있는 국가 요약 폴더다. 이번 점검에서 직접 노출 위험어 검색이 나오지 않았고, 프론티아와 암디르 문서처럼 정사 기준, 국가 감각, 생활 문화, 도시 연결이 산문형으로 정리되어 있다.

따라서 국가별 설정 폴더는 구판 격리 대상이 아니라, `02`와 도시별 문서를 이어 주는 보조 기준으로 둔다.

## 4. 실제 집필 전 사용 순서

상위 구판 문서에서 무언가를 가져와야 할 때는 아래 순서를 지킨다.

1. 구판 문서에서 이름과 관계 방향만 뽑는다.
2. `15`에서 작중어를 찾는다.
3. `02` 또는 국가별 설정에서 장소와 생활 규칙을 확인한다.
4. 인물과 연결되면 개별 인물 문서의 정사 카드와 대조한다.
5. 설명 문장이 남으면 장면 행동, 물성, 금기, 소품 흔적으로 바꾼다.

이 기준으로 보면 구판 문서는 위험한 충돌원이 아니라, 정사 세계를 더 두껍게 만드는 씨앗 창고가 된다.
