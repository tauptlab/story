# 모순 2차 파일별 점검

이 문서는 `08_모순_정리_로그.md`의 1차 모순 후보를 현재 파일 상태 기준으로 다시 분류한다. 목적은 "설정이 틀렸다"를 찾는 것이 아니라, 이미 정사 문서에서 해결된 항목과 향후 집필 전까지 남아 있는 설정 위험을 분리하는 것이다.

---

## 1. 상태 표기

| 상태 | 의미 |
|---|---|
| 해결 완료 | 정사 성경 또는 개별 인물 문서에 기준이 반영되어 추가 결정이 필요 없음 |
| 부분 해결 | 정사 방향은 정해졌지만 구판 설정, 세부 회차, 향후 집필 설계에는 아직 확인/반영이 필요함 |
| 원고 점검 필요 | `웹소설작성` 본문에 직접 반영되었는지 아직 확인해야 함 |
| 설정 추출 필요 | 부분 구체화 문서에서 요소를 뽑아 정사 카드/도감/아틀라스에 연결해야 함 |
| 설정 설계 필요 | 실제 원고 작성이 아니라 향후 집필을 위한 위치, 기능, 회수 방향을 설계해야 함 |
| 부분 구체화 확인 | `웹소설작성` 또는 `장면시퀀스`의 샘플에서 설정 기능을 확인했으나 최종 연재문 완료로 보지 않음 |
| 구판 정리 필요 | 과거 설정 문서에 직접어, 오래된 표현, 새 정사와 다른 문체가 남아 있어 정사 성경 기준으로 정리해야 함 |
| 기준 문서 | 다른 문서를 판단할 때 우선 적용할 상위 기준 |

## 4. 현재 가장 위험한 잔류 모순

### 4.1 포보스 원고 체감 문제

정사 문서에서는 포보스가 수호자/관리자라는 기준이 잡혔다. 하지만 원고에서 독자가 초반 포보스를 "그냥 가학적인 악역"으로만 기억하면 후반 반전이 약해진다.

반드시 보강할 흔적:

- 죽일 수 있는데 죽이지 않는 선택
- 질문 의식이 처벌이 아니라 선별/관리라는 작은 단서
- 풀을 밟지 않는 발끝
- 기억의 돌과 불꽃에 대한 조심스러운 태도
- 셀레네의 목소리 또는 문장 흔적

`74_포보스초반수호자단서_점검_1차.md`에서 현재 단서와 부족한 단서를 분리했다. 부분 원고 기준으로 질문 의식과 풀잎 증언은 이미 좋고, 실제 연재문에서는 셀레네 풍경 소리와 죽이지 않는 손짓을 소량 추가하는 방향이 가장 안전하다.

### 4.2 검은 달과 흑요석의 시각적 혼동

정사 기준:

- 흑요석 파편: 포보스 개인 유물의 파손 단서
- 흑요석 일식 가면: 포보스의 생존 정체성
- 검은 달: 프론티아가 공포 물질을 질서 장치로 차용한 인공물

원고에서는 색만으로 구분하면 약하다. 검은 달은 프론티아식 계산식, 순은 격자, 허가된 방향감을 함께 보여줘야 한다.

### 4.3 태고/파생 설명의 직접 노출

설정 문서에서는 태고와 파생을 설명해도 되지만, 원고에서는 "이 인물은 어떤 바깥 기능의 파생이다" 같은 문장이 나오면 세계의 마법이 깨진다. 세부 기준은 `76_태고파생_구판문서_운용기준_1차.md`를 따른다.

원고 기준:

- 태고의 힘 -> 오래된 세르, 첫 물결, 왕좌 아래의 숨
- 파생 인물 -> 특정 경험이 세르와 엉겨 생긴 얼굴
- 통합 -> 삼켜짐이 아니라 서로를 통과시키는 공명

### 4.4 신규 확장 인물의 기능 중복

새 인물이 많아졌기 때문에 각자의 장면 기능을 겹치지 않게 유지해야 한다.

| 인물 | 겹치면 안 되는 대상 | 구분 기준 |
|---|---|---|
| 살리엔 | 포보스 | 살리엔은 위험의 크기를 가르는 주의, 포보스는 공포를 관리하는 수호자 |
| 프리오르 | 카우투스 | 프리오르는 가능성 예측, 카우투스는 부정적 미래 앞에서 행동을 늦추는 신중함 |
| 에라타 | 리버 | 에라타는 틀림을 통한 배움, 리버는 방향 있는 자유 |
| 에토스 | 프락시스 | 에토스는 살아 있는 반복, 프락시스는 결과를 향한 강제 실행 |
| 아우토노에 | 파불라 | 아우토노에는 자기 삶의 첫 문장, 파불라는 사실 사이에 놓인 이야기 |
| 인터라 | 살루스 | 인터라는 몸의 신호를 듣는 인물, 살루스는 쉬어야 할 때를 알려주는 안식 |
| 아나트 | 이놉스 | 아나트는 고정 자아를 느슨하게 하는 놓아두기, 이놉스는 아무것도 할 수 없다는 무력감 |

`에피소드/5부` EP21~25는 `34_5부_통합요소_추출과_누락카드.md`에서 1차 추출되었고, 상위 카드 9개는 `35_5부_상위누락카드_상세초안.md`에서, 중위 카드 14개는 `36_5부_중위누락카드_상세초안.md`에서 상세화되었다. 핵심 위험은 루멘을 만능 구원자로 만들거나, 셀로를 새 악역으로 고정하거나, 태초의 메아리/최초의 순환/위대한 왜곡을 설정 해설로만 소비하거나, 5부 전장을 단순 배경으로 소비하는 점이다.

확인 기준:

- 살루스, 레툼, 쿠스토스 세넥스는 5부에서 답처럼 등장하지 않고, 내려놓음과 새 순환의 전조로만 남긴다.

## 6. 당장 확정해도 되는 기준

- 새 문서나 원고에서 충돌이 나면 `정사 성경` 폴더의 번호가 높은 최신 점검 문서를 우선 참조한다.
- 인물 해석은 개별 `등장인물/*.md`의 `정사 카드 요약`을 우선한다.
- 소품/물질 해석은 `07_핵심소품_장치_개별카드.md`와 `12_무기물_핵심물질_2차카드.md`를 우선한다.

- 구판 문서의 바깥 학문식 용어는 설정 연구용으로만 허용하고, 본문에는 옮기지 않는다.
