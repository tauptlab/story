# 등장인물 근원축 잔류 표식 확인 1차

## 1. 목적

`16_모순_2차_파일별점검.md`에 남아 있던 `기존 사유/근원 축 인물` 항목이 실제 위험인지, 이전 점검에서 남은 표식인지 확인한 기록이다.

## 2. 확인 결과

`등장인물` 폴더 전체에서 직접 노출 위험어를 다시 검색했고, 현재 추가 정리가 필요한 항목은 나오지 않았다.

확인 중 발견된 표현은 주로 `핵심 은유`, `상징`, `내면`, `바깥` 같은 설정 카드용 표지다. 이들은 실제 원고 문장이 아니라 인물 문서의 정리 방식이므로, 지금 단계에서는 구판 위험으로 보지 않는다.

## 3. 운용 기준

개별 인물 문서의 `핵심 은유`는 집필자용 표지로 둔다. 실제 원고에서는 아래 방식으로 바꾼다.

| 설정 카드 표지 | 원고 변환 |
|---|---|
| 핵심 은유 | 첫 등장 몸짓, 피하는 장소, 반복하는 말끝 |
| 상징 | 손에 쥔 소품, 자주 어기는 금기, 주변 사람이 붙인 별명 |
| 내면 | 행동 전 멈춤, 못 보는 방향, 끝내 하지 못하는 말 |
| 바깥 | 국경, 문턱, 닫힌 문, 돌아오지 않는 길 |

## 4. 판정

`기존 사유/근원 축 인물` 항목은 해결 완료로 본다.

남은 일은 문서 정리가 아니라 실제 집필 단계의 장면화다. 인물 문서의 표지를 원고 문장으로 옮길 때는 설명문이 아니라 물성, 선택, 대가, 침묵으로 바꾼다.
