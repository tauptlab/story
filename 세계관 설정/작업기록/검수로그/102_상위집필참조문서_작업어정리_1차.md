## 3. 이번에 남겨 둔 범위

- `15_작중어_전용_설정집.md`는 변환표 성격이 강하므로 별도 정리 대상으로 둔다. 이 문서는 과거 말과 정사 말을 함께 보여 주는 역할이 있어, 무조건 삭제하면 검수 기준을 잃을 수 있다.
- 구판 로드맵, 바깥 해설 문서, 오래된 루트 문서는 정사 성경 본류와 분리해서 다룬다.
- `99_고도화_작업_현황과_완성_체크리스트.md`에는 예전 작업 이력 문장이 많이 남아 있으므로, 새로 추가하는 줄은 정사 표시를 따르되 과거 전체 이력 문장 정리는 별도 작업으로 둔다.

## 4. 후속 기준

실제 연재문을 작성하기 전에는 아래 순서로 본다.

1. `06`과 `100`에서 회차권과 장면 투입 후보를 확인한다.
2. 이번 문서에 적힌 정사 표시를 우선한다.
3. 구판 초안에서 다른 이름이 나오면 곧바로 새 요소로 만들지 않고, 기존 정사 표시의 하위 흔적인지 확인한다.
4. 회색 숨, 카산드라 연락망, 넥서스 사업은 설명어보다 물건과 손버릇으로 먼저 보인다.

현재 기준으로 상위 집필 참조 문서 11개는 이번 작업어 검수 기준을 통과했다.
