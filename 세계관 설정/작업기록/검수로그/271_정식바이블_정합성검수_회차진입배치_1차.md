# 정식바이블 정합성 검수와 회차 진입 배치 1차

작성일: 2026-06-25

역할: 정식바이블 01~07 문서만으로 현재 정사를 파악할 수 있는지 점검하고, 실제 연재 진입용 회차 배치를 상위표로 재정렬한다.

---

## 1. 검수 결론

| 항목 | 결론 |
|---|---|
| 01~07 문서만으로 현재 정사 파악 가능 여부 | 가능. 단, `07_회차_투입_색인.md`가 작업 장부 어투였으므로 정식 회차 투입 상위표로 재작성했다 |
| 인물/도시/도감 충돌 | 라쿠스 최근 계열 기준으로 치명 충돌 없음. 메리/지그몬드, 아흐, 켈, 루지에로, 미니, 시몬의 정식명은 도시 문서와 도감에 정렬됨 |
| 작업 로그 전용 설정 | 작업 로그의 구판명과 임시명은 보존하되, 정식 참조에 흡수되지 않은 경우 원고 기준으로 쓰지 않는 원칙을 07에 명시함 |
| 회차 배치 | 부별 첫 노출, 핵심 결투, Mnemosyn 귀환 빌드업, 라쿠스 물가 투입 순서를 `07_회차_투입_색인.md`에 정리함 |

## 3. 운영 원칙

- 실제 원고 작성자는 `07_회차_투입_색인.md`를 우선 본다.
- 07A~07F는 세부 추출과 후보 배치를 확인할 때만 연다.
- 작업 로그의 숫자 코드와 구판명은 근거 표식이지 작중명이나 현재 정식명이 아니다.
- 한 회차에는 새 설정 하나, 물성 하나, 인물 손버릇 하나를 넘기지 않는 것을 기본값으로 둔다.
- Mnemosyn 귀환은 5부 EP24~25에서 조건이 확정되지만, 빌드업은 1부부터 빈자리와 대체 가치의 왜곡으로 누적한다.
