# 도시물성 도감교차감사 1차

정사 등급: 작업 장부

역할: 62개 `도시별설정` 문서의 소품, 생태, 무기물, 잔류물 표에 등장하는 이름들이 도감 계열 문서와 어떻게 연결되어 있는지 점검하고, 반복 등장하거나 결말부 장면 기능이 큰 후보를 우선 회수한 기록이다.

## 2. 판정 원칙

- 반복 등장한 항목은 도감 계열에 이름을 직접 올린다.
- 한 번만 등장해도 결말부, 근원부, 국가 핵심 상처와 연결되면 도감 계열에 올린다.
- 한 도시에서만 쓰이는 작은 물건은 도시 문서 안에서 낮의 쓰임과 장면 기능이 있으면 로컬 물성으로 둔다.
- 로컬 물성이 같은 부 안에서 다시 등장하거나 특정 생활 인물의 정체성과 묶이면 다음 감사에서 07, 11, 12 중 하나로 승격한다.
- 도감에 올린 항목은 물건 자체보다 누가 만지고, 밤에는 무엇으로 변하고, 어떤 선택을 어렵게 만드는지를 먼저 적는다.

## 4. 로컬 물성 보류 기준

로컬 도시 물성으로 남은 항목은 버려진 것이 아니다. 아래 기준 중 하나에 해당하면 다음 차수에서 승격한다.

| 승격 조건 | 예시 처리 |
|---|---|
| 같은 항목이 두 도시 이상에서 다시 확인됨 | 반복 소품/생태/잔류물 표로 승격 |
| 특정 인물의 선택을 바꿈 | 07 또는 12에 개별 카드 추가 |
| 특정 국가의 생활 윤리를 대표함 | 11 또는 04에 국가별 대표 항목으로 추가 |
| 결말부의 이름 놓기, 회색 숨, 보존, 안식, 장례와 연결됨 | 12 또는 07의 후반 물성으로 승격 |
| 독립 장소 문서의 금기와 직접 연결됨 | 도시 문서와 도감 문서를 함께 갱신 |
