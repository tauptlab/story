# 연재문작성 참조동선 1차

이 문서는 실제 연재문을 채우기 전, 어떤 자료를 어떤 순서로 열지 정하는 동선표다. 목적은 설정을 더 많이 넣는 것이 아니라, 한 화에 필요한 인물, 장소, 물건, 금기, 회수 장면을 서로 충돌 없이 고르는 것이다.

현재 운용 주의: 이 문서는 1차 작업동선이므로 과거 작업번호가 일부 남아 있다. 실제 집필 진입은 `세계관 설정/정식바이블/00_집필진입점.md`가 우선이며, 회차 배치는 `세계관 설정/정식바이블/07_회차_투입_색인.md`를 먼저 본다. 아래 표는 현재 정식바이블 구조에 맞춰 보정한 참조 순서다.

## 3. 한 화 준비표

| 질문 | 적을 내용 |
|---|---|
| 이번 화의 핵심 질문은 무엇인가 | 예: 지키려는 마음이 언제 닫힌 문이 되는가 |
| 이번 화의 중심 장소는 어디인가 | 도시 문서에서 규칙 하나와 금기 하나를 고른다 |
| 중심 인물은 무엇을 얻고 무엇을 잃는가 | 선물, 상처, 대가를 한 줄씩 적는다 |
| 반복 회수할 물건은 무엇인가 | 낮 용도와 밤 변형을 둘 다 확인한다 |
| 이 장면은 무엇을 되찾거나 다시 배열하는가 | 버린 가치관, 대체 가치관, 왜곡된 가치관, 다시 받아들일 가치관 중 하나로 적는다 |
| 녹티오라에 남을 흔적이 있는가 | 강하게 남은 말, 표정, 실패, 약속, 결핍 중 하나로 낮춘다 |
| 생활 인물이나 작은 생태가 필요한가 | 필요하면 한 명 또는 한 종만 넣는다 |
| 후속 화로 남길 흔적은 무엇인가 | 이름표, 장부, 흠, 빛, 냄새 중 하나로 남긴다 |
| 금지해야 할 쉬운 설명은 무엇인가 | 정사 표현으로 바꿀 문장을 미리 적는다 |

## 5. 장면 밀도 조절

| 너무 많을 때 | 덜어내는 기준 |
|---|---|
| 새 이름이 세 개 이상 나온다 | 하나만 남기고 나머지는 물성으로 낮춘다 |
| 인물의 설명이 길어진다 | 손버릇, 시선, 물건 반응 중 하나로 바꾼다 |
| 도시 설명이 멈춰 선다 | 규칙을 어기는 작은 행동을 넣는다 |
| 물건이 장식처럼 보인다 | 선택을 늦추거나 바꾸는 대가를 준다 |
| 적대자가 납작해진다 | 그가 지키려 한 것 하나를 먼저 보여준다 |
| 회수 요소가 많다 | 이번 화에서 갚을 것과 남길 것을 분리한다 |

## 7. 실제 작성 전 마지막 동선

1. `에피소드`에서 사건의 뼈대를 본다.
2. `정식바이블/01C_Mnemosyn_녹티오라_결투_상위기준.md`에서 버린 가치관, 대체 가치관, 밤의 흔적, 결투 뒤 재배치 중 무엇에 닿는지 고른다.
3. `정식바이블/07_회차_투입_색인.md`에서 이미 배치된 첫 노출, 핵심 결투, 귀환 빌드업을 확인한다.
4. 중심 인물 문서 하나, 중심 도시 문서 하나, 소품 문서 하나만 먼저 연다.
5. 필요할 때만 `07A`~`07F`에서 작은 보강 요소 하나를 고른다.
6. `정식바이블/08_원고용_작중어_빠른검수표.md`로 표현을 맞춘다.
7. `09A`~`09F`의 정밀표로 시간선과 관계 충돌을 확인한다.
8. 작성 뒤에는 등장한 새 요소를 `02`~`06`, 인물 문서, 도시 문서, 도감 계열 중 하나에 되돌릴지 판단한다.

## 8. 완료 판정

한 화 준비가 끝났다고 보려면 아래가 모두 확인되어야 한다.

- 핵심 질문이 한 줄로 정리되어 있다.
- 중심 인물의 선물, 상처, 대가가 장면 안에서 보인다.
- 중심 장소의 규칙이나 금기가 사건을 바꾼다.
- 물건이나 생태 하나가 장식이 아니라 선택을 바꾼다.
- 귀환, 밤 변형, 결투가 있다면 `01C`의 연결 축 중 하나가 확인된다.
- 새 보강 요소는 한 장면에 하나만 들어간다.
- 바깥말 없이 정사 표현만 남는다.
- 다음 화로 넘어갈 흔적이 이름표, 장부, 흠, 빛, 냄새 중 하나로 남는다.
