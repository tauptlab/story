## 2. 검수 기준

| 연결 | 잠금 |
|---|---|
| EP8 -> EP9 | 상실이 바로 기록 자료가 되지 않게, 끊어진 노멘 매듭과 기록 사이 빈칸을 유지 |
| EP9 -> EP10 | 마이더스 가격 질서의 선의를 먼저 보이고, 가격 없는 칸을 남김 |
| EP10 -> EP11 | 3부 공감이 정답처럼 튀어나오지 않게, 디아파손 홀은 고통의 위치만 낮춤 |
| EP11 -> EP12 | 넥서스 선로와 후원자들의 선의를 먼저 설득 |
| EP12 -> EP13 | 코로나 내정원을 처음부터 감옥처럼 보이지 않게 함 |

## 3. 결론

- 2부에서 3부로 넘어가는 핵심 물성은 `낡은 동전`, `계약서 여백`, `흔들리는 물컵`이다.
- 3부 회복 가치는 2부 대체 가치를 비웃지 않고, 2부가 감당하지 못한 빈칸을 낮은 조건으로 맡는다.
- 다음 작업은 4부 EP16~17 파편 전투와 EP18~20 보호처 정치의 감정선 연결 검수다.
