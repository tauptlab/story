# 정식바이블 00~08 원고전통합점검 1차

작성일: 2026-06-25

역할: 실제 연재분 작성에 들어가기 전 `00_집필진입점.md`에서 `08_원고용_작중어_빠른검수표.md`까지 이어지는 참조 흐름이 끊기지 않는지 확인한 기록이다. 이번 작업은 연재분 작성이 아니라 집필 전 운영 구조 보강이다.

---

## 1. 감사 결론

| 점검 항목 | 결론 |
|---|---|
| 00 진입성 | 유지 가능. `00`이 현재 첫 진입점 역할을 한다 |
| 01~07 정사 파악 | 가능. 상위규칙, 도시, 인물, 갈등, 밤, 소품, 회차 배치가 각각 분리되어 있다 |
| 08 마지막 필터 | 유지 가능. 다만 `08`의 작성 직전 단계가 `00`의 전체 진입 흐름과 완전히 묶이지 않아 보강함 |
| 09 계열 | 매 화 여는 문서가 아니라 죽음/귀환/후반 회수/번호 충돌이 걸릴 때만 여는 정밀표로 유지 |
| 작업 로그 전용 설정 | 원고 직접 사용 금지 원칙 유지 |

---

## 3. 새 통합 게이트 요약

| 단계 | 확인 |
|---|---|
| 00 | 이번 화가 EP 안에서 맡는 기능을 확인한다 |
| 01C/01E | 귀환/밤/결투 원칙과 명칭 직접성을 확인한다 |
| 02 | 공간이 배경 설명이 아니라 선택 압박 구조인지 확인한다 |
| 03 | 강한 능력, 약한 발현, 사소한 감각, 능력 없음의 층위를 확인한다 |
| 04 | 결투가 능력/장치/선택/피해로 보이는지 확인한다 |
| 05/06 | 밤 변형과 소품이 선택 시험과 잔류 흔적을 갖는지 확인한다 |
| 07/07G | EP를 2~6개 연재 회차로 나누고, 각 회차 6000자 이상을 버틸 장면 보상을 확인한다 |
| 08/08A | 바깥 설명어를 작중 물성으로 낮추고, 말미는 소품 1~2개로 닫는다 |

## 4. 보류/잔여

| 항목 | 판단 |
|---|---|
| 실제 회차 원고 | 아직 작성하지 않는다 |
| 도시별 문서 | 다음 고도화에서 87개 도시의 대표 물성이 `02`, `05`, `06`, `07B`와 충돌하지 않는지 2차 감사한다 |
| 인물 본문 표기 | `발현 표지`와 하위 운용이 실제 문장 후보에서 기술명처럼 보이지 않는지 2차 감사를 남긴다 |
