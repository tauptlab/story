# EP1-1 설정 투입량과 독자 보상 게이트 2차

작성일: 2026-06-25

목적: 실제 연재분을 작성하지 않고, EP1-1이 첫 화에서 과도한 설정 설명으로 무거워지지 않도록 고유명사, 물성, 중심 질문, 장면 보상의 허용량을 잠근다.

---

## 1. 투입량 판정

| 항목 | 허용량 | 판정 |
|---|---:|---|
| 전면 고유명사 | 1개 | `셀로`만 허용 |
| 후면 호칭 | 2개 이하 | 노인, 회색 조끼 무리 정도만 허용 |
| 전면 물성 | 3개 | 지워진 표지판 획, 손톱 밑 잔가루, 물동이 |
| 중심 질문 | 1개 | 빠진 획이 왜 셀로에게 붙는가 |
| 주인공 보상 | 1개 이상 | 물동이를 받치는 행동 |
| 말미 훅 | 1개 | 털어도 사라지지 않는 손톱 밑 획 |

---

## 2. 회차 보상 박자

| 위치 | 기능 |
|---|---|
| 첫 500~800자 | 표지판 획이 사라지는 즉시 사건 |
| 1200~1800자 | 셀로의 몸이 먼저 반응하는 감각 |
| 2500~3300자 | 광장 박수와 반복 동작의 불편한 긴장 |
| 4000~5000자 | 물동이를 받치는 첫 호감 행동 |
| 마지막 100~200자 | 손톱 밑 획 잔류 훅 |

---

## 3. 뒤로 미루는 항목

| 항목 | 이동 |
|---|---|
| 포에두스/콘코르디아 이름 | EP1-2 |
| 카멜레온즈/아르티잔스 | EP2-4 이후 또는 2부 회상 |
| 우물 수면과 자기 얼굴 | EP1-2 |
| 밤 전환/유리막/깨진 등롱 | EP1-3 |
| 에이도스/초록 칼날 | EP2-1~EP2-3 |
| `오컴의 면도날` | EP2-3 |
