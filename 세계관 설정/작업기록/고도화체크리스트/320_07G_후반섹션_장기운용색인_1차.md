# 07G 후반 섹션 장기 운용 색인 1차

상태: 작업 로그
대상 정식 참조: `정식바이블/07G_회차별_장면카드.md`
작성 목적: 5부 후반~6부 결말 원고 준비 시 `07G`의 후반 섹션을 빠르게 찾아가도록 운용 색인을 보강한다.

---

## 1. 작업 배경

`07G_회차별_장면카드.md`에 41~49번 섹션이 누적되면서, 후반부 집필자가 실제 원고 전 어느 카드를 먼저 열어야 하는지 다시 묶을 필요가 생겼다. 이번 작업은 새 설정을 추가하는 것이 아니라, 이미 정식바이블에 들어간 귀환 조건, 후폭풍, 이름 놓기, 과밀도 감사 카드를 후반 원고 준비 순서로 재배치한 것이다.

---

## 2. 반영 내용

| 항목 | 반영 위치 | 내용 |
|---|---|---|
| 후반 원고 전 먼저 열 카드 | `07G` 50.1 | EP24, EP25, EP26, EP27, EP28별로 먼저 열 07G 섹션과 보조 문서를 지정했다 |
| 후반 운용 순서 | `07G` 50.2 | `07` -> `07G` 50번 -> 해당 EP 카드 -> 46~49번 게이트 -> 05/06 -> 08/08A 순서로 정리했다 |
| 원고 전 경고선 | `07G` 50.3 | 귀환 의식화, 강한 능력명 과밀, 결말 해설화, 도감 물성 충돌 위험을 점검표로 만들었다 |
| 우선순위 표 갱신 | `07G` 41번 | 9번 작업을 반영 완료로 바꾸고, 다음 작업을 `00_집필진입점` 보강으로 넘겼다 |

---

## 3. 정합성 판단

| 검수 항목 | 판단 |
|---|---|
| 설정 변경 여부 | 없음. 후반부 사건, 인물, 물성의 정사 내용은 바꾸지 않았다 |
| 작중 설명 누수 | 없음. 표는 작가용 운용 색인이며 실제 원고 문장으로 쓰지 않는다 |
| Mnemosyn 귀환 원리 | 유지. 귀환은 봉인 해제가 아니라 조건이 누적된 뒤 발생하는 가치의 재편입으로 운용한다 |
| 녹티오라 원리 | 유지. 밤 변형과 소품 물성은 05/06 도감에서 역참조하도록 남겼다 |
