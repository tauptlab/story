# 핵심 결투 능력/소품/후폭풍표 보강 1차

작성일: 2026-06-25

역할: `07_회차_투입_색인.md`의 핵심 결투 배치를 가치관 설명 중심에서 능력/장치 충돌과 장면 흔적 중심으로 보강한 기록이다.

---

## 1. 보강 이유

결투는 A 안의 가치관 충돌을 은유하지만, 실제 원고에서는 가치관 토론처럼 보이면 안 된다. 독자가 먼저 보는 것은 기술, 방어, 실패, 피해, 반전, 남은 소품이어야 한다.

기존 `07`의 핵심 결투표는 회차 위치와 재배치 방향은 분명했지만, 어떤 능력과 장치가 어떻게 부딪히는지가 짧았다. 이번 작업에서는 같은 표 안에 `능력/장치 충돌`, `대응과 반전`, `남는 소품/밤 흔적`, `전투 뒤 재배치`를 나누어 넣었다.

## 2. 반영 범위

| 구간 | 보강 축 |
|---|---|
| 2부 EP4 전후 | 콘코르디아의 `파동모사`가 소속 압력과 부딪히는 방식 |
| 2부 EP6 | 프락시스의 `텔로스 창`과 카우투스의 `선견의 주사위`가 결과와 보류를 충돌시키는 방식 |
| 2부 EP7~8 | `아미쿠스 맹약`, `피의 계약선`, `피데스 가시`의 전투/방어 기능 |
| 2부 EP10 | `아우룸 가격표`와 `오컴의 면도날`이 가격 언어와 빈자리를 만드는 방식 |
| 3부 EP11~15 | `리라 현`, `코로나 내정원`, `아미쿠스 맹약`, 페쿤디아 씨앗 계열의 장면 기능 |
| 4부 EP17 | `닫힌문장`, `솔리움 루브라`, `프리즈마 알바`가 셀로 분광체를 여는 방식 |
| 5부 EP21~25 | `알레테이아`, `문고리울림`, `무지갯빛 호수`, `프리마 에코`가 Mnemosyn 귀환 조건을 쌓는 방식 |
| 6부 EP26~28 | `고요한 조류`, `울티마 열쇠`, 장례 들판, 노멘 계단이 귀환 후폭풍을 남기는 방식 |

---

## 3. 운용 원칙

1. 표의 앞쪽 네 열은 원고 장면 설계용이다.
2. 작가용 가치관 의미는 표 아래 주석에만 둔다.
3. 결투 승패보다 결투 뒤 바뀐 소품 반응, 금기, 도시 습관을 우선한다.
4. 능력이 없는 생활 인물은 억지로 기술명을 받지 않는다. 그들은 소품, 예법, 손버릇, 생태 반응으로 결투의 결과를 받는다.
5. Mnemosyn 귀환은 5부 EP24~25에서 즉시 치유가 아니라 재편입 조건의 성립으로만 보인다.

## 4. 다음 작업

`09A`~`09F` 정밀표가 이번 결투표와 충돌하지 않는지 확인해야 한다. 특히 아미키티아/드라쿨 죽음과 귀환, 루멘/셀로 빛과 파동, 카우투스/프리오르/쿠스토스 축은 `07`의 회차 배치와 직접 연결된다.
