# 인물속성 모듈 운용규칙

역할: 인물 문서를 길게 늘리지 않고, 각 인물의 절대 속성만 얇게 남긴 뒤 필요한 공통 속성 문서와 관계 문서를 조립해 페르소나를 만드는 기준이다.

---

## 1. 기본 원칙

개별 인물 문서는 장문 설정집이 아니라 참조 허브다. 모든 장면에서 반드시 고려해야 하는 정보만 인물 문서에 남기고, 공통 해석은 모듈 문서로 뺀다.

인물 문서에 남길 정보:

| 항목 | 기준 |
|---|---|
| 정사 등급 | 장기 서사에서의 비중 |
| 존재 위상 | 낮의 사람, 밤의 사람, 태고의 존재, 약한 발현자, 전승 인물 등 |
| 밤 진입 시점 | 언제 밤의 사람이 되었는지, 미정이면 해당 EP 작성 전 확정 |
| 능력/발현 | 고유 특수능력, 약한 발현, 사소한 감각, 능력 없음 |
| 애니어그램 후보 | 현재 성격의 고정 라벨이 아니라 성장/퇴행 궤도 |
| 지역성 | 출신/거주/소속 도시가 말투와 사고방식에 남기는 압력 |
| 절대 행동 원칙 | 어떤 장면에서도 지켜야 하는 핵심 습관 |
| 절대 금지 | 이 인물답지 않은 대사, 행동, 결론 |

모듈 문서로 빼는 정보:

| 문서 | 맡는 기능 |
|---|---|
| `03C_존재위상/00_존재위상_색인.md` | 낮/밤/태고/발현 단계와 밤 충동. 종류별 개별 문서를 링크한다 |
| 본 문서 7절 `성장·상태 요소 정의` | 성격 유형이 아니라 불안, 생존법, 퇴행, 성장 선택, 시기상태를 나눠 본다. 인물별 시기 잠금은 인물 문서의 `회차 운용 잠금` 절과 관계페르소나 카드에 둔다 |
| `국가별 설정/*.md`, `도시별설정/*.md`, `단체/*.md` | 지역별 말버릇, 사고 순서, 금기. 별도 지역성 문서로 빼지 않고 기존 장소/집단 문서에 흡수한다 |
| `03F_관계문서_페르소나_서브에이전트_운용규칙.md` | 관계 문서 작성 기준과 장면별 페르소나 생성 절차 |

인물 문서가 길어지는 순간에는 먼저 "이 정보가 모든 장면에서 필요한가"를 묻는다. 특정 상대, 특정 도시, 특정 회차에서만 필요한 정보는 관계 문서, 도시 문서, EP 문서로 분리한다.

## 2. 인물 문서 참조 블록 형식

개별 인물 문서 앞부분에는 아래 형식의 참조 블록을 둔다.

```md
## 인물 속성 참조

- 속성 모듈: [인물속성 모듈 운용규칙](../세계관%20설정/정식바이블/03B_인물속성_모듈_운용규칙.md)
- 존재 위상(정사 전체 기준): [밤의 사람](./03C_존재위상/밤의_사람.md), [고유 특수능력자](./03C_존재위상/고유_특수능력자.md)
- 밤 진입 시점: EP1 이전 얼마 되지 않은 시점
- 발현/능력(정사 전체 기준): 고유 특수능력 `파동모사`
- 현재 회차 사용 가능 범위: EP1~EP3은 신규 밤의 사람의 흔들림, 모방 습관, 박자 읽기 중심으로 제한한다. `파동모사`는 EP4에서도 완전 각성이 아니며, 초반에는 능숙한 설명이나 자유로운 전투 기술처럼 쓰지 않는다.
- 성장궤도: 4번 후보. 요소 정의는 본 문서 7절, 시기별 잠금은 인물 문서의 `회차 운용 잠금` 절과 관계페르소나 카드를 따른다
- 지역성: [눅세린](../세계관%20설정/국가별%20설정/눅세린.md), [벨라사르](../세계관%20설정/도시별설정/눅세린-벨라사르.md), [아르티잔스-카멜레온즈](../세계관%20설정/단체/아르티잔스-카멜레온즈.md)
- 관계 문서: 필요 시 [관계문서/페르소나 운용](../세계관%20설정/정식바이블/03F_관계문서_페르소나_서브에이전트_운용규칙.md)
```

참조 블록은 작가가 장면에 들어가기 전 빠르게 여는 색인이다. 여기서 장문 설명을 반복하지 않는다.

## 3. 비중별 문서 밀도

| 비중 | 운용 |
|---|---|
| S/A급 | 개별 인물 문서 + 참조 블록 + 필요 관계 문서 |
| B급 | 개별 인물 문서 + 참조 블록. 관계 문서는 장면 판단에 실제 영향을 줄 때만 만든다 |
| C급 | 짧은 인물 카드 + 참조 블록. 공통 속성은 모듈 문서로 처리한다 |
| 단회 생활 인물 | `03A_국가별_생활인물_명부.md`에 남기고, 반복 등장 시 개별 문서로 승격한다 |

작중 조금이라도 비중이 있는 인물은 참조 블록을 가진다. 그러나 모든 인물이 긴 생애사를 가져야 하는 것은 아니다. 중요한 것은 "장면에서 이 인물이 왜 이렇게 말하고 움직이는가"를 빠르게 회수하는 일이다.

이 밀도 차이의 작가용 근거는 강과 지류의 원칙이다(원고 노출 금지).

- 주요 인물은 여러 지류가 합쳐진 강이다. 하나의 가치관은 단 하나의 경험이 아니라 수많은 작은 경험(지류)이 모여 만들어지고, 능력은 그 강의 유속과 힘 자체를 상징한다.
- 거대한 사건은 분수령이다. A의 인생을 바꾼 큰 사건 하나는 여러 인물에게 동시에 서로 다른 강(죄책감, 진실 탐구, 상실과 수호 등)을 만들어 낸다.
- 단역과 발현체는 지류 하나 그 자체다. 사소한 기억이나 생각 하나로 충분하며, 탄생 비화를 만들 필요가 없다.

이 원칙 덕분에 비중에 따라 인물의 깊이와 능력의 복잡성을 조절할 수 있고, 주요 인물에게만 서사를 집중할 수 있다.

## 4. 밤 진입 시점 기록 기준

`밤의 사람` 여부는 현재 상태만으로 충분하지 않다. 이 세계에서는 밤의 사람이 늘어나는 시대적 양상이 중요하므로, 각 인물 문서에는 밤 진입 시점을 따로 둔다.

| 상태 | 기록 방식 |
|---|---|
| 처음부터 밤 진입 | 태고의 존재 또는 특수 예외. 사유를 함께 적는다 |
| EP 이전에 이미 밤의 사람 | "EP N 이전"처럼 독자가 만나는 시점을 기준으로 적는다 |
| EP 안에서 밤의 사람이 됨 | 해당 회차와 장면을 적는다 |
| 발현은 있으나 시점 미정 | "미정. 현재 회차에서 밤의 사람으로 쓰지 말 것. 해당 EP 작성 전 발현/각성 시점 확정 필수"로 남긴다 |
| 낮의 사람 | "밤 진입 없음. 발현시키려면 해당 EP 문서와 인물 문서에 시점 추가"로 둔다 |

초반 0~10대 시기에는 셀로 외의 밤의 사람이 극히 드물다. 따라서 초반 조연을 밤의 사람으로 올릴 때는 반드시 장면 기능이 있어야 한다. 중반에는 밤의 사람이 늘어나 사회적 긴장이 커지고, 후반에는 다시 줄어드는 양상으로 운용한다.

## 5. 페르소나 조립 순서

장면을 쓰기 전에는 아래 순서로 인물 페르소나를 조립한다.

1. 현재 EP 문서에서 회차 시점과 공개 정보만 확인한다.
2. 개별 인물 문서의 참조 블록을 확인한다.
3. 인물 문서의 `현재 회차 사용 가능 범위`를 먼저 보고, 정사 전체 능력명을 현재 숙련도로 쓰지 않는다.
4. 존재위상 개별 문서, 본 문서 7절의 성장·상태 요소 정의, 국가/도시/단체 문서 중 필요한 것만 연다.
5. 특정 상대와의 말투가 달라지는 장면이면 관계 문서를 연다.
6. 페르소나 서브에이전트에는 현재 회차 시점까지의 정보만 준다.
7. 미래 복선, 결말 연결, 세계관 장기 구조는 메인 에이전트가 관리한다.

서브에이전트가 새 설정을 발명하면 안 된다. 대사와 행동은 만들 수 있지만, 새 법칙, 새 능력, 새 과거사는 메인 에이전트의 승인을 받아야 한다.

## 6. 장면 후 업데이트

원고 작성 뒤 아래 중 하나가 바뀌면 문서를 갱신한다.

| 변경 | 갱신 위치 |
|---|---|
| 밤의 사람이 되는 장면 발생 | 해당 인물 문서의 밤 진입 시점 |
| 특정 관계에서 말투가 바뀜 | 관계 문서 |
| 지역 말투/관습이 새로 확정됨 | 해당 국가/도시/단체 문서 |
| 성장/퇴행이 명확해짐 | 해당 인물 문서(`회차 운용 잠금` 절 포함)와 관계페르소나 카드 |
| 능력/발현이 새 단계에 들어감 | 해당 인물 문서와 `03_인물_바이블.md` |

설정은 원고를 위해 존재한다. 새 문서가 늘어났는데 장면에서 더 좋은 대사와 행동을 만들지 못하면 그 문서는 실패한 것이다.

## 7. 성장·상태 요소 정의

성장궤도는 성격 유형 라벨이 아니라 아래 요소로 쪼개어 장면 좌표로 쓴다. 애니어그램 번호는 독자에게 설명하지 않고, 장면에서는 손, 시선, 말의 순서, 멈칫거림, 물건을 다루는 방식으로만 드러낸다. 같은 번호라도 인물마다 다르게 움직이므로, 번호보다 불안, 생존법, 현재 시기가 우선이다.

| 요소 | 정의 | 운용 기준 |
|---|---|---|
| 불안 | 인물이 피하려는 감정 | 대사로 설명하지 않고 먼저 보는 것, 먼저 만지는 것으로 장면화한다. 첫 등장에서는 한 가지 물성으로 낮추고, 현재 회차에서 아직 모를 불안을 미래 지식으로 말하지 않는다 |
| 생존법 | 불안을 피하려고 반복해 온 행동 | 처음에는 실제로 쓸모 있어야 뒤의 위험이 설득된다. 초반에는 틀린 방식이라 단정하지 않고, 밤 장면에서는 말보다 몸에 먼저 올라온다 |
| 퇴행 | 압박 속에서 익숙한 생존법을 과하게 쓰는 현상 | 성격 붕괴가 아니다. 한 장면 편의로 반대 성격을 만들지 않고, 악역화하지 않으며, 미래에 겪을 상처를 현재 회차 대사로 미리 고백하게 하지 않는다 |
| 성장선택 | 같은 욕구를 덜 해로운 방식으로 쓰는 장면 | 능력을 더 잘 쓰는 장면이 아니라 쓰지 않거나 늦추는 장면일 수 있다. 작을수록 좋고, 성장 뒤에도 옛 생존법은 남는다 |
| 밤에서 과장되는 것 | 밤 진입 시 본능화되는 충동 | 밤은 성격을 바꾸지 않고 낮에서 다스리던 생존법을 빠르고 원초적으로 만든다. EP1 첫 밤의 위험은 으스스한 자연현상이 아니라 신규 밤의 사람들의 충동과 다툼이며, 물리 위험은 그 다툼의 여파로 쓴다 |
| 관계변형 | 기본 생존법이 특정 상대 앞에서 다른 모양으로 꺾이는 것 | 새로운 성격이 아니다. 특정 상대에게만 쓰는 말버릇과 몸짓은 인물 본문이 아니라 관계 문서에 둔다. 개별 관계 문서가 항상 우선한다 |
| 시기상태 | 같은 인물의 초반, 중반, 후반 상태를 섞지 않는 회차 범위 구분 | 범위 문장은 `지금 써도 되는 것`과 `지금 쓰면 안 되는 것`을 반드시 함께 적는다 |

시기별 운용 정보(관계페르소나 카드, 인물 문서의 `회차 운용 잠금` 절)는 시기, 사용 가능 정보, 불안, 생존법, 퇴행 신호, 성장 선택, 밤에서 과장되는 것, 대사/행동 허용, 금지 필드로 쓰고, 한 카드는 한 시기만 다룬다. `금지` 칸은 반드시 쓴다. 이 칸이 없으면 초반 정보량이 쉽게 무너진다. 미래 복선의 정답은 카드에 넣지 않으며, 카드가 없는 조연은 해당 EP 작성 전에 짧게라도 만든다.

카드의 시기는 인물 문서의 명시 EP/부 단서를 우선하고, 단서가 없으면 지역/도시 링크와 국가 생활권 기준으로 좁힌다. 낮의 사람은 밤을 직접 본 것처럼 말하지 않으며, 생활 인물의 장면 기능은 세계관 법칙 해설이 아니라 손, 물건, 말끝으로 드러난다.
