# 연재화별 등장 요소 인덱스

이 문서는 `웹소설작성` 폴더의 부분 구체화 원고를 기준으로, 각 화에 등장하거나 암시된 장소, 인물, 소품, 장치, 생물, 건축물을 추적한다. 목적은 본문에 나온 모든 요소를 정식바이블의 인물/아틀라스/도감과 연결하는 것이다.

현재 단계에서 `웹소설작성`은 최종 연재분이 아니라 부분 구체화 원고/샘플이다. 따라서 이 인덱스는 완성 원고 검수가 아니라, 향후 세계 성경 편찬을 위한 요소 추출 장부로 사용한다.

운용 주의: 이 문서 안의 `19`, `40`, `84`, `100` 같은 숫자 표기는 작업기록 근거 코드다. 실제 연재분 작성자는 `07_회차_투입_색인.md`를 우선 보고, 이 문서는 특정 요소가 어디서 추출되었는지 역추적할 때만 연다.

추가 주의: 이 문서의 `연결 완료`는 정식바이블의 인물/도시/도감 축에 흡수되었다는 뜻이지, 해당 회차 원고에 반드시 투입한다는 뜻이 아니다. 실제 회차 투입은 `07_회차_투입_색인.md`, `07B`, `07C`, `07E`의 과밀도 규칙을 통과한 후보만 선택한다.

상태 표기:

| 상태 | 의미 |
|---|---|
| 연결 완료 | 정식바이블의 인물/도시/도감 문서에 이미 연결됨 |
| 보강 필요 | 기본 설정은 있으나 정식바이블/도감/아틀라스 본문에서 장면 기능을 더 분명히 해야 함 |
| 본류 흡수 필요 | 본문에는 등장하지만 아직 도감/아틀라스에 충분히 흡수되지 않음 |

## 0. 번호 체계 주의
