# 무기물 핵심물질

이 문서는 네우로시아의 핵심 광물, 결정, 도구, 기록 매체, 의식 물질을 개별 카드로 정리한다. `04_유기물_무기물_도감.md`가 전체 도감이고 `07_핵심소품_장치_개별카드.md`가 장치 중심이라면, 이 문서는 물질의 촉감, 파손 방식, 오염 방식, 거래 방식, 장면 삽입법을 담당한다.

운용 주의: 무기물 카드는 능력의 하위 기술 목록이 아니다. 반지, 칼집, 잔, 깃, 끈, 방패 같은 사물명은 독자가 붙잡는 물성으로 보존하고, 작가용 의미는 장면 기능과 사용 금기 안에만 둔다. 같은 이름이 능력 주변에 놓여도 능력명과 소품명을 섞지 않는다.

---

## 1. 공통 원칙

### 1.1 물질은 감정의 침전물이다

네우로시아의 무기물은 죽은 배경 재료가 아니다. 오래 반복된 생각, 선택, 상처, 믿음이 굳어 물성이 된 것이다.

| 물성 | 작중 의미 | 장면 기능 |
|---|---|---|
| 단단함 | 한 가치가 오래 버틴 시간 | 인물의 고집, 신념, 방어를 보여줌 |
| 투명함 | 스스로를 설명할 수 있다고 믿는 정도 | 진실처럼 보이지만 왜곡 가능 |
| 무게 | 피한 선택, 쌓인 책임, 미뤄둔 감정 | 인물이 더는 도망칠 수 없게 함 |
| 균열 | 처음 생긴 의심이나 변화 가능성 | 적대자의 인간성, 주인공의 성장 |
| 빛 | 타인에게 보이고 싶은 가치 | 욕망, 희망, 인정 욕구 |
| 온도 | 기억과 감정의 생존 여부 | 애도, 위로, 상실, 회복 |
| 소리 | 말하지 않은 의미의 잔류 | 오라쿨룸, 센시르, 아르스 모투스 연결 |

### 1.2 같은 물질도 국가에 따라 다르게 쓰인다

| 물질 | 프론티아 | 눅세린 | 암디르 | 센시르 | 히프노레아 | 오라쿨룸 | 아르스 모투스 |
|---|---|---|---|---|---|---|---|
| 결정 | 분류와 통제 | 가격과 지위 | 봉인과 방어 | 조율과 치유 | 보존과 열람 | 이름과 해석 | 자세와 반복 |
| 금속 | 직각 구조 | 장식과 과시 | 갑옷과 족쇄 | 의료 도구 | 서고 보강재 | 악기 부품 | 무기와 수련구 |
| 실 | 경계선 | 거래 약속 | 묶인 생존 | 연결과 공감 | 므네마 매듭 | 이름 의식 | 동작 궤적 |
| 물 | 냉각과 정화 | 향락과 소비 | 피와 땀 | 반향과 성찰 | 망각과 보존 | 음성의 전달 | 호흡과 리듬 |

### 1.3 카드 사용 규칙

새 회차에서 아래 물질을 사용할 때는 최소 하나를 장면에 심는다.

1. 손에 닿는 촉감.
2. 낮의 용도.
3. 밤에 남는 흔적.
4. 어떤 인물의 거짓 믿음을 찌르는지.
5. 장면이 끝난 뒤 남는 파편, 얼룩, 소리, 빚.

## 2. 눅세린 계열

### 아우룸 결정체

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 소속 | 눅세린, 아르도르, 피데스 거래소, 마이더스 체계 |
| 작중 별칭 | 빛나는 빚, 작은 왕좌, 손바닥의 태양 |
| 낮의 물성 | 투명한 황금 결정. 들고 있으면 손바닥이 따뜻하고, 주변 시선이 잠깐 부드러워진다. 마이더스가 값을 당기면 액면가보다 그 안의 소유 이력, 인정 욕구, 빚의 무게에 비례해 장벽, 중장병, 포격 장치, 황금 문의 핵이 된다 |
| 밤에 남는 흔적 | 갚아도 작아지지 않는 빚덩이. 빛이 강할수록 그림자도 선명해지고, 갚히지 않은 욕망은 작은 전투체처럼 결정 안쪽에서 선다 |
| 은유 | 인정 욕구와 비교 가치가 굳은 물질 |
| 일상 사용자 | 상인, 계약 중개인, 그리세오 구역 주민, 마이더스 추종자 |
| 주요 인물 | 마이더스, 세리엘, 나르키소스, 소키우스, 레마니스, 아미키티아 |
| 대가 | 관계를 숫자로 세는 습관이 생긴다. 오래 지니면 호의도 투자처럼 느껴진다 |
| 파손 방식 | 억지로 깨면 황금 가루가 되어 주변의 작은 욕망을 과장한다 |
| 회복 방식 | 계산되지 않은 도움을 받거나 줄 때 빛이 잠시 흐려진다 |
| 장면 기능 | 마이더스 체제의 매혹을 보여주되, 값과 욕망이 실제 장벽과 병력이 되는 폭력성까지 장면화한다. 동전과 지폐가 속도와 수를 맡는다면, 아우룸 결정체는 전장의 무게와 중심을 맡는다. 단순한 황금 장식이 아니라 가격표가 읽어 낸 욕망을 몸으로 세우는 물질이다 |

삽입 장면 씨앗:

- 그리세오 구역 아이가 아우룸 결정체를 혀 밑에 숨겨 굶주림을 잊는다. 셀로는 그것을 빼앗을 수 없어서, 대신 아이가 먹을 수 있는 진짜 빵을 찾는다.
- 세리엘이 계약서를 찢을 때, 결정체가 깨지는 대신 더 맑아진다. 그의 배신은 탐욕만이 아니라 "믿고 싶었던 마음"의 뒤틀림이었다는 단서가 된다.
- 마이더스의 왕좌 아래에는 가장 큰 결정체가 없다. 수많은 작은 결정체가 서로를 비추며 왕좌처럼 보일 뿐이다.
- 마이더스가 결정체를 바닥에 굴리면, 그 결정에 기대던 사람들의 인정 욕구가 금빛 중장병의 무게로 일어난다.

- 셀로가 가시를 뽑으려 하면 세리엘보다 셀로의 손이 먼저 베인다. 믿음은 대신 처리해 줄 수 없다는 뜻이다.

## 3. 히프노레아 계열

### 겹문장 결정

정사 등급: A급

| 항목 | 설정 |
|---|---|
| 소속 | 히프노레아, 이렐사드, 기록의 대전당 |
| 작중 별칭 | 얼어붙은 장면, 기억의 렌즈, 침묵한 증언 |
| 낮의 물성 | 차갑고 투명한 결정. 가까이 대면 내부에 빛의 층이 여러 겹 보인다 |
| 밤에 남는 흔적 | 열람자를 기록된 순간 안에 가두는 방 또는 복도 |
| 은유 | 과거의 보존과 해석의 위험 |
| 일상 사용자 | 사서, 복원사, 므네몬의 기록관, 에이도스 |
| 주요 인물 | 므네몬, 로고스, 에이도스, 셀로, 레기나 페쿤디아, 아미키티아, 블라드 드라쿨 |
| 대가 | 오래 열람하면 현재의 감정보다 기록된 감정을 더 진짜처럼 느낀다 |
| 파손 방식 | 충격이 아니라 모순된 증언이 동시에 닿을 때 금이 간다 |
| 회복 방식 | 한 가지 정답으로 봉합하지 않고, 여러 관점을 층으로 보존해야 한다 |
| 장면 기능 | 기억이 사실이면서 동시에 편집된 것일 수 있음을 드러낸다 |

삽입 장면 씨앗:

- 에이도스가 겹문장 결정을 논리적으로 분류하려 할수록, 결정 안의 인물들이 그를 바라본다. 분석 대상이던 과거가 윤리적 부담으로 변한다.
- 셀로가 손을 대면 결정체 속 장면이 완전히 재생되지 않는다. 대신 빠진 부분이 무지갯빛으로 흔들려, 셀로 자신이 그 공백과 관련 있음을 암시한다.
- EP9에서는 드라쿨 쪽 증언도 한 층으로 열린다. 그가 자기 이름을 말하지 못하면 결정은 답을 대신 내리지 않고, 말하지 못한 자리만 검은 투명층으로 남긴다.

## 12. 정사 연결 메모

- `흑요석 파편`, `흑요석 일식 가면`, `검은 달`은 같은 물건이 아니다. 흑요석 파편은 포보스 개인 유물의 파손 단서이고, 일식 가면은 포보스의 생존 정체성이며, 아테르 루나은 프론티아가 공포 물질을 통제 장치로 차용한 인공물이다.
- `므네메 돌`과 `겹문장 결정`은 다르다. 므네메 돌은 체온과 애도의 물질이고, 겹문장 결정은 사건과 해석의 저장 매체다.
- `디아파손 실`과 `노멘 매듭`은 모두 끈 형태지만 기능이 다르다. 디아파손 실은 감정 상태를 연결하고, 노멘 매듭은 정체성의 역할과 응답을 묶는다.
- `오차분필`의 프론티아 변종은 수치심을 남기고, 아르스 모투스 원형은 배움의 시작점을 남긴다. 같은 물질의 문화적 사용 차이를 보여주는 좋은 대비 요소다.
- `프리즘 결정`과 `이리스 수정체`도 구분한다. 프리즘 결정은 통합을 통과시키는 도구이고, 이리스 수정체는 셀로가 과청취 끝에 굳은 상태다.
- `디아파손 실`과 `아미쿠스 맹약 실`도 구분한다. 디아파손 실은 감정 상태를 조율하고, 아미쿠스 맹약 실은 동의한 관계와 책임을 묶는다.
- `손때 묻은 잔`과 눅세린 `마지막 잔`은 연결되지만 같지 않다. 눅세린의 마지막 잔은 대가 없는 환대의 컵이고, 프론티아 노비스 잔 선술집의 마누스 잔은 무명 노동의 기억 장치다.
- `황금중추`는 단순 오염 물질이 아니다. 넥서스 선로가 실제로 누군가를 살렸기 때문에, 잔류물은 성공과 대가가 함께 남은 흔적이다.
- `타키투스 껍질`, `영혼 침식증`, `이리스 수정체`는 모두 고요하거나 굳은 이미지가 있지만 같은 상태가 아니다. 타키투스 껍질은 강제 치유/판결 실패 피해, 아니마 침식증은 넥서스 후폭풍 질병, 이리스 수정체는 셀로의 과청취 결정화다.
- `페이단 씨앗`은 악성 감염 물질이 아니라 이미 존재하던 분노가 자기 열을 찾는 촉매다.
- `중심불 검격`은 포르티스의 필살기가 아니라 므네메 돌, 결돌림, 공포의 경계선이 함께 맞을 때 가능한 중심 타격이다.

- `카르보 레스타`는 비극의 증거품이 아니라 다시 자라야 할 땅에 남은 먼지다.
- 오리고 샘 물성은 설명용 상징으로 길게 해설하지 않는다. 손등, 물결, 늦은 빛, 말 전의 떨림으로 먼저 보인다.
