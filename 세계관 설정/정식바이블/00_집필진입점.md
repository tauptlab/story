# 집필진입점

역할: 실제 연재분을 쓰기 전에 가장 먼저 여는 문서다. 이 문서는 작업 장부가 아니라, 지금 원고에 필요한 정사만 빠르게 고르는 입구다.

---

## 1. 최우선 참조

실제 집필 전에는 아래 핵심 묶음을 본류로 삼는다. 나머지 문서는 보조 자료이며, 필요한 장면에서만 연다.

| 우선 | 문서 | 역할 |
|---|---|---|
| 1 | `01_세계관_상위규칙.md` | 정식바이블의 참조 우선순위와 이야기 고도화 기준 |
| 2 | `01B_정사_연표와_시대구분.md` | A의 생애 흐름을 작중 세계사와 시대 변화로 바꾼 기준 |
| 3 | `01C_Mnemosyn_녹티오라_결투_상위기준.md` | 귀환, 밤 변형, 결투의 의미 |
| 4 | `01D_인물공감_가치관장면_운용헌장.md` | 인물을 사랑받게 만드는 장면 기준 |
| 5 | `01E_명칭_능력_소품_작명기준.md` | 능력명, 소품명, 장소명 판단 |
| 6 | `02_국가_도시_아틀라스.md` | 국가, 도시, 장소의 본류 |
| 7 | `03_인물_바이블.md` | 주요 인물, 능력 보유 기준, 인물의 상처와 선물 |
| 8 | `04_가치관_갈등_매트릭스.md`, `05_녹티오라_밤변형_도감.md`, `06_소품_생태_물성_도감.md` | 결투, 밤, 물성의 본류 |
| 9 | `07_회차_투입_색인.md` | 회차별 첫 노출, 숨길 정보, 장면 보상 |

`01A`, `03A`, `06A`, `06B`, `07A`~`07G`, `08`, `08A`, `09A`~`09F`는 보조 참조다. 정사에서 제외된다는 뜻이 아니라, 원고 첫 진입 때 매번 열 문서는 아니라는 뜻이다.

실제 한 화를 준비할 때는 `00 -> 01B의 해당 시대 한 단락과 2.4 부별 사건 엔진 -> 07 -> 해당 에피소드 문서`까지만 먼저 열고, 장면이 과밀하거나 후반부 오독 위험이 있을 때만 `07G`의 해당 절을 추가로 본다. 도시, 밤 변형, 소품을 고를 때는 `02`, `05`, `06`의 `0`번 체급 섹션을 먼저 보고, 핵심 사건 장소/핵심 밤 변형/A급 핵심 장치인지부터 판단한다. `01C`, `01D`, `01E`는 원칙이 흔들릴 때 확인하고, 보조 참조는 후보나 충돌을 확인할 때만 연다. 정식바이블 안에서도 모든 문서가 같은 무게가 아니다.

`에피소드` 폴더의 현재 EP 문서는 실제 연재분 초안 직전의 EP별 작업대다. 먼저 `전면 목표`, 회차별 `첫 장면`, `이야기 기능 요약`, `도시 변화`, 다음 질문/연결 문단을 본다. `녹티오라 씨앗`은 밤 장면을 쓰는 회차에서만 하나 골라 전면화한다. 예전 초안 흔적이나 번호가 `07`, `07G`와 충돌하면 `07`을 우선하고, 해당 EP 문서의 장면을 다시 낮춘다.

`웹소설작성`과 `장면시퀀스` 폴더는 실제 문장감과 전투 동선을 얻기 위한 원천 초안이다. 그대로 정식 원고로 승격하지 않는다. 그 안의 좋은 장면은 `07`의 EP 기능과 해당 EP 문서의 전면 목표에 맞춰 재배열하고, 장면이 과밀할 때만 `07G`의 첫 장면/말미 잔류물을 확인한다. 구식 능력명/직접 가치어/작가용 해설은 `01E`, `08`, `08A` 기준으로 낮춘 뒤 사용한다.

### 1.1 현재 이야기 척추

집필자는 매번 모든 설정을 다시 펼치지 말고 아래 척추만 먼저 든다. 세부명, 국가사, 도감 물성은 이 흐름을 장면으로 세울 때만 내려가 확인한다.

- 1부의 동사는 `끼어들다`와 `따라 하다`다. 세계 설명이 아니라 환대와 배제가 같은 손에서 오는 사건으로 시작한다. 셀로는 먼저 사랑받고, 그 다음 안전한 무리가 왜 검은 낙인을 남기는지 보인다.
- 2부의 동사는 `판정하다`와 `값을 붙이다`다. 맞는 말, 규칙, 가격이 실제로 사람을 구한 뒤 늦게 온 얼굴과 값 없는 빵을 설명하지 못하는 이야기다.
- 3부의 동사는 `낮추다`다. 치유와 보호가 필요하다는 사실을 인정한 뒤, 고통을 없애려는 손이 목소리와 질문까지 낮추는 위험을 보인다.
- 4부의 동사는 `보호하다`와 `결정권을 나누다`다. 위험한 파편을 처치하지 않고 이름, 경계, 빈 의자로 보호처 정치에 넘기는 이야기다.
- 5부의 동사는 `기다리다`와 `채우지 않다`다. 진실과 귀환이 만능 해결책처럼 보이는 순간, 낮춘 빛과 열지 않은 문과 빈그릇이 귀환의 조건을 만든다.
- 6부의 동사는 `분류하다`, `열지 않다`, `자리를 나누다`다. 돌아온 가치가 만든 후폭풍을 감당하는 이야기다. 장례 들판, 수리 인계표, 돌아가지 않은 울티마 열쇠, 차갑지 않은 물컵 자국, 중심석의 당김은 평화의 증거가 아니라 남은 책임의 형태다.

이 척추에 닿지 않는 새 설정은 지금 원고에 넣지 않는다. 매력적인 세부라도 현재 EP의 선택, 피해, 반전, 말미 잔류물을 바꾸지 못하면 보조 후보로만 둔다.

### 1.2 원고 진입 압축 기준

문서가 많아 보일 때는 새 자료를 더 찾지 말고, 아래 순서로 현재 회차를 좁힌다.

1. 먼저 해당 EP의 동사를 하나로 고른다. 예: `분류한다`, `열지 않는다`, `자리를 나눈다`.
2. 그 동사를 실제로 강제하는 표면 사건을 하나 고른다. 예: 젖으면 사라질 길표, 돌아가지 않는 열쇠, 중심으로 미끄러지는 이름표.
3. 그 사건 때문에 선택해야 하는 인물을 하나 세운다.
4. 그 선택에 부딪히는 능력, 소품, 도시 압력 중 하나만 전면화한다.
5. 나머지 설정은 배경 반응, 손버릇, 소품 잔류물로 낮춘다.

원고 진입 직전의 질문은 "이 설정을 더 넣을까"가 아니라 "이 인물이 이 장면에서 무엇을 잃을 위험이 있는가"다. 잃을 것이 없는 설정은 아무리 멋있어도 이번 회차의 전면에 세우지 않는다.

## 2. 한 화를 준비할 때

한 화를 준비할 때는 모든 설정을 펼치지 않는다. 아래 네 가지만 먼저 정한다.

1. 이번 화의 독자 보상: 사건, 관계 변화, 능력 시연, 밤 변형, 물성 발견 중 무엇을 줄 것인가.
2. 이번 화에서 감출 것: A의 뇌 은유, 버린 가치관, 귀환 조건, 작가용 의미는 직접 말하지 않는다.
3. 이번 화의 중심 인물 하나: 그 인물이 옳은 이유와 위험한 이유를 행동으로 보인다.
4. 이번 화의 중심 물성 하나: 문, 끈, 잔, 칼, 의자, 빛, 물 같은 물건 하나가 선택을 바꾸게 한다.

EP 하나는 실제 플랫폼 연재 회차 2~6개로 쪼개진다. 각 연재 회차는 6000자 이상을 버틸 작은 목표, 중반 반전, 말미 잔류물을 가져야 한다. 설명이 아니라 장면 보상으로 독자를 붙잡는다.

현재 EP 문서는 28개 모두 원고 전 기준으로 같은 말미 구조를 가진다. 원고를 쓰기 전에는 해당 EP 문서의 `이야기 기능 요약`으로 이번 묶음의 동사를 정하고, `도시 변화`로 결투 뒤 남길 후폭풍을 고른 뒤, `녹티오라 씨앗`에서 밤에 살아남을 흔적 하나만 전면화한다.

## 3. 보조 문서를 여는 경우

보조 문서는 아래 상황에서만 연다.

| 필요 | 여는 문서 |
|---|---|
| 생활 인물이나 배경 인물의 감각이 필요할 때 | `03A_국가별_생활인물_명부.md` |
| 특정 생태/무기물의 세부 물성이 필요할 때 | `06A`, `06B` |
| 특정 EP의 보조 후보가 필요할 때 | `07A`~`07F` |
| 원고에 작가용 설명이 새는지 확인할 때 | `08`, `08A` |
| 죽음, 귀환, 시간선, 빛/파동, 신중함/보존축이 충돌할 때 | `09A`~`09F` |
| 과거 장면 초안의 감정선이나 전투 동선이 필요할 때 | `웹소설작성`, `장면시퀀스` |

보조 문서는 본류를 대체하지 않는다. 보조 문서에만 있는 설정은 아직 원고 기준이 아니다.
`웹소설작성`과 `장면시퀀스`도 마찬가지다. 그 안의 문장과 사건은 재료이지 정식 문장이나 최종 명칭이 아니다.

## 4. 고도화 방향

앞으로의 고도화는 새 감사표, 대응표, 검수표, 체크리스트를 늘리는 방식으로 진행하지 않는다. 이미 문서는 충분히 많다. 새 정보가 필요하면 먼저 기존 정식바이블, 개별 인물 문서, 도시 문서, 도감 문서 안의 한 문단으로 흡수한다.

고도화의 우선순위는 아래 순서다.

1. 인물이 더 사랑받는가.
2. 장면이 설명보다 선택과 행동으로 움직이는가.
3. 결투가 가치관 해설이 아니라 능력, 소품, 물성, 피해, 반전으로 보이는가.
4. 지금 알아야 할 설정과 나중에 알아도 되는 설정이 분리되는가.
5. 새 문서가 늘어나는 대신 기존 참조 동선이 짧아지는가.

## 5. 새 설정 추가 원칙

새 인물, 도시, 소품, 생태, 역사, 결투를 만들 때도 새 기록 문서를 먼저 만들지 않는다.

1. 먼저 본류 문서 중 어디에 들어갈지 정한다.
2. 인물이라면 `03` 또는 개별 `등장인물` 문서에 흡수한다.
3. 도시/국가라면 `02`, 도시별 문서, 국가별 문서에 흡수한다.
4. 소품/생태/밤 변형이라면 `05` 또는 `06`에 흡수한다.
5. 회차 투입이라면 `07` 또는 해당 EP 문서에 흡수하고, 세부 흐름 보조가 필요할 때만 `07G`에 남긴다.
6. 작업기록은 꼭 필요한 근거 보존용으로만 남긴다.

작업기록만 있고 본류 문서에 흡수되지 않은 설정은 집필 참조 불가다.

## 6. 빠른 판단

| 막히는 지점 | 먼저 볼 곳 |
|---|---|
| 세계 법칙 | `01`, `01C` |
| 인물 | `03`, 필요하면 개별 `등장인물` 문서 |
| 도시/장소 | `02`, 필요하면 도시별 문서 |
| 결투 | `04`, `07`, 필요하면 `07G` |
| 밤 변형 | `05` |
| 소품/물성 | `06` |
| 회차 배치 | `07`, 필요하면 `07G` |
| 작중 표현 | `08`, `08A` |

가장 중요한 원칙은 하나다. 원고에서는 설정을 증명하지 말고, 인물이 어떤 선택을 하게 만들었는지만 보여준다.
