# 집필진입점

역할: 실제 연재분을 쓰기 전에 여는 집필 진입 문서다. 단, 사용자가 특정 파일 하나만 링크하거나 작업 요구를 짧게 던진 경우에는 반드시 `000_작업전_필수숙지_최상위인덱스.md`를 먼저 열고 이 문서로 들어온다. 이 문서는 작업 장부가 아니라, 지금 원고에 필요한 정사만 빠르게 고르는 입구다.

---

## 1. 최우선 참조

실제 한 화를 쓰기 시작할 때의 최소 동선은 `000_작업전_필수숙지_최상위인덱스.md -> 00_집필진입점.md -> 07_회차_투입_색인.md -> 에피소드/해당 부/EP N.md`다. 이 흐름으로 현재 회차의 동사, 표면 사건, 말미 잔류물이 잡히면 곧장 원고 설계로 들어간다. 각 EP 작업대 상단의 `회차 참조` 블록에는 `07`의 해당 절과 `07G`의 해당 절로 바로 가는 링크가 있으므로, 색인 전체를 읽지 않고 그 링크로 진입한다.

본류 검수와 실제 집필 진입은 아래처럼 나눈다.

| 목적 | 여는 문서 | 확인할 것 |
|---|---|---|
| 전체 1~6부 흐름 검수 | `00 -> 01B -> 07` | 시대 전환, 부별 사건 엔진, EP1~28의 표면 사건과 다음 질문 |
| 한 EP 묶음 준비 | `000 -> 00 -> 07 -> 에피소드/해당 부/EP N.md` | 해당 EP의 동사, 첫 사건, 선택 인물, 말미 잔류물 |
| 실제 연재 한 화 준비 | `해당 EP 문서` | 회차별 첫 장면, 기한, 중반 반전, 회차 끝 질문 |
| 장면이 흔들릴 때 | `01C~01E`, `02~06`, 필요 시 `07G` | 귀환/밤/결투 원리, 인물 호감, 명칭, 도시/도감 충돌, 세부 장면 흐름 |

아래 표의 다른 문서는 매번 모두 여는 목록이 아니라, 동사와 장면이 흔들릴 때 내려가는 기준층이다. 생애 지층이 흐리면 `01B`, 귀환/밤/결투 원리가 흔들리면 `01C`, 인물 호감이 약하면 `01D`, 명칭과 능력/소품 경계가 흔들리면 `01E`, 도시/인물/도감 충돌이 있을 때만 `02`~`06`으로 내려간다.

문서별 참조 등급(상시 진입, 본류 기준층, 보조 참조, 원천 초안)과 사용 방식은 `01_세계관_상위규칙.md` 1절 참조 우선순위가 정본이다. 이 문서에서는 반복하지 않는다.

실제 한 화를 준비할 때는 먼저 `000 -> 00 -> 07 -> 해당 에피소드 문서`까지만 연다. 이 흐름으로 현재 회차의 동사, 선택 인물, 표면 사건, 말미 잔류물이 잡히지 않을 때만 기준층을 추가로 확인한다. 도시, 녹티오라 잔류 흔적, 소품을 고를 때는 `02`, `05`, `06`의 `0`번 체급 섹션을 먼저 보고, 핵심 사건 장소/핵심 녹티오라 잔류 흔적/A급 핵심 장치인지부터 판단한다. 후반부 EP21~28도 기본 진입은 `07`의 5B~6C와 해당 EP 문서이며, 장면이 과밀하거나 오독 위험이 있을 때만 `07G`의 해당 절을 추가로 본다. 정식바이블 안에서도 모든 문서가 같은 무게가 아니다.

`에피소드` 폴더의 현재 EP 문서는 실제 연재분 초안 직전의 EP별 작업대다. 먼저 `전면 목표`, 회차별 `첫 장면`, `이야기 기능 요약`, `도시 변화`, 다음 질문/연결 문단을 본다. `녹티오라 씨앗`은 밤 장면을 쓰는 회차에서만 하나 골라 전면화한다. 예전 초안 흔적이나 번호가 `07`, `07G`와 충돌하면 `07`을 우선하고, 해당 EP 문서의 장면을 다시 낮춘다.

`웹소설작성`과 `장면시퀀스` 폴더는 실제 문장감과 전투 동선을 얻기 위한 원천 초안이다. 그대로 정식 원고로 승격하지 않는다. 그 안의 좋은 장면은 `07`의 EP 기능과 해당 EP 문서의 전면 목표에 맞춰 재배열하고, 장면이 과밀할 때만 `07G`의 첫 장면/말미 잔류물을 확인한다. 구식 능력명/직접 가치어/작가용 해설은 `01E`, `08`, `08A` 기준으로 낮추고, 어색한 한국어 조어와 번역체도 `08` 기준으로 다시 본 뒤 사용한다.

### 1.1 현재 이야기 척추

집필자는 매번 모든 설정을 다시 펼치지 말고 아래 척추만 먼저 든다. 세부명, 국가사, 도감 물성은 이 흐름을 장면으로 세울 때만 내려가 확인한다.

- 1부의 동사는 `끼어들다`와 `따라 하다`다. 세계 설명이 아니라 환대와 배제가 같은 손에서 오는 사건으로 시작한다. 셀로는 먼저 사랑받고, 그 다음 안전한 무리가 왜 검은 낙인을 남기는지 보인다.
- 2부의 동사는 `판정하다`와 `값을 붙이다`다. 맞는 말, 규칙, 가격이 실제로 사람을 구한 뒤 늦게 온 얼굴과 값 없는 빵을 설명하지 못하는 이야기다.
- 3부의 동사는 `낮추다`다. 치유와 보호가 필요하다는 사실을 인정한 뒤, 고통을 없애려는 손이 목소리와 질문까지 낮추는 위험을 보인다.
- 4부의 동사는 `보호하다`와 `결정권을 나누다`다. 위험한 파편을 처치하지 않고 이름, 경계, 빈 의자로 보호처 정치에 넘기는 이야기다.
- 5부의 동사는 `기다리다`와 `채우지 않다`다. 진실과 귀환이 만능 해결책처럼 보이는 순간, 낮춘 빛과 열지 않은 문과 빈그릇이 귀환 직전의 낮은 자리를 만든다.
- 6부의 동사는 `분류하다`, `열지 않다`, `자리를 나누다`다. 돌아온 가치가 만든 후폭풍을 감당하는 이야기다. 장례 들판, 수리 인계표, 돌아가지 않은 울티마 열쇠, 차갑지 않은 물컵 자국, 중심석의 당김은 평화의 증거가 아니라 남은 책임의 형태다.

이 척추에 닿지 않는 새 설정은 지금 원고에 넣지 않는다. 매력적인 세부라도 현재 EP의 선택, 피해, 반전, 말미 잔류물을 바꾸지 못하면 보조 후보로만 둔다.

### 1.2 원고 진입 압축 기준

문서가 많아 보일 때는 새 자료를 더 찾지 말고, 아래 순서로 현재 회차를 좁힌다.

1. 먼저 해당 EP의 동사를 하나로 고른다. 예: `분류한다`, `열지 않는다`, `자리를 나눈다`.
2. 그 동사를 실제로 강제하는 표면 사건을 하나 고른다. 예: 젖으면 사라질 길표, 돌아가지 않는 열쇠, 중심으로 미끄러지는 이름표.
3. 그 사건 때문에 선택해야 하는 인물을 하나 세운다.
4. 그 선택에 부딪히는 능력, 소품, 도시 압력 중 하나만 전면화한다.
5. 나머지 설정은 배경 반응, 손버릇, 소품 잔류물로 낮춘다.

원고 진입 직전의 질문은 "이 설정을 더 넣을까"가 아니라 "이 인물이 이 장면에서 무엇을 잃을 위험이 있는가"다. 잃을 것이 없는 설정은 아무리 멋있어도 이번 회차의 전면에 세우지 않는다.

### 1.3 장소/밤/소품 전면화 기준

도시, 녹티오라 잔류 흔적, 소품을 동시에 전면에 올리지 않는다. 먼저 이번 회차의 위험을 만드는 것이 장소 압력인지, 세르 반응인지, 손에 잡히는 물건인지 고른다. 장소가 선택지를 좁히면 `02`, 밤에 남은 흔적이 선택을 시험하면 `05`, 손을 멈추게 하는 물건이 말미 잔류물이 되면 `06`을 우선한다.

나머지는 배경 반응으로 낮춘다. 예를 들어 핵심 장소가 루이나 시장터라면 빈그릇이나 물컵은 시장을 설명하는 물성으로 쓰고, 핵심 소품이 마시지 않은 물컵이라면 도시는 컵을 씻을 손이 필요한 생활 압력으로만 남긴다. 셋 모두를 설명하려 들면 장면은 넓어지지만 선택은 흐려진다.

## 2. 한 화를 준비할 때

한 화를 준비할 때는 모든 설정을 펼치지 않는다. 아래 네 가지만 먼저 정한다.

1. 이번 화의 독자 보상: 사건, 관계 변화, 능력 시연, 녹티오라 잔류 흔적, 물성 발견 중 무엇을 줄 것인가.
2. 이번 화에서 감출 것: A의 뇌 은유, 버린 가치관, 귀환 원리, 작가용 의미는 직접 말하지 않는다.
3. 이번 화의 중심 인물 하나: 그 인물이 옳은 이유와 위험한 이유를 행동으로 보인다.
4. 이번 화의 중심 물성 하나: 문, 끈, 잔, 칼, 의자, 빛, 물 같은 물건 하나가 선택을 바꾸게 한다.

EP 하나는 실제 플랫폼 연재 회차 2~6개로 쪼개진다. 각 연재 회차는 6000자 이상을 버틸 작은 목표, 중반 반전, 말미 잔류물을 가져야 한다. 설명이 아니라 장면 보상으로 독자를 붙잡는다.

현재 EP 문서는 28개 모두 원고 전 기준으로 같은 말미 구조를 가진다. 원고를 쓰기 전에는 해당 EP 문서의 `이야기 기능 요약`으로 이번 묶음의 동사를 정하고, `도시 변화`로 결투 뒤 남길 후폭풍을 고른 뒤, `녹티오라 씨앗`에서 밤에 살아남을 흔적 하나만 전면화한다.

### 2.1 초반 10화 잠금

초반 10화의 목적은 세계관을 이해시키는 것이 아니라, 셀로와 동료들의 작은 선택을 계속 따라가고 싶게 만드는 것이다. 첫 장면마다 독자가 즉시 이해할 기한이 있어야 한다. 마감 전 임시 잠자리나 일거리를 얻기, 마지막 종 전에 깃발문을 통과하기, 밀물 전 지도를 받기, 검문이 끝나기 전 접힌 지도를 통행선으로 바꾸기처럼 실패 비용이 한 문장 안에 잡혀야 한다.

초반에는 "왜 이런 세계인가"보다 "이 손이 지금 무엇을 잃을 수 있는가"가 먼저다. 셀로는 자기 정체보다 떨리는 손목을 먼저 보고, 에이도스는 이론보다 거짓 가지를 먼저 자르며, 콘코르디아는 자기 이름보다 늦은 손목을 먼저 알아차린다. 독자가 세 사람의 손을 기억하면 국가명과 세계사는 뒤에 와도 버틴다.

초반 말미 훅은 거대한 비밀 공개가 아니라 남은 물성 하나로 닫는다. 손톱 밑 획, 빠진 발자국, 새 코트의 검은 안감, 어긋난 박수, 흰 타일 같은 작은 잔류물이 다음 화의 이유가 되어야 한다. 이 원칙이 흔들리면 설정을 더 넣지 말고 해당 화의 기한, 중반 반전, 말미 잔류물을 먼저 고친다.

### 2.2 원고 완성 후 검수 동선

초안이 끝나면 아래 순서로 확인한다. 이 동선은 세계 사실과 금지선을 보는 참조 순서이지, 문장이나 전개 방식을 고치라는 틀이 아니다.

1. 회차 시점: 중심 인물들의 개별 `등장인물/*.md`에서 밤 진입 시점과 `회차 운용 잠금` 절(있는 경우)을 확인한다. 이번 화 시점에 아직 일어나지 않은 성장, 관계 변화, 능력 전면화가 새었는지 본다.
2. 작중어·금지선: `08_원고용_작중어_빠른검수표.md`의 불변선과 한국어 금지선을 통과시키고, `000` 6절의 레거시 오염 검색어(초반 호칭 오류 포함)를 원고에서 검색한다.
3. 감출 것: A의 뇌 은유, 능력 원리, 작가용 의미가 대사나 해설로 새지 않았는지 본다.
4. 신규 설정 등록: 이번 화에서 새로 만든 인물, 소품, 명칭, 장소가 있으면 5절 원칙대로 본류 문서에 먼저 흡수한다. 원고에만 존재하는 설정을 남기지 않는다.

## 3. 보조 문서를 여는 경우

보조 문서는 아래 상황에서만 연다.

| 필요 | 여는 문서 |
|---|---|
| 생활 인물이나 배경 인물의 감각이 필요할 때 | `03A_국가별_생활인물_명부.md` |
| 인물의 대사, 행동, 밤 위상, 성장, 지역 말투, 관계 반응이 흔들릴 때 | `03B`, `03C_존재위상`, `03F`, 해당 `등장인물/*.md`(회차 운용 잠금 절 포함), 필요 관계 문서와 국가/도시/단체 문서 |
| 특정 생태/무기물의 세부 물성이 필요할 때 | `05`, `06B` |
| 특정 EP의 보조 후보가 필요할 때 | `07`, `07G`, 해당 `에피소드/부/EP N.md` |
| 원고에 작가용 설명이나 어색한 문체가 새는지 확인할 때 | `08`, `08A` |
| 죽음, 귀환, 시간선, 빛/파동, 신중함/보존축이 충돌할 때 | 먼저 `01B`, `01C`, `07`, `07G`를 본다 |
| 과거 장면 초안의 감정선이나 전투 동선이 필요할 때 | `웹소설작성`, `장면시퀀스` |

보조 문서는 본류를 대체하지 않는다. 보조 문서에만 있는 설정은 아직 원고 기준이 아니다.
`웹소설작성`과 `장면시퀀스`도 마찬가지다. 그 안의 문장과 사건은 재료이지 정식 문장이나 최종 명칭이 아니다.

## 4. 고도화 방향

앞으로의 고도화는 새 감사표, 대응표, 검수표, 체크리스트를 늘리는 방식으로 진행하지 않는다. 이미 문서는 충분히 많다. 새 정보가 필요하면 먼저 기존 정식바이블, 개별 인물 문서, 도시 문서, 도감 문서 안의 한 문단으로 흡수한다.

고도화의 우선순위는 아래 순서다.

1. 인물이 더 사랑받는가.
2. 장면이 설명보다 선택과 행동으로 움직이는가.
3. 결투가 가치관 해설이 아니라 능력, 소품, 물성, 피해, 반전으로 보이는가.
4. 지금 알아야 할 설정과 나중에 알아도 되는 설정이 분리되는가.
5. 새 문서가 늘어나는 대신 기존 참조 동선이 짧아지는가.

세션 말미에는 반드시 단계 전환 판정을 한다. 이번 단계의 핵심 산출물이 본류 문서에 흡수되었는지, 남은 문제가 다음 단계 진행을 막는지, 이제 다음 단계로 넘어갈 수 있는지를 판단한다. 판정은 `다음 단계 이동`, `같은 단계 1회 추가`, `보류 후 이동` 중 하나로 끝낸다. 다음 단계 진행을 막지 않는 명칭, 표현, 소품, 문장 감각 문제는 보류하고 넘어간다.

## 5. 새 설정 추가 원칙

새 인물, 도시, 소품, 생태, 역사, 결투를 만들 때도 새 기록 문서를 먼저 만들지 않는다.

1. 먼저 본류 문서 중 어디에 들어갈지 정한다.
2. 인물이라면 `03` 또는 개별 `등장인물` 문서에 흡수한다.
3. 도시/국가라면 `02`, 도시별 문서, 국가별 문서에 흡수한다.
4. 소품/생태/녹티오라 잔류 흔적이라면 `05` 또는 `06`에 흡수한다.
5. 회차 투입이라면 `07` 또는 해당 EP 문서에 흡수하고, 세부 흐름 보조가 필요할 때만 `07G`에 남긴다.
6. 작업기록은 꼭 필요한 근거 보존용으로만 남긴다.

작업기록만 있고 본류 문서에 흡수되지 않은 설정은 집필 참조 불가다.

## 5.1 사용자 피드백 반영 작업 방식

사용자 발화 처리 규칙과 반영 순서는 `000_작업전_필수숙지_최상위인덱스.md` 0절(절대 순서)과 3절(사용자 발화 우선 원칙)이 정본이다. 여기서는 반복하지 않는다. 핵심만 남기면: 사용자 발화가 요구사항의 1차 근거이고, 작업자가 이전에 만든 요약은 참고 자료일 뿐이며, 수정 시 같은 설정을 참조하는 색인/흐름/문체 문서까지 함께 맞춘다.

## 6. 빠른 판단

| 막히는 지점 | 먼저 볼 곳 |
|---|---|
| 세계 법칙 | `01`, `01C` |
| 인물 | `03`, 필요하면 `03B`, `03C_존재위상`, `03F`와 개별 `등장인물` 문서 |
| 도시/장소 | `02`, 필요하면 도시별 문서 |
| 결투 | `01D` 8절, `07`, 필요하면 `07G` |
| 녹티오라 잔류 흔적 | `05` |
| 소품/물성 | `06` |
| 회차 배치 | `07`, 필요하면 `07G` |
| 작중 표현과 문체 | `08`, `08A` |

가장 중요한 원칙은 하나다. 원고에서는 설정을 증명하지 말고, 인물이 어떤 선택을 하게 만들었는지만 보여준다.
