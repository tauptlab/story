# 집필진입점

역할: 실제 연재분을 쓰기 전에 여는 집필 진입 문서다. 단, 사용자가 특정 파일 하나만 링크하거나 작업 요구를 짧게 던진 경우에는 반드시 `000_작업전_필수숙지_최상위인덱스.md`를 먼저 열고 이 문서로 들어온다. 이 문서는 작업 장부가 아니라, 지금 원고에 필요한 정사만 빠르게 고르는 입구다.

---

## 1. 최우선 참조

실제 한 화를 쓰기 시작할 때의 최소 동선은 `000_작업전_필수숙지_최상위인덱스.md -> 00_집필진입점.md -> 07_회차_투입_색인.md -> 에피소드/해당 부/EP N.md`다. 이 흐름으로 현재 회차의 동사, 표면 사건, 말미 잔류물이 잡히면 곧장 원고 설계로 들어간다. 각 EP 작업대 상단의 `회차 참조` 블록에는 `07`의 해당 절과 `07G`의 해당 절로 바로 가는 링크가 있으므로, 색인 전체를 읽지 않고 그 링크로 진입한다.

본류 검수와 실제 집필 진입은 아래처럼 나눈다.

| 목적 | 여는 문서 | 확인할 것 |
|---|---|---|
| 전체 1~6부 흐름 검수 | `00 -> 01B -> 07` | 시대 전환, 부별 사건 엔진, EP1~28의 표면 사건과 다음 질문 |
| 한 EP 묶음 준비 | `000 -> 00 -> 07 -> 에피소드/해당 부/EP N.md` | 해당 EP의 동사, 첫 사건, 선택 인물, 말미 잔류물 |
| 실제 연재 한 화 준비 | `해당 EP 문서` | 회차별 첫 장면, 기한, 중반 반전, 회차 끝 질문 |
| 장면이 흔들릴 때 | `01C~01E`, `02~06`, 필요 시 `07G` | 귀환/밤/결투 원리, 인물 호감, 명칭, 도시/도감 충돌, 세부 장면 흐름 |

아래 표의 다른 문서는 매번 모두 여는 목록이 아니라, 동사와 장면이 흔들릴 때 내려가는 기준층이다. 생애 지층이 흐리면 `01B`, 귀환/밤/결투 원리가 흔들리면 `01C`, 인물 호감이 약하면 `01D`, 명칭과 능력/소품 경계가 흔들리면 `01E`, 도시/인물/도감 충돌이 있을 때만 `02`~`06`으로 내려간다.

문서별 참조 등급(상시 진입, 본류 기준층, 보조 참조, 원천 초안)과 사용 방식은 `01_세계관_상위규칙.md` 1절 참조 우선순위가 정본이다. 이 문서에서는 반복하지 않는다.

`에피소드` 폴더의 현재 EP 문서는 실제 연재분 초안 직전의 EP별 작업대다. 먼저 `전면 목표`, 회차별 `첫 장면`, `이야기 기능 요약`, `도시 변화`, 다음 질문/연결 문단을 본다. `녹티오라 씨앗`은 밤 장면을 쓰는 회차에서만 하나 골라 전면화한다. 예전 초안 흔적이나 번호가 `07`, `07G`와 충돌하면 `07`을 우선하고, 해당 EP 문서의 장면을 다시 낮춘다.

`웹소설작성`과 `장면시퀀스` 폴더는 실제 문장감과 전투 동선을 얻기 위한 원천 초안이다. 그대로 정식 원고로 승격하지 않는다. 그 안의 좋은 장면은 `07`의 EP 기능과 해당 EP 문서의 전면 목표에 맞춰 재배열하고, 장면이 과밀할 때만 `07G`의 첫 장면/말미 잔류물을 확인한다. 구식 능력명/직접 가치어/작가용 해설은 `01E`, `08`, `08A` 기준으로 낮추고, 어색한 한국어 조어와 번역체도 `08` 기준으로 다시 본 뒤 사용한다.


### 2.2 원고 완성 후 검수 동선

초안이 끝나면 아래 순서로 확인한다. 이 동선은 세계 사실과 금지선을 보는 참조 순서이지, 문장이나 전개 방식을 고치라는 틀이 아니다.

1. 회차 시점: 중심 인물들의 개별 `등장인물/*.md`에서 밤 진입 시점과 `회차 운용 잠금` 절(있는 경우)을 확인한다. 이번 화 시점에 아직 일어나지 않은 성장, 관계 변화, 능력 전면화가 새었는지 본다.
2. 작중어·금지선: `08_원고용_작중어_빠른검수표.md`의 불변선과 한국어 금지선을 통과시키고, `000` 6절의 레거시 오염 검색어(초반 호칭 오류 포함)를 원고에서 검색한다.
3. 감출 것: A의 뇌 은유, 능력 원리, 작가용 의미가 대사나 해설로 새지 않았는지 본다.
4. 신규 설정 등록: 이번 화에서 새로 만든 인물, 소품, 명칭, 장소가 있으면 5절 원칙대로 본류 문서에 먼저 흡수한다. 원고에만 존재하는 설정을 남기지 않는다.

## 3. 보조 문서를 여는 경우

보조 문서는 아래 상황에서만 연다.

| 필요 | 여는 문서 |
|---|---|
| 생활 인물이나 배경 인물의 감각이 필요할 때 | `03A_국가별_생활인물_명부.md` |
| 인물의 대사, 행동, 밤 위상, 성장, 지역 말투, 관계 반응이 흔들릴 때 | `03B`, `03C_존재위상`, `03F`, 해당 `등장인물/*.md`(회차 운용 잠금 절 포함), 필요 관계 문서와 국가/도시/단체 문서 |
| 특정 생태/무기물의 세부 물성이 필요할 때 | `05`, `06B` |
| 특정 EP의 보조 후보가 필요할 때 | `07`, `07G`, 해당 `에피소드/부/EP N.md` |
| 원고에 작가용 설명이나 어색한 문체가 새는지 확인할 때 | `08`, `08A` |
| 죽음, 귀환, 시간선, 빛/파동, 신중함/보존축이 충돌할 때 | 먼저 `01B`, `01C`, `07`, `07G`를 본다 |
| 과거 장면 초안의 감정선이나 전투 동선이 필요할 때 | `웹소설작성`, `장면시퀀스` |

보조 문서는 본류를 대체하지 않는다. 보조 문서에만 있는 설정은 아직 원고 기준이 아니다.
`웹소설작성`과 `장면시퀀스`도 마찬가지다. 그 안의 문장과 사건은 재료이지 정식 문장이나 최종 명칭이 아니다.


## 5. 새 설정 추가 원칙

새 인물, 도시, 소품, 생태, 역사, 결투를 만들 때도 새 기록 문서를 먼저 만들지 않는다.

1. 먼저 본류 문서 중 어디에 들어갈지 정한다.
2. 인물이라면 `03` 또는 개별 `등장인물` 문서에 흡수한다.
3. 도시/국가라면 `02`, 도시별 문서, 국가별 문서에 흡수한다.
4. 소품/생태/녹티오라 잔류 흔적이라면 `05` 또는 `06`에 흡수한다.
5. 회차 투입이라면 `07` 또는 해당 EP 문서에 흡수하고, 세부 흐름 보조가 필요할 때만 `07G`에 남긴다.
6. 작업기록은 꼭 필요한 근거 보존용으로만 남긴다.

작업기록만 있고 본류 문서에 흡수되지 않은 설정은 집필 참조 불가다.

## 5.1 사용자 피드백 반영 작업 방식

사용자 발화 처리 규칙과 반영 순서는 `000_작업전_필수숙지_최상위인덱스.md` 0절(절대 순서)과 3절(사용자 발화 우선 원칙)이 정본이다. 여기서는 반복하지 않는다. 핵심만 남기면: 사용자 발화가 요구사항의 1차 근거이고, 작업자가 이전에 만든 요약은 참고 자료일 뿐이며, 수정 시 같은 설정을 참조하는 색인/흐름/문체 문서까지 함께 맞춘다.

---


가장 중요한 원칙은 하나다. 원고에서는 설정을 증명하지 말고, 인물이 어떤 선택을 하게 만들었는지만 보여준다.
