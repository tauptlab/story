# 퇴행

역할: 압박이 커질 때 인물이 더 좁아지는 방향을 잡는다. 퇴행은 성격 붕괴가 아니라 익숙한 생존법을 과하게 쓰는 현상이다.

| 유형 | 퇴행 장면 |
|---|---|
| 1번 | 규칙과 옳음을 앞세워 사람의 숨을 줄인다 |
| 2번 | 도움을 핑계로 상대의 선택을 대신한다 |
| 3번 | 성과와 박수로 자기 빈칸을 가린다 |
| 4번 | 결핍과 특별함에 머물며 현재 관계를 밀어낸다 |
| 5번 | 관찰과 해석으로 관계를 통제한다 |
| 6번 | 대비, 의심, 충성 시험으로 관계를 확인한다 |
| 7번 | 농담과 새 이야기로 아픔을 닫는다 |
| 8번 | 힘과 분노로 먼저 장면을 점유한다 |
| 9번 | 자기 욕구를 지우고 낮은 평화에 숨는다 |

## 금지

- 한 장면 편의를 위해 갑자기 반대 성격으로 만들지 않는다.
- 퇴행을 악역화하지 않는다. 독자가 "그래서 저랬구나"까지 이해해야 한다.
- 미래에 겪을 상처를 현재 회차 대사로 미리 고백하게 하지 않는다.
