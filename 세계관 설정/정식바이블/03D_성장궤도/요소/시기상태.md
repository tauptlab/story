# 시기상태

역할: 같은 인물의 초반, 중반, 후반 상태를 섞지 않게 한다.

## 시기상태 필드

| 필드 | 내용 |
|---|---|
| 시기 | EP 또는 회차 범위 |
| 사용 가능 정보 | 그 시점의 독자와 인물이 아는 것 |
| 불안 | 이 시기에 가장 앞서는 두려움 |
| 생존법 | 이 시기에 주로 쓰는 대응 |
| 퇴행 신호 | 압박 속에서 좁아지는 행동 |
| 성장 선택 | 이 시기에 가능한 작은 변화 |
| 밤에서 과장되는 것 | 밤 진입 시 몸에 먼저 올라오는 충동 |
| 대사/행동 허용 | 써도 되는 말투와 행동 |
| 금지 | 미래 정보, 완성 능력, 해설화 금지 |

## 현재 회차 사용 가능 범위 작성법

좋은 문장:

> EP1에서는 신규 밤의 사람이며, 조건과 훈련으로 충동을 억누르려는 단계다. 완성된 기술 해설 대신 손짓, 조건문, 통제 욕구의 미숙한 압력으로 먼저 보인다.

나쁜 문장:

> 해당 EP 문서와 회차별 공개 정보 우선.

범위 문장은 반드시 `지금 써도 되는 것`과 `지금 쓰면 안 되는 것`을 함께 적는다.
