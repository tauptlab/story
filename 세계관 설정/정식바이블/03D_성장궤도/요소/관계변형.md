# 관계변형

역할: 같은 인물도 상대에 따라 생존법이 다르게 올라오는 지점을 관리한다.

## 기준

관계변형은 개별 관계 문서가 우선한다. 이 문서는 관계 문서 작성 전 임시 판단 기준이다.

| 상황 | 확인할 것 |
|---|---|
| 상대가 보호 대상일 때 | 도움과 통제가 섞이지 않는가 |
| 상대가 경쟁자일 때 | 증명 욕구가 대사를 앞지르지 않는가 |
| 상대가 과거 상처를 건드릴 때 | 현재 회차에서 알 수 없는 고백을 하지 않는가 |
| 상대가 비슷한 결핍을 가질 때 | 공감이 자기 투사로 변하지 않는가 |
| 상대가 권위자일 때 | 순종, 반항, 계산 중 어느 생존법이 먼저 오르는가 |

## 작성 기준

- 관계변형은 새로운 성격이 아니다. 기본 생존법이 특정 상대 앞에서 다른 모양으로 꺾이는 것이다.
- 특정 상대에게만 쓰는 말버릇이나 몸짓은 인물 본문보다 관계 문서에 둔다.
- 서브에이전트에는 현재 회차까지의 관계 정보만 준다.
