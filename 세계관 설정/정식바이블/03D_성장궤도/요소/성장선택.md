# 성장선택

역할: 성장 장면에서 인물이 같은 욕구를 덜 해로운 방식으로 쓰게 한다. 성장선택은 능력을 더 잘 쓰는 장면이 아니라, 능력을 쓰지 않거나 늦추는 장면일 수 있다.

| 유형 | 성장선택 |
|---|---|
| 1번 | 옳음보다 회복 가능한 다음 행동을 본다 |
| 2번 | 돕지 않는 시간도 관계임을 인정한다 |
| 3번 | 실패 뒤에도 남는 자기 목소리를 선택한다 |
| 4번 | 특별하지 않아도 진짜인 감정을 받아들인다 |
| 5번 | 모르는 채로도 곁에 남는다 |
| 6번 | 불확실한 상태에서 선택한 신뢰를 지킨다 |
| 7번 | 즐거움 뒤에 남은 질문을 견딘다 |
| 8번 | 힘을 내려놓고도 지킬 수 있음을 배운다 |
| 9번 | 침묵 대신 작은 선택을 말한다 |

## 작성 기준

- 성장선택은 작게 보일수록 좋다. 손을 내리는 일, 대답을 기다리는 일, 문을 열지 않는 일이 더 강할 수 있다.
- 성장 뒤에도 옛 생존법은 남는다. 완전히 다른 사람이 되지 않는다.
- 성장선택은 관계나 장소의 물성과 연결한다.
