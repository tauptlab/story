# 불안

역할: 인물이 피하려는 감정을 정한다. 불안은 대사로 설명하지 않고, 먼저 보는 것과 먼저 만지는 것으로 장면화한다.

| 유형 | 핵심 불안 | 장면 신호 |
|---|---|---|
| 1번 | 틀리거나 잘못될까 두려움 | 선, 규칙, 고칠 곳을 먼저 본다 |
| 2번 | 필요 없는 사람이 될까 두려움 | 상대가 원하는 것을 먼저 맞힌다 |
| 3번 | 실패해 가치 없게 보일까 두려움 | 박수, 성과, 표정을 확인한다 |
| 4번 | 자기만의 얼굴을 잃을까 두려움 | 이름, 결핍, 비교에 민감하다 |
| 5번 | 모르면 침범당할까 두려움 | 거리, 단서, 기록을 먼저 잡는다 |
| 6번 | 기반이 무너질까 두려움 | 출구, 책임자, 보증을 찾는다 |
| 7번 | 고통에 갇힐까 두려움 | 농담, 전환, 새 가능성으로 몸을 뺀다 |
| 8번 | 약해져 지배당할까 두려움 | 먼저 밀고 들어가거나 지킬 대상을 선점한다 |
| 9번 | 갈등으로 관계가 깨질까 두려움 | 자기 요구를 낮추고 자리만 남긴다 |

## 작성 기준

- 불안은 나약함이 아니라 오래 살아남은 방식의 출발점이다.
- 현재 회차에서 아직 모를 불안을 미래 지식으로 말하지 않는다.
- 첫 등장 장면에서는 불안을 한 가지 물성으로 낮춘다. 설명문으로 "두려웠다"를 반복하지 않는다.
