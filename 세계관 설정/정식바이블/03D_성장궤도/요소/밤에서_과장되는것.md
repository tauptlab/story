# 밤에서 과장되는 것

역할: 밤의 사람에게 올라오는 본능적 충동을 성장궤도와 연결한다. 밤은 성격을 바꾸지 않는다. 밤은 낮에서 다스리던 생존법을 빠르고 원초적으로 만든다.

| 유형 | 밤에서 과장되는 것 |
|---|---|
| 1번 | 판정, 교정, 금지선 |
| 2번 | 보호 충동, 허락 없는 도움 |
| 3번 | 증명, 경쟁, 박수 집착 |
| 4번 | 이름, 얼굴, 결핍, 비교 |
| 5번 | 단서, 구조, 거리 두기 |
| 6번 | 경계, 감시, 방어, 책임 |
| 7번 | 회피, 속도, 이야기 전환 |
| 8번 | 보호와 지배의 혼합 |
| 9번 | 안식, 지연, 자리 비움이 자기소거로 기우는 현상 |

## EP1 주의

EP1 첫 밤의 위험은 으스스한 자연현상이 아니라 밤의 사람이 된 지 얼마 안 된 인물들의 충동과 다툼이다. 표적판, 줄, 기둥, 짐이 무너지는 물리 위험은 그 다툼의 여파로 쓴다.
