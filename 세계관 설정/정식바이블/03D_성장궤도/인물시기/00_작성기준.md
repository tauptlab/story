# 인물시기 카드 작성기준

역할: 인물별 성장궤도를 장문 생애사가 아니라 회차별 운용 카드로 만든다.

## 템플릿

```md
# 인물명 EP범위

시기:

사용 가능 정보:

불안:

생존법:

퇴행 신호:

성장 선택:

밤에서 과장되는 것:

대사/행동 허용:

금지:
```

## 작성 원칙

1. 한 카드는 한 시기만 다룬다.
2. 미래 복선의 정답을 카드에 넣지 않는다.
3. 인물 문서가 길어지면 이 카드로 분리한다.
4. 카드가 없는 조연은 해당 EP 작성 전에 짧게라도 만든다.
5. `금지` 칸은 반드시 쓴다. 이 칸이 없으면 초반 정보량이 쉽게 무너진다.
