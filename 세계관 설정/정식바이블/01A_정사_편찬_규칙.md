# 네우로시아 정사 편찬 규칙

이 문서는 네우로시아를 하나의 완성된 세계와 역사로 확장하기 위한 보조 정사 규칙이다. 실제 연재분을 준비할 때는 `00`, `01`, `01B`, `02`~`07`, 해당 EP 문서를 먼저 보고, 이 문서는 새 인물, 도시, 건축물, 생물, 물건, 의식, 전투 기술이 본류 문서 안에 흡수될 수 있는지 확인할 때만 연다.

---

## 1. 최상위 전제

네우로시아는 A의 보이지 않는 선택, 기억, 감정, 몸의 반응(심장 박동, 숨, 긴장 같은 신체 감각)을 세계로 은유한 판타지 세계다. 그러나 작중 인물은 자신이 어떤 바깥 원형의 일부인지 알지 못한다.

따라서 새 요소를 만들 때는, 그 요소가 A의 어떤 내면 기능을 은유하는지 먼저 정하고, 아래 표에서 같은 기능을 이미 맡고 있는 국가/장소/사물에 붙인다. 표의 왼쪽은 A의 내면 기능이고, 오른쪽은 그 기능이 작중에서 이미 확정된 국가, 장소, 사물이다.

| A의 내면 기능 | 이를 은유하는 확정 요소 (국가, 장소, 사물) | 작가용 뇌 은유 (원고 노출 금지) |
|---|---|---|
| 질서와 판단 | 프론티아, 아스트리온, 넥서스, 규약, 계산탑 | 전전두피질 — 논리, 계획, 통제, 결과주의 |
| 욕망과 보상 | 눅세린, 아르도르, 아우룸 결정체, 환락 시장 | 보상회로(측좌핵) — 쾌락, 인정, 소속, 거래 |
| 경보와 공포 | 암디르, 아미시오 계곡, 경보성, 그림자 계약 | 편도체 — 공포, 분노, 생존 본능 |
| 갈등 조율 | 센시르, 리테노아, 디아파손 실, 에코 샘 | 전대상피질 — 공감, 고통 처리, 갈등 중재 |
| 기억과 길찾기 | 히프노레아, 이렐사드, 겹문장 결정, 망각의 강 | 해마 — 기억의 저장, 관리, 왜곡 |
| 몸의 균형 | 아르스 모투스, 모르파 전당, 균형 첨탑 | 소뇌 — 운동 제어, 숙련, 절차 기억 |
| 의미와 언어 | 오라쿨룸, 별의 언어, 타키투스 도서관 | 측두엽 — 언어, 의미, 정체성, 신비 탐구 |
| 밤의 재배열 | 녹티오라, 밤의 세 작동 원리(01C 4C), 퀘스티오 | REM 수면의 꿈 — 기억 정리와 감정 재처리 |
| 자기 이야기(자기서사) | 아이덴티아 회랑, 아노니마 초상화 | - |
| 앞읽기와 오차(예측과 예측 실패) | 프라이오라 관측소, 프라이오라 격자, 어긋난 번개 자국 | - |
| 중요도 판정(무엇을 먼저 처리할지 정하는 감각) | 살리엔티아 관문, 경보성, 묻힌 종소리 | - |
| 신체 감각(박동, 숨, 몸이 먼저 아는 반응) | 몸울림 우물, 박동수, 숨의 세르 | - |

작가용 어원 메모: `세르(Cere)`는 뇌(Cerebrum)와 Core/Err의 조합에서 온 작가용 조어다. A가 경험으로 얻은 감정/가치관 상태가 세계에 물성으로 굳은 것이며, 이 어원은 원고에 노출하지 않는다.

## 2. 세계 구성요소의 기본 기록 항목

모든 작중 요소를 별도 문서나 카드로 만들지 않는다. 반복 등장하거나 회차의 선택, 결투, 녹티오라 잔류 흔적, 귀환 빌드업을 바꾸는 요소만 아래 항목을 갖춘다. 스쳐 지나가는 배경 요소는 관련 본류 문서의 한 문장, 생활 인물의 손버릇, 장소 물성 하나로 충분하다.

기록의 목적은 설정을 늘리는 것이 아니라 원고에서 무엇을 보여주고 무엇을 감출지 정하는 데 있다. 핵심도가 낮은 요소까지 같은 깊이로 정리하면 정보가 섞이므로, A/B급 정사와 회차 기능이 있는 요소부터 우선 기록한다.

새 요소가 특정 EP의 사건, 결투, 녹티오라 잔류 흔적, 도시 변화, 말미 질문을 바꾼다면 `07_회차_투입_색인.md`와 해당 `에피소드` 문서에 먼저 흡수한다. 작업기록에만 남은 요소는 정사 후보일 수 있어도 아직 실제 원고 기준은 아니다.

### 2.1 인물 항목

| 항목 | 설명 |
|---|---|
| 이름 | 작중 이름과 별칭 |
| 소속 | 국가, 도시, 층계, 단체 |
| 은유 | 어떤 생각, 감정, 가치관, 기능을 은유하는가 |
| 선물 | 이 인물이 세계에 주는 긍정적 기능 |
| 상처 | 이 가치관이 왜 과도해졌는가 |
| 거짓 믿음 | 자신을 지키기 위해 믿는 틀린 명제 |
| 사랑스러운 습관 | 독자가 인물로 사랑하게 되는 작은 행동 |
| 능력 | 세르가 어떤 전투/비전투 형태로 발현되는가 |
| 대가 | 능력을 쓸 때 잃는 것 |
| 반작용 | 어떤 가치관이나 상황 앞에서 약해지는가 |
| 관계 | 반드시 얽혀야 하는 인물 |
| 스토리 기능 | 어느 장면에서 어떤 선택을 강요하는가 |

### 2.2 장소 항목

| 항목 | 설명 |
|---|---|
| 이름 | 도시, 마을, 건축물, 성역, 폐허 |
| 소속 | 국가 또는 층계 |
| 은유 | 어떤 생각, 가치관, 상처를 공간화했는가 |
| 첫인상 | 독자가 처음 봤을 때 떠올릴 감각 |
| 구조 | 길, 층, 중심부, 외곽, 금지 구역 |
| 거주자 | 주류 주민, 소수 집단, 떠돌이 |
| 유기물 | 식물, 동물, 균류, 생체 구조물 |
| 무기물 | 광물, 금속, 건축 소재, 장치 |
| 갈등 | 이 장소가 낳는 딜레마 |
| 장면 기능 | 추격, 전투, 고백, 수련, 반전, 휴식 등 |
| 변화 | 1부~6부를 지나며 어떻게 변하는가 |

### 2.3 물건/장치 항목

| 항목 | 설명 |
|---|---|
| 이름 | 작중 명칭 |
| 재료 | 어떤 세르/물질/기억으로 만들어졌는가 |
| 소속 | 국가, 단체, 제작자 |
| 용도 | 일상, 전투, 의식, 통제, 치유 |
| 상징 | 어떤 생각을 물체화했는가 |
| 작동 조건 | 언제 켜지고 언제 멈추는가 |
| 부작용 | 사용할수록 무엇이 왜곡되는가 |
| 관련 인물 | 누가 만들고, 누가 빼앗고, 누가 망가뜨리는가 |
| 장면 기능 | 발견, 추리 단서, 전투 변수, 감정 폭발 장치 |

### 2.4 유기물 항목

| 항목 | 설명 |
|---|---|
| 이름 | 식물, 동물, 공생체, 변이체 |
| 서식지 | 어느 국가/층계에서 자라는가 |
| 먹이/세르 | 어떤 세르를 흡수하는가 |
| 행동 | 어떤 감정이나 자극에 반응하는가 |
| 상징 | 무엇을 은유하는가 |
| 활용 | 약, 음식, 장신구, 위험, 의식, 복선 |
| 스토리 기능 | 인물 성격을 드러내거나 사건을 촉발하는 방식 |

## 3. 정사 등급

설정이 계속 늘어나도 모순을 피하기 위해 모든 항목은 정사 등급을 가진다.

| 등급 | 의미 | 사용 규칙 |
|---|---|---|
| A급 정사 | 이미 세계관/연재본/에피소드 설계에 반복 등장한 핵심 설정 | 변경 금지. 변경 시 반드시 연표와 관련 문서도 수정 |
| B급 정사 | 아직 본문에는 적게 나왔지만 앞으로 사용하기로 확정한 설정 | 이름과 기능 유지. 세부는 보강 가능 |
| C급 후보 | 확장 아이디어, 신규 인물/지역/장치 | 향후 회차에 넣기 전 검토 |
| 폐기/대체 | 모순되거나 더 좋은 설정으로 대체된 요소 | 삭제하지 말고 이유를 기록 |

현재 A급 정사:

- 네우로시아는 한 사람의 안쪽에 펼쳐진 세계다.
- 국가는 공간, 층계는 머무는 깊이와 밝기다.
- 세르는 마음의 결이 돌, 빛, 물, 숨으로 굳은 것이다.
- 일루미라와 녹티오라는 낮/밤 순환을 이룬다.
- 므네모신은 잊히거나 억압된 가치관의 저류다.
- Mnemosyn 귀환은 봉인 해제나 단순 부활이 아니라, A가 과거 생존을 위해 버렸던 가치관을 성장 뒤 다시 감당 가능한 형태로 받아들이는 사건이다.
- 녹티오라는 낮의 모든 설정이 아니라, A에게 강하게 남은 상처, 욕망, 후회, 약속, 미완의 질문이 밤에 형태를 얻는 층계다.
- 결투는 외부 사건인 동시에 가치관 충돌이며, 승패보다 전투 뒤 어떤 가치가 재배치되는지가 중요하다.
- 근원층은 오래된 세르가 처음 이름을 얻는 자리다.
- 셀로는 스스로를 버리지 않으려는 세르가 이름을 얻은 인물이며, 마지막에는 여러 울림을 서로 통과시키는 공명으로 간다.
- 악역은 반드시 지키려는 것이 있다.

## 4. 국가와 층계의 구분

국가와 층계는 절대 같은 분류가 아니다.

| 분류 | 항목 | 의미 |
|---|---|---|
| 국가 | 프론티아, 눅세린, 암디르, 센시르, 히프노레아, 아르스 모투스, 오라쿨룸 | 기능과 가치관이 공간/문화로 굳어진 영역 |
| 층계 | 미라마인드, 일루미라, 녹티오라, 므네모신, 근원층 | 머무는 깊이, 밝기, 밤의 결 |
| 접경 지대 | 살리엔티아 관문, 프라이오라 관측소, 아이덴티아 회랑, 몸울림 우물 | 국가와 층계 사이에서 기능을 중계하는 특수 구역 |

층계의 작가용 의미(원고 노출 금지)는 아래와 같다. A의 외부 경험은 `미라마인드 -> 일루미라 -> 녹티오라` 순서로 내면의 사건으로 번역되고 처리된다.

| 층계 | 작가용 의미 |
|---|---|
| 미라마인드 | 현실반사계. 외부 자극(사건, 말, 감각)이 내면의 언어(상징, 물성)로 번역되는 최초의 경계이자 필터. A의 방어기제에 따라 번역이 왜곡될 수 있다 |
| 일루미라 | 낮, 표면의식. 번역된 정보가 국가들에 의해 인식되고 상호작용하는 주 무대. 해결되지 않은 문제는 녹티오라로 넘어간다 |
| 녹티오라 | 밤, 무의식적 재처리. 세부 원리는 `01C` 4~4D |
| 므네모신 | 새벽녘, 망각과 억압의 기록보관소. 패배하거나 억압된 가치관이 격리되는 독립 차원. 시간의 흐름이 거의 정체되어 있고, 큰 충격이나 각성 같은 특별한 계기로만 귀환이 열린다 |
| 근원층 | 신경심핵계, 존재의 근원. 태고 존재의 원형이 있는 최심부. 셀로가 도달해 합일하는 장소가 아니라, 이름을 중심에서 내려놓고 자리를 나누는 마지막 압력으로 운용한다 |

작가용 개념 `왕좌`: 국가의 통치자가 아니라, 현재 A의 행동과 가치판단에 가장 강한 영향을 미치는 가치관이 차지한 의식의 주도권을 뜻한다. "지금 A를 움직이는 핵심 가치관이 무엇인가"가 곧 왕좌의 주인이다. 원고에서는 왕좌를 개념으로 설명하지 않고 권력, 자리, 의자, 이름표의 물성으로만 보인다.

한 인물은 `눅세린 출신`이면서 `므네모신으로 가라앉을` 수 있다. 이것은 출신 국가와 존재 상태가 다르기 때문이다.

## 5. 스토리에 녹이는 규칙

설정은 설명문으로 소비하지 않는다. 반드시 장면에서 기능해야 한다.

| 설정 유형 | 장면에 녹이는 방법 |
|---|---|
| 도시 | 인물이 선택을 잘못하게 만드는 구조로 작동 |
| 건축물 | 인물의 가치관을 움직임으로 강제 |
| 물건 | 감정의 대가를 눈앞에 보이게 함 |
| 식물/동물 | 말로 못하는 감정 상태를 먼저 반응 |
| 의식 | 가치관 충돌을 규칙 있는 전투로 변환 |
| 역사 | 현재 인물의 편견과 상처를 설명 |
| 단역 | 주연이 보지 못한 생활 현실을 드러냄 |
| 귀환 | 버렸던 가치관이 지금 다시 필요해졌음을 사건으로 증명 |
| 녹티오라 잔류 흔적 | 낮의 기능이 A에게 남은 흔적 때문에 비틀리거나 과장됨 |
| 결투 | 두 가치관의 옳음과 위험을 모두 드러낸 뒤 세계 질서를 재배치 |

좋은 예:

- 아스트리온의 감정 여과기는 셀로가 느끼는 답답함을 실제 호흡 곤란처럼 만든다.
- 아르도르의 아우룸 그라두스패는 콘코르디아가 남의 평가를 의식하는 습관을 자극한다.
- 리테노아의 에코 샘은 셀로가 자신의 선의가 남긴 피해를 보게 만든다.
- 아미시오 계곡의 므네메 돌은 포보스가 공포를 연기하는 이유를 말없이 보여준다.

나쁜 예:

- "이 도시는 바깥 원형을 상징한다"라고 설명한다.
- "이 인물은 탐욕이다"라고 설정만 붙인다.
- 전투가 단순히 더 센 빛과 더 큰 폭발로 끝난다.
- 귀환이 전투 보상처럼 즉시 일어난다.
- 녹티오라가 공포 분위기 배경으로만 쓰인다.

세부 설계 기준은 `01C_Mnemosyn_녹티오라_결투_상위기준.md`를 우선한다.

## 6. 모든 요소가 가져야 할 이야기 역할

작품 안에 등장하는 모든 요소(인물, 장소, 물건, 생물, 의식)는 아래 역할 중 하나 이상을 맡는다. 역할 판정이 흔들리면 각 역할의 검증 질문에 답해 본다. 답이 안 나오면 그 역할이 아니다.

| 역할 | 판정 기준 | 검증 질문 |
|---|---|---|
| 촉발자 | 이 요소가 등장해야 사건이 시작된다 | 이 요소를 지우면 이 사건 자체가 일어나지 않는가 |
| 압박자 | 기한, 비용, 위험을 만들어 인물이 선택을 미루지 못하게 한다 | 이 요소 때문에 인물이 "지금" 결정해야 하는가 |
| 거울 | 인물이 자기 문제를 말하지 않아도, 이 요소에 대한 반응으로 독자가 그 문제를 본다 | 인물의 반응만 보고 독자가 그 인물의 상처를 짐작할 수 있는가 |
| 유혹 | 당장 문제를 풀어 주지만 나중에 더 큰 값을 치르게 한다 | 이 요소를 쓴 대가가 무엇인지 문서에 적혀 있는가 |
| 증거 | 나중에 돌아보면 진실을 가리키고 있던 물증이다 | 처음 등장 시점에는 답을 말하지 않고, 회수 시점이 정해져 있는가 |
| 쉼터 | 위험이 잠시 멈춰 인물이 본심을 드러내는 자리다 | 이 장면에서 관계 변화나 다음 사건의 대비가 심어지는가 |
| 상처 | 과거 사건의 값이 아직 청산되지 않았음을 현재 장면에서 보이게 한다 | 흉터, 금기, 피하는 장소, 못 하는 말 중 무엇으로 보이는가 |
| 다리 | 서로 만날 일 없는 두 국가나 가치관을 같은 장면에 세운다 | 연결의 대가나 마찰이 함께 설계되어 있는가 |
| 잔류자 | 낮 사건의 흔적을 밤이나 다음 장면에 남겨 선택을 다시 묻게 한다 | 그 흔적이 다음 장면에서 실제로 다시 등장하는가 |

## 7. 신규 설정 흡수 전 질문

아래 질문은 새 양식이나 별도 문서를 만들기 위한 목록이 아니다. 새 설정을 기존 본류 문서, 개별 인물 문서, 도시 문서, 도감 문서, 해당 EP 문서 중 어디에 흡수할지 정하기 위한 압축 질문이다.

- 이 요소는 어느 국가/층계/접경 지대에 속하는가?
- 이것은 어떤 생각, 감정, 가치관을 판타지의 물성, 도시 구조, 능력 충돌로 바꾼 것인가?
- 독자가 이 요소를 처음 봤을 때 어떤 감각을 느껴야 하는가?
- 누가 이 요소를 사랑하고, 누가 두려워하고, 누가 이용하는가?
- 이 요소는 어떤 회차에서 사건을 실제로 바꿀 수 있는가?
- 이 요소가 사라지면 어떤 인물이 상처를 받는가?
- 이 요소는 1부와 6부에서 다르게 보이는가?

답할 수 없다면 아직 정사에 넣지 않는다. 답할 수 있어도 새 작업표를 만들기보다, 먼저 기존 본류 문서의 한 문단이나 해당 EP의 사건/도시 변화/말미 잔류물에 흡수한다.
