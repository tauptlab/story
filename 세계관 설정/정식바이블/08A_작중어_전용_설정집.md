# 작중어 전용 설정집

이 문서는 네우로시아의 현실 기반 은유를 실제 원고 문장으로 바꾸기 위한 번역 규칙이다. 설정 작업용 문서에서는 뇌과학, 심리학, 철학 용어를 사용할 수 있지만, `웹소설작성` 본문에서는 아래 작중어만 사용한다.

핵심 원칙: 독자는 네우로시아가 A의 뇌라는 사실을 설명으로 듣지 않는다. 대신 도시의 구조, 세르의 물성, 인물의 습관, 밤에 남는 흔적을 통해 느낀다.

---

## 1B. 압축 개념 조어 폐기 표준 (원고·설정문서 공통)

**문제:** `수식어 + 추상명사` 꼴의 압축 조어(예: 느린 질문, 자기 박자, 빈 기록줄)가 정의 없이 설정문서에까지 굴러다녀, 작가조차 첫 읽기에 뜻을 못 잡는다. 이는 `08` §2C 축1(작가 내부 언어 금지)·§3(어색한 조어)의 위반이며, **원고뿐 아니라 설정문서에도 적용한다.**

**규칙:**
- 설정문서에서도 **새 압축 추상 조어를 만들지 않는다.** 개념은 평이한 말로 풀어 쓴다.
- 아래 표의 폐기어는 문서에서 **이름 자체를 지우고** 오른쪽 평이 표현으로 바꾼다(문장에 맞게 어미·조사 조정). 이름을 보존하지 않는다.
- **구체 물성어는 예외(유지).** 눈에 그려지는 물건은 08의 권장 방식이므로 바꾸지 않는다: 빈 의자, 식은 문고리, 낡은 동전, 젖은 종이, 낮은 물컵, 식은 잔, 값 없는 빵, 검은 낙인, 접힌 빵 봉투, 마시지 않은 물컵, 느슨한 가면 끈, 낮은 북 등.
- 판정이 애매하면(개념인가 물건인가) 서브에이전트에 묻고, 확실치 않으면 더 쉬운 생활어로 풀어 쓴다.

**폐기 조어 → 평이 교체:**

| 폐기 조어 | 평이 교체(기본형; 문맥에 맞게 조정) |
|---|---|
| 느린 질문 / 늦은 질문 | 성급히 답 내지 않고 묻기 (모르는 채 묻고 도움 청하기) |
| 자기 박자 | 자기만의 속도 |
| 빈 기록줄 | 비워 둔 기록칸 |
| 닫힌 질문 | 답을 정해 놓고 던지는 물음 |
| 낮춘 빛 / 낮은 빛(개념일 때) | 세기를 줄인 빛 (다 밝히지 않은 빛) |
| 낮은 응답 / 빈자리 응답 | 빈자리에서 오는 약한 반응 |
| 반 마디 고백 | 하다 만 고백 |
| 무대응 여백 | 일부러 응하지 않은 침묵 |
| 늦은 도착(선의가) | 뒤늦게 온 선의 |
| 낮은 이름 | 낮춰 말한 요구 |
| 낮은 빛(물성) | 은은한 빛 |
| 낮춘 진실 | 다 밝히지 않은 진실 |
| 보류 흔적 | 판단을 아직 안 내리고 남겨 둔 자국 |
| 빈 이름 | 지워져 빈 이름 (단 '빈 이름표/빈 이름패' 소품은 유지) |
| 어긋난 박수 | 박자를 놓친 박수 |
| 빈 박자 | 박자 사이 빈틈 |
| 닫힌 문장(조각) | 고쳐 쓸 수 없게 굳은 문장(조각) |
| 낮은 반사면 | 안개 낀 반사면 (루멘 어스름의 은거울) |
| 낮은 기록칸 | 작은 신호 칸 |
| 낮은 반응 | 약한 반응 |
| 낮은 신호 | 작은 신호 |
| 낮은 판결 | 누그러뜨린 판결 |
| 낮은 형태(귀환) | 작아진 형태 |
| 낮은 온기 | 은은한 온기 |
| 낮은 사건 | 사소한 사건 |
| 낮은 반례 | 작은 반례 |
| 낮은 장례 | 조용한 장례 |

**`낮은 X` 처리 원칙:** **물리·감각 표현은 정상이라 유지**한다 — 낮은 의자·탁자·문·종·난간·목소리·북·온도·속삭임·울림·불빛 등(= a low chair/low voice). **오직 '낮은 + 추상명사'**(반응·신호·판결·형태·온기·사건·반례·장례·기록칸)만 위 표대로 교체한다. `낮은 안식/청취`는 문맥 확인 후 보류.

**혼동 금지(유지):** `닫힌 문`(방어기제 작중어)·`닫힌 증언`·`닫힌 창구/문턱`·`닫힌 논리`는 '닫힌 문장'과 다르니 그대로 둔다. 나르키소스·드라쿨의 그냥 `반사면`(반사하는 면)은 구체어라 유지.

이 표는 확정 발견분이며, 새 폐기어가 나오면 같은 원칙으로 이 표에 추가한다.

## 2. 국가별 작중어

| 현실 기반 기능 | 국가/지역 | 원고에서 쓰는 감각 |
|---|---|---|
| 계획, 통제, 판단 | 프론티아 | 은색 직선, 계산 소리, 차가운 숨, 규약 문장 |
| 인정, 욕망, 보상 | 눅세린 | 황금빛, 향, 군중의 시선, 가격표, 초대장 |
| 공포, 분노, 생존 | 암디르 | 재, 열기, 낮은 북, 검은 바위, 거친 숨 |
| 공감, 고통 중재 | 센시르 | 물결, 실, 반향, 손바닥의 온기, 낮은 화음 |
| 기억, 기록, 망각 | 히프노레아 | 결정, 서가, 수로, 먼지, 깨진 문장 |
| 숙련, 절차, 몸의 반복 | 아르스 모투스 | 박자, 발자국, 원무, 자세끈, 아쿠스꽃 |
| 언어, 의미, 소리 | 오라쿨룸 | 별말, 풍경 소리, 수정벽, 타키투스 문자, 음계꽃 |

작성 규칙:

- 국가 이름만 적고 기능을 설명하지 않는다.
- 기능은 물성으로 보여준다.
- 한 문단에 국가 이름과 현실 기반 기능을 같이 쓰지 않는다.

나쁜 예:

> 프론티아는 전전두피질을 상징하는 국가라서 모든 것을 통제하려 했다.

좋은 예:

> 프론티아의 길은 모두 같은 각도로 꺾였다. 셀로가 한 걸음 비스듬히 딛자, 바닥의 은선이 먼저 숨을 들이켰다.

## 3. 층계별 작중어

| 현실 기반 상태 | 층계 | 원고 표현 |
|---|---|---|
| 외부 자극의 최초 번역 | 미라마인드 | 현실의 파편, 첫 반사, 아직 이름 없는 빛 |
| 표면 의식 | 일루미라 | 낮, 아침, 또렷한 거리, 주민의 시간 |
| 밤의 재처리 | 녹티오라 | 밤, 퀘스티오, 뒤집힌 건축, 감정의 전장 |
| 억압/망각된 저류 | 므네모신 | 새벽녘, 잠든 이름, 가라앉은 서가, 망각의 강 |
| 근원 원형 | 근원층 | 가장 아래의 방, 프리마 에코, 왕좌 이전의 고요 |

나쁜 예:

> 녹티오라는 A의 꿈 처리 단계였다.

좋은 예:

> 밤이 오자 낮의 골목은 같은 자리에 남아 있었지만, 방금까지 바구니를 포개던 사람들만 보이지 않았다. 접다 만 안대와 흐트러진 표적판 줄 아래에서, 아이의 손목이 뒤늦게 떨렸다.

원고에서는 밤 자체가 괴현상을 일으킨다고 쓰지 않는다. 밤은 같은 좌표의 다른 위상이며, 긴장은 사라진 낮 주민, 남아 버린 발현자, 가까워지는 정체 모를 기척, 낮까지 이어지는 상처에서 만든다.

## 4. 과정별 변환표

### 4.1 주의와 경보

| 기반 개념 | 작중어 | 장면화 |
|---|---|---|
| 중요도 판정 | 살리엔티아 관문 | 모든 소리가 종소리로 오르는 도시 |
| 주의 집중 | 경보등, 묻힌 종소리 | 한 사람의 손떨림이 광장 전체에 비친다 |
| 경보 피로 | 너무 많이 울린 종 | 진짜 위험이 소음 속에 묻힌다 |

문장 기준:

- "주의가 과도하다" 대신 "종이 너무 자주 울려 아무도 뛰지 않았다"라고 쓴다.
- "중요한 자극" 대신 "묻혀 있던 종소리"라고 쓴다.

### 4.2 예측과 오차

| 기반 개념 | 작중어 | 장면화 |
|---|---|---|
| 사전 예측 | 프리오르, 프라이오라 격자 | 아직 오지 않은 사람의 의자에 가능성 표식이 붙는다 |
| 예측 모델 | 관측 회랑 | 미래가 은색 가지처럼 갈라진다 |
| 예측 오류 | 에라타, 어긋난 번개 자국 | 완벽한 격자 위에 작은 번개가 떨어진다 |
| 학습 | 에라타 분필 | 틀린 선이 다음 발의 시작점이 된다 |

문장 기준:

- "예측이 틀렸다" 대신 "격자의 한 칸이 아직 울지 않은 이름으로 금 갔다"라고 쓴다.
- "오류를 수정했다" 대신 "분필선이 지워지지 않고 다음 발 앞에 남았다"라고 쓴다.

### 4.3 자기서사와 자기 수용

| 기반 개념 | 작중어 | 장면화 |
|---|---|---|
| 자기서사 | 아우토노에, 첫줄방 | 삶의 첫 문장을 고친다 |
| 자아 이미지 | 아노니마 초상화 | 초상화가 보는 사람마다 다른 얼굴을 건넨다 |
| 무아/무상 | 아나트, 아나타 정원 | 얼굴이 지나가지만 존재는 사라지지 않는다 |
| 자기 수용 | 느슨해진 노멘 매듭 | 이름이 끊어지지 않고 숨 쉴 틈을 얻는다 |

문장 기준:

- "셀로의 자아가 흔들렸다" 대신 "초상화 속 얼굴이 하나로 멈추지 않았다"라고 쓴다.
- "고정된 자기 이미지에서 벗어났다" 대신 "왕관이 머리에서 떨어지지 않고, 조금 느슨해졌다"라고 쓴다.

### 4.4 몸의 신호와 안식

| 기반 개념 | 작중어 | 장면화 |
|---|---|---|
| 내감각 | 인터라, 내감의 우물 | 괜찮다는 말보다 숨이 먼저 어긋난다 |
| 소포르 베개 | 소포르 베개 | 기억의 모서리가 흐린 수채화가 된다 |
| 무감각 | 팔리다 아우룸 결정 | 아무것도 원하지 않으니 갈증도 희미해진다 |
| 회복 | 호흡고사리 | 잎이 상대의 숨에 맞춰 접혔다 펴진다 |

문장 기준:

- "몸의 신호를 읽었다" 대신 "인터라는 말보다 먼저 손목의 박자를 들었다"라고 쓴다.
- "고통이 사라졌다" 대신 "아픈 곳은 조용해졌지만, 기쁨도 조금 멀어졌다"라고 쓴다.

### 4.5 습관과 실행

| 기반 개념 | 작중어 | 장면화 |
|---|---|---|
| 절차 기억 | 모르파 전당 | 몸이 먼저 길을 기억한다 |
| 습관 | 에토스 춤마당 | 같은 길을 걸어도 바닥 문양이 조금씩 바뀐다 |
| 반복 학습 | 아쿠스꽃 | 여러 번 스친 손길이 있어야 꽃이 핀다 |
| 균형 | 박자석, 리브라검 | 맞는 박자가 아니면 칼이 무겁다 |

## 5. 인물 설명 변환 규칙

인물을 설명할 때는 `은유`를 직접 말하지 않고, 아래 순서로 보여준다.

1. 사랑스러운 습관
2. 그 습관이 생긴 상처
3. 능력이 작동하는 물성
4. 능력의 대가
5. 남는 질문

나쁜 예:

> 이놉스는 무력감을 상징하는 인물이다.

좋은 예:

> 이놉스가 지나간 자리에서는 재가 떨어져도 소리가 나지 않았다. 그는 꺼진 불씨 앞에서만 잠시 멈췄다.

나쁜 예:

> 프리오르는 예측 처리의 인격화다.

좋은 예:

> 프리오르는 아직 오지 않은 손님을 위해 바쿠아 의자에 위험 표식을 붙여두었다.

나쁜 예:

> 아나트는 불교의 무아 사상을 상징한다.

좋은 예:

> 아나트의 얼굴은 셀로가 기억하려는 순간마다 조금씩 지나갔다. 그런데도 그는 사라지지 않았다.

| 무상 | 아나타 정원, 흐르는 얼굴 | 허무주의처럼 보이면 안 됨 |
| 덕 윤리 | 에토스의 반복, 아쿠스꽃 | 훈계처럼 보이면 안 됨 |
| 실존적 공포 | 포보스, 아테르 루나, 아미시오 계곡 | 과장된 악역 미학으로만 보이면 안 됨 |
| 공리/법칙 | 로고스, 알바 타일, 별 없는 별자리 | 차가운 공식만 남으면 안 됨 |
| 회의/의심 | 카산드라, 속삭이는 회랑 | 음모론으로만 보이면 안 됨 |
| 관용 | 리타스, 포쿠스 울티마 | 무경계 착취 허용처럼 보이면 안 됨 |
| 정의로운 분노 | 페이단, 가름불 | 단순 파괴 쾌감으로 보이면 안 됨 |

> 각 철학 개념이 어떤 밤 충동의 성숙형인지는 `정식바이블/03C_존재위상/밤충동/00_밤충동_색인.md`의 대응표를 함께 본다.

## 7. 문단 변환 예시

### 예시 A: 분석과 통제

초안:

> 에이도스는 전전두엽적 분석 능력으로 셀로의 감정 반응을 평가했다.

작중어:

> 에이도스의 눈동자에 은색 격자가 떠올랐다. 셀로의 숨결은 격자 위에서 두 번 꺾였고, 세 번째 선에서야 이름 없는 열로 번졌다.

### 예시 B: 보상과 인정

초안:

> 눅세린 주민들은 보상회로가 과활성화된 것처럼 인정에 중독되어 있었다.

작중어:

> 눅세린 사람들은 서로의 손목에서 결정의 밝기를 먼저 보았다. 누군가 웃으면, 그 웃음보다 결정에 비친 금빛이 더 빨리 팔렸다.

### 예시 C: 예측 오류

초안:

> 에라타는 예측 오류를 통해 셀로의 고정된 자기 모델을 흔들었다.

작중어:

> 셀로의 왕관 위에 먼지 한 점이 앉았다. 그가 손끝으로 지우자, 먼지는 둘이 되었고 둘은 웃음소리가 되었다.

### 예시 D: 내감각

초안:

> 인터라는 내감각을 읽어 셀로가 괜찮지 않다는 사실을 알아냈다.

작중어:

> "괜찮아." 셀로가 말했다. 인터라는 대답하지 않고 자기 손목을 짚었다. 우물 아래에서 돌아온 박자가 셀로의 말보다 한 박자 늦었다.

### 예시 E: 무아와 자기 수용

초안:

> 아나트는 셀로에게 고정된 자아가 없다는 사실을 가르쳤다.

작중어:

> 아나트의 얼굴은 물 위의 달처럼 바뀌었다. 셀로는 그중 어느 것도 붙잡지 못했지만, 이상하게도 대화는 끊어지지 않았다.

## 8. 원고 검수 기준

원고에 새 설정을 반영한 뒤 아래를 확인한다.

- `뇌`, `신경`, `전전두`, `편도체`, `해마`, `소뇌`, `도파민`, `보상회로`, `예측 처리`, `내감각`, `무아`, `무상` 같은 직접어가 남아 있지 않은가.
- 직접어가 필요하다면 설정 문서에만 두고 본문에서는 작중어로 바꿨는가.
- 한 문단에 새 작중어를 3개 넘게 넣지 않았는가.
- 설명보다 물성, 행동, 습관, 대가가 먼저 나오는가.
- 독자가 은유를 몰라도 장면을 이해할 수 있는가.
- 은유를 아는 독자는 두 번째 층위를 읽을 수 있는가.

## 9. 빠른 검색 명령

원고 반영 뒤에는 아래 검색으로 직접어 노출을 점검한다.

```powershell
rg -n "뇌|신경|전전두|편도체|해마|소뇌|도파민|보상회로|예측 처리|내감각|무아|무상|철학|심리학|뇌과학" "웹소설작성"
```

검색 결과가 나오면 다음 중 하나로 처리한다.

1. 설정 메모라면 `세계관 설정` 쪽으로 옮긴다.
2. 본문 설명이라면 작중어로 고친다.
3. 인물 대사라면 그 인물이 그 용어를 알 수 있는지 확인한다. 원칙적으로 알 수 없다.
