# 도시 문서 회차 배치표 1차

정사 등급: 보조 정식 색인

역할: 보강된 도시/장소 문서가 어느 설계 EP, 부분 원고, 장면시퀀스에 연결되는지 역방향으로 추적하는 보조 색인이다. 최종 회차 투입 판단은 `07_회차_투입_색인.md`를 우선한다.

---

## 1. 표기 기준

| 표기 | 의미 |
|---|---|
| 설계 EP | `에피소드` 폴더의 큰틀 설계 문서 |
| 부분 원고 | `웹소설작성` 폴더의 부분 구체화 원고 |
| 장면 | `장면시퀀스` 폴더의 전투/전환 장면 |
| 실등장 | 이미 해당 원고나 설계에 중심 장소로 들어간 상태 |
| 삽입 후보 | 향후 실제 연재문에서 보강 장면으로 넣을 자리 |
| 재방문 | 한 번 등장한 장소를 후반의 다른 질문으로 다시 쓰는 자리 |
