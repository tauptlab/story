# 원고용 작중어 빠른 확인

이 문서는 실제 연재문 작성 직전에 옆에 두는 빠른 확인 문서다. `08A_작중어_전용_설정집.md`가 작업자용 변환 장부라면, 이 문서는 바깥말을 반복하지 않고 정사 표현만 모아 둔 현장용 참조다. 문장 자연스러움, 번역체, 어색한 조어, 서브에이전트 검수 절차도 이 문서 안에서 함께 본다.

원칙은 하나다. 세계의 숨은 바탕은 설명하지 않는다. 공간의 구조, 인물의 습관, 세르의 물성, 밤에 남는 흔적, 선택의 대가로만 느끼게 한다.

## 9. 원고 문체와 사용자 피드백 기준

이 절은 실제 원고를 쓰거나 고칠 때 문장 단위로 적용한다. 기준은 작업자가 정리한 요약이 아니라 사용자가 직접 남긴 피드백이다.

### 9.1 최상위 문체 원칙

1. 먼저 그려지게 쓴다.
2. 그다음 낯설게 만든다.
3. 마지막에 생각하게 만든다.

첫 문단에서 독자가 알아야 할 것은 설정의 뜻이 아니라 위치, 몸, 위험이다. 누가 어디에 있고, 무엇을 보고 있으며, 어떤 방향에서 무엇이 다가오거나 사라지는지 보여 준 뒤에만 상징을 얹는다.

| 사용자가 반복해서 요구한 것 | 원고 처리 |
|---|---|
| 머릿속에 장면이 그려져야 한다 | 위치, 방향, 거리, 사람의 동선을 먼저 쓴다 |
| 1화부터 긴장감과 내용이 있어야 한다 | 첫 500자 안에 즉시 문제와 실패 비용을 둔다 |
| 상징을 위한 상징은 싫다 | 소품은 사건을 실제로 바꾸거나 세계 법칙을 보여 줄 때만 쓴다 |
| 한국어가 자연스러워야 한다 | 번역체, 과한 명사화, 어색한 조어, 직역식 추상어를 우선 제거한다 |
| 이영도식 장점은 원하지만 베끼면 안 된다 | 이야기 우선성, 말맛, 사유의 장면화만 가져오고 문장 습관이나 장면 구조는 모방하지 않는다 |
| 외부 작품의 접근법은 재창조해야 한다 | 특정 장면이나 감정 표지를 그대로 가져오지 않고 기능만 분석해 이 세계관의 물성으로 바꾼다 |
| 설정 충돌을 그냥 고치지 말고 방향성을 잡아야 한다 | 원인, 세계 법칙, 인물 위치를 먼저 확정한 뒤 문장을 고친다 |
| 폐기한 설정을 남기지 말아야 한다 | 후보나 보존 주석으로 남기지 않고 현재 정식 장치로 완전히 교체한다 |

### 9.2 한국어 윤문 기준

`korean-humanizer` 기준을 원고용으로 압축하면 아래와 같다.

| 유형 | 피할 표현 | 원고 처리 |
|---|---|---|
| 번역체 조사 | `~에 대해`, `~을 통해`, `~에 있어서`, `~와 관련하여` | 조사나 동사로 줄인다 |
| 직역 동사 | `의미를 가진다`, `영향력을 가지고 있다`, `가능하게 한다` | `의미가 있다`, `영향이 크다`, `하게 한다` 또는 더 짧게 |
| 이중 피동 | `보여진다`, `되어진다`, `여겨진다` | `보인다`, `된다`, `여긴다` |
| 추상 명사화 | `상황의 발생`, `감정의 처리`, `선택의 수행` | `상황이 벌어졌다`, `감정을 다뤘다`, `선택했다` |
| AI 결산어 | `결론적으로`, `시사하는 바가 크다`, `본질적으로`, `궁극적으로` | 삭제하거나 장면 결과로 바꾼다 |
| 3단 공식 | `A, B, 그리고 C`, `먼저/반면/결국` 반복 | 필요한 둘만 남기거나 문단에 녹인다 |
| 대명사 직역 | `그/그녀/그것/그들` 반복 | 주어 생략, 이름, 호칭으로 처리한다 |

원고에서는 `상징한다`, `의미한다`, `철학적 사유를 유도한다` 같은 작가용 설명을 쓰지 않는다. 해설어는 감각과 행동으로 바꾼다. 예를 들어 `고립을 상징한다`가 아니라 `막 밖의 목소리가 한 박자 늦었다`라고 쓴다.

### 9.3 어색한 조어 금지선

아래 표현 계열은 원고에서 금지하거나 생활어로 낮춘다.

| 표현/계열 | 문제 | 대체 방향 |
|---|---|---|
| `늦은 얼굴`, `늦은 이름` | 한국어 결합이 흐리고 장면이 늦게 온다 | `늦게 떠오른 얼굴`, `이름이 뒤늦게 떠올랐다`처럼 동작으로 푼다 |
| `손톱 선`, `숨의 획`, `목소리의 자국` | 실제 흔적이 보이지 않는다 | 보이는 자국, 손끝, 흙먼지, 실오라기 같은 물성으로 바꾼다 |
| `낮은 문` | 장소명처럼 쓰면 어색하다 | `낮게 난 출입구`, `고개를 숙여야 지나가는 좁은 문`처럼 구조를 설명한다 |
| `문이 박수를 쳤다` | 원인 없는 사물 의인화다 | `잠금쇠가 차례로 돌아갔다`, `문들이 차례로 닫혔다` |
| `밤의 압력이 말했다` | 추상 주어가 행동한다 | 사람, 소리, 빛, 온도, 실제 움직임을 주어로 세운다 |
| `마감 종 때문에 밤이 온다` | 생활 신호와 밤 전환의 인과가 잘못 붙는다 | 마감 종은 하루 일과 종료, 밤 전환은 별도 규칙으로 분리한다 |
| `달을 본 사람` | 초반 낮 주민은 달을 직접 본 적이 없다 | `밤의 사람`, `사라졌다 돌아온 사람`, `방금 없던 사람` |

새 조어가 `형용사 + 추상명사` 구조이거나, 사물이 사람처럼 행동하거나, 읽고 바로 위치가 그려지지 않으면 원고에 강행하지 않는다.

### 9.4 의심 표현 서브에이전트 검수

내가 어색한 한국어를 스스로 완전히 판정할 수 없다는 전제를 둔다. 아래 조건 중 하나라도 걸리면 서브에이전트에게 짧게 묻는다.

- 새 조어가 `형용사 + 추상명사` 구조다.
- `늦은/낮은/닫힌/젖은/식은`이 얼굴, 이름, 마음, 문장, 기억처럼 추상 대상에 붙었다.
- 사물이 사람처럼 행동한다.
- 문장을 읽고 장면 위치가 바로 그려지지 않는다.
- 대체어 후보가 둘 이상인데 한국어 자연스러움 판단이 애매하다.
- 사용자가 한 번이라도 어색하다고 지적한 표현과 닮았다.

서브에이전트가 없으면 의심 표현을 보류하고 더 쉬운 생활어로 쓴다. 이 경우 새 상징어를 강행하지 않는다.

## 10. 실제 작성 직전 선택표

| 회차권 | 먼저 고를 것 | 피해야 할 것 |
|---|---|---|
| 1부 | 셀로의 방어, 초반 소품, 프론티아의 길 | 능력을 설명으로 먼저 밝히기 |
| 2부 | 틈, 기록, 금빛 소리, 판단의 원 | 정답과 오답을 너무 빨리 나누기 |
| 3부 | 정원과 후원자, 노멘패, 손바닥의 금가루, 선술집의 빈 잔 | 어느 한쪽만 악하게 만들기 |
| 4부 | 왕관, 밤문, 황금 심장로, 회색 숨 | 셀로의 분열을 단순 적으로 만들기 |
| 5부 | 낮은 빛, 거울, 등불, 첫 물결, 빈칸의 기록 | 빛을 만능 해결로 쓰기 |
| 6부 | 열쇠, 끝돌, 바쿠아 의자, 낮은 물결 | 결말을 완전한 삭제나 완전한 정답으로 닫기 |

## 11. 마지막 검수 질문

- 이 장면은 설명 없이도 냄새, 빛, 소리, 손의 움직임으로 이해되는가.
- 새 이름이 너무 많이 들어갔다면 하나를 덜어낼 수 있는가.
- 인물의 약점이 결함만이 아니라 사랑받을 수 있는 습관으로도 보이는가.
- 적대하는 쪽에도 지키려는 것이 남아 있는가.
- 물건과 장소가 한 번 쓰고 사라지는 장식이 아니라 다음 선택을 바꾸는가.
- 숨은 바탕을 아는 사람은 두 번째 층위를 읽고, 모르는 사람도 이야기를 따라갈 수 있는가.
