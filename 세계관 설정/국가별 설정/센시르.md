# 센시르 (Sensir) 국가 설정

정사 등급: A급 국가

핵심 은유: 공감, 조율, 책임, 메아리, 고통을 대신 판결하지 않는 느린 이해

정사 기준: `정식바이블/02_국가_도시_아틀라스.md`, `정식바이블/03A_국가별_생활인물_명부.md`, `정식바이블/07_회차_투입_색인.md`, 센시르 도시별설정, 카시안/살루스 문서를 우선한다.

## 1. 국가 정체성

센시르는 서로 다른 고통이 한자리에 모여도, 어느 한쪽을 지우지 않고 둘 다 듣는 법을 배운 나라다. 이곳의 사람들은 말보다 반향을 조심한다. 누군가 한 행동은 사라지지 않고 물결, 실, 음, 작은 통증으로 돌아온다. 그래서 센시르의 정의는 빠른 처벌보다 느린 정산에 가깝다.

하지만 공감도 위험할 수 있다. 남의 말을 너무 많이 들으면 자기 목소리를 잃고, 너무 빨리 화해시키려 하면 피해자의 분노까지 아름다운 화음으로 덮어 버린다. 카시안의 센시르는 그래서 "모두 이해하자"보다 더 어려운 질문을 붙든다.

> 나는 너를 들을 수 있지만, 너 대신 결론을 내릴 권리는 있는가.

## 2. 국가 감각

| 감각 | 정사 표현 |
|---|---|
| 색 | 물빛 파랑, 부드러운 분홍, 은색 실, 흙색, 낮은 금빛 |
| 소리 | 리라 현, 작은 종, 물결, 딸꾹질, 발걸음이 늦춰지는 소리 |
| 냄새 | 물가 풀, 젖은 돌, 따뜻한 차, 쉬는 담요의 천 냄새 |
| 촉감 | 리라석의 떨림, 디아파손 실, 에코 샘가의 차가운 물기 |

## 3. 사람들

센시르 사람들은 스스로를 `레조나`라 부른다. 이들은 타인의 감정에 민감하지만, 그것이 곧 완전한 이해를 뜻하지 않는다는 사실도 배워야 한다.

| 인물군 | 설정 | 장면 기능 |
|---|---|---|
| 조율자 | 갈등 당사자들의 반향을 천천히 맞추는 사람 | 카시안 계열 책임 윤리 |
| 조율 견습생 | 남의 감정을 너무 빨리 받아들여 실수하는 학생 | 미렌의 딸꾹질 같은 인간적 약점 |
| 고요의 정원지기 | 무타이끼와 타키투스 껍질을 관리하는 사람들 | 선악이 아닌 내부 논쟁 |
| 안식 마을 의사 | 쉬어도 된다는 허락을 주는 사람들 | 살루스의 다정함과 위험 |
| 망명자/교류자 | 암디르, 눅세린 등에서 온 상처 입은 방문자 | 센시르가 타인의 고통을 어떻게 맞이하는지 시험 |

## 4. 역사

| 시대 | 핵심 사건 | 정사 기능 |
|---|---|---|
| 초기 조율기 | 큰 갈등 뒤 서로의 고통을 듣기 위한 공동체 형성 | 센시르의 기원 |
| 카시안 성숙기 | 제거보다 조율과 책임 정산을 선택 | 카시안의 과거와 현재 연결 |
| 프락시스 후폭풍기 | 빠른 결과가 남긴 생태 부채와 고통을 정산 | 프론티아/눅세린 피해 회수 |
| 디아파손 홀 갈등기 | 무타이끼, 타키투스 껍질, 빠른 심판 충동이 충돌 | 셀로의 심판 욕망 비판 |
| 살루스 영향기 | 쉬어도 된다는 다정함이 커지지만 선명함이 흐릴 위험도 생김 | 안식과 무감각의 경계 |
| 고요한 조류 이후 | 국가 경계가 낮아지며 센시르가 듣는 범위도 넓어짐 | 후반 세계 조율 |

| 드레오렌 | 농담과 옆문으로 감정을 피하는 회피 생활 도시 | 실렌, 하빈, 도르스, 라모, 오누엘, 페루, 리수스 다리와 리수스 보자기, 옆문열쇠, 프론스 손잡이 |
| 라세드린 | 프론티아 접경의 논쟁 도시 | 감정의 이유와 근거 있는 말의 도착 방식을 함께 묻는 장소 |

## 6. 건축과 물건

| 요소 | 설정 | 장면 기능 |
|---|---|---|
| 공명실 | 마주 앉도록 만든 둥근 방 | 말을 늦추고 듣게 함 |
| 디아파손 홀 | 물길, 현, 공명판이 겹친 심부 공간 | 서로 다른 고통을 섞지 않고 동시에 듣게 함 |
| 에코 샘 | 깊은 물웅덩이 | 행동의 결과를 물결로 보여줌 |
| 디아파손 실 | 갈등 당사자 사이에 느슨히 걸리는 실 | 동의 없는 묶음과 느슨한 연결의 경계 |
| 리라 현 | 카시안의 악기이자 조율 도구 | 책임을 음으로 드러냄 |
| 소포르 베개 | 통증을 낮추는 물건 | 과도하면 기억도 흐려짐 |
| 프리마 소누스 교실 | 첫 반응을 판정하기 전 함께 확인하는 둥근 교실 | 울림의 주인과 시작점을 먼저 묻는다 |
| 레스폰사 모래시계 | 바로 대답하지 않아도 되는 시간을 보여 주는 작은 모래시계 | 기다림을 처벌이 아니라 선택의 시간으로 둔다 |
| 리토르노 손수건 | 이미 던진 말을 주워 닦을 때 쓰는 손수건 | 사과를 말이 아니라 다시 만지는 책임으로 낮춘다 |
| 모라 클렙시드라실 | 모라 레스폰사 의원 안쪽의 물방울 방 | 맞힌 말을 말하기 전 한 방울을 기다리게 한다 |
| 데클리나 난간 | 모라 스페쿨룸방 바깥의 낮은 난간 | 본 것을 부정하지 않되 허락 전에는 들어가지 않는 예법을 만든다 |
| 라쿠나 기록대 | 눈결 골목 안쪽의 두 칸짜리 기록대 | 본 것과 아직 말해지지 않은 것을 같은 높이에 둔다 |
| 여백 난간 | 라쿠나 기록대 뒤쪽의 좁은 난간 | 적지 않은 칸도 보호된 자리임을 보인다 |
| 트레몰로잔 | 손의 떨림을 물결로 남기는 찻잔 | 감지자가 말하기 전 멈추게 한다 |
| 모라 클렙시드라 | 읽은 뒤 물방울 하나가 떨어질 때까지 말을 늦추는 작은 물시계 | 기다림과 회피의 차이를 물성으로 보인다 |
| 데클리나 핀 | 본 것을 부정하지 않되 곁에 남기 위해 다는 옷깃 핀 | 일부러 모르는 채 떠나지 않는 예법을 보인다 |
| 빈칸 먹줄 | 모르는 부분 둘레에 선을 긋되 안쪽을 채우지 않는 먹줄 | 빈칸을 방치하지 않고 지키게 한다 |
| 반칸 기록판 | 본 것의 칸과 아직 말해지지 않은 칸이 나뉜 기록판 | 증언과 소유의 차이를 보인다 |
| 모라 레스폰사패 | 오늘은 말하지 않겠다는 작은 표 | 침묵을 무례가 아니라 권리로 둔다 |
| 세이르의 바쿠아 잔 | 자기 대답을 뒤로 미루던 잔 | 남을 편하게 하기 전 자기 말할 자리를 남기게 한다 |
| 아욱실리움 끈 | 한 걸음만 함께 넘겠다는 느슨한 손목끈 | 도움을 소유나 빚으로 만들지 않게 한다 |
| 반걸음 목발 | 문턱 하나를 넘을 만큼만 받쳐 주는 낮은 목발 | 기대는 일과 길을 빼앗기는 일을 구분한다 |
| 두손다리 | 양쪽 난간의 손 홈이 서로를 당기는 접경 다리 | 포용과 속박의 차이를 손목 통증으로 보인다 |
| 두물빛 잔 | 두 물빛을 한 잔 안에 나누어 담는 접경 찻잔 | 모두를 품는 마음이 언제 목막힘이 되는지 보인다 |
| 한쪽발 돌 | 선택을 시작하는 낮은 디딤돌 | 한 발을 딛고 남은 쪽을 지우지 않는 책임을 보인다 |
| 남은손 표식 | 선택받지 못한 쪽에 남기는 손바닥 무늬 | 선택이 삭제가 아니라 인사를 남기는 책임임을 보인다 |
| 두빛 풀 | 선택받지 못한 길을 저주하지 않는 말 앞에서 두 색 잎을 펴는 풀 | 선택 뒤에도 남은 쪽을 지우지 않는 책임을 보인다 |
| 리수스 다리 | 불편한 말을 건널 때 먼저 농담 한 줄을 놓는 낮은 다리 | 충돌을 완충하되 돌아올 길표를 남기게 한다 |
| 리수스 보자기 | 어려운 말을 잠시 싸 두는 얇은 천 | 웃음 뒤 남은 말이 사라지지 않게 한다. 다 마르기 전에는 밝은 곳에 널지 않고, 주인에게 돌려줄 때 매듭을 지어 그 손에서 계속 마르게 둔다 |
| 옆문열쇠 | 정문을 열 준비가 안 된 사람에게 잠시 주는 작은 열쇠 | 피할 권리와 돌아올 책임을 함께 묻는다 |
| 프론스 손잡이 | 한 문장만 직접 말하면 낮게 따뜻해지는 손잡이 | 회피 뒤 귀환의 시작을 물성으로 보인다. 감당 가능한 한 문장이나 빈칸만 남겨도 되고, 그 문장이 자발적인지 압박인지 낮은 등불로 가른다 |
| 이유 원탁 | 감정의 이유와 말의 근거를 같은 높이에 놓는 둥근 탁자 | 듣기와 따지기가 서로를 지우지 않게 한다 |
| 결론보류끈 | 아직 결론 내리지 못한 문장 묶음에 느슨히 매는 끈 | 보류를 회피가 아니라 남은 책임으로 둔다 |

| 실렌/하빈/도르스/라모/오누엘/페루 | 드레오렌 생활 인물 | 농담과 옆문과 돌림말이 보호인지 고립인지, 웃음 뒤 한 문장을 남기고 밖으로만 열린 옆문을 정면문 쪽으로 돌려 놓는 행동으로 보이게 함 |
| 엘리올/레사르/미논/하엘디/오르베아/다니온/베리안 | 라세드린 접경 인물 | 감정이 근거를 지우지 않고 근거가 상처를 지우지 않게 하는 논쟁 예법을 보임 |

## 8. 주요 인물군

| 인물/유형 | 역할 | 장면 기능 |
|---|---|---|
| 카시안 | 균형 조율자 | 공감과 책임의 중심 |
| 살루스 | 쉬게 하는 다정함 | 안식과 무감각의 경계 |
| 세이르 | 맞춰 주다 자기 대답을 잃은 귀환 인물 | 공감의 선의가 자기 지움으로 기울 때의 회복 |

| 엘레시아 | 선택해도 남은 쪽을 지우지 않는 귀환 인물 | 모두를 듣는 다정함이 결정 회피로 굳지 않도록 한쪽발 돌과 남은손 표식을 남김 |
| 도르스 | 돌아올 길표를 다시 다는 귀환 인물 | 회피가 한때 사람을 지켰지만, 돌아올 문을 잃으면 고립이 된다는 점을 보임 |
| 실렌 | 웃음 뒤 한 문장을 남기는 귀환 인물 | 농담이 한때 사람을 지켰지만, 마지막 말을 덮으면 고립이 된다는 점을 보임 |
| 하빈 | 마지막 농담 뒤 물컵을 남기는 이야기꾼 | 유쾌함이 사람을 지키지만, 모든 끝을 웃음으로 닫으면 질문이 사라진다는 점을 보임 |
| 셀로 | 빠른 심판의 유혹을 가져오는 주인공 | 센시르의 느림과 충돌 |
| 세리엘 | 회복과 주체성의 시험자 | 누가 대신 열어도 되는가를 묻는 인물 |
| 프락시스/마이더스 피해자들 | 정산의 당사자 | 결과의 비용을 몸으로 증언 |

## 9. 원고 운용 원칙

1. 센시르는 모두를 착하게 만드는 나라가 아니다. 듣는 일이 느리고 어렵고 때로는 실패하는 나라다.
2. 공감은 결론이 아니라 시작이다. 카시안은 대신 용서하거나 대신 판결하면 안 된다.
3. 살루스의 안식은 실제로 다정해야 한다. 독자가 잠깐 쉬고 싶다고 느껴야 그 위험도 설득된다.
4. 에코 샘과 디아파손 홀은 해설 장치가 아니라 책임을 물성으로 보여주는 장소다.
5. 좋은 센시르 장면은 누군가의 말을 아름답게 정리하는 순간보다, 정리하지 않고 곁에 두는 순간을 남긴다.
6. 세이르 장면에서는 맞춰 주는 다정함을 부정하지 않는다. 다만 그 다정함 앞에 자기 잔 하나가 놓였는지를 확인한다.
7. 나리안 장면에서는 도움을 미덕으로 강요하지 않는다. 한 걸음만 필요한 사람에게 두 걸음을 대신 걸어 주면 안 된다.
8. 실렌 장면에서는 농담을 얕은 회피로만 쓰지 않는다. 농담이 실제로 사람을 지켰다는 사실과, 웃음 뒤 남은 한 문장이 필요하다는 사실을 함께 둔다.
9. 하빈 장면에서는 크게 웃기는 장면만 쓰지 않는다. 마지막 농담 뒤에도 아직 웃지 못한 사람의 자리와 남은 한마디가 남아 있는지를 확인한다.
10. 엘레시아 장면에서는 모두를 품는 태도를 비겁함으로만 쓰지 않는다. 다만 한쪽발 돌 위에 서지 못한 다정함이 양쪽 모두를 기다리게 만들 수 있음을 남긴다.

## 10. 미조명 생활권 정사 반영

| 생활권 | 핵심 감각 | 연결 인물 | 원고 운용 |
|---|---|---|---|
| 모라 우체국 | 보류 우편함, 반쪽 봉인초, 아직 읽을 수 없는 사과문 | 노엘 | 사과보다 먼저 도착해야 할 것은 읽을 권리라는 원칙으로 쓴다 |
| 반향 빨래터 | 물결 빨랫줄, 무거운 빨래돌, 남의 얼룩을 대신 말하지 않는 물길 | 유라 | 밝히기 전 먼저 불려야 하는 시간을 생활 장면으로 보여준다 |

이 두 생활권은 센시르의 느림을 미덕 표어로 끝내지 않는다. 사과도 회복도, 상대가 견딜 수 있는 속도를 먼저 묻는 손길이어야 한다.

## 말투와 사고방식 운용

센시르 말투는 결론을 늦추는 말투다. 느리게 말하는 척이 아니라, 상대가 지금 감당할 수 있는 높이를 먼저 고른다.

| 항목 | 운용 |
|---|---|
| 먼저 보는 것 | 호흡, 상처의 높이, 회복 속도, 손이 견디는 거리 |
| 문장 방향 | 답보다 받을 준비를 먼저 확인한다 |
| 금기 | 준비되지 않은 사람에게 답을 강요하는 말 |
| 권장 대사 | "말해도 된다. 아직 아니면, 컵만 여기 두고." |
| 비권장 대사 | "상처를 인정해."처럼 고백을 재촉하는 말 |

센시르 인물은 공감으로 판결하지 않는다. 듣는 일은 대신 정리하는 일이 아니라, 정리되지 않은 말을 잠시 둘 자리를 만드는 일이다.
