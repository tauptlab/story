# 에비안

정사 등급: B급 상위 생활 인물

소속: 프론티아 아르세니아, 칼리브레 대기실, 안티로고스 우편함 광장

핵심 은유: 비교 앞에서 사라지는 목소리, 나란히 놓이는 순간 작아지는 자기 자리, 침묵을 동의로 읽지 않는 공정

---

## 1. 인물 개요

에비안은 아르세니아 칼리브레 대기실의 조용한 대기자다. 그는 늘 자기 순서가 와도 바로 일어나지 않고, 옆 의자의 사람이 먼저 숨을 고르는지 살핀다. 에비안에게 비교는 처음부터 나쁜 일이 아니었다. 서로를 나란히 놓아야 숨은 차별이 보이고, 같은 조건이라는 말이 거짓인지도 드러난다.

하지만 오래 비교당한 사람은 자기 말을 시작하기 전에 옆 사람의 표정을 먼저 읽게 된다. 에비안은 누군가와 나란히 앉는 순간 자기 목소리가 얇아진다. 그는 틀릴까 봐 침묵하는 것이 아니라, 이미 비교표 위에 놓인 이상 자기 말이 자기 말로 들리지 않을까 봐 멈춘다.

에비안의 귀환은 비교를 없애는 사건이 아니다. 비교가 사람을 지키려면, 먼저 각자의 자기칸과 첫 문장을 돌려주어야 한다는 사실을 다시 받아들이는 사건이다.

---

## 2. 독자가 먼저 사랑할 지점

- 자기 번호가 불려도 옆 사람이 놀라면 먼저 의자 다리를 조용히 붙잡아 준다.
- 비교표를 받으면 가장 먼저 자기 이름이 아니라 빈칸의 가장자리를 손가락으로 문질러 본다.
- 말이 막힌 사람 앞에서는 대답을 대신하지 않고, 보크스 표찰를 뒤집어 "아직"이라는 면을 보이게 둔다.
- 누가 더 나은지 묻는 자리에서, 누가 아직 자기 말을 시작하지 못했는지 먼저 본다.
- 결투에서 이겨도 자기 의자를 가운데로 끌어오지 않고, 나란히 놓인 의자 사이에 작은 통로를 남긴다.

## 3. 상처와 거짓 믿음

| 항목 | 설정 |
|---|---|
| 원상처 | 나란히 놓인 순간 자기 말이 사라지고, 침묵이 동의나 패배로 읽힌 사건 |
| 배운 생존법 | 먼저 옆 사람의 표정과 점수를 읽으면 덜 다칠 수 있다 |
| 거짓 믿음 | 비교표에 오른 말은 이미 내 말이 아니라 남보다 나은지 못한지만 남는다 |
| 숨은 욕망 | 남과 나란히 있어도 자기 첫 문장을 잃지 않고 싶다 |
| 두려움 | 자기 말을 꺼내는 순간 누군가의 기준 아래 놓여 다시 작아질까 두렵다 |
| 성장 질문 | 나는 비교를 피하지 않으면서도, 내 목소리를 남의 줄에 묶지 않을 수 있는가 |

## 4. 예법과 소품

능력 등급: 없음

| 예법/소품 | 설명 | 한계 |
|---|---|---|
| 파렐라 의자 조율 | 나란히 놓인 의자 사이의 간격을 조용히 조절한다 | 겁을 내면 의자가 너무 멀어져 아무도 서로를 보지 못한다 |
| 라테라 거울 | 옆 사람만 먼저 비추는 거울을 돌려 자기 얼굴의 윤곽을 찾는다 | 오래 들여다보면 자기 표정이 가장자리로 밀린다 |
| 칼리브레 빈칸 | 비교표에서 아직 말하지 못한 자기칸을 비워 둔다 | 방치하면 남의 글씨가 빈칸에 스며든다 |
| 보크스 표찰 | 말할 순서표를 뒤집어 "아직"과 "지금"을 구분한다 | 순서에만 기대면 말의 이유가 사라진다 |
| 리멘 남김 | 나란히 놓인 자리 사이에 작게 지나갈 길을 남긴다 | 통로가 너무 넓으면 만남이 회피가 된다 |

에비안의 힘은 비교표에서 가장 좋은 칸을 차지하는 데 있지 않다. 그의 가장 강한 순간은, 비교가 시작되기 전에 아직 말하지 못한 사람의 자기칸을 비워 두는 순간이다.

| 마르빌 | 마르빌의 가표가 사람을 비교표의 한 줄로 굳히면, 에비안은 아직 말하지 않은 빈칸을 가리킨다 |
| 티레인 | 티레인의 라보르추가 수고의 비교로 기울 때, 에비안은 고생의 순위보다 말하지 못한 피로를 본다 |
| 엘리올 | 엘리올의 완벽한 문장은 에비안을 더 조용하게 만들 수 있지만, 늦봉투 우편은 그의 첫 문장을 되찾을 봉투가 된다 |
| 알디레아 | 알디레아의 라쿠나 표찰은 에비안에게 비교 전에 이름을 늦게 붙일 수 있는 시간을 준다 |
| 에르바크 | `158` 기준 에르바크는 에비안이 끝내 말하지 못한 질문을 늦은 봉투로 돌려보낼 수 있게 하되, 침묵한 모든 자리를 자동으로 대신 제출하지는 않는다 |

## 6. 결투 설계

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 숨은 차이를 드러내는 비교 vs 자기 목소리를 보존하는 공정 |
| 비교의 옳음 | 나란히 놓아야 숨은 특혜와 불공정이 드러난다 |
| 비교의 위험 | 사람을 옆 사람보다 나은지 못한지만 남은 줄로 만들 수 있다 |
| 자기 목소리 보존의 옳음 | 비교 전에 각자의 첫 문장과 자기칸을 돌려준다 |
| 자기 목소리 보존의 위험 | 비교 자체를 피하면 실제 차별이나 불공정도 보지 못할 수 있다 |
| 전투 뒤 재배치 | 비교표는 사라지지 않고, 말하기 전 자기칸과 통로를 먼저 남기는 표로 낮아진다 |
| 남는 변화 | 칼리브레 대기실에는 파렐라 의자 사이에 작은 통로가 생기고, 보크스 표찰의 "아직" 면이 정식 표지가 된다 |

## 7. 녹티오라 흔적

에비안의 밤은 끝없이 나란한 의자들이 놓인 대기실로 나타난다. 의자는 가까워질수록 서로를 밀어내고, 멀어질수록 목소리가 늦게 도착한다.

- 파렐라 의자는 비교가 시작되면 서로 같은 높이로 맞춰지지만, 앉은 사람의 발이 바닥에서 살짝 뜬다.
- 라테라 거울은 처음에는 옆 사람만 비추다가, 자기 이름을 부르면 가장자리에 희미한 윤곽을 돌려준다.
- 칼리브레 빈칸은 침묵을 동의로 읽으면 남의 글씨로 채워지고, "아직 말하지 않았다"는 표지를 놓으면 비어 있는 빛을 낸다.
- 보크스 표찰는 순서를 재촉하면 목에 달라붙고, 말할 준비를 확인하면 손바닥 위로 낮게 내려온다.
- 카논꽃은 의자가 똑같이 맞춰질 때 향을 잃고, 각자의 자기칸이 열리면 다른 향을 낮게 낸다.

## 9. 작중 사용 규칙

1. 에비안을 단순히 소심한 인물로 쓰지 않는다. 그는 비교가 사람을 살릴 수도 있다는 사실을 알기 때문에 더 조심스럽다.
2. 그의 성장은 비교를 거부하는 것이 아니라, 비교 전에 자기칸과 첫 문장을 되찾는 것이다.
3. 에비안 장면은 설명보다 파렐라 의자, 라테라 거울, 칼리브레 빈칸, 보크스 표찰, 의자 사이의 작은 통로로 먼저 보인다.
4. 그가 침묵할 때도 공백으로 취급하지 않는다. 침묵은 동의가 아니라, 아직 자기 말이 도착하지 못한 상태다.
