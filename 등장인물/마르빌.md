# 마르빌

정사 등급: B급 상위 생활 인물

소속: 프론티아 아르세니아, 눈금 계단, 안티로고스 우편함 광장

핵심 은유: 잠깐 붙은 판정이 이름처럼 굳는 두려움, 결과 앞에서 지워진 과정, 틀리지 않으려는 평가의 선의와 폭력

---

## 1. 인물 개요

마르빌은 아르세니아 눈금 계단의 가표 관리자다. 그는 확정 판결이 내려지기 전, 사람들이 아무 기준 없이 소문과 눈치에 밀려나는 일을 막기 위해 얇은 표를 붙인다. 마르빌에게 가표는 사람을 닫는 딱지가 아니라, 더 거친 판정이 달려들기 전 잠깐 세우는 낮은 울타리였다.

그가 그런 사람이 된 이유는 단순하다. 성실했던 과정이 보이지 않는 결과표 하나로 닫힌 날이 있었고, 그때 아무도 "어떻게 여기까지 왔는가"를 묻지 않았다. 마르빌은 다시는 그런 침묵이 생기지 않게 하려고 표를 만들었다. 하지만 표가 많아질수록, 그는 사람을 보호하려 붙인 얇은 종이가 사람의 이름처럼 굳을 수 있다는 사실을 늦게 알아차린다.

마르빌의 귀환은 판정을 없애는 사건이 아니다. 판정은 필요하지만, 사람을 끝내는 이름이 아니라 아직 읽는 중인 문장으로 낮아지는 사건이다.

---

## 2. 독자가 먼저 사랑할 지점

- 표를 붙이기 전 반드시 종이 모서리를 손톱으로 둥글게 눌러, 받는 사람이 베이지 않게 한다.
- 여분 가표를 세 번 접어 주머니에 넣지만, 매번 마지막 접힘에서 망설인다.
- 계단에서 넘어지는 아이를 보면 먼저 표를 떼는 대신 신발끈을 묶어 준다.
- 틀린 표를 회수한 날에는 버리지 않고 밤새 물빛 풀에 적셔 다시 부드럽게 만든다.
- 누군가 "나는 그 표가 아니야"라고 말하면 바로 반박하지 않고, 자기 손등에 붙은 표부터 내려다본다.

## 3. 상처와 거짓 믿음

| 항목 | 설정 |
|---|---|
| 원상처 | 성실한 과정과 빠진 사정이 보이지 않는 결과표 하나로 닫힌 날, 아무도 다시 묻지 않았다 |
| 배운 생존법 | 먼저 얇은 기준을 붙이면 더 거친 판정이 사람을 덜 다치게 한다 |
| 거짓 믿음 | 이름 붙지 않은 과정은 결국 아무도 지켜 주지 못한다 |
| 숨은 욕망 | 누구도 한 장의 표로 끝나지 않는 계단을 만들고 싶다 |
| 두려움 | 묻는 일이 늦어지는 사이 누군가 영원히 부족함으로 불릴까 두렵다 |
| 성장 질문 | 나는 판정을 붙이기 전에, 아직 읽지 못한 과정을 남겨 둘 수 있는가 |

## 4. 예법과 소품

능력 등급: 없음

| 예법/소품 | 설명 | 한계 |
|---|---|---|
| 가표 붙이기 | 사건이나 행동에 확정 전의 얇은 판정을 붙여 소문이 먼저 굳지 않게 한다 | 오래 붙이면 가표도 이름처럼 딱딱해진다 |
| 알테라 표 접기 | 아직 맞는 이름을 찾지 못한 판정을 접어 두고, 바로 붙이지 않을 시간을 번다 | 너무 많이 접으면 모서리가 날카로워져 손가락을 벤다 |
| 타르수스 표 회수 | 밤에 발등이나 손등에 붙은 표를, 빠진 사정 한 줄과 함께 천천히 떼어 낸다 | 당사자의 말을 듣지 않고 떼면 표가 더 깊이 붙는다 |
| 세미칸 눈금 | 합격과 실패, 옳음과 틀림 사이에 아직 읽는 중인 반칸을 만든다 | 책임을 피하려는 사람이 쓰면 눈금이 휘어진다 |
| 눈금 낮춤 | 눈금 계단의 단을 낮춰 판정을 받는 사람이 도망치지 않고 다시 말할 수 있게 한다 | 마르빌이 틀릴까 두려워하면 계단이 다시 높아진다 |

마르빌의 힘은 사람을 정확히 분류하는 데 있지 않다. 그의 가장 강한 순간은, 이미 붙인 표를 부끄러워하면서도 떼어 내고 "아직 읽는 중"이라고 다시 적는 순간이다.

| 알디레아 | 알디레아는 좋은 제목 아래의 차가운 사실을 보게 하고, 마르빌은 결과표 아래의 보이지 않은 과정을 보게 한다 |
| 티레인 | 티레인은 인정받지 못한 수고의 무게를 밀어 올리고, 마르빌은 그 수고를 점수표 하나로 닫을 수 없음을 배운다. 반대로 티레인은 마르빌에게서 수고도 표처럼 사람의 이름이 될 수 있음을 본다 |
| 솔렌 | 솔렌은 기운 선을 다시 그리게 하고 지운 자리 먼지를 읽게 하며, 마르빌은 표가 선을 바로잡는지 사람을 재단하는지 시험받는다 |
| 에비안 | 비교 앞에서 말이 사라지는 에비안을 보며, 마르빌은 표가 침묵한 사람의 동의처럼 쓰일 수 있음을 깨닫는다 |
| 에르바크 | `158` 기준 늦게 온 봉투를 받는 에르바크는, 마르빌에게 굳은 가표 뒤에도 도착한 말이 표 가장자리를 다시 적실 수 있음을 보여 준다 |
| 프락시스 | 결과를 빠르게 요구하는 힘 앞에서 마르빌의 가표는 쉽게 최종표로 굳는다 |

## 6. 결투 설계

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 빠른 임시 판정의 보호 vs 아직 읽히지 않은 과정의 존엄 |
| 빠른 임시 판정의 옳음 | 아무 기준 없는 소문과 무작위 배제를 막고, 다시 물을 첫 자리를 만든다 |
| 빠른 임시 판정의 위험 | 얇은 표가 사람의 이름처럼 굳어 과정과 선의를 지운다 |
| 읽히지 않은 과정의 옳음 | 결과 밖에 남은 수고, 사정, 망설임을 사람의 일부로 되돌린다 |
| 읽히지 않은 과정의 위험 | 모든 판정을 끝없이 미루면 당장 보호해야 할 사람을 놓칠 수 있다 |
| 전투 뒤 재배치 | 판정은 사라지지 않고, 떼어 낼 수 있고 다시 읽을 수 있는 임시 문장으로 낮아진다 |
| 남는 변화 | 눈금 계단 첫 단이 낮아지고, 가표 가장자리에 "아직 읽는 중"이라는 빈칸이 생긴다 |

## 7. 녹티오라 흔적

마르빌의 밤은 시험장이나 벌점표가 아니라, 낮에 잠깐 붙였던 표들이 손등과 발등에 돌아와 붙는 눈금 계단으로 나타난다.

- 가표는 밤이 되면 벽에서 떨어져 발등에 붙고, 최종 이름처럼 부르면 발걸음을 무겁게 만든다.
- 알테라 표 주머니는 묻지 않고 접어 둔 표를 계속 불려 낸다. 마르빌이 빠진 사정을 들으면 주머니 안에 한 장만 남는다.
- 세미칸 눈금자는 합격과 실패 사이에 낮은 틈을 열지만, 핑계로 쓰이면 눈금이 휘어져 계단을 비뚤게 만든다.
- 카논꽃은 꽃잎 수를 유지하지만 향이 표 아래 갇힌다. 표를 떼고 사정을 물으면 향이 낮게 돌아온다.
- 안티카논 이끼는 타르수스 표 아래에서 자라, 늦은 봉투가 도착할 길을 발밑에 만든다.

## 9. 작중 사용 규칙

1. 마르빌을 차가운 평가자로만 쓰지 않는다. 그는 거친 판정으로부터 사람을 지키려다 얇은 표에 매달린 사람이다.
2. 그의 성장은 기준을 버리는 것이 아니라, 기준이 사람보다 먼저 이름이 되지 않게 낮추는 것이다.
3. 마르빌 장면은 설명보다 표의 모서리, 접힌 주머니, 손등에 붙은 종이, 낮아지는 계단으로 먼저 보인다.
4. 그가 틀릴 때도 조롱하지 않는다. 틀린 표를 떼는 손이 바로 그의 사랑스러운 핵심이다.
