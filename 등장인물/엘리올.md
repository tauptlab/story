# 엘리올

정사 등급: B급 상위 생활 인물

소속: 프론티아 아르세니아, 안티로고스 우편함 광장, 프론티아-센시르 라세드린

핵심 은유: 정답을 요구하는 성실함, 빠진 사정을 다시 받는 공정, 완벽한 문장을 고칠 수 있는 용기

---

## 1. 인물 개요

엘리올은 틀린 판정과 애매한 책임을 누구보다 싫어한다. 그는 누군가가 "어쩔 수 없었다"는 말 뒤에 숨어 사라지는 것을 보았고, 그 뒤로 모든 말에는 기준이 있어야 한다고 믿게 되었다. 그의 차가움은 무관심이 아니라, 아무도 다시 억울하게 밀려나지 않게 하려는 과잉 성실함이다.

그는 아르세니아 안티로고스 우편함 광장에서 논쟁문을 쓰는 사람이다. 문장을 쓰기 시작하면 한 줄도 흐트러뜨리지 않지만, 빠진 봉투가 하나 더 떨어지면 다시 첫 줄로 돌아간다. 엘리올의 진짜 매력은 고집이 아니라, 고치기 싫어하면서도 고치는 손에 있다.

엘리올의 귀환은 "답을 내리는 힘"이 아니라 "답을 고치는 힘"이 다시 세계 질서 안으로 들어오는 사건이다.

---

## 2. 독자가 먼저 사랑할 지점

- 완벽한 문장을 쓰려다 새 봉투가 떨어지면 한숨을 쉬고도 첫 줄부터 다시 고친다.
- 남의 주장을 반박하기 전 자기 잉크가 너무 진한지 종이 뒷면을 비춰 본다.
- 줄 서는 사람을 볼 때 새치기보다 말 못 하고 뒤로 밀린 사람을 먼저 찾는다.
- 카논꽃을 돌볼 때 꽃잎 수는 세지만, 향이 달라진 이유도 작은 글씨로 남긴다.
- 논쟁에서 이긴 뒤에도 물컵이 비어 있으면 승리 표식을 받지 않는다.

## 3. 상처와 거짓 믿음

| 항목 | 설정 |
|---|---|
| 원상처 | 애매한 결론과 흐린 책임 때문에 억울한 사람이 끝내 다시 묻지 못한 사건 |
| 배운 생존법 | 답을 선명하게 만들면 책임도 선명해진다 |
| 거짓 믿음 | 빠진 사정이 남아 있으면 어떤 답도 말해서는 안 된다 |
| 숨은 욕망 | 틀리지 않는 사람이 아니라, 늦게 온 사람까지 놓치지 않는 사람이 되고 싶다 |
| 두려움 | 결론을 늦추는 동안 누군가 또 버려질까 두렵다 |
| 성장 질문 | 나는 정답을 지키면서도, 늦게 온 봉투를 다시 받을 수 있는가 |

---

## 4. 약한 발현과 예법

능력 등급: 약한 발현
발현 표지: 되쓴 첫줄
작동: 결론선을 잠시 고정하고 늦은 반례가 들어오면 첫 줄만 되돌려 다시 쓰게 한다
대가/약점: 늦은 봉투를 계속 받으면 몸과 문장이 함께 무거워진다. 반대로 은핀을 오래 꽂아 두면 결론은 안전해 보이지만 사람에게 닿지 않는 차가운 문장으로 굳는다.

| 예법/소품 | 설명 | 한계 |
|---|---|---|
| 사건 은선 | 말과 사건 사이에 은색 선을 그어 책임의 위치를 분명히 한다 | 선 밖의 사정이 보이지 않을 수 있다 |
| 늦봉투 우편 | 빠진 사정이 담긴 봉투를 불러와 결론 앞에 떨어뜨린다 | 봉투가 많아질수록 몸이 무거워지고 말이 늦어진다 |
| 첫줄 되쓰기 | 결투 중에도 논쟁의 첫 문장을 다시 쓸 수 있다 | 자존심이 강할수록 손끝이 굳는다 |
| 은핀 | 아직 결론 내리지 못한 문장을 잠시 고정한다 | 오래 꽂아 두면 문장이 차갑게 굳어 사람에게 닿지 않는다 |

엘리올의 힘은 결론을 빨리 내리는 데 있지 않다. 그의 가장 강한 순간은, 이미 이긴 문장을 내려놓고 첫 줄을 다시 쓰는 순간이다.

## 5. 관계선

| 인물 | 관계 기능 |
|---|---|
| 에이도스 | 근거를 읽는 성실함과 사람을 잃지 않는 관찰의 차이를 나란히 보여 준다 |
| 카우투스 | 반례를 너무 많이 보아 결단이 늦어지는 가치와 공명한다 |
| 프락시스 | 빠른 결과를 요구하는 힘 앞에서 엘리올의 유예가 약해 보이지만, 그 약함이 사람을 살릴 수 있다 |
| 카시안 | 말의 맞고 틀림보다 도착 방식을 묻는 법을 가르친다 |
| 레사르 | 이기는 문장의 매혹과 첫 이유를 다시 보는 습관을 공유한다 |
| 하엘디 | 상처와 이유를 같은 줄에 적게 하며, 엘리올의 분리 습관을 흔든다 |
| 알디레아 | 사실만 붙드는 귀환 후보. 엘리올과 함께 사실, 답, 빠진 사정의 차이를 후반 결투로 만들 수 있다 |
| 마르빌 | 가표가 빠진 봉투를 막고 있었는지 보게 하며, 답을 고치는 일과 표를 떼는 일을 연결한다 |
| 셀로 | 빠른 심판을 내리려는 마음 앞에서 안티로고스 우편함을 낮추는 역할을 한다 |

## 6. 결투 설계

| 항목 | 내용 |
|---|---|
| 충돌 가치 | 명확한 정답의 보호 vs 빠진 사정을 다시 받는 공정 |
| 명확한 정답의 옳음 | 흐린 책임 뒤에 숨는 사람을 막고, 억울함이 사라지지 않게 한다 |
| 명확한 정답의 위험 | 답에서 빠진 사람을 예외로 밀어내고, 감정을 증명대에 세울 수 있다 |
| 다시 받는 공정의 옳음 | 늦게 도착한 사정도 판결 앞에 설 수 있게 한다 |
| 다시 받는 공정의 위험 | 결론이 끝없이 늦어져 당장 보호해야 할 사람을 놓칠 수 있다 |
| 전투 뒤 재배치 | 답은 버려지지 않고, 고칠 수 있는 문장으로 낮아진다 |
| 남는 변화 | 안티로고스 우편함은 더 낮아지고, 첫줄 지우개는 문장을 없애지 않고 첫 줄만 부드럽게 흐린다 |

---

## 7. 녹티오라 흔적

엘리올의 밤은 끝없는 시험장이 아니라, 늦게 도착한 봉투가 계속 떨어지는 광장으로 나타난다.

- 안티로고스 우편함은 낮아져 누구나 봉투를 넣을 수 있지만, 엘리올이 결론을 서두르면 입구가 다시 높아진다.
- 완벽한 논쟁문은 밤마다 첫 줄만 은색으로 굳어, 손으로 만지면 손끝이 차가워진다.
- 카논꽃은 꽃잎 수를 유지하지만 향을 잃고, 빠진 이름이 불리면 향이 아주 낮게 돌아온다.
- 라세드린의 이유잔은 엘리올의 첫 이유를 비추며, "맞아서"가 아니라 "버려질까 봐"라는 문장을 남긴다.

## 8. Mnemosyn 귀환 연결

| 기준 | 엘리올의 연결 |
|---|---|
| A가 버린 가치관 | 모르는 상태를 견디고, 늦게 온 사정을 다시 받을 수 있다는 믿음 |
| A를 보호하던 대체 가치관 | 선명한 답과 기준을 세워 흐린 책임을 막는 습관 |
| 시간이 지나 왜곡된 가치관 | 답이 사람보다 먼저 움직이고, 느린 사정은 예외로 밀려난다 |
| 다시 받아들여야 하는 가치관 | 답을 없애지 않고, 고칠 수 있는 문장으로 낮추는 용기 |
| 녹티오라에 남은 흔적 | 낮아지는 안티로고스 우편함, 은색으로 굳은 첫 줄, 향을 잃은 카논꽃 |
| 귀환 갈등 | 명확한 정답의 보호와 빠진 사정을 다시 받는 공정이 결투 뒤 서로의 자리를 다시 정한다 |

## 9. 작중 사용 규칙

1. 엘리올을 냉정한 말싸움꾼으로만 쓰지 않는다. 그는 억울함을 줄이려다 문장에 매달린 사람이다.
2. 그의 성장은 정답을 포기하는 것이 아니라, 정답을 다시 쓸 수 있게 낮추는 것이다.
3. 엘리올의 장면은 논리 설명보다 봉투, 첫 줄, 물컵, 은핀, 카논꽃의 향으로 먼저 보인다.
4. 그가 틀릴 때도 조롱하지 않는다. 틀림을 인정하는 손이 바로 그의 사랑스러운 핵심이다.
